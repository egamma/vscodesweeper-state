# Issue #281576 — Visual flickering UI elements after latest update

[https://github.com/microsoft/vscode/issues/281576](https://github.com/microsoft/vscode/issues/281576)

## Summary

Legitimate upstream Chromium rendering regression introduced with the Electron 39 / Chromium 142 update, confirmed by multiple users on Windows and Linux with multi-monitor setups. A team member (deepak1556) is already engaged. Should stay open as an upstream tracking issue.

## Change summary

VS Code UI elements flicker, shift, and redraw incorrectly on larger/multi-monitor displays after the Electron 39 / Chromium 142 update.

## Best solution

Track the upstream Chromium compositor bug and land a fix or workaround once Chromium addresses it.
