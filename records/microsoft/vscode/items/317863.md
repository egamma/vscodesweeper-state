# Issue #317863 — Copilot Inline Suggestions block IntelliSense suggestions to automatically pop up when pressing letters

[https://github.com/microsoft/vscode/issues/317863](https://github.com/microsoft/vscode/issues/317863)

## Summary

Hard keep-open: assigned to 'On Deck' milestone (a real scheduling milestone), and a maintainer explicitly stated 'Let's keep this issue open for now for discussions.' The default change to `offWhenInlineCompletions` is intentional and still active on main, but community feedback is ongoing.

## Change summary

The new default `editor.quickSuggestions.other: "offWhenInlineCompletions"` suppresses the suggest widget while inline completions are active, breaking workflows that rely on IntelliSense for auto-imports, symbol discovery, and parameter names.

## Best solution

The team should decide whether to refine the `offWhenInlineCompletions` behavior (e.g. still show suggest widget for imports/trigger characters more aggressively), improve discoverability of the setting change (a one-time notification), or accept the current state with the workaround documented. The issue is appropriately parked on 'On Deck' for that decision.
