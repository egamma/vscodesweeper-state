# Issue #131484 — [Bug] "Toggle Search Details" doesn't show keybinding when hovering over

[https://github.com/microsoft/vscode/issues/131484](https://github.com/microsoft/vscode/issues/131484)

## Summary

Still reproduces on main: the Toggle Search Details button tooltip is set to a plain label string without the keybinding. An open PR (#311859) is already addressing this — keeping open for that PR to land.

## Change summary

The "Toggle Search Details" button tooltip does not show the Ctrl+Shift+J keybinding, unlike other similar buttons in the search view.

## Best solution

Land or review the open PR https://github.com/microsoft/vscode/pull/311859, which adds the keybinding to the tooltip. The fix is in `src/vs/workbench/contrib/search/browser/searchView.ts` around line 510 where `setupManagedHover` uses a plain label instead of including the keybinding from `IKeybindingService`.
