# Issue #245853 — Discard Changes dialog buttons inconsistent with meaning

[https://github.com/microsoft/vscode/issues/245853](https://github.com/microsoft/vscode/issues/245853)

## Summary

Valid polish issue, still present on main. The 'Discard File' button label is misleading for tracked files — the action discards changes, not the file. Issue is milestoned 'On Deck' so it stays open; #285361 is a duplicate of this one.

## Change summary

The 'Discard Changes' confirmation dialog shows a 'Discard File' button for tracked files, which is inconsistent with the actual action of discarding changes.

## Best solution

Rename the button from 'Discard File' to 'Discard Changes' for tracked single-file discard (line ~2306 in `extensions/git/src/commands.ts`), and similarly adjust the multi-file variant. Close https://github.com/microsoft/vscode/issues/285361 as a duplicate of this issue.
