## Change summary

Allow dragging selected text from one editor panel and dropping it into another editor panel in a split layout.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/149184, which captures this exact request more comprehensively and already has community engagement. Cross-editor text DnD would require extending the internal drag handler in `src/vs/editor/browser/controller/mouseHandler.ts` to emit platform drag events usable by other editor instances.
