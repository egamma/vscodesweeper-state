# Issue #319666 — Feature Request: Add a confirmation dialog to the "Archive/Clear" button in Copilot Chat

[https://github.com/microsoft/vscode/issues/319666](https://github.com/microsoft/vscode/issues/319666)

## Summary

The archive action is non-destructive and fully reversible — sessions can be unarchived at any time. Bulk archive operations already show confirmation dialogs. Adding a confirmation to the single-session archive button would add friction to a safe, undoable action, which conflicts with VS Code's simple/uniform UI value.

## Change summary

Request to add a confirmation dialog before archiving a single chat session in Copilot Chat.

## Best solution

Current behavior is appropriate: archiving is reversible (unarchive exists), so no confirmation is needed for a single-session archive. Bulk operations already confirm. Close as as-designed.
