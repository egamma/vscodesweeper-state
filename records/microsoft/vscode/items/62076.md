# Issue #62076 — User needs to be able to control whether or not the issue report seaches duplicates

[https://github.com/microsoft/vscode/issues/62076](https://github.com/microsoft/vscode/issues/62076)

## Summary

Team-authored issue; the requested control still does not exist on current main — the `searchDuplicates` method in the issue reporter unconditionally calls the external probot service with no user-facing setting or checkbox to opt out.

## Change summary

Add a user-facing control (setting or checkbox) to disable the issue reporter's online duplicate-search call to the external probot service.

## Best solution

Add a setting (or a checkbox in the issue reporter UI) gating the `searchDuplicates` call in `src/vs/workbench/contrib/issue/browser/baseIssueReporterService.ts`, consistent with the existing `@tag:usesOnlineServices` pattern for other online-service toggles. The setting should appear under `@tag:usesOnlineServices` in the settings editor.
