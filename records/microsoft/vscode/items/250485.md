# Issue #250485 — Bug: removing suggested attachment with keyboard doesn't work

[https://github.com/microsoft/vscode/issues/250485](https://github.com/microsoft/vscode/issues/250485)

## Summary

Team-authored polish request: the discussion between members clarified that suggested attachments aren't toggles, and the follow-up ask is to surface the Delete command hint (e.g. in a tooltip) for discoverability. Keeping open per team-author policy.

## Change summary

Keyboard users cannot discover how to remove a chat attachment once it transitions from a suggested to a real attachment; a tooltip or hint showing the Delete shortcut would help.

## Best solution

Add a tooltip or keyboard hint on real chat attachments indicating that Delete/Backspace removes them. The likely area is `src/vs/workbench/contrib/chat/browser/attachments/chatAttachmentWidgets.ts` and/or `implicitContextAttachment.ts`, where attachment rendering and tooltip setup already exist.
