# Issue #202964 — Dynamic softwrap margin whilst retaining horizontal scroll.

[https://github.com/microsoft/vscode/issues/202964](https://github.com/microsoft/vscode/issues/202964)

## Summary

Keep this open as an editor feature request: current wrapping options cover viewport wrapping, fixed-column wrapping, and bounded wrapping, but not a separate dynamic soft-wrap margin while preserving horizontal scrolling. There is no duplicate or concrete implementation evidence in the provided issue history.

## Change summary

Add an editor wrapping mode or option that allows a dynamic soft-wrap margin without disabling horizontal scrolling behavior.

## Best solution

Route to the editor area for an owner decision on whether this belongs in core wrapping/layout options, likely around `src/vs/editor/common/config/editorOptions.ts` and the view model line breaking pipeline.
