## Summary

Team-authored UX feature request to improve discoverability of custom agents in the chat input. The terminology still reads "Configure Custom Agents" on current main, and no null/empty state has been added — the request remains unimplemented. Keeping open per the hard keep-open rule for team members.

## Change summary

Add a disabled null state to the custom agent dropdown when no agents exist and rename "Configure" to "Manage" to signal that new agents can be created.
