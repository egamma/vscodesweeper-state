# Issue #303015 — Sessions: Edit pill appears slightly different than other pills

[https://github.com/microsoft/vscode/issues/303015](https://github.com/microsoft/vscode/issues/303015)

## Summary

Team-authored polish issue — the edit pill and inline-anchor (Read) pill use different border widths (1px vs 0.5px) and slightly different font-size inheritance, causing a visible size/weight inconsistency. Keeping open for maintainer judgment.

## Change summary

File-name pills next to 'Edited' status appear slightly larger and with a thicker border than pills next to 'Read' status.

## Best solution

Unify the border width and font-size declarations between `chatCodeBlockPill.css` (`.chat-codeblock-pill-widget`) and `chatInlineAnchorWidget.css` (`.chat-inline-anchor-widget`) so both pill types render identically. Likely change: set the edit pill border to `0.5px solid` to match the inline anchor, or vice versa, and ensure `font-size` is applied at the same level.
