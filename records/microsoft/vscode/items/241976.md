# Issue #241976 — Support alternative suggest mode (insert/replace) for terminal suggestions

[https://github.com/microsoft/vscode/issues/241976](https://github.com/microsoft/vscode/issues/241976)

## Summary

The terminal suggest widget does not support an alternative insert/replace mode analogous to the editor's suggest modes. This remains unimplemented on current main and is a valid feature request in the terminal-suggest area.

## Change summary

Add support for insert vs. replace acceptance modes in terminal suggestions, mirroring the editor's suggest behavior.

## Best solution

Implement an insert/replace mode toggle (and keybinding, e.g. shift+tab) in the terminal suggest addon, similar to how `editor.suggest.insertMode` works. The work lives in `src/vs/workbench/contrib/terminalContrib/suggest/browser/terminalSuggestAddon.ts`.
