# Issue #240571 — Git - Honor `git.autoStash` when switching branches

## Summary

Valid feature request with an open PR (#305697) that addresses it. The `git.autoStash` setting is currently only honored during pull/rebase, not during branch checkout. Keeping open while the PR is in flight.

## Change summary

When `git.autoStash` is enabled, switching branches with uncommitted changes should automatically stash and pop instead of showing a modal dialog.
