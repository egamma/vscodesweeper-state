## Summary

Team-authored issue tracking a known limitation in the indentation engine where multiple closing braces on one line don't outdent correctly. Must remain open per policy.

## Change summary

Reindent Lines does not outdent by more than one level when multiple closing braces appear on the same line.

## Best solution

Extend the indentation logic (likely via a new enum value like `IndentByNumberOfCaptureGroups`) to support multi-level outdentation based on regex capture groups, enabling correct reindentation for lines with multiple closing braces.
