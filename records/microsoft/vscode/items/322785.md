# Issue #322785 — Show total session token usage

[https://github.com/microsoft/vscode/issues/322785](https://github.com/microsoft/vscode/issues/322785)

## Summary

Team-authored feature request to surface cumulative session token usage in the context widget alongside the recently added session cost. The widget currently shows per-turn token counts and session cost (credits) but not a session-level token total. Keeping open — team member issue requiring maintainer judgment.

## Change summary

Show total session-level token usage (input + output tokens across all turns) in the chat context usage widget.

## Best solution

Extend `IChatContextUsageData` in `chatContextUsageDetails.ts` with cumulative session token fields (total input tokens, total output tokens) and render them in `ChatContextUsageDetails`, mirroring the existing session-cost row pattern. The accumulation logic would live in `ChatContextUsageWidget.update()` where `_sessionCost` is already tracked.
