# Issue #233991 — Editor GPU: Render link underline decorations on GPU

[https://github.com/microsoft/vscode/issues/233991](https://github.com/microsoft/vscode/issues/233991)

## Summary

Still an open gap on current main: the GPU renderer explicitly only supports `line-through` for text-decoration-line (line 284 of viewGpuContext.ts), not `underline`. This is a team-authored plan-item in active development.

## Change summary

Render link underline decorations via the GPU renderer instead of falling back to DOM.

## Best solution

Implement underline drawing in the GPU glyph rasterizer (with descender dodging) and extend `supportsCssRule` to accept `underline` for `text-decoration-line`. The outline in the issue comment lays out the work clearly.
