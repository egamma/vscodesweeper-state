Plausible bug in agent terminal lifecycle: when a completed agentic chat leaves a background terminal running and the user closes it, the agent resumes the already-finished chat. No detailed reproduction steps or logs were provided, but the described behavior is specific enough to route to the chat-terminal area for investigation.

## Change summary

Copilot resumes and restarts a completed agentic chat session when the user closes or stops the background terminal the agent had been using.
