## Summary

Still present on main: the quick pick width scales to 62% of the window width with no minimum, so at very narrow widths the OK button is clipped. A team member confirmed and suggested the fix direction but no implementation landed.

## Change summary

Quick pick OK button is clipped/inaccessible when the VS Code window is narrow (especially on Windows where controls are wider).

## Best solution
