Team-authored feature request. The base `Chat: Review` command exists for selections, but the author's updated ask — a quick pick surfacing multiple review targets (current selection, current diff, HEAD against main) — remains unimplemented. Add discoverable command-palette commands for code review that offer a quick pick of review targets (selection, diff, HEAD vs main).
