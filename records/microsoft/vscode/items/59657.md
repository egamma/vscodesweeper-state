# Issue #59657 — [folding] clicking on on line number of folded line should select full folding range

[https://github.com/microsoft/vscode/issues/59657](https://github.com/microsoft/vscode/issues/59657)

## Summary

This is a valid, accepted feature request (labeled `help wanted`, milestone `Backlog`) that has not been implemented on current main. The line-number click handler in mouseHandler.ts has no folding-aware selection logic.

## Change summary

Clicking on the line number of a folded line should select the entire folded range, not just the first line.

## Best solution

In the editor's mouse handler (src/vs/editor/browser/controller/mouseHandler.ts around line 330), when a click targets a line number that is part of a collapsed folding range, expand the selection to cover the full range. The folding model can supply the range boundaries. The `help wanted` label makes this a good community contribution candidate.
