# Issue #251576 — Feature Request: Enhanced Tool Information Display in MCP Configuration

[https://github.com/microsoft/vscode/issues/251576](https://github.com/microsoft/vscode/issues/251576)

## Summary

Valid feature request: the MCP server hover widget on current main shows server label, scope, stars, description, and status but does not list individual tool names. The requested capability (hover over tool count to see tool names) remains unimplemented.

## Change summary

Show a tooltip listing individual MCP tool names when hovering over the tools count indicator in the MCP configuration panel.

## Best solution

Enhance `McpServerHoverWidget.getHoverMarkdown()` in `src/vs/workbench/contrib/mcp/browser/mcpServerWidgets.ts` (or the relevant tools-count UI element) to enumerate tool names from the server's tool list. The tools are already available via `listTools` on the server instance. Low complexity, good discoverability improvement.
