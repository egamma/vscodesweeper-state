Confirmed inconsistency on current main: the merge-conflict extension's CodeLens labels say "Accept Current Change" / "Accept Incoming Change" / "Accept Both Changes" while the same extension's command palette entries say "Accept Current" / "Accept Incoming" / "Accept Both" (without "Change").
