# Issue #268102 — Chat fetch tool can't read content from my private GitHub repos

[https://github.com/microsoft/vscode/issues/268102](https://github.com/microsoft/vscode/issues/268102)

## Summary

Valid feature request to have the fetch tool authenticate against private GitHub repos. A team member explicitly decided to keep this open for potential future work. No implementation has landed on main.

## Change summary

The chat fetch tool cannot access private GitHub repo content despite the user being authenticated with GitHub in VS Code.

## Best solution

Either translate GitHub URLs into authenticated API calls using the user's existing GitHub auth token, or rely on built-in GitHub MCP becoming the standard path for private repo content. The fetch tool code lives in `src/vs/workbench/contrib/chat/electron-browser/builtInTools/fetchPageTool.ts`.
