# Issue #151931 — Exception has occurred message looks out of place

[https://github.com/microsoft/vscode/issues/151931](https://github.com/microsoft/vscode/issues/151931)

## Summary

The merged PR #152002 only fixed the close button vertical alignment. The broader visual polish request — restyling the exception widget to match the peek widget's layout and appearance — remains unaddressed on current main. The widget still uses its custom styling with hardcoded platform font sizes and a non-peek-style title bar.

## Change summary

The debug exception widget looks visually inconsistent with VS Code's peek widget — different title layout, close button placement, and font sizing.

## Best solution

Restyle the exception widget in `src/vs/workbench/contrib/debug/browser/exceptionWidget.ts` and its CSS to adopt the peek widget's title bar pattern (right-aligned close button, consistent font sizing). The discussion in comments 4–7 converged on reusing the peek widget style while potentially keeping differentiated coloring. This is a localized polish task in the debug contribution.
