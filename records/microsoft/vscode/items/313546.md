# Issue #313546 — Support reasoning effort configuration in custom .agent.md files

[https://github.com/microsoft/vscode/issues/313546](https://github.com/microsoft/vscode/issues/313546)

## Summary

This feature request asks for a `reasoning_effort` frontmatter property in `.agent.md` files. It is not implemented on current main (the prompt file parser has no such attribute). The issue carries the 'On Deck' milestone, so it must stay open.

## Change summary

Allow .agent.md files to specify a reasoning effort level (e.g., low/medium/high) for the configured model.

## Best solution

Add a `reasoningEffort` attribute to `PromptHeaderAttributes` in `src/vs/workbench/contrib/chat/common/promptSyntax/promptFileParser.ts`, wire it through the prompt file attributes definitions, and pass it to the model request pipeline where `reasoningEffort` is already handled (e.g., `endpointProvider.ts`). The plumbing from agent host sessions already supports the concept — it just needs to be exposed in the frontmatter schema.
