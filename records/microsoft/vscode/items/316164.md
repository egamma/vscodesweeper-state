## Summary

Legitimate UX design discussion backed by usability study evidence; already correctly labeled under-discussion in the agents-window area. Needs a product/design decision on how to simplify the changes view categories.

## Change summary

The three change categories (Branch Changes, All Changes, Uncommitted Changes) in the Agents Window changes view are confusing and redundant to users.

## Best solution

The Agents Window team should decide on a simplified information architecture for the changes view — likely a single default view with optional filters rather than three separate categories. This is a design decision that needs maintainer judgment informed by the usability study findings.
