# Issue #280076 — when downvoting copilot chat response multi selection is not possible

[https://github.com/microsoft/vscode/issues/280076](https://github.com/microsoft/vscode/issues/280076)

## Summary

The downvote feedback-reasons UI is owned by the Copilot extension, not VS Code core. VS Code core only provides a binary thumbs-up/thumbs-down vote via ChatAgentVoteDirection and fires the result to the extension via onDidReceiveFeedback. Any multi-select reason picker would be implemented in the extension. This belongs as a feature request on the Copilot extension repo.

## Change summary

Allow selecting multiple reasons when downvoting a Copilot chat response.

## Best solution

File this as a feature request against the GitHub Copilot extension (https://github.com/microsoft/vscode-copilot-release) — VS Code core provides only the binary vote signal; the feedback-reason UI and its multi-select behavior are the extension's responsibility.
