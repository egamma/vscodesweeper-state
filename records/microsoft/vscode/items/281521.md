# Issue #281521 — Copilot status bar hover flickers while agent is running

[https://github.com/microsoft/vscode/issues/281521](https://github.com/microsoft/vscode/issues/281521)

## Summary

The flickering symptom is gone on current main — the status bar entry no longer listens to `onDidChangeInProgress` at all (the spinner was fully removed in commit 53b86cfbfa8). However, the root cause (`onDidChangeInProgress` firing excessively) was never fixed, and this is a team-authored issue, so keeping open for maintainer judgment on whether the underlying event issue still matters.

## Change summary

The Copilot status bar hover flickered every time `onDidChangeInProgress` fired during an agent session, even when the in-progress count had not changed.

## Best solution

The visible symptom is resolved — the status bar spinner was removed entirely in https://github.com/microsoft/vscode/commit/53b86cfbfa8. The underlying `onDidChangeInProgress` event still fires excessively (no count-change guard at the source in `chatSessions.contribution.ts`). A maintainer should decide whether to close this as symptomatically resolved or keep it open to fix the noisy event at its source.
