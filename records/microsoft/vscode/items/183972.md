# Issue #183972 — Nonce does not appear to work on fish shell

[https://github.com/microsoft/vscode/issues/183972](https://github.com/microsoft/vscode/issues/183972)

## Summary

Still reproduces on main: the nonce is scoped locally inside an if-block and unavailable to the functions that need it. An open PR (#313195) addresses the fix. Team-authored issue — keeping open.

## Change summary

Fish shell nonce variable is scoped to the if-block and not accessible when commands are executed, causing unnecessary confirmation dialogs on rerun.

## Best solution

Land PR https://github.com/microsoft/vscode/pull/313195 which changes `set -l __vsc_nonce` to `set -g __vsc_nonce` (or moves the assignment outside the if-block) so the nonce is available in `__vsc_cmd_executed` and `__vsc_cmd_clear`.
