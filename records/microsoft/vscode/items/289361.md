# Issue #289361 — A little lighter color tone for comparision files

[https://github.com/microsoft/vscode/issues/289361](https://github.com/microsoft/vscode/issues/289361)

## Summary

The diff colors are already intentionally theme-driven and user-customizable through existing diff editor color tokens, so changing the default red/green tone globally is not the right fix for this specific preference.

## Change summary

Make the default comparison/diff file red and green colors lighter to reduce eye strain during long reviews.

## Best solution

Keep the default diff styling theme-controlled and point users who need softer colors to `workbench.colorCustomizations` for the existing `diffEditor.*` color keys rather than adding another built-in tone preference.
