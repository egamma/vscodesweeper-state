# Issue #139305 — Swedish keyboard layout is identified as German and this breaks Cmd+Z to undo

[https://github.com/microsoft/vscode/issues/139305](https://github.com/microsoft/vscode/issues/139305)

## Summary

Team-authored issue (alexdima, MEMBER) with milestone 'On Deck' — must keep open. The bug remains valid: Swedish keyboard is misidentified as German on web, breaking Cmd+Z and other shortcuts.

## Change summary

On vscode.dev/Codespaces with a Swedish keyboard on macOS, the layout is incorrectly guessed as German, causing Cmd+Z (undo) and other shortcuts to fail because Z/Y are swapped in the German layout mapping.

## Best solution

Fix the keyboard layout guessing heuristic in the web keymap service to correctly distinguish Swedish from German layouts — likely in `src/vs/workbench/services/keybinding/browser/keyboardLayoutService.ts` or the keyboard layout data. The Swedish layout shares physical key positions with German for some keys but not Z/Y, so the scoring/matching algorithm needs better differentiation.
