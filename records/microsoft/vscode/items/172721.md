# Issue #172721 — Wrap not correct

[https://github.com/microsoft/vscode/issues/172721](https://github.com/microsoft/vscode/issues/172721)

## Summary

Keep open: the report is plausible for editor wrapping of Vietnamese text with decomposed combining marks, and I did not find a concrete duplicate or fixing commit that would justify closing it.

## Change summary

Vietnamese text with combining diacritics wraps at visually incorrect positions compared with Word.

## Best solution

Editor wrapping should either account for grapheme/combining-mark width in the default simple wrapping path or clearly route affected users to `editor.wrappingStrategy: advanced` if the browser-based wrapping path is the intended solution.
