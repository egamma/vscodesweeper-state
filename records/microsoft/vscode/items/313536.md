## Summary

Valid polish request — the different icons are an intentional design choice (arrowRight for side-by-side directional semantics, discard for inline), but the reporter's argument for consistency has merit. Keeping open for the diff editor owner to decide.

## Change summary

The diff editor uses `Codicon.arrowRight` in side-by-side view and `Codicon.discard` in inline view for the same revert action, creating visual inconsistency.

## Best solution

The relevant code is in `src/vs/editor/browser/widget/diffEditor/diffEditor.contribution.ts` (lines 49-51) and `src/vs/editor/browser/widget/diffEditor/features/revertButtonsFeature.ts` (line 129).
