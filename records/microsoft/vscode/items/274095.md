# Issue #274095 — Extension Host terminated unexpectedly when GitHub Copilot is enabled

[https://github.com/microsoft/vscode/issues/274095](https://github.com/microsoft/vscode/issues/274095)

## Summary

Extension host crash with a native N-API fatal error when GitHub Copilot is enabled. Two team members are actively investigating — deepak1556 requested crash dumps and abpbackup posted crash analysis. Keeping open as the team is mid-investigation and the root cause (VS Code runtime vs. extension native module) is not yet determined.

## Change summary

Extension host terminates repeatedly with FATAL ERROR: Error::New napi_get_last_error_info when GitHub Copilot is enabled, exit code 134.

## Best solution

Team should complete crash dump analysis (abpbackup's screenshot suggests progress) to determine whether the N-API failure is in VS Code's Node/Electron runtime layer or in a native module shipped by the Copilot extension. If the latter, route to the Copilot extension team; if the former, fix the runtime compatibility issue in VS Code core.
