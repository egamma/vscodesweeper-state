# Issue #83196 — Enteraction appending text before the new line

[https://github.com/microsoft/vscode/issues/83196](https://github.com/microsoft/vscode/issues/83196)

## Summary

Valid feature request for an `appendTextBefore` (or similar) property on EnterAction — not yet implemented on main. An open community PR (#320286) is in progress.

## Change summary

Allow EnterAction to append text to the line above (before the newline), enabling string-split-on-enter workflows in language extensions.

## Best solution

Extend the `EnterAction` interface in `src/vs/editor/common/languages/languageConfiguration.ts` with a property like `appendTextBeforeNewLine` and wire it through the onEnter command handling. The open PR https://github.com/microsoft/vscode/pull/320286 is attempting this; review and land it if it meets quality.
