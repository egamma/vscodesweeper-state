# Issue #83196 — Enteraction appending text before the new line

[https://github.com/microsoft/vscode/issues/83196](https://github.com/microsoft/vscode/issues/83196)

## Summary

Keep open as a still-valid editor command/API feature request. Current main still exposes `EnterAction.appendText` only for text after the newline/indentation, and the recent comment indicates contributors are interested in working on it.

## Change summary

Allow an `EnterAction` to insert or append text before the newline when Enter splits existing text.

## Best solution

If accepted, extend the editor language configuration/VS Code API for enter handling and update `cursorTypeEditOperations.ts` so enter actions can transform text on both sides of the inserted newline. This likely needs API/design review because it changes public `EnterAction` behavior.
