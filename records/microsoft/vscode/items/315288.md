# Issue #315288 — Editor Scroll Lock Button

[https://github.com/microsoft/vscode/issues/315288](https://github.com/microsoft/vscode/issues/315288)

## Summary

This is a valid editor UI feature request and should stay open for maintainer consideration rather than be closed. Current main has the locked-scrolling command and an active-state status bar entry, but I did not find an always-available GUI button for turning it on.

## Change summary

Add a GUI button to toggle locked scrolling across editors without using the Command Palette or a custom keybinding.

## Best solution

Route to the editor/scroll-locking owner to decide whether the toggle belongs in the editor title, layout/title toolbar, or another lightweight workbench affordance. The implementation would likely extend `src/vs/workbench/contrib/scrollLocking/browser/scrollLocking.ts` with an appropriate menu/action contribution if accepted.
