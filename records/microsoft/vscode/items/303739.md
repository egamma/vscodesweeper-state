# Issue #303739 — Copilot crashes on CNB.cool docker workspaces

[https://github.com/microsoft/vscode/issues/303739](https://github.com/microsoft/vscode/issues/303739)

## Summary

The crash originates entirely within the Copilot Chat extension (github.copilot-chat-0.40.1) — its bundled @github/copilot/sdk fails to load a native pty.node module for the container's platform. This is an extension packaging/platform-support issue, not a VS Code core defect.

## Change summary

Copilot Chat extension crashes on CNB.cool Docker workspaces because the bundled pty.node native module is missing for the linux-x64 container environment.

## Best solution

File this against the Copilot Chat extension (https://github.com/microsoft/vscode-copilot-release) so the extension team can ensure pty.node prebuilds are packaged for all supported remote platforms.
