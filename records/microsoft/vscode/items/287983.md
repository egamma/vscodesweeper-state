# Issue #287983 — OpenType font features (such as alternate characters) not working in the integrated terminal

[https://github.com/microsoft/vscode/issues/287983](https://github.com/microsoft/vscode/issues/287983)

## Summary

Fixed on main and shipped in 1.125.0. The ligatures addon config change detection was broken — it never stored the config, causing the addon to churn on every refresh. A separate reporter confirmed the same symptom fixed in 1.125.1.

## Change summary

OpenType font feature settings (e.g. stylistic sets) do not display in the integrated terminal despite correct configuration.

## Best solution

Keep the shipped fix in commit 4ac5322601c (1.125.0); no further change needed. The ligatures addon now properly tracks its config and only recreates when settings actually change.
