# Issue #136748 — option for bracket pair vertical line depending on indentation

[https://github.com/microsoft/vscode/issues/136748](https://github.com/microsoft/vscode/issues/136748)

## Summary

Valid feature request for an option to decouple bracket pair vertical guides from indentation level. Not implemented on current main. Has 19 upvotes (just below the 20 threshold) and is in the Backlog milestone; keeping open for the community process.

## Change summary

Request an option so bracket pair vertical guides render at the bracket's column rather than at the indentation level, helping C/C++ users whose preprocessor directives sit at column 0.

## Best solution

Add a setting (e.g. `editor.guides.bracketPairs.indentationAware: true|false`) that controls whether vertical bracket guides use indentation-based positioning or bracket-column positioning. The implementation would likely touch the `getBracketPairsInRange` call path and the guide rendering in `src/vs/editor/common/model/bracketPairsTextModelPart/`. Given the niche audience (C/C++ with non-indented preprocessor directives) and hediet's stated preference for minimal settings, this could stay in the backlog until demand grows.
