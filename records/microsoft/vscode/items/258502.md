# Issue #258502 — Quick chat doesn't reflow horizontally when model changes

[https://github.com/microsoft/vscode/issues/258502](https://github.com/microsoft/vscode/issues/258502)

## Summary

Still present on main: the quick chat widget doesn't re-center horizontally when the model picker changes width. The registerListeners method in chatQuick.ts only handles vertical layout changes (height, sash, container resize) but has no listener for model-picker width changes to trigger a horizontal reflow.

## Change summary

Quick chat doesn't reflow its horizontal position when the selected model changes, leaving it visually offset.

## Best solution

Add a listener in `chatQuick.ts` (registerListeners or render) that detects when the model picker selection changes and triggers a horizontal re-centering of the quick chat container.
