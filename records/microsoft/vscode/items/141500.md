# Issue #141500 — QuickPick allow the mixing of checkbox items with non-checkbox items

[https://github.com/microsoft/vscode/issues/141500](https://github.com/microsoft/vscode/issues/141500)

## Summary

Team-authored feature request (hard keep-open). The `pickable: false` property (added in c170760ec54, March 2025) partially addresses this by hiding checkboxes for specific items in `canSelectMany` mode, but the broader vision of separator-based 'checkbox groups' vs 'normal groups' may not be fully realized.

## Change summary

Allow QuickPick to mix checkbox items with non-checkbox items, using separators to define which groups show checkboxes.

## Best solution

Evaluate whether the existing `pickable: false` property on `IQuickItem` (which hides checkboxes per-item in `canSelectMany` mode) fully satisfies this request, or whether a higher-level separator/group-based API is still needed. If satisfied, close as implemented; otherwise, define the remaining gap.
