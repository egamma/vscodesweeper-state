# Issue #164029 — Inline completions and snippets collide

[https://github.com/microsoft/vscode/issues/164029](https://github.com/microsoft/vscode/issues/164029)

## Summary

Implemented on main: the setting `editor.inlineSuggest.suppressInSnippetMode` (defaulting to `true`) now suppresses inline completions while in snippet mode, directly addressing this collision.

## Change summary

Inline completions interfere with snippet Tab navigation — pressing Tab to advance placeholders inadvertently accepts inline completions.

## Best solution

Keep the shipped behavior: `editor.inlineSuggest.suppressInSnippetMode` defaults to `true` since commit dd57d238873, suppressing inline completions during snippet sessions. No further change needed.
