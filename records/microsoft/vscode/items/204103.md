# Issue #204103 — mouse text selection cursor turned white on light themes

[https://github.com/microsoft/vscode/issues/204103](https://github.com/microsoft/vscode/issues/204103)

## Summary

Properly triaged upstream Chromium/GPU compositing bug affecting Windows users with AMD GPUs on light themes. Already labeled upstream/mitigated/chromium and serving as the canonical tracker for many duplicates — no action needed.

## Change summary

Mouse text-selection cursor renders white on light themes due to a Chromium direct-composition bug, making it invisible against white backgrounds.

## Best solution

Keep open as the canonical upstream tracker. The root cause is in Chromium's direct composition layer (https://issues.chromium.org/issues/40239916); the mitigation is `--disable_direct_composition=1` or disabling MPO at the OS level. Resolution depends on an upstream Chromium fix or AMD driver update.
