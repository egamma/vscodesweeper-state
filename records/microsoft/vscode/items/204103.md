# Issue #204103 — mouse text selection cursor turned white on light themes

[https://github.com/microsoft/vscode/issues/204103](https://github.com/microsoft/vscode/issues/204103)

## Summary

Well-triaged upstream Chromium/DirectComposition bug on Windows with AMD GPUs. Already labeled upstream/mitigated with 115 upvotes and a linked Chromium issue — should stay open as the tracking issue for VS Code users until the upstream fix lands.

## Change summary

Mouse text-selection cursor renders white on light themes in VS Code on Windows, caused by a Chromium DirectComposition bug with AMD GPUs.

## Best solution

Continue tracking the upstream Chromium bug (https://issues.chromium.org/issues/40239916). The workaround `--disable_direct_composition=1` is documented. When Chromium ships a fix and VS Code upgrades Electron to include it, this issue can be closed. No VS Code-side code change is needed — the CSS simply uses `cursor: text`.
