# Issue #271261 — Chat: Current file selection is debounced to the point of feeling sluggish

[https://github.com/microsoft/vscode/issues/271261](https://github.com/microsoft/vscode/issues/271261)

## Summary

Still valid on main: the 500ms debounce on implicit context updates in chatImplicitContext.ts has not been changed since the issue was filed. The sluggish current-file selection update in chat is a real polish issue affecting the feel of a core chat interaction.

## Change summary

The chat panel's current file context indicator updates with a 500ms debounce, making it feel sluggish when switching editors.

## Best solution

Reduce the debounce in `src/vs/workbench/contrib/chat/browser/attachments/chatImplicitContext.ts` (lines 64–71, 81–87, 91–97, 101) from 500ms to a shorter value (e.g. 100–150ms) for model/editor change events while possibly keeping a longer debounce for scroll/selection changes that fire at high frequency.
