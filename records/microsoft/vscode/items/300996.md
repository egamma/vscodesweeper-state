# Issue #300996 — Display two buttons for adding configurations

[https://github.com/microsoft/vscode/issues/300996](https://github.com/microsoft/vscode/issues/300996)

## Summary

Confirmed bug: the 'Add Configuration' button appears duplicated in the diff view of launch.json. A team member reproduced it. The menu registration in debugCommands.ts uses a regex path match on EditorContent but doesn't account for diff editor scenarios where the button may render in both panes.

## Change summary

When viewing changes to launch.json in the diff editor, duplicate 'Add Configuration' buttons are displayed.

## Best solution

The `AddConfigurationAction` menu registration at `src/vs/workbench/contrib/debug/browser/debugCommands.ts:994-999` should add a condition to exclude diff editors (e.g., checking that the editor is not in a diff context), or the button should only appear in the modified side of the diff.
