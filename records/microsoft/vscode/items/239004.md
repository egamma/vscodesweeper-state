# Issue #239004 — Filter input on the Output view hides all other panel tabs

[https://github.com/microsoft/vscode/issues/239004](https://github.com/microsoft/vscode/issues/239004)

## Summary

Fixed on main and shipped in 1.105.0 — the panel filter container's max-width was reduced from 400px to 200px by default, expanding only on focus, which prevents it from hiding other panel tabs.

## Change summary

The filter input in the Output panel title area grew too wide by default, pushing other panel tabs (Terminal, Problems, etc.) out of view.

## Best solution

Keep the shipped fix (commit 3605c9c0ded, 1.105.0). The filter container is now capped at 200px and expands to 400px only when focused. No further change needed.
