## Summary

Valid bug with a confirmed code path: the 800ms `passFocusScheduler` in debugSession.ts swaps focus away from the stepping thread when the step takes longer than 800ms in multi-target scenarios. An open PR (#324631) already addresses this — keep open pending that PR's review and merge.
