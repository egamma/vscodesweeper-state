# Issue #242494 — VSCode does not retrieve SSH passphrase from kwallet/ksshaskpass

[https://github.com/microsoft/vscode/issues/242494](https://github.com/microsoft/vscode/issues/242494)

## Summary

Verified duplicate of #57488: VS Code's integrated askpass overrides `SSH_ASKPASS` and sets `SSH_ASKPASS_REQUIRE: 'force'`, bypassing the user's ksshaskpass/kwallet setup. The canonical issue #57488 tracks this exact problem and has been open since 2018 with ongoing discussion.

## Change summary

VS Code's built-in askpass replaces the user's SSH_ASKPASS (ksshaskpass) so SSH passphrases are not retrieved from kwallet; the user is re-prompted on every Git operation.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/57488, which tracks the broader problem of VS Code's integrated askpass overriding the system's ssh-askpass configuration on Linux. The workaround (setting `GIT_ASKPASS` manually and disabling `git.useIntegratedAskPass`) is already documented there.
