# Issue #303993 — Context widget is not scrollable

[https://github.com/microsoft/vscode/issues/303993](https://github.com/microsoft/vscode/issues/303993)

## Summary

Team-authored issue — keeping open. The context usage details widget is rendered inside a workbench hover (`IHoverService.showInstantHover`) with no scrollable wrapper, so at higher zoom levels the content clips without a way to scroll.

## Change summary

The chat context usage details popup is not scrollable, making its content inaccessible at higher zoom levels.

## Best solution

Wrap the `ChatContextUsageDetails` content in a scrollable container (e.g. a `DomScrollableElement` or CSS `overflow-y: auto` with a `max-height`) so the hover content is scrollable when it exceeds the available viewport height. The fix lives in `src/vs/workbench/contrib/chat/browser/widgetHosts/viewPane/chatContextUsageDetails.ts` and its companion CSS file `media/chatContextUsageDetails.css`.
