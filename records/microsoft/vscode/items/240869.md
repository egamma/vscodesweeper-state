## Change summary

Terminal output wrapped at the visible column width breaks ProblemMatcher regex matching, causing diagnostics to be missed or truncated.

## Best solution

A proper fix would need to ensure the line data event always reconstructs the full unwrapped line regardless of terminal width or ConPTY cursor-move sequences.
