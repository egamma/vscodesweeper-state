# Issue #155874 — File Decorations for Search Folders 

[https://github.com/microsoft/vscode/issues/155874](https://github.com/microsoft/vscode/issues/155874)

## Summary

Team-authored feature request for search folder decorations — must stay open per hard keep-open rule (MEMBER author). The request remains valid: folders in search results should show decorations that reflect only the visible contained matches, not the full directory contents.

## Change summary

Add file decorations to search result folders that reflect only the decorations of contained search matches, not all files in the directory.

## Best solution

Implement folder-level decoration aggregation in the search results view (src/vs/workbench/contrib/search/browser/searchResultsView.ts) that computes a composite decoration from only the visible child matches, rather than inheriting the full directory's decorations.
