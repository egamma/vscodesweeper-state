# Issue #278109 — Debugger completions from debug adapter are not highlighting

[https://github.com/microsoft/vscode/issues/278109](https://github.com/microsoft/vscode/issues/278109)

## Summary

Still reproduces on main: DAP completion items without `start`/`length` get a zero-length range and no filterText, so the suggest widget cannot compute fuzzy-match highlights. An open PR (#308029) is in progress to fix this.

## Change summary

Debug console completions from DAP adapters do not highlight matching characters because range and filterText are not correctly derived when the DAP item omits `start`/`length`.

## Best solution

Land or refine the open PR https://github.com/microsoft/vscode/pull/308029 which fixes the highlight matching logic in `repl.ts`.
