## Summary

Still present on main: `bracketGuides[lineNumber - visibleStartLineNumber]` at indentGuides.ts:207 can be `undefined` when the model changes between visible-range computation and guide fetching, causing the `ArrayQueue` constructor to crash on `this.items.length`.

## Change summary

Editor indent guides crash with 'Cannot read properties of undefined (reading length)' during rendering when bracket guides array is shorter than expected.

## Best solution

Guard the array access at indentGuides.ts:207 so that an out-of-bounds result falls back to `[]` instead of passing `undefined` to `ArrayQueue`. E.g. `const bracketGuidesInLine = bracketGuides ? (bracketGuides[lineNumber - visibleStartLineNumber] ?? []) : [];`
