# Issue #318198 — Default themes should not include markdown suffix

[https://github.com/microsoft/vscode/issues/318198](https://github.com/microsoft/vscode/issues/318198)

## Summary

This is a valid theming cleanup request, not a confirmed bug: current default themes still use Markdown-specific punctuation scopes in several places, and the request needs theme-owner judgment about compatibility and token-color precedence.

## Change summary

Use generic markup TextMate scopes in default themes so other markup grammars can reuse list and quote coloring without matching Markdown-specific suffixes.

## Best solution

Route to the themes owners to decide whether default token color rules should prefer generic `markup.list` and `markup.quote` scopes, then update the relevant default theme JSON and colorize baselines if accepted.
