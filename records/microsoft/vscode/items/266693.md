# Issue #266693 — Improve Manage Trusted Extensions Settings Dialog

[https://github.com/microsoft/vscode/issues/266693](https://github.com/microsoft/vscode/issues/266693)

## Summary

Valid polish request to improve the Manage Trusted Extensions dialog UX when multiple accounts are signed in. A team member (TylerLeonhardt) engaged and the request remains actionable — the current dialog doesn't surface which account an extension is actively using without clicking through per-extension.

## Change summary

Show which account each extension is currently using in the Manage Trusted Extensions dialog and allow inline account switching without opening a separate dialog per extension.

## Best solution

Enhance the quick-pick items in `manageTrustedExtensionsForAccountAction.ts` to display the active account label per extension (from `accountQuery.extension(ext.id).getUsage()`) and consider inlining account-preference selection rather than requiring the separate gear-button flow.
