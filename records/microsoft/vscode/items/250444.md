# Issue #250444 — Add a custom icon for prompt, chat mode and instruction files

[https://github.com/microsoft/vscode/issues/250444](https://github.com/microsoft/vscode/issues/250444)

## Summary

Team-authored feature request still open — custom icons for .prompt, .chatmode, and .instructions files have not been added to the seti font. Currently these extensions map to the generic markdown icon.

## Change summary

Add dedicated custom icons (in the seti icon font) for .prompt, .chatmode, and .instructions file types so they are visually distinct in the file explorer.

## Best solution

Design and contribute new glyphs to the seti icon font for these file types, then wire the extensions in `extensions/theme-seti/build/update-icon-theme.js` to reference the new glyphs instead of `_markdown`.
