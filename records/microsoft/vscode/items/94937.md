# Issue #94937 — Truncate and copy large values from the debug console

[https://github.com/microsoft/vscode/issues/94937](https://github.com/microsoft/vscode/issues/94937)

## Summary

Team-authored feature request with 26 upvotes; the requested UI-side truncation with copy/show-all buttons for large debug console values remains unimplemented on current main. Keep open — needs maintainer scheduling.

## Change summary

Add UI-side truncation of large values in the debug console with inline copy and show-all buttons, rather than relying on debug adapters to truncate before sending data.

## Best solution

Implement UI-side truncation in the debug console tree renderer so large strings/objects are clipped with inline 'Copy' and 'Show All' affordances, using the `evaluateName`/`evaluate` clipboard context from DAP to retrieve the full value. The work lives in `src/vs/workbench/contrib/debug/browser/` (console rendering and commands). The existing `COPY_VALUE_ID` command infrastructure can be extended.
