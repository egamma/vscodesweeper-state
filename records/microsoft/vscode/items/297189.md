# Issue #297189 — Code folding arrows are overlaid by git diff vertical line

[https://github.com/microsoft/vscode/issues/297189](https://github.com/microsoft/vscode/issues/297189)

## Summary

The dirty-diff gutter decoration uses z-index: 5 (in dirtydiffDecorator.css) and overlays the folding control icons (which have no z-index in folding.css), making fold arrows hard to see and click. Still present on current main.

## Change summary

Git dirty-diff gutter indicator visually overlaps folding arrows, making them hard to see and target.

## Best solution

Adjust z-index or layout so that folding controls render above the dirty-diff gutter line, or shift the dirty-diff indicator to avoid overlapping the fold icon hit area. The fix would touch `src/vs/workbench/contrib/scm/browser/media/dirtydiffDecorator.css` and/or `src/vs/editor/contrib/folding/browser/folding.css`.
