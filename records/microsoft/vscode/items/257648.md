## Change summary

When mouse hovers over a sticky folder in the file explorer, a partially-overlapped secondary sticky row visually reveals itself.

## Best solution

Likely needs the container to clip the last row's overflow at exactly the container height boundary, or ensure each row's stacking context prevents bleed-through on hover.
