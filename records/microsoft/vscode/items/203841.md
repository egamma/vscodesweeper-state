# Issue #203841 — Git - Display blame information in the editor gutter

[https://github.com/microsoft/vscode/issues/203841](https://github.com/microsoft/vscode/issues/203841)

## Summary

Still a valid open feature request with 49 upvotes. The built-in git blame now shows inline decoration for the current line and a status bar item (shipped ~2024-Q4), but the full gutter blame view (all lines, left margin) requested here has not been implemented. Team members have discussed it in iteration plans but it remains pending.

## Change summary

Display git blame information for all lines in the editor gutter (left margin), similar to traditional blame views.

## Best solution

Implement a gutter-based blame view in `extensions/git/src/blame.ts` that decorates all visible lines with authorship metadata in the editor margin, complementing the existing single-line inline decoration. The team has already signaled intent (iteration plans Feb/Mar 2025); this needs scheduling.
