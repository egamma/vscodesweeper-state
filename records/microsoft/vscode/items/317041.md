# Issue #317041 — Chat terminals are nowhere to be found

[https://github.com/microsoft/vscode/issues/317041](https://github.com/microsoft/vscode/issues/317041)

## Summary

Team-authored issue (MEMBER) reporting that chat-used terminals are not discoverable in terminal tabs or via clicking commands in chat. Hard keep-open applies.

## Change summary

Terminals used by chat/agent sessions are not visible in terminal tabs and clicking commands in chat doesn't reveal them.

## Best solution

Improve discoverability of agent-spawned terminals — either surface them reliably in terminal tabs, ensure click-to-reveal works from chat commands, or make the hidden-terminal indicator more persistent. The workaround via `chat.tools.terminal.outputLocation` setting exists but the default UX should be better. Related work in PR #322121 and issue #321049.
