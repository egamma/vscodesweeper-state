# Issue #282251 — Allow Ctrl and Shift select functions when deleting breakpoints

[https://github.com/microsoft/vscode/issues/282251](https://github.com/microsoft/vscode/issues/282251)

## Summary

Valid feature request: multi-select in the breakpoints view is explicitly disabled (`multipleSelectionSupport: false` in breakpointsView.ts). The request to enable Ctrl/Shift selection for bulk deletion is reasonable and well-scoped, but has only 3 upvotes — keeping open for community traction.

## Change summary

Allow Ctrl+click and Shift+click multi-selection in the breakpoints view to enable selective bulk deletion.

## Best solution

Enable `multipleSelectionSupport: true` in `src/vs/workbench/contrib/debug/browser/breakpointsView.ts` and wire the delete action to operate on all selected items. This would also require updating context menu actions to handle multi-selection properly.
