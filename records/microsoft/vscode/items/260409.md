[https://github.com/microsoft/vscode/issues/260409](https://github.com/microsoft/vscode/issues/260409)

## Summary

Likely an upstream Electron/Chromium rendering regression introduced with Electron 37 (VS Code 1.103.0). The bug is confirmed by multiple users on Windows with display scaling below 150%, does not occur in Chrome/vscode.dev, and correlates exactly with the Electron version bump. Current main now runs Electron 42 — the issue may have been resolved by the upgrade, but this needs verification.
