# Issue #155101 — [json] Show key value pairs of JSON Array in `Outline` view

[https://github.com/microsoft/vscode/issues/155101](https://github.com/microsoft/vscode/issues/155101)

## Summary

Valid feature request to show array element values (not just indices) in the JSON Outline view. Still unimplemented — the document symbol provider in vscode-json-languageservice only emits index-based names for array items. Keeping open as a legitimate enhancement in the Backlog.

## Change summary

JSON arrays in the Outline view show only numeric indices (0, 1, 2) instead of meaningful value previews for their elements.

## Best solution

Enhance the `findDocumentSymbols`/`findDocumentSymbols2` implementation in `microsoft/vscode-json-languageservice` to emit a short value preview (e.g. first string property or primitive value) as the symbol name for array items, rather than just the numeric index. The VS Code side delegates via `extensions/json-language-features/server/src/jsonServer.ts` and would pick this up automatically once the upstream library ships it.
