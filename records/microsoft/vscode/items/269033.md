# Issue #269033 — Inline chat for a chat session model picker does not apply to delegated sessions

[https://github.com/microsoft/vscode/issues/269033](https://github.com/microsoft/vscode/issues/269033)

## Summary

The inline chat model picker does not reflect or defer to the model chosen in the delegated parent session. A MEMBER comment confirms the picker is 'meaningless' in this context and that a better UX for inline delegation is planned but not yet implemented. No fix found on main.

## Change summary

When sending additional input to a chat session (e.g. from a diff hunk), the inline chat model picker should show the model selected in the parent chat session rather than an independent/meaningless selection.

## Best solution

The inline chat delegation path should either hide the model picker entirely when delegating (since it has no effect) or sync it to display the parent session's model. Likely involves `src/vs/workbench/contrib/inlineChat/browser/inlineChatController.ts` and the model picker widget integration in `src/vs/workbench/contrib/chat/browser/widget/input/`.
