# Issue #284175 — Copilot Edits: Arrow up should not change attached files

[https://github.com/microsoft/vscode/issues/284175](https://github.com/microsoft/vscode/issues/284175)

## Summary

Team-authored issue requesting that arrow up/down in the chat input (Copilot Edits) cycle through previous prompts without restoring the attached file context. Hard keep-open: author is MEMBER.

## Change summary

Arrow up/down in the Copilot Edits chat input restores both the prompt text and the attached files context, making it difficult to retry a request with different context.

## Best solution

Decouple prompt history navigation (arrow up/down) from context/attached-files restoration in the chat widget, so users can cycle through previous prompts while keeping their current file attachments unchanged.
