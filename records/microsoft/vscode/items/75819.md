# Issue #75819 — [folding] Fold current level

[https://github.com/microsoft/vscode/issues/75819](https://github.com/microsoft/vscode/issues/75819)

## Summary

Valid feature request for a 'Fold Current Level' command that would fold all regions at the same nesting level as the cursor. Not yet implemented on main — no such action exists in the folding contribution. Parked in Backlog with 9 upvotes, below the 20-upvote promotion threshold but still a reasonable ask.

## Change summary

Request a built-in command to fold all folding regions at the same indentation/nesting level as the cursor position, without needing to manually count the level number.

## Best solution

Add a new FoldingAction (e.g. `FoldCurrentLevelAction`) in `src/vs/editor/contrib/folding/browser/folding.ts` that determines the folding level at the cursor and folds all regions at that level, similar to the existing `FoldLevelAction` but cursor-aware. Until it reaches 20 upvotes or a maintainer champions it, it can remain in the Backlog.
