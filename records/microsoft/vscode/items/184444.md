# Issue #184444 — Test Explorer: Sort tests by name

[https://github.com/microsoft/vscode/issues/184444](https://github.com/microsoft/vscode/issues/184444)

## Summary

Valid feature request for a 'Sort by Name' option in Test Explorer. The issue carries the 'On Deck' milestone (a scheduling decision), so it must be kept open. Current code only offers Sort by Status, Location, and Duration — no dedicated name sort exists.

## Change summary

Add a 'Sort by Name' option to the Test Explorer tree view, so tests in the same file can be alphabetically sorted for easier scanning.

## Best solution

Add a `ByName` entry to `TestExplorerViewSorting` in `src/vs/workbench/contrib/testing/common/constants.ts`, a corresponding `TestingSortByNameAction` in `testExplorerActions.ts`, and handle it in the `TreeSorter.compare` method in `testingExplorerView.ts` to sort purely by label (ignoring location). This is a small, self-contained change.
