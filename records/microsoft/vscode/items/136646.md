# Issue #136646 — Feature request: Option to colorize only active bracket pairs

[https://github.com/microsoft/vscode/issues/136646](https://github.com/microsoft/vscode/issues/136646)

## Summary

Valid feature request still unimplemented on main — bracket pair colorization only supports a boolean `enabled`, not an `"active"` mode like guides do. Has 15 upvotes and a related open issue (#150648). Keeping open in the backlog.

## Change summary

Add an option to colorize only the active (cursor-enclosing) bracket pair, analogous to the existing `"active"` mode for bracket pair guides.

## Best solution

Extend `editor.bracketPairColorization.enabled` (or add a new setting) to accept `"active"` alongside `true`/`false`, reusing the active-bracket-pair detection already used by `editor.guides.bracketPairs: "active"`. The implementation would live in `src/vs/editor/browser/viewParts/` (bracket colorization rendering) and `src/vs/editor/common/config/editorOptions.ts`. Issue #150648 requests essentially the same thing and could be consolidated.
