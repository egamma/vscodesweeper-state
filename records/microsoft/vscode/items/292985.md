## Summary

The requested setting now exists on main: `chat.agent.thinkingStyle` offers three modes including `fixedScrolling` (now the default), which shows reasoning in a visible, streaming panel instead of collapsing it. This directly addresses the request for more transparent thinking display.

## Change summary

Request for a setting to keep chat reasoning/thinking content expanded and visible instead of auto-collapsing into grey boxes.

## Best solution

The `chat.agent.thinkingStyle` setting (with `collapsed`, `collapsedPreview`, and `fixedScrolling` modes) already ships on current main, with `fixedScrolling` as the default — this keeps reasoning visible in a fixed-height streaming panel. The `chat.agent.thinking.collapsedTools` setting further controls tool call visibility. No additional work is needed.
