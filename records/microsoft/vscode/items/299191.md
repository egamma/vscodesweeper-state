# Issue #299191 — Ask questions hover should render to the side

[https://github.com/microsoft/vscode/issues/299191](https://github.com/microsoft/vscode/issues/299191)

## Summary

Team-authored polish issue — the ask-questions hover uses HoverPosition.BELOW and can overlap items; changing to a side position would fix it. Still present on main.

## Change summary

The ask-questions tool hover renders below the item, overlapping adjacent content, instead of rendering to the side.

## Best solution

Change `HoverPosition.BELOW` to `HoverPosition.RIGHT` (or `LEFT`) in `chatQuestionCarouselPart.ts` at lines 1163 and 1374 so the hover doesn't overlap adjacent list items.
