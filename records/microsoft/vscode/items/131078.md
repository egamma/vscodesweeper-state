# Issue #131078 — Show bracket in ruler

[https://github.com/microsoft/vscode/issues/131078](https://github.com/microsoft/vscode/issues/131078)

## Summary

Valid feature request with 27 upvotes, promoted to Backlog. The bracket pair colorization system on current main has no overview ruler integration — only the older bracket matching feature shows a single matched pair at the cursor in the ruler. The requested colored-bracket-pairs-in-ruler capability remains unimplemented.

## Change summary

Show bracket pair colorization markers in the overview ruler so users can see bracket scope boundaries when scrolling long files.

## Best solution

Extend the bracket pair colorization contribution (under `src/vs/editor/common/model/bracketPairsTextModelPart/`) to emit overview ruler decorations with bracket-level colors, gated behind a setting like `editor.bracketPairColorization.showInOverviewRuler`. The existing `editorOverviewRuler.bracketMatchForeground` color in `bracketMatching.ts` is only for cursor-adjacent matching, not for all colored pairs.
