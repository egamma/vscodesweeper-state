# Issue #230128 — how to disable Word Wrap globally

[https://github.com/microsoft/vscode/issues/230128](https://github.com/microsoft/vscode/issues/230128)

## Summary

This appears fixed on current main: the long-line automatic wrapping that matched the reporter's 10000+ character repro is now gated to screen-reader mode instead of overriding normal `editor.wordWrap: off` behavior.

## Change summary

A plain text file or notebook cell with a very long line reopened wrapped even when global word wrap was off.

## Best solution

Treat this as completed by commit 983397225279176dbeecfaf0379ec285699e8f84, shipped in 1.103.0, which preserves the accessibility-specific long-line wrapping behavior while avoiding it for normal editor sessions.
