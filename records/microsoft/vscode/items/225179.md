# Issue #225179 — Python: Improved Bracket Pair Detection for strings that contain `{{` 

[https://github.com/microsoft/vscode/issues/225179](https://github.com/microsoft/vscode/issues/225179)

## Summary

Valid feature request in the 60-day community voting window (started 2025-06-02). Only 4 upvotes so far, needs 20 to be promoted. A concrete solution exists (adding `unbalancedBracketScopes` for `constant.character.escape` in the Python grammar) but has not been implemented.

## Change summary

Bracket pair colorization incorrectly colors escaped `{{` and `}}` inside Python f-strings as bracket pairs instead of treating them as escape characters.

## Best solution

Add `"unbalancedBracketScopes": ["constant.character.escape"]` to the Python grammar entry in `extensions/python/package.json`, as suggested in comment 11. This would tell the bracket matching engine to ignore brackets scoped as escape characters, matching behavior already used by other language extensions (C#, YAML, Perl, etc.).
