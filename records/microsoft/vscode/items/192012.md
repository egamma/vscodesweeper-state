# Issue #192012 — Add option to turn off rich content notebook search results

[https://github.com/microsoft/vscode/issues/192012](https://github.com/microsoft/vscode/issues/192012)

## Summary

Team-authored feature request for a setting to disable rich content notebook search results once they become the default. The experimental setting `search.experimental.closedNotebookRichContentResults` still defaults to `false` on current main, so the feature hasn't graduated to default-on yet — the request remains forward-looking and valid.

## Change summary

Provide a user-facing setting to opt out of rich content notebook search results once they are enabled by default.

## Best solution

When `search.experimental.closedNotebookRichContentResults` graduates from experimental to default-on, add a permanent non-experimental setting (or repurpose the existing one) so users who need plain-text metadata search can disable rich content rendering. The search area owner should decide whether the setting name/scope changes at graduation time.
