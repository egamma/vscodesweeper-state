# Issue #322935 — Model picker: add settings to disable "Auto" grouping and expand "Other Models" by default

[https://github.com/microsoft/vscode/issues/322935](https://github.com/microsoft/vscode/issues/322935)

## Summary

Valid feature request for two new settings to customize the Copilot model picker: disabling the Auto/recently-used grouping, and defaulting the 'Other Models' section to expanded. The request is well-articulated, distinct from existing tracked issues, and the model picker area is actively maintained. Keeping open for community traction and maintainer evaluation.

## Change summary

Add settings to optionally disable the 'Auto' recently-used grouping in the model picker and to expand the 'Other Models' section by default.

## Best solution

Evaluate adding configuration for the model picker's grouping behavior in `src/vs/workbench/contrib/chat/browser/widget/input/chatModelConfigurationLogic.ts` and related files. The two proposed settings are independently useful: one controls whether recently-used models are promoted into a separate group (with deduplication), the other controls the collapsed/expanded default of 'Other Models'. Related issues exist (https://github.com/microsoft/vscode/issues/266950 for persistent model control, https://github.com/microsoft/vscode/issues/314513 for the collapsed-section discrepancy, https://github.com/microsoft/vscode/issues/312908 for the deduplication bug) but none fully absorbs both requests here.
