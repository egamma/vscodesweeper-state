# Issue #66468 — Shrink empty lines

[https://github.com/microsoft/vscode/issues/66468](https://github.com/microsoft/vscode/issues/66468)

## Summary

The underlying variable-line-height engine is merged and enabled on main, but the extension API to set per-line heights via decorations is not yet exposed in vscode.d.ts. The feature request remains valid until extensions can actually implement 'shrink empty lines' behavior.

## Change summary

Request to shrink empty/whitespace-only lines in the editor to reclaim vertical space, similar to a Visual Studio extension.

## Best solution

Expose `lineHeight` (or equivalent) on `DecorationRenderOptions` in the public extension API (`src/vscode-dts/vscode.d.ts`), enabling extensions to implement shrink-empty-lines. Once the API ships, this issue and the linked https://github.com/microsoft/vscode/issues/98410 can both be closed.
