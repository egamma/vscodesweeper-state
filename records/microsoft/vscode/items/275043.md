# Issue #275043 — Agents menu - Configure Custom Agents item

[https://github.com/microsoft/vscode/issues/275043](https://github.com/microsoft/vscode/issues/275043)

## Summary

Still reproduces on main: the "Configure Custom Agents" link in the agent picker is rendered as an action-bar action with text-link styling and `background-color: transparent !important` on hover, while list items above get a proper hover background via `.monaco-list-row.action.focused`. Team-authored issue — keeping open.

## Change summary

The "Configure Custom Agents" item in the agent picker dropdown lacks a hover background highlight, making it visually inconsistent with the list items above it.

## Best solution

Either render the "Configure Custom Agents" action as a regular list item (via `ActionListItemKind.Action` in the main list rather than via `actionBarActionProvider`) so it inherits the same hover styling, or add a hover background to `.action-widget-action-bar .action-label:hover` instead of the current `background-color: transparent !important`. The fix lives in `src/vs/platform/actionWidget/browser/actionWidget.css` (hover rule) or in how `modePickerActionItem.ts` provides the action.
