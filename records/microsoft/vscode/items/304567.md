# Issue #304567 — Agent Sessions: Increase padding at bottom of session list group

[https://github.com/microsoft/vscode/issues/304567](https://github.com/microsoft/vscode/issues/304567)

## Summary

Team-authored polish request for the Agents window session list. The session-section header currently has no bottom padding/margin on the preceding group's last item; adding 12px spacing between the last item and the next section header would improve visual grouping. Keeping open — team member issue.

## Change summary

Increase the padding between the last item in a session list group and the next section header to 12px for better visual separation.

## Best solution

Add a bottom margin or padding (12px) to the last `.session-item` row before a `.session-section` header, likely in `src/vs/sessions/contrib/sessions/browser/media/sessionsList.css`. Could be done via a CSS rule on the last row in each group or by adjusting the list delegate's height calculation for section rows.
