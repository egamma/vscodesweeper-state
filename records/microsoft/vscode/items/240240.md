# Issue #240240 — Terminal suggest: Support configuring fig specs

[https://github.com/microsoft/vscode/issues/240240](https://github.com/microsoft/vscode/issues/240240)

## Summary

Valid, unimplemented feature request from the terminal maintainer to allow users to provide custom fig specs for terminal completions. Active discussion and clear direction — keep open in the Backlog.

## Change summary

Allow users to add their own fig completion specs (user-level and workspace-level) so terminal suggest can be extended beyond built-in completions.

## Best solution

Implement a configuration point (e.g. a user-data-dir path like `<user_data_dir>/terminalcompletions` plus workspace-level specs) that the terminal-suggest extension reads at activation, merging user specs with built-in ones. The work lives in `extensions/terminal-suggest/`.
