# Issue #180156 — [linked editing] Linked editing stops syncing on quick edits

[https://github.com/microsoft/vscode/issues/180156](https://github.com/microsoft/vscode/issues/180156)

## Summary

Fixed on main by commit fbefb06cae2 (PR #242993, shipped in 1.104.0). The root cause — unconditional `clearRanges()` during recompute causing desync on rapid edits — is addressed by only clearing when the cursor moves outside the current decoration range.

## Change summary

Linked editing stops syncing tags when typing quickly because ranges are prematurely cleared during recomputation.

## Best solution

Keep the shipped fix (commit fbefb06cae2, 1.104.0). No further change needed — the desync during rapid editing is resolved.
