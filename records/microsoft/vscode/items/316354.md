# Issue #316354 — Profile support for Agent Window

[https://github.com/microsoft/vscode/issues/316354](https://github.com/microsoft/vscode/issues/316354)

## Summary

The Agents window now has a dedicated internal profile (commit acc18a057fe), but the request for user-switchable profiles and CLI --profile support remains unimplemented. Keeping open in Backlog Candidates for the community upvote process.

## Change summary

Allow the Agents window to adopt user profiles, switch between them, and accept a --profile CLI flag.

## Best solution

If the team decides profile customization fits the Agents window vision, extend the current internal-profile mechanism (src/vs/platform/userDataProfile/common/userDataProfile.ts and src/vs/sessions/) to support user-created profiles. Otherwise, close as *as-designed if the opinionated fixed profile is the intended end state. The community process will decide.
