# Issue #247401 — No way to accept individual change in merge editor using keyboard?

[https://github.com/microsoft/vscode/issues/247401](https://github.com/microsoft/vscode/issues/247401)

## Summary

Team-authored accessibility bug; the merge editor's individual-conflict accept/toggle commands lack default keybindings, so keyboard-only users cannot accept a single change. Still unfixed on main.

## Change summary

No default keyboard shortcut exists to accept an individual change in the merge editor.

## Best solution

Add default keybindings to `ToggleActiveConflictInput1` and `ToggleActiveConflictInput2` (and possibly a dedicated 'accept current combination' command) in `src/vs/workbench/contrib/mergeEditor/browser/commands/commands.ts`, and ensure the merge editor's accessibility help dialog documents them.
