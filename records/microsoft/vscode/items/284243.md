## Summary

Legitimate feature request already labeled under-discussion and routed to editor-core. A team member noted the tab context menu is already overloaded; the decision rests with the editor team. No implementation exists on main.

## Change summary

Add an 'Open in Integrated Terminal' option to the editor tab right-click context menu to open a terminal at the file's parent directory.

## Best solution

The editor team should decide whether to add this menu item (registering a terminal command in `MenuId.EditorTitleContext` gated on `ResourceContextKey.IsFileSystemResource`) or decline it as menu overload.
