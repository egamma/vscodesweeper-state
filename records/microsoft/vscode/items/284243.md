## Summary

This is a duplicate of #194791 which asked for the identical feature and was closed as *extension-candidate — the team's position is that adding 'Open in Integrated Terminal' to the editor tab context menu belongs in an extension rather than core.

## Change summary

Add an 'Open in Integrated Terminal' entry to the editor tab right-click context menu.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/194791 — the team already decided this belongs as an extension. The `openInIntegratedTerminal` command exists and works; an extension can register it in `MenuId.EditorTitleContext` via the `menus` contribution point in `package.json`.
