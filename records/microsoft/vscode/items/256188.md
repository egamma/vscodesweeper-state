# Issue #256188 — Minimized windows cannot be restored

[https://github.com/microsoft/vscode/issues/256188](https://github.com/microsoft/vscode/issues/256188)

## Summary

VS Code intentionally does not persist minimized window state — the code explicitly comments "we don't want to save minimized state, only maximized or normal" and the WindowMode.Minimized enum is marked as no longer used. This is a deliberate design choice, not a bug.

## Change summary

When VS Code windows are restored after closing all windows, previously minimized windows appear in their normal (non-minimized) state instead of remaining minimized.

## Best solution

Close as designed. Persisting minimized state is intentionally excluded — restoring windows as minimized would be confusing for most users (invisible windows). The related Electron feature request (electron/electron#37494) is still open, suggesting platform-level support is also limited. If this were to change, it would be a feature request requiring careful UX consideration.
