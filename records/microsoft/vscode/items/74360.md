# Issue #74360 — [folding] Fold All in selection

[https://github.com/microsoft/vscode/issues/74360](https://github.com/microsoft/vscode/issues/74360)

## Summary

Keep open as a low-traction Backlog feature request in the existing folding area. Current main still keeps `Fold All` as whole-file behavior, while the related `Fold Recursively` command handles selected lines, so this needs editor-folding owner/product judgment rather than cleanup closure.

## Change summary

Make `Fold All` apply only within the current selection when a selection exists.

## Best solution

Editor-folding owners should decide whether `Fold All` should remain whole-file behavior and point users to `Fold Recursively`, or whether `FoldAllAction` should be changed to pass selected lines into the existing folding model helper.
