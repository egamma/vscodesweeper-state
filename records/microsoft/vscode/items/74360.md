# Issue #74360 — [folding] Fold All in selection

[https://github.com/microsoft/vscode/issues/74360](https://github.com/microsoft/vscode/issues/74360)

## Summary

This feature request for a 'Fold All in Selection' command remains unimplemented on main. The existing 'Fold All Except Selected' does the inverse (folds everything outside the selection). The request is valid and distinct.

## Change summary

Request a 'Fold All in Selection' command that folds only ranges within the current text selection, rather than folding the entire file.

## Best solution

Add a new `FoldAllInSelectionAction` in `src/vs/editor/contrib/folding/browser/folding.ts` that calls `setCollapseStateLevelsDown` constrained to the line range of the current selection. This would complement the existing 'Fold All Except Selected' command.
