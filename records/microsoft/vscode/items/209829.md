# Issue #209829 — Mouse middle button could close a pinned tab

## Summary

The requested capability already exists via the `workbench.editor.preventPinnedEditorClose` setting (values: `keyboardAndMouse`, `keyboard`, `mouse`, `never`). Setting it to `keyboard` allows middle-click to close pinned tabs while still protecting against keyboard shortcuts. However, since the author is a MEMBER, keeping open for maintainer judgment.
