## Summary

The SCM commit message input already supports inline completions (ghost text autocomplete) via the InlineCompletionsController, which is wired into the editor. Extensions like GitHub Copilot provide the actual AI-powered suggestions. This feature already exists — the distinction the reporter draws between 'full message generation' (sparkle button) and 'autocomplete as you type' is exactly what inline completions deliver.
