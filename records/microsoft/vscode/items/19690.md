# Issue #19690 — [folding] show folding lines

[https://github.com/microsoft/vscode/issues/19690](https://github.com/microsoft/vscode/issues/19690)

## Summary

This is a multi-part feature request in Backlog with 17 upvotes (below the 20 threshold). Some sub-features have been partially addressed (folding highlight background via `editor.foldingHighlight`, bracket pair guides via `editor.guides.bracketPairs`), but the primary request — VS-style folding lines in the gutter connecting fold start/end — remains unimplemented. Keeping open as a valid feature request.

## Change summary

Show folding region lines/guides in the editor gutter connecting the start and end of foldable blocks, with options for highlight, peek, and collapse from bottom bracket.

## Best solution

The issue bundles several sub-features. Sub-requests #1 (single-line collapse) is tracked in https://github.com/microsoft/vscode/issues/3352. The folding highlight (request #4) is partially addressed by `editor.foldingHighlight`. The core remaining gap is gutter fold-lines (request #2) and peek-at-top-bracket (request #3). This could be evaluated as a future enhancement to the existing `editor.guides` system or the folding decoration provider in `src/vs/editor/contrib/folding/browser/foldingDecorations.ts`.
