# Issue #188312 — Terminal tab home CWD is not handled correctly in remote workspaces

[https://github.com/microsoft/vscode/issues/188312](https://github.com/microsoft/vscode/issues/188312)

## Summary

Still reproducible on main: the terminal tab label logic in `computeLabel` doesn't account for CWD being the user's home directory in remote workspaces — it only compares against the workspace folder URI, so it incorrectly shows the folder name when CWD is home.

## Change summary

Terminal tab incorrectly shows folder name when CWD is the user's home directory in a remote workspace.

## Best solution

In `TerminalLabelComputer.computeLabel`, add a check that suppresses `cwdFolder` when the resolved CWD equals `instance.userHome` — the `userHome` property is already available on the instance parameter but unused in the showCwd logic.
