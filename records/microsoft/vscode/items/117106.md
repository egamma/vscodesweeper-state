# Issue #117106 — Advanced wrapping strategy causes hangs for larger files

[https://github.com/microsoft/vscode/issues/117106](https://github.com/microsoft/vscode/issues/117106)

## Summary

The report describes a real editor wrapping performance gap that was acknowledged as tied to `editor.wrappingStrategy: advanced`; I did not find enough concrete evidence in the provided discussion to close it as fixed, duplicate, or as-designed.

## Change summary

Large text files can hang or become very slow to open/resize when word wrap uses the advanced wrapping strategy.

## Best solution

Keep this routed to the editor wrapping/performance owners to decide whether large-file heuristics should switch away from advanced wrapping, time out browser wrapping computation, or otherwise degrade gracefully for large files.
