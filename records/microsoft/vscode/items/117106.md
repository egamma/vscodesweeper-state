# Issue #117106 — Advanced wrapping strategy causes hangs for larger files

[https://github.com/microsoft/vscode/issues/117106](https://github.com/microsoft/vscode/issues/117106)

## Summary

The issue is real but already documented behavior — the advanced wrapping strategy explicitly warns about freezes on large files. The constructive suggestion (auto-fallback to simple strategy for large files, like tokenization limits) remains unimplemented on main. Keeping open as a valid improvement request.

## Change summary

Advanced wrapping strategy (`editor.wrappingStrategy: advanced`) causes multi-second freezes when opening or resizing large files (80k+ lines).

## Best solution

Implement an automatic fallback from advanced to simple wrapping for files above a size threshold (similar to how tokenization is limited for large files). The logic would live in `src/vs/editor/common/viewModel/viewModelLines.ts` where `wrappingStrategy` is applied. This is a small, well-scoped change that would eliminate the freeze without user intervention.
