## Summary

Team-authored flaky smoke test issue with multiple confirmed occurrences from team members; must remain open for maintainer judgment.

## Change summary

The 'install and enable vscode-smoketest-check extension' smoke test intermittently times out waiting for the 'Disable' button to appear after extension installation.

## Best solution

Investigate the race condition in the extension install flow — the test waits 20s for the Disable button in the extension editor, which suggests either slow gallery fetches, slow extension activation, or a UI rendering delay. Likely needs a longer timeout or a more resilient wait condition in `test/automation/src/extensions.ts`.
