## Change summary

Terminal in a floating (auxiliary) window does not echo typed characters in real time when the main VS Code window is minimized during a remote connection (WSL/SSH).

## Best solution

Investigate the remote connection data flow when the main window is minimized — likely the renderer or remote data relay pauses/throttles when the main window's document becomes hidden, without considering that an auxiliary window is still visible and active.
