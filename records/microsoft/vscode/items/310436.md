# Issue #310436 — Restrict skills to specific agents

[https://github.com/microsoft/vscode/issues/310436](https://github.com/microsoft/vscode/issues/310436)

## Summary

Valid feature request to scope skills to specific agents. Instructions already support `excludeAgent`, but skills (PromptsType.skill) lack any agent-restriction attribute. Low community traction so far (3 upvotes).

## Change summary

Allow skills to be restricted to specific agents rather than being globally accessible to all agents.

## Best solution

Add an `agents` or `excludeAgent` attribute to skill file headers (PromptsType.skill in promptValidator.ts / promptFileParser.ts) so that skill resolution in computeAutomaticInstructions.ts filters skills by the active agent. This mirrors the existing `excludeAgent` pattern on instruction files.
