# Issue #128923 — Allow exact and excluded matches in fuzzy search

[https://github.com/microsoft/vscode/issues/128923](https://github.com/microsoft/vscode/issues/128923)

## Summary

The quoted/exact-match portion was partially addressed via contiguous-match support (PR #131292), but the exclusion feature (`-term`) was never implemented. The issue remains valid for the outstanding exclusion request, and the related issue #131431 is also still open.

## Change summary

Allow exact matches (quoted strings) and excluded terms (minus prefix) in the quick-open fuzzy search.

## Best solution

Implement exclusion-term support in `src/vs/base/common/fuzzyScorer.ts` (extending `prepareQuery` to parse `-prefix` tokens and filtering out matches that contain those tokens). The quoted-match part is already shipped via `expectContiguousMatch`. The open issue #131431 tracks related contiguous-match refinements and could potentially absorb this, but the exclusion feature is distinct enough to keep here.
