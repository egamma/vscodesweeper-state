## Summary

Active design discussion between team members about the UX of the `chat.mcp.serverSampling` setting — the confusing key format and separation from the main MCP config. Alternatives proposed in comments and linked issue #250442. Keeping open for resolution.

## Change summary

The `chat.mcp.serverSampling` setting uses an opaque 'prefix: label' key format and lives far from the MCP server config it relates to, making it confusing to users.
