## Summary

A Clock_Watchdog_Timeout BSOD is a Windows kernel-level error indicating a CPU core failed to respond to an interrupt in time — this is a hardware/driver/OS issue that no user-mode application like VS Code can cause. The WinToGo setup on an external drive adds further system-level instability risk.
