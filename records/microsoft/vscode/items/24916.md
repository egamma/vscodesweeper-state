# Issue #24916 — SCM - Support keyboard shortcuts

[https://github.com/microsoft/vscode/issues/24916](https://github.com/microsoft/vscode/issues/24916)

## Summary

Valid, well-supported feature request (73 upvotes) for default keyboard shortcuts to stage/unstage files directly from the SCM view. No default keybinding exists on current main for git.stage in the SCM tree context; only editor-based range staging has keybindings. The gap remains real.

## Change summary

Add default keyboard shortcuts for staging, unstaging, and reverting files when the SCM view tree has focus, enabling a fully keyboard-driven git workflow.

## Best solution

Add default keybindings for `git.stage` and `git.unstage` scoped to `when: focusedView == 'workbench.scm'` (or the appropriate list-focus context), so users can stage/unstage the focused item without reaching for the mouse. The git extension's `package.json` keybindings section and possibly the SCM view pane's inline action handling need updates.
