# Issue #266038 — requesting terminal icon and terminal color control via launch.json

[https://github.com/microsoft/vscode/issues/266038](https://github.com/microsoft/vscode/issues/266038)

## Summary

Valid feature request with 23 upvotes, promoted to Backlog. Tasks already support icon/color in tasks.json but launch.json has no equivalent for debug terminals — the gap is real and unimplemented on current main.

## Change summary

Allow configuring terminal icon and color for debug sessions via launch.json, similar to how tasks.json already supports icon/color.

## Best solution

Extend the debug launch configuration schema (src/vs/workbench/contrib/debug/common/debugSchemas.ts) with optional `icon` and `color` properties, and wire them through to the terminal creation in the debug terminal system, mirroring the existing pattern in tasks (src/vs/workbench/contrib/tasks/browser/terminalTaskSystem.ts lines 1233–1248).
