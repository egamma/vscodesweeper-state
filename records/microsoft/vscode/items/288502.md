# Issue #288502 — Skills should support tools references and enforce them in system prompt

[https://github.com/microsoft/vscode/issues/288502](https://github.com/microsoft/vscode/issues/288502)

## Summary

Feature request for skills to support #tool: references with enforcement in the system prompt. Infrastructure for parsing tool references in skill files already exists on main (promptFileParser.ts, promptsServiceImpl.ts), and a related PR in vscode-copilot-chat was noted by a contributor, but there's no clear evidence the full enforcement loop requested here is complete.

## Change summary

Allow SKILL.md files to reference specific tools via #tool: syntax and have those tools enforced/available in the system prompt when the skill is loaded.

## Best solution

Verify whether the existing tool reference parsing in skill files (promptFileParser.ts line 491, promptsServiceImpl.ts line 1550-1561) and the copilot-chat PR #3014 fully cover the requested behavior. If enforcement on skill load is still missing, wire up the toolReferences from IAgentInstructions when a skill is activated so the referenced tools are added to the tool set. Route to the prompt/skill customization area.
