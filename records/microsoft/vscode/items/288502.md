# Issue #288502 — Skills should support tools references and enforce them in system prompt

[https://github.com/microsoft/vscode/issues/288502](https://github.com/microsoft/vscode/issues/288502)

## Summary

Tool references (`#tool:toolName`) in skill/prompt files were already implemented before this issue was filed (commit 31dca034fe, Oct 2025) and continue to work on current main. The system parses, resolves, and enforces tool references in skill files.

## Change summary

Request for skills to support `#tool:` references (e.g., `#tool:runSubagent`) and enforce them in system prompts.

## Best solution

Keep the shipped behavior — `#tool:` references in skill files already resolve to tool entries that are added to the request context. No further change needed beyond the existing implementation.
