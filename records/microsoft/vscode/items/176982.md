# Issue #176982 — Json sort command does not honor indentation with tabs

[https://github.com/microsoft/vscode/issues/176982](https://github.com/microsoft/vscode/issues/176982)

## Summary

The bug was fixed as a side effect of the refactoring in commit 9c184b60077 (PR #188480, August 2023). The old ternary `insertSpaces ? Boolean(insertSpaces) : true` always yielded `true` when `insertSpaces` was `false`; the new code uses `Boolean(insertSpaces)` which correctly preserves `false`.

## Change summary

The JSON Sort Document command always used spaces for indentation, ignoring the editor's `insertSpaces: false` (tab) setting.

## Best solution

Keep the shipped fix in commit 9c184b60077 (VS Code 1.82); no further change is needed.
