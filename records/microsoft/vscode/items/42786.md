# Issue #42786 — [json] Automatically add required fields to object

[https://github.com/microsoft/vscode/issues/42786](https://github.com/microsoft/vscode/issues/42786)

## Summary

This feature request to auto-populate required JSON properties in completions remains unimplemented. The completion logic lives in the upstream vscode-json-languageservice package; the current completion code does not generate snippets with required fields pre-filled.

## Change summary

When JSON intellisense suggests an object value, it should pre-fill required properties as a snippet instead of offering just `{}`.

## Best solution

Implement in the upstream `vscode-json-languageservice` package (microsoft/vscode-json-languageservice) in `jsonCompletion.ts` — when suggesting an object type, inspect the schema's `required` array and generate a snippet body containing those properties with tabstops. The VS Code JSON extension would pick it up via a dependency bump.
