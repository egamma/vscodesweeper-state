# Issue #292000 — Claude Agent - Auto-expand the plan while in plan mode

[https://github.com/microsoft/vscode/issues/292000](https://github.com/microsoft/vscode/issues/292000)

## Summary

Team-authored feature request, kept open. The plan section in Claude Agent plan mode still collapses by default behind a 'Show more' link. Commit 3b3b067a1d8 raised the height threshold from 45→100px but did not add an auto-expand option. The underlying request — expanding the confirmation message by default when in plan mode — remains unimplemented.

## Change summary

Auto-expand the Claude Agent plan confirmation section so users see the full plan without clicking 'Show more'.

## Best solution

Add an `expandedByDefault` option to `IToolConfirmationMessages` (or a parallel mechanism in `ToolConfirmationSubPart`) so that the `vscode_get_confirmation` tool can request that its message section starts expanded. The Claude Agent's plan-mode confirmation would set this flag. The rendering code in `chatToolConfirmationSubPart.ts` would skip adding the `can-see-more` class when the flag is set.
