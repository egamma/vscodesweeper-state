# Issue #303777 — Support skill-scoped hooks (on-load) in SKILL.md

[https://github.com/microsoft/vscode/issues/303777](https://github.com/microsoft/vscode/issues/303777)

## Summary

Valid feature request. Hooks are currently supported for agents (AGENT.md) and subagent definitions but NOT for skills (SKILL.md). The `skillAttributes` in `promptFileAttributes.ts` has no `hooks` entry, confirming this capability gap.

## Change summary

Request to support lifecycle hooks (e.g., onLoad) in SKILL.md frontmatter, similar to how hooks work in agent files.

## Best solution

Add a `hooks` attribute to `skillAttributes` in `src/vs/workbench/contrib/chat/common/promptSyntax/languageProviders/promptFileAttributes.ts` and wire skill-scoped hooks through the hook execution pipeline. Needs product decision on which hook types apply to skills (onLoad is new — existing hooks are session/tool lifecycle events).
