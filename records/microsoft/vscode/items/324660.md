# Issue #324660 — "New Browser" keybinding conflict

[https://github.com/microsoft/vscode/issues/324660](https://github.com/microsoft/vscode/issues/324660)

## Summary

Team-authored issue identifying a real keybinding conflict: the 'New Browser' tab chord (Cmd+Shift+K, B) clashes with the widely-used 'Delete Line' command (Cmd+Shift+K). Keeping open for team resolution.

## Change summary

The 'New Browser' tab keybinding (Cmd+Shift+K, B chord) conflicts with 'Delete Line' (Cmd+Shift+K), causing a common editor shortcut to stop working immediately.

## Best solution

Change the 'New Browser' tab chord in `src/vs/sessions/contrib/editor/browser/addTabActions.ts` (line 90) to use a non-conflicting chord prefix — for example reusing the existing `Cmd+N` chord prefix already used by sibling actions in that file, or `Ctrl+Alt+/` as kycutler suggested.
