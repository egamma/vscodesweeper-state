## Summary

Confirmed visual bug on Windows with custom titlebar: window control buttons show a duplicate tooltip (one from VS Code/browser, one from the OS). An open PR (#297100) attempts to fix it but has not merged.

## Change summary

Window control buttons (close, minimize, maximize/restore) show two overlapping tooltips — one from the browser layer and one from the OS — on Windows with a custom titlebar.
