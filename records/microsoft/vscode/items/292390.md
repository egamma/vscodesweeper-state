# Issue #292390 — Should we unify the workspace dropdown and add context widget on welcome page

[https://github.com/microsoft/vscode/issues/292390](https://github.com/microsoft/vscode/issues/292390)

## Summary

Team-authored design discussion about consolidating two overlapping UI surfaces on the welcome page (workspace dropdown vs. add-context widget). Keeping open for maintainer judgment.

## Change summary

Consider unifying the workspace dropdown and the add-context widget on the welcome page, since users expect to choose a project target via add-context and the two controls feel redundant.

## Best solution

The welcome-page area owner should evaluate whether the workspace dropdown can be folded into the add-context widget (or vice versa) to reduce redundancy and match user expectations. This is a design/UX decision that needs team discussion and likely ties into the broader welcome-experience rollout tracked internally.
