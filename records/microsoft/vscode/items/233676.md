# Issue #233676 — Highlight all found text in DEBUG console

[https://github.com/microsoft/vscode/issues/233676](https://github.com/microsoft/vscode/issues/233676)

## Summary

Valid feature request with 25 upvotes, already promoted to Backlog. The debug console filters to show matching output elements but does not highlight the matched text within them — the gap is confirmed on current main.

## Change summary

Highlight the matched/found text within debug console output messages when using the find/filter feature.

## Best solution

Propagate the FuzzyScore filter data from the tree node into the ReplOutputElementRenderer and apply highlight decorations to the rendered value text, similar to how ReplEvaluationInputsRenderer already uses createMatches(element.filterData). The main challenge is that output values are rendered via DebugExpressionRenderer.renderValue (which handles ANSI codes), so highlights need to be woven into that rendering path. Likely files: src/vs/workbench/contrib/debug/browser/replViewer.ts and possibly src/vs/workbench/contrib/debug/browser/debugExpressionRenderer.ts.
