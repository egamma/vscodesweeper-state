# Issue #202204 — Debug toolbar can be hidden by window resize

[https://github.com/microsoft/vscode/issues/202204](https://github.com/microsoft/vscode/issues/202204)

## Summary

Fixed on main across multiple commits: the debug toolbar now uses CSS clamping to stay within window bounds on resize, and stores position as an x-percentage so it scales with window width. Both related duplicates (#202516, #205986) were already closed as duplicates of this family of fixes.

## Change summary

Debug toolbar gets clipped or hidden when resizing the window smaller while it is positioned on the right side.

## Best solution

Keep the shipped fix — the toolbar now uses CSS `min`/`max`/`calc(100vw …)` clamping (commit 86bf11fe769, refined in 27c1b449122) to prevent overflow. No further change needed.
