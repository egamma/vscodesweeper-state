# Issue #174044 — Latest update broke automatic whitespace

[https://github.com/microsoft/vscode/issues/174044](https://github.com/microsoft/vscode/issues/174044)

## Summary

Still reproduces on main: Ruby's `increaseIndentPattern` excludes `{` via `[^#\{;]`, so `def method(opts={})` fails to trigger indent increase, breaking all subsequent auto-indentation in that method body.

## Change summary

Ruby auto-indentation breaks when a method signature contains `{}` (e.g., hash default parameter), causing incorrect indentation for lines inside that method.

## Best solution

Fix the Ruby `increaseIndentPattern` in `extensions/ruby/language-configuration.json` to allow balanced `{}` within method signatures — either by adjusting the trailing exclusion pattern to tolerate matched braces, or by leveraging the indentation line processor (d309e11579e) which now strips brackets from string/comment tokens (though this case is in code context, so the regex itself needs updating).
