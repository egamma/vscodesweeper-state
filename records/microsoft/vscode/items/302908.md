# Issue #302908 — GitHub Copilot chat "Copy All" truncates the copied conversation.

[https://github.com/microsoft/vscode/issues/302908](https://github.com/microsoft/vscode/issues/302908)

## Summary

The 'Copy All' code path in VS Code has no truncation logic — it joins all items and passes the full string to the clipboard service. The truncation at ~19,000 lines likely stems from a platform clipboard limit (Electron/Chromium/Windows) or from the conversation model itself not retaining all content. Without knowing the total conversation size and whether the data is truncated before or after the clipboard write, more info is needed to diagnose.

## Change summary
