# Issue #47331 — [html][css] Investigate cross css/html, multi file support

[https://github.com/microsoft/vscode/issues/47331](https://github.com/microsoft/vscode/issues/47331)

## Summary

Team-authored feature request (aeschli is a MEMBER) for cross-file CSS/HTML language intelligence — must remain open for maintainer judgment. The feature is still unimplemented on main and has sustained community interest.

## Change summary

CSS/HTML language servers should provide cross-file IntelliSense: class/id completions between HTML and CSS files, go-to-definition across files, and variable completions from imported stylesheets.

## Best solution

Merge or bridge the HTML and CSS language servers to build a cross-file index of class names, IDs, and CSS variables, enabling completions and references across HTML ↔ CSS/LESS/SCSS boundaries. This lives in `extensions/css-language-features` and `extensions/html-language-features`.
