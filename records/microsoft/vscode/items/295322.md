# Issue #295322 — Add Support for Pasting Images and Referencing Files in GitHub Copilot's Ask Questions Input Box

[https://github.com/microsoft/vscode/issues/295322](https://github.com/microsoft/vscode/issues/295322)

## Summary

Valid feature request to support image pasting and file references in the askQuestions tool's input carousel. The main chat input already supports this, but the specialized askQuestions carousel uses basic InputBox widgets that lack attachment capabilities. Low community traction (1 upvote) — keeping open for the community process.

## Change summary

The askQuestions tool's input box in Copilot Chat does not support pasting images or referencing files, unlike the main chat input.

## Best solution

Enhance ChatQuestionCarouselPart (src/vs/workbench/contrib/chat/browser/widget/chatContentParts/chatQuestionCarouselPart.ts) to support image paste and file attachment in its InputBox, potentially reusing the PasteImageProvider and attachment model already used by the main chat input. Alternatively, this may be out of scope for the askQuestions tool if its inputs are meant to be simple text/selection answers.
