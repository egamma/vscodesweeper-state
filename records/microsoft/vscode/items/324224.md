# Issue #324224 — Copilot Code Review button missing

[https://github.com/microsoft/vscode/issues/324224](https://github.com/microsoft/vscode/issues/324224)

## Summary

This is a support/configuration question, not a VS Code bug. The Code Review button visibility is gated by both a user setting (`github.copilot.chat.reviewAgent.enabled`) and a server-side entitlement flag (`isCopilotCodeReviewEnabled`) on the Copilot authentication token. The user's token likely does not carry the code-review entitlement, or the setting is disabled.

## Change summary

The Copilot Code Review button does not appear in the Source Control panel toolbar despite having a Business plan with the feature enabled.

## Best solution

The user should verify that the setting `github.copilot.chat.reviewAgent.enabled` is `true`, and that their Copilot subscription actually provisions the code-review entitlement for VS Code (it may be enabled for Visual Studio but not yet for VS Code on their plan). This is a GitHub Copilot entitlement/support question best handled through GitHub Support, not a VS Code core issue.
