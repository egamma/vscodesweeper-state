# Issue #257285 — Users need to be guided to make a proper instruction file

[https://github.com/microsoft/vscode/issues/257285](https://github.com/microsoft/vscode/issues/257285)

## Summary

Team-authored feature request for better onboarding/guidance when users create instruction files without the required frontmatter. Keeping open per policy — team members' issues require explicit maintainer judgment.

## Change summary

Users who create `.instructions.md` files aren't guided to add the required `applyTo` frontmatter, so their custom instructions silently fail to activate.

## Best solution

Add discoverability guidance (e.g., via IntelliSense/suggest, a diagnostic, or a CodeAction) when a `.instructions.md` file lacks the `applyTo` frontmatter block, so users understand the file format requirements.
