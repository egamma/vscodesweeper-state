# Issue #249421 — Git - Show a rich hover over the branch picker

[https://github.com/microsoft/vscode/issues/249421](https://github.com/microsoft/vscode/issues/249421)

## Summary

Valid, unimplemented feature request from a core team member (bpasero) for a rich hover on the SCM branch picker status bar item. The current status bar tooltip is a plain string; no rich hover with copy-branch or PR info has been added on main. Keeping open — the request is in Backlog and needs design/implementation work.

## Change summary

Show a rich MarkdownString hover over the branch picker status bar item, with a copy-branch-name action and optionally PR description or last commit info.

## Best solution

Extend the SCM status bar contribution in `src/vs/workbench/contrib/scm/browser/activity.ts` to provide a `MarkdownString` tooltip (similar to the blame hover in `extensions/git/src/hover.ts`) that displays the branch name with a copy command, and optionally the associated PR description or the latest commit on the branch. This likely also touches the Git extension for supplying the enriched data.
