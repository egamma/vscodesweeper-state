# Issue #24869 — [json] option to format code with leading commas

[https://github.com/microsoft/vscode/issues/24869](https://github.com/microsoft/vscode/issues/24869)

## Summary

This is a long-standing feature request for comma-first formatting, not a bug. It has some community interest but is below the 20-upvote Backlog promotion threshold, and a maintainer already clarified that a global formatter option is not realistic because formatting is language-owned.

## Change summary

Add an option, ideally global but at least for JSON, to format comma-separated lists with leading commas instead of trailing commas.

## Best solution

Keep the request scoped to the JSON formatter if the team still wants it; any implementation would belong in the JSON language service formatter rather than a global editor option. The broader cross-language behavior should remain a formatter-by-formatter decision or an extension concern.
