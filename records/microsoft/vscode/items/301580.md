# Issue #301580 — Copilot help menu covers line numbers immediately upon selection

[https://github.com/microsoft/vscode/issues/301580](https://github.com/microsoft/vscode/issues/301580)

## Summary

This is a duplicate of the already-triaged inline chat affordance questions. The popup covering line numbers on selection is the `inlineChat.affordance` feature, which can be turned off via that setting. The sub-issue of it showing even when AI features are disabled was separately tracked and fixed in #300500.

## Change summary

The inline chat affordance popup appears on text selection and covers line numbers, interfering with editing.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/300417 — the answer is `"inlineChat.affordance": "off"` in settings. The separate concern about respecting `chat.disableAIFeatures` was already fixed in https://github.com/microsoft/vscode/issues/300500 (verified, shipped).
