## Change summary

Allow dragging chat editor tabs back into the chat panel to recover chat sessions from the editor area.

## Best solution

Implement `toUntyped` on `ChatEditorInput` to provide drag data identifying the session, then extend `ChatDragAndDrop` with a new `ChatDragAndDropType` enum value, `guessDropType` logic for session identification, and a `drop` handler that opens the session in the panel.
