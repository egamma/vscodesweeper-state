## Summary

This crash was reported on VS Code 1.84 (Electron 25) in Dec 2023 and was traced to third-party software modifying code.exe on disk, triggering Chromium's sandbox image-base check (error_code=65). The investigation stalled without a final root cause, and VS Code has since moved to Electron 42. Requesting confirmation on current builds before further action.
