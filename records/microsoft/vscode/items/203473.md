# Issue #203473 — Focus indicator is cropped on checkboxes under a dropdown of multi select command parameter in command center.

[https://github.com/microsoft/vscode/issues/203473](https://github.com/microsoft/vscode/issues/203473)

## Summary

Team-authored issue; the bug is still present on main — `overflow: hidden` on `.quick-input-list .quick-input-list-entry` (quickInput.css:210) clips the focus outline on checkboxes that sit flush against the container edge.

## Change summary

Focus indicator (outline/border) on checkboxes in the quick pick multi-select list is cropped on the left side due to overflow clipping.

## Best solution

Add left padding or margin to the checkbox (or use `outline-offset: -1px` / inset outline) so the focus ring isn't clipped by the parent's `overflow: hidden`. The fix belongs in `src/vs/platform/quickinput/browser/media/quickInput.css` around the `.quick-input-list .quick-input-list-entry` or `.monaco-checkbox` rules.
