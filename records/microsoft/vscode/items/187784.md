# Issue #187784 — No way to copy full value from debugger console (repl)

[https://github.com/microsoft/vscode/issues/187784](https://github.com/microsoft/vscode/issues/187784)

## Summary

Partially addressed by a merged PR but explicitly left incomplete — the code still has a TODO citing this exact issue. Copying full values from the debug console works for user-evaluated expressions but not for console.log output, which requires a DAP protocol extension.

## Change summary

Allow copying full (non-truncated) values of objects/arrays from the debug console REPL, including console.log output.

## Best solution

Extend the Debug Adapter Protocol with a method to request clipboard representations of arbitrary variables/output items (not just re-evaluable expressions). The TODO at `src/vs/workbench/contrib/debug/browser/repl.ts:1205` marks the exact gap. This is an upstream DAP coordination item as well as a core debug feature.
