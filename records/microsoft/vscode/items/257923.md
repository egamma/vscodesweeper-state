# Issue #257923 — Managed hovers leak `hoverwidget` disposables

[https://github.com/microsoft/vscode/issues/257923](https://github.com/microsoft/vscode/issues/257923)

## Summary

Team-authored debt issue identifying a real disposable leak in the managed hover system. The team is aware and plans to address it either via a fix in the new hover API or by removing the old code entirely.

## Change summary

Managed hover widgets are cleared but never disposed when the mouse leaves the originating element, leaking disposables.

## Best solution

Add proper disposal of the hover widget in the new hover API (which is replacing the deprecated managed hover path), or ensure the old code disposes correctly until it is fully removed. Connor4312 indicated intent to add this for an upcoming feature.
