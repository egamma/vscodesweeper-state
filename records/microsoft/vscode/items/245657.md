# Issue #245657 — Hover from terminal warning icon clips `Show Environment Contribution` statusbar option

[https://github.com/microsoft/vscode/issues/245657](https://github.com/microsoft/vscode/issues/245657)

## Summary

Still present on main: the hover widget's `overflow: hidden` combined with `adjustHoverMaxHeight` can clip the action buttons row when the terminal status icon hover renders near the bottom edge of the viewport. No fix commit found.

## Change summary

The hover tooltip from the terminal warning icon clips the 'Show Environment Contributions' action button at the bottom of the hover widget.

## Best solution

The hover widget's max-height calculation in `hoverWidget.ts` (`adjustHoverMaxHeight`) should account for the status-bar action row height when constraining the hover; alternatively, the status-bar row could be excluded from the scrollable content area so it's never clipped by the `overflow: hidden` on the container.
