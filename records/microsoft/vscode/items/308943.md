# Issue #308943 — Exclude .jj (Jujutsu VCS) directory by default

[https://github.com/microsoft/vscode/issues/308943](https://github.com/microsoft/vscode/issues/308943)

## Summary

Valid feature request with 36 upvotes, promoted to Backlog. The `.jj` directory (Jujutsu VCS) is not yet excluded by default in current main — the default `files.exclude` in `files.contribution.ts` lists `.git`, `.svn`, `.hg`, `.DS_Store`, and `Thumbs.db` but not `.jj`. The request is straightforward and analogous to existing VCS directory exclusions.

## Change summary

Add `.jj` (Jujutsu VCS directory) to the default `files.exclude` and `files.watcherExclude` settings, alongside `.git`, `.svn`, and `.hg`.

## Best solution

Add `'**/.jj': true` to the default `files.exclude` object and `'.jj/store/**'` patterns to `files.watcherExclude` in `src/vs/workbench/contrib/files/browser/files.contribution.ts`, mirroring the existing `.hg` entries.
