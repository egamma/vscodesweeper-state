# Issue #315511 — TypeScript IntelliSense popup not showing while typing

[https://github.com/microsoft/vscode/issues/315511](https://github.com/microsoft/vscode/issues/315511)

## Summary

Duplicate of the closed and verified #315373. The suggest widget was suppressed by the `offWhenInlineCompletions` default change (initially rolled out as an A/B experiment, then made permanent in PR #316900). The canonical issue tracked the fix, which shipped in 1.122.0.

## Change summary

IntelliSense suggestions stopped appearing while typing in TypeScript (and other languages) when Copilot inline completions were active.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/315373, which tracked and verified the fix for the suggest widget not working with Copilot inline completions enabled. The workaround (`editor.quickSuggestions.other: "on"`) is documented for users who want the old behavior; the new default (`offWhenInlineCompletions`) is intentional but the suggest widget now works correctly when manually triggered.
