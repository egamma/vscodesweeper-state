# Issue #318872 — Copilot Chat debug file logging omits per-turn events for empty-window chat sessions

[https://github.com/microsoft/vscode/issues/318872](https://github.com/microsoft/vscode/issues/318872)

## Summary

Already fixed and shipped. Commit c0cc2ba1dcd added a globalStorageUri fallback so debug file logging works in empty-window sessions; shipped in 1.125.0.

## Change summary

Copilot Chat debug file logging did not write per-turn events when Chat ran in an empty-window session because the logger only used workspace-scoped storageUri, which is undefined without a workspace.

## Best solution

Keep the shipped fix (globalStorageUri fallback in chatDebugFileLoggerService.ts). No further change needed — update to VS Code 1.125.0+ to get it.
