# Issue #111133 — [css] Support for CSS3 speech properties voice-family, voice-pitch, voice-range and voice-stress

[https://github.com/microsoft/vscode/issues/111133](https://github.com/microsoft/vscode/issues/111133)

## Summary

Valid feature request for CSS Speech Module properties in the CSS language service. The properties are not in the upstream vscode-css-languageservice data; low community traction (1 upvote in 4+ years). Keeping open as a backlog feature request.

## Change summary

CSS3 speech properties (voice-family, voice-pitch, voice-range, voice-stress) trigger 'Unknown property' warnings despite being valid W3C CSS.

## Best solution

Add CSS Speech Module properties to the property data in https://github.com/microsoft/vscode-css-languageservice, then bump the dependency in VS Code. Alternatively, users can use the `css.customData` setting to register these properties today.
