# Issue #111133 — [css] Support for CSS3 speech properties voice-family, voice-pitch, voice-range and voice-stress

[https://github.com/microsoft/vscode/issues/111133](https://github.com/microsoft/vscode/issues/111133)

## Summary

Valid feature request — CSS Speech Module properties are still absent from the CSS language service data on current main. The fix belongs upstream in microsoft/vscode-css-languageservice's web custom data. Keeping open as a tracked request.

## Change summary

CSS3 speech properties (voice-family, voice-pitch, voice-range, voice-stress) produce 'Unknown property' warnings because they are not in the CSS language service property database.

## Best solution

Add the CSS Speech Module properties to the web custom data in microsoft/vscode-css-languageservice (the data source consumed by the built-in CSS extension). Alternatively, the user can work around it today via a custom data JSON file contributed through the `css.customData` setting.
