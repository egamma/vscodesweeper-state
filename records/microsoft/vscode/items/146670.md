# Issue #146670 — Inline suggestions are not displayed as ghosted text with HC themes

[https://github.com/microsoft/vscode/issues/146670](https://github.com/microsoft/vscode/issues/146670)

## Summary

Ghost text in HC themes still uses null foreground (relying on a border instead), so the core request — a visible ghosted foreground for HC — remains unresolved. The discussion ended with a maintainer noting contrast-ratio concerns; keeping open as under-discussion is appropriate.

## Change summary

Inline suggestions (ghost text) lack a visible foreground color in high-contrast themes, appearing as plain white/black text instead of grayed out.

## Best solution

Define `hcDark`/`hcLight` defaults for `editorGhostText.foreground` in `src/vs/editor/common/core/editorColorRegistry.ts` that meet the 4.5:1 contrast ratio requirement for HC themes. The existing border workaround (0.8 opacity white/dark) partially helps but a proper foreground default would be the complete fix. A PR with contrast-compliant values and before/after screenshots is needed.
