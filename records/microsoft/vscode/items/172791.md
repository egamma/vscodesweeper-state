# Issue #172791 — Uncaught Error: Illegal value for lineNumber

[https://github.com/microsoft/vscode/issues/172791](https://github.com/microsoft/vscode/issues/172791)

## Summary

Fixed on main by commit cc1f5b21370 which guards lineNumber bounds in the hover path. Keeping open because this is a team-authored error-telemetry issue that needs explicit maintainer closure.

## Change summary

Hovering after lines are deleted can throw 'Illegal value for lineNumber' because the debounced anchor references a stale line number that no longer exists in the model.

## Best solution

Already fixed: commit cc1f5b2137057d7c1c628628ac5f0553a7335723 adds a bounds check in both contentHoverComputer.computeSync and markdownHoverParticipant.computeSync. A maintainer can close this as completed once the fix ships.
