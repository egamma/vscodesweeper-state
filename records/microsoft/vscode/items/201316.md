# Issue #201316 — Sort Lines Ascending command messes up trailing commas

[https://github.com/microsoft/vscode/issues/201316](https://github.com/microsoft/vscode/issues/201316)

## Summary

The Sort Lines command is intentionally a plain text-line sort with no language awareness — it doesn't parse syntax or adjust trailing commas. The request is really a feature request for language-aware sorting, which belongs in an extension.

## Change summary

Sort Lines Ascending command doesn't fix trailing commas after reordering lines in languages like TypeScript.

## Best solution

Close as designed. The built-in sort is a generic text operation; language-aware list sorting (understanding commas, semicolons, etc.) is an extension-candidate capability. Extensions or Copilot inline edits can handle this.
