## Summary

Valid feature request with a real version milestone (1.130.0) and a team member's explicit commitment to add the option under `thinkingStyle` settings. The existing `chat.agent.thinkingStyle` setting offers `collapsed`, `collapsedPreview`, and `fixedScrolling` but no 'always expanded' mode — the requested gap is real and acknowledged.

## Change summary

Add a thinking-style option that prevents automatic collapsing of thinking sections in Copilot Chat, keeping them expanded after the response completes.

## Best solution
