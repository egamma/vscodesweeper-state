# Issue #305798 — Sessions: Git branch not found despite existing

[https://github.com/microsoft/vscode/issues/305798](https://github.com/microsoft/vscode/issues/305798)

## Summary

Team-authored issue; the bug is confirmed in source — `_loadBranches` only queries `refs/heads` (local branches) and never fetches or shows remote branches, so any branch not checked out locally is invisible in the picker.

## Change summary

The Sessions branch picker does not show remote Git branches that are not checked out locally, and does not fetch from the remote.

## Best solution

Extend `_loadBranches` in `copilotChatSessionsProvider.ts` to also query `refs/remotes` (or trigger a fetch first), deduplicate against local branches, and surface them in the picker so users can target any branch that exists on the remote.
