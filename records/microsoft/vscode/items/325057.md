# Issue #325057 — The btn 'Generate Commit Message' is arbitrary not working after updating to 1.128.0

[https://github.com/microsoft/vscode/issues/325057](https://github.com/microsoft/vscode/issues/325057)

## Summary

The 'Generate Commit Message' button in VS Code's SCM view is a thin shell that delegates entirely to the Copilot Chat extension via `product.defaultChatAgent.generateCommitMessageCommand`. The intermittent failure (works a few times then stops) is a Copilot extension issue — VS Code core only invokes the command; the generation logic, rate limiting, and session management live in the extension.

## Change summary
