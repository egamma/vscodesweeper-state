# Issue #308609 — chat interface now giving problems with any form of attachments 

[https://github.com/microsoft/vscode/issues/308609](https://github.com/microsoft/vscode/issues/308609)

## Summary

The error originates from Claude's API rejecting an image exceeding its 8000px dimension limit. The attachment-to-image conversion and API request formatting is handled by the Copilot extension/backend service, not VS Code core — this is a Copilot service-side issue where a CSV attachment is being rendered as an oversized image before submission to Claude.

## Change summary

Attaching a CSV file to Copilot Chat fails with Claude models because the file is converted to an image exceeding Claude's 8000px dimension limit.

## Best solution

The Copilot extension or backend service should resize/downsample large image representations of non-image attachments before sending them to Claude, or send CSV content as text rather than an image. This should be reported to the Copilot team (GitHub Copilot extension) rather than VS Code core.
