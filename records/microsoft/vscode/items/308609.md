## Best solution

Investigate whether the Copilot backend (or client) is incorrectly routing non-image file attachments as image content blocks to Claude's Messages API. The client-side `isImage` check in `chatPasteProviders.ts` uses magic-byte detection and would not misclassify a CSV, so the bug likely lives in the service layer's file-to-message conversion for Anthropic models.
