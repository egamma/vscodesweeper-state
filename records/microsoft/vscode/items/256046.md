# Issue #256046 — Adopting variable font sizes in `createDecorationsCollection` code

[https://github.com/microsoft/vscode/issues/256046](https://github.com/microsoft/vscode/issues/256046)

## Summary

Team-authored issue (MEMBER) tracking a design discussion about whether `createDecorationsCollection` should support variable font sizes. The author noted they need to discuss with their team whether this is intended behavior. Keeping open per policy.

## Change summary

Font-related decoration properties (fontSize, fontFamily, fontWeight) set via `createDecorationsCollection` or `model.deltaDecorations()` have no visible effect because those APIs don't modify the CSS stylesheet.

## Best solution

Team should decide whether to extend `createDecorationsCollection` to support font properties (by generating CSS rules similar to `createTextEditorDecorationType`), or to explicitly document/type-narrow that font properties are only supported via the decoration type API. PR https://github.com/microsoft/vscode/pull/256048 contains investigation.
