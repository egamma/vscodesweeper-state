# Issue #308464 — Copilot extension breaks auto complete

[https://github.com/microsoft/vscode/issues/308464](https://github.com/microsoft/vscode/issues/308464)

## Summary

The reported behavior is the result of an intentional default change: `editor.quickSuggestions.other` was changed from `'on'` to `'offWhenInlineCompletions'` (initially via A/B test, then permanently in commit ef680109cc3). The suggest widget now suppresses itself when an inline completion (Copilot ghost text) is active. Users can restore the old behavior via settings.

## Change summary

Auto-complete suggestions stop appearing when the Copilot extension is enabled, because inline completions suppress the suggest widget by default.

## Best solution

Close as designed — the default was deliberately changed after A/B testing (tracked in https://github.com/microsoft/vscode/issues/316899, now closed and verified). Users who prefer the old behavior should set `"editor.quickSuggestions": "on"` in their settings. The OP's claim in comment 14 that even this workaround fails is unconfirmed by other reporters and may warrant a separate, more detailed bug report if reproducible.
