## Summary

Still valid on main: the `ToggleBetweenInputs` command only cycles between input 1 and input 2; there is no command or keybinding to focus the result editor pane within the merge editor. Team-authored issue — keeping open.

## Change summary

Add a keyboard command to focus/navigate to the result editor pane inside the merge editor.
