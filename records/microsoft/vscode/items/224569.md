# Issue #224569 — Auto indent when move line up or down

[https://github.com/microsoft/vscode/issues/224569](https://github.com/microsoft/vscode/issues/224569)

## Summary

Still reproduces on main: the built-in Python extension has no indentationRules in its language-configuration.json, so move-line auto-indent cannot function for Python. The `tree-sitter-autoindent-candidate` label already tracks this for a future tree-sitter-based solution.

## Change summary

Moving lines up/down with Alt+Arrow does not auto-adjust indentation in Python files because the built-in Python extension lacks indentation rules.

## Best solution

Add `indentationRules` (increaseIndentPattern / decreaseIndentPattern) to `extensions/python/language-configuration.json`, or resolve this via the planned tree-sitter auto-indent work that the `tree-sitter-autoindent-candidate` label tracks.
