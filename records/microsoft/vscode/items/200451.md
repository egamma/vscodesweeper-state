# Issue #200451 — [themes] globally change saturation and contrast

[https://github.com/microsoft/vscode/issues/200451](https://github.com/microsoft/vscode/issues/200451)

## Summary

Valid feature request with 28 upvotes, properly promoted to Backlog. No implementation exists on current main — the theming system has no global saturation/contrast filter mechanism.

## Change summary

Request for global saturation, brightness, and contrast adjustment controls for color themes, avoiding per-color manual tuning.

## Best solution

Introduce a set of theme modifier settings (e.g. `workbench.colorTheme.saturation`, `workbench.colorTheme.contrast`) that apply HSL/filter transforms to all resolved theme colors before rendering. Implementation would live in `src/vs/workbench/services/themes/` and the color registry resolution path.
