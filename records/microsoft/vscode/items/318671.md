# Issue #318671 — Show active model immediately after submitting a prompt

[https://github.com/microsoft/vscode/issues/318671](https://github.com/microsoft/vscode/issues/318671)

## Summary

Valid feature request to surface the active model name at the start of a response rather than only after generation completes. Currently in the Backlog Candidates community upvote window (10 days remaining, 2 of 20 upvotes). No duplicate found; keeping open for the community process.

## Change summary

Display the active model name immediately when a chat prompt is submitted, rather than waiting until the response finishes generating.

## Best solution

Show the selected model identifier in the response header or as an early turn pill as soon as streaming begins, so users can verify the model before waiting for a full response. The change would likely touch the chat response rendering pipeline (chatListRenderer.ts, chatContentParts area) and the response model (chatModel.ts where selectedModel is tracked). Alternatively, the existing model picker in the input area could provide a stronger visual indicator of the model in use during streaming.
