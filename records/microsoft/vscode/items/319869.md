# Issue #319869 — Add a configurable delay for inline suggestions

[https://github.com/microsoft/vscode/issues/319869](https://github.com/microsoft/vscode/issues/319869)

## Summary

The requested feature already exists: `editor.inlineSuggest.minShowDelay` is a configurable delay (in ms) before inline suggestions appear after typing. It shipped in VS Code 1.104.

## Change summary

Add a configurable delay before Copilot inline suggestions appear after the user stops typing.

## Best solution

The setting `editor.inlineSuggest.minShowDelay` (0–10000 ms, default 0) already delivers exactly this. The user can set it to e.g. 2000 for a 2-second pause. No further change needed.
