Team-authored bug reporting that the large-file confirmation dialog triggers inappropriately when an agent applies a moderately sized diff to a large custom agent file. Keeping open for maintainer judgment.

## Change summary

The 'very large file' confirmation dialog appears when an agent edits a large custom agent file, even though the diff itself is moderate in size.
