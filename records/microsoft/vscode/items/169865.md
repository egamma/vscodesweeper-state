# Issue #169865 — When zoomed out in PWA mode, the title bar gets smaller than WCO

[https://github.com/microsoft/vscode/issues/169865](https://github.com/microsoft/vscode/issues/169865)

## Summary

This appears fixed on main by commit 5ed967a8e2a (ensuring titlebar is at least as tall as the WCO bounding rect), shipped in 1.100.0. However, the author is a team member (MEMBER), so keeping open for explicit maintainer judgment.

## Change summary

When zoomed out in PWA mode, the title bar becomes shorter than the fixed-height Window Controls Overlay because counter-zoom doesn't work in web.

## Best solution

This was fixed in commit 5ed967a8e2a (shipped 1.100.0) by using `getWCOTitlebarAreaRect()` to ensure `minimumHeight` never drops below the WCO height. The author (sbatten) can close this as completed.
