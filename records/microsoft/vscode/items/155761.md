# Issue #155761 — Upstream find filter and replace improvements into workbench SimpleFindWidget

[https://github.com/microsoft/vscode/issues/155761](https://github.com/microsoft/vscode/issues/155761)

## Summary

Still valid: the notebook find widget uses its own `SimpleFindReplaceWidget` base class (in `notebookFindReplaceWidget.ts`) rather than the workbench `SimpleFindWidget`, so the consolidation work has not been done. The base `SimpleFindWidget` still lacks replace support and the notebook-specific filter features.

## Change summary

Consolidate notebook find/replace and filter features into the shared SimpleFindWidget so terminal, webview, and other consumers can reuse them.

## Best solution

Upstream replace support and find filters from the notebook's `SimpleFindReplaceWidget` into the shared `SimpleFindWidget` in `src/vs/workbench/contrib/codeEditor/browser/find/simpleFindWidget.ts`, then migrate the notebook find widget to extend it — related to https://github.com/microsoft/vscode/issues/155762 (move SimpleFindWidget into base).
