# Issue #149282 — indent guides highlight in HTML interrupted in script/style sections when "editor.guides.bracketPairs": true

[https://github.com/microsoft/vscode/issues/149282](https://github.com/microsoft/vscode/issues/149282)

## Summary

Still reproduces on main: bracket pair guides replace indent guides at the same column, causing visual interruption in HTML files where bracket pairs in embedded script/style sections only span part of the indentation scope.

## Change summary

Indent guide highlighting is interrupted in HTML script/style sections when bracket pair guides are enabled, because bracket guides replace indent guides at the same column but only cover the bracket pair range.

## Best solution

The guide merging logic in `indentGuides.ts` (line 222) suppresses the indent guide when a bracket guide occupies the same column. A fix would either render both guides overlapping, or extend the bracket guide to cover the full indentation scope, or allow the indent guide to show through where no bracket guide is active. This is a design question about how the two guide systems should compose.
