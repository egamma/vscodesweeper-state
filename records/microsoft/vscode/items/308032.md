Team-authored UX improvement request for the Agents window customizations view — the target selector should default to the user's current session context rather than requiring manual discovery. Keeping open per team-author policy.

## Change summary

The customizations view in the sessions app should default its target to the current agent host session instead of showing an unrelated context (e.g. Copilot CLI).
