Already fixed and shipped in 1.86.0. The commit explicitly addresses rename in the multi-diff editor by considering the overflowWidgetsDomNode for focus/blur events.

F2 rename in the multi-diff editor silently did nothing because the editor lost focus when the rename widget appeared in the overflow container.
