# Issue #306080 — Feature Request: "Send to AI Chat" from Markdown/Text Editor Use Case

[https://github.com/microsoft/vscode/issues/306080](https://github.com/microsoft/vscode/issues/306080)

## Summary

The requested functionality already exists in VS Code: 'Add File to Chat' (right-click editor tab or editor context menu, or Command Palette) and 'Add Selection to Chat' (right-click selected text) attach file/selection content as context to the Chat view. These actions are registered in `chatContextActions.ts` with menus on `EditorContext`, `EditorTitleContext`, `ExplorerContext`, and more. This is effectively as-designed — the feature ships today.

## Change summary

Add a 'Send to AI Chat' action to attach file or selected text from the editor as context to the Chat view.

## Best solution

No change needed — VS Code already ships 'Add File to Chat' (`workbench.action.chat.attachFile`) and 'Add Selection to Chat' (`workbench.action.chat.attachSelection`) in editor context menus, editor title tab context menus, explorer context menus, and the Command Palette. The reporter's workflow is fully supported today.

  > Thanks for the feature request! This capability actually already exists in VS Code today:
  > 
  > - **Add File to Chat** — right-click an editor tab or use the editor context menu (or run "Add File to Chat" from the Command Palette). This attaches the entire file as context to the Chat view.
  > - **Add Selection to Chat** — select text in the editor, right-click, and choose "Add Selection to Chat". This attaches the selected text as context.
  > - **Add Context…** — in the chat input, use the attach button or run "Add Context…" from the Command Palette to pick files, symbols, etc.
  > 
  > You can also drag and drop files into the chat input area.
  >
