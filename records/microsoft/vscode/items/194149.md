## Summary

Still unimplemented on main: the terminal tabs list toolbar still uses a raw ActionBar rather than WorkbenchToolBar/MenuWorkbenchToolBar, so the right-click 'hide item' configurability hasn't been added. The issue is valid, labeled help wanted, and awaiting a contributor.

## Change summary

Allow users to hide individual toolbar items (Split, Kill) in the terminal tabs list via right-click context menu.
