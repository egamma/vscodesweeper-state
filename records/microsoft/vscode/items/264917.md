# Issue #264917 — Chat - different modes use different working rendering

[https://github.com/microsoft/vscode/issues/264917](https://github.com/microsoft/vscode/issues/264917)

## Summary

Team-authored polish issue about visual inconsistency between Ask mode (animated '...') and Agent mode (spinner + static 'Working...') progress indicators. Author clarified it's about the look, not position. The related PR #264924 fixed Ask mode styling but didn't unify the two modes. Keeping open per team-authored rule.

## Change summary

Ask mode and Agent mode use visually different progress/working indicators — animated ellipsis vs. spinner with static text.

## Best solution

Unify the working indicator rendering across chat modes in the chat list renderer (`chatListRenderer.ts`) and progress content part (`chatProgressContentPart.ts`) so both modes use a consistent visual style. A design decision is needed on which style to adopt uniformly.
