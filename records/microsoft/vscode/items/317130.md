# Issue #317130 — Can't disable commit message suggestions

[https://github.com/microsoft/vscode/issues/317130](https://github.com/microsoft/vscode/issues/317130)

## Summary

The behavior is working as designed — inline completions in the SCM input are provided by the Copilot extension and can be disabled via `"github.copilot.enable": {"scminput": false}`. The request for a more discoverable GUI toggle belongs with the Copilot extension, not VS Code core.

## Change summary

User wants an easier way to disable Copilot inline suggestions specifically in the commit message input box.

## Best solution

This is a Copilot extension UX concern — the per-language enable/disable setting exists but the reporter (and likely others) find it undiscoverable. A toggle or context menu in the SCM input widget would be the extension's responsibility. File as a feature request on the Copilot extension (or github/copilot-docs).
