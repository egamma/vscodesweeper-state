# Issue #304554 — Put "GitHub Copilot" somewhere

[https://github.com/microsoft/vscode/issues/304554](https://github.com/microsoft/vscode/issues/304554)

## Summary

The Chat view is intentionally titled "Chat" because it is a generic chat surface in VS Code that can host multiple providers — not just GitHub Copilot. The current model/agent name is visible in the chat input area's model picker. This is a deliberate design choice, so proposing close as as-designed.

## Change summary

User requests the Chat tab or view be renamed or branded to indicate it comes from GitHub Copilot.

## Best solution

Keep the current "Chat" title. The view is a generic VS Code chat surface, not exclusively a GitHub Copilot surface. The active model/agent identity is already visible in the chat input's model picker dropdown. Renaming the container to a specific provider would be incorrect when multiple chat providers can contribute.

## Vision fit

- **Reason:** The Chat view is a platform surface designed to host any chat provider via extensions; branding it to a single provider conflicts with the extension-ecosystem value of keeping core surfaces generic.
