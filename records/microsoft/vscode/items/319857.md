# Issue #319857 — Add “Open Repository…” action to Source Control Repositories view title

[https://github.com/microsoft/vscode/issues/319857](https://github.com/microsoft/vscode/issues/319857)

## Summary

Valid feature request: the `git.openRepository` command exists but is not surfaced in the `scm/repositories/title` menu. Currently that menu only shows `git.reopenClosedRepositories`. Adding it would be a small package.json change in the git extension.

## Change summary

Add the existing 'Git: Open Repository' command to the Source Control Repositories view title area for discoverability.

## Best solution

Add a menu contribution for `git.openRepository` in the `scm/repositories/title` group in `extensions/git/package.json`, similar to how `git.reopenClosedRepositories` is already placed there. A one-line addition with appropriate `when` clause (`config.git.enabled && !git.missing`).
