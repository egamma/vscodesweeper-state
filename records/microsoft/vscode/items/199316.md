# Issue #199316 — Multi-editor diff: Hard to collapse/expand file

[https://github.com/microsoft/vscode/issues/199316](https://github.com/microsoft/vscode/issues/199316)

## Summary

Team-authored issue; must keep open. Additionally, the fix is only partially landed: header click-to-collapse is implemented but gated behind `IsSessionsWindowContext` (Agents window only) — the main workbench multi-diff editor still does not have this behavior enabled.

## Change summary

Make the entire header row in the multi-diff editor clickable to collapse/expand a file entry, with pointer cursor and hover feedback.

## Best solution

Enable `headerClickToCollapse` unconditionally (or by default) in the workbench `WorkbenchUIElementFactory` at `src/vs/workbench/contrib/multiDiffEditor/browser/multiDiffEditor.ts:233`, and add `cursor: pointer` to the `.header-content` rule in `style.css` so both the interaction and visual affordance work in the standard multi-diff editor.
