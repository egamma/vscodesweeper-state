# Issue #35785 — Input box completion support

[https://github.com/microsoft/vscode/issues/35785](https://github.com/microsoft/vscode/issues/35785)

## Summary

The original request's basic scenario (dynamically updating quick pick items as you type) is now possible via `createQuickPick` + `onDidChangeValue`, but the broader ask — true inline completion/autocomplete within an input box (like IntelliJ's flag completion) — remains unimplemented. With 27 upvotes and Backlog milestone, this stays open as a valid feature request.

## Change summary

Expose an extension API for showing completion suggestions (autocomplete) within an input box or quick pick text field.

## Best solution

The `createQuickPick` API addresses the simpler dynamic-filter use case. The remaining gap is inline completion within input boxes — a richer UI that would let extensions offer word-level suggestions inside the text field. This is a meaningful extension-API enhancement in the `quick-pick` area; a maintainer should decide whether the remaining ask justifies continued tracking or whether `createQuickPick` is sufficient and the issue can be closed as addressed.
