# Issue #323840 — Expose themeVariables of the built-in mermaid-markdown-features extension in user settings

[https://github.com/microsoft/vscode/issues/323840](https://github.com/microsoft/vscode/issues/323840)

## Summary

Valid feature request to expose Mermaid themeVariables as user-configurable settings. The extension currently derives themeVariables from the VS Code color theme but offers no user override.

## Change summary

Allow users to customize Mermaid themeVariables (e.g., primaryColor, lineColor) via VS Code settings in the built-in mermaid-markdown-features extension.

## Best solution

Add a `markdown-mermaid.themeVariables` object setting in `extensions/mermaid-markdown-features/package.json` and merge user-provided overrides into the resolved themeVariables in `preview-src/shared/vsCodeTheme.ts`.
