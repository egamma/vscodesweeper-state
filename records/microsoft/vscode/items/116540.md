# Issue #116540 — [json] support link on $ref for external references

[https://github.com/microsoft/vscode/issues/116540](https://github.com/microsoft/vscode/issues/116540)

## Summary

This feature request is still valid — external $ref links are not supported on current main. The JSON language service only resolves local (#-prefixed) JSON pointer references. The issue is labeled help-wanted and has clear maintainer guidance on how to implement it.

## Change summary

Support clickable document links for external $ref references (e.g. pointing to other files) in JSON schemas.

## Best solution

Implement external $ref link support in vscode-json-languageservice's `findLinks` (jsonLinks.ts), adding a `resolveLink` API that uses the `schemaRequestService` to resolve external targets on click, then wire `connection.onDocumentLinkResolve` in jsonServer.ts. The maintainer guidance in the comments provides a clear implementation path.
