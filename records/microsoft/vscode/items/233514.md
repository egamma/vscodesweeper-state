# Issue #233514 — Line digit highlighting has no effect if the line has been moved

[https://github.com/microsoft/vscode/issues/233514](https://github.com/microsoft/vscode/issues/233514)

## Summary

This appears fixed on current main by a targeted editor line-number rendering change, so it can be closed as completed rather than with a triage `*` label.

## Change summary

Active line-number highlighting was lost when the cursor was on a wrapped continuation of the same model line.

## Best solution

Treat this as fixed by commit 503bace89cb48c7ef8cbee8109e262889bf99907, which shipped in 1.101.0; no further editor-rendering change is needed unless the bug still reproduces on a current build.
