# Issue #324963 — Copilot Chat side-panel input font is too small and cannot be independently resized

[https://github.com/microsoft/vscode/issues/324963](https://github.com/microsoft/vscode/issues/324963)

## Summary

Verified duplicate of #298292, which is the canonical request for an independent chat input font size setting. The reporter describes the same gap — the chat input editor hardcodes fontSize to 13 and ignores chat.fontSize — and #298292 already has community discussion and a volunteer implementer.

## Change summary

The chat side-panel input editor does not respect chat.fontSize and has no independent font size setting.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/298292, which tracks adding a chat.inputEditor.fontSize setting (and possibly font family/weight). The fix would wire `chat.fontSize` or a new dedicated setting into the input editor construction at src/vs/workbench/contrib/chat/browser/widget/input/chatInputPart.ts line 3243, where fontSize is currently hardcoded to 13.
