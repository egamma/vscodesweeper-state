# Issue #308446 — Missing RTL (Right-to-Left) text direction support for Farsi/Arabic in Chat panel

[https://github.com/microsoft/vscode/issues/308446](https://github.com/microsoft/vscode/issues/308446)

## Summary

Valid request for RTL text direction support in the Chat panel. An open community PR (https://github.com/microsoft/vscode/pull/314365) explicitly lists this issue in its fixes. No RTL/bidi support has been merged into the chat CSS on current main — the gap remains real.

## Change summary

Chat panel forces LTR alignment for all text, breaking readability for RTL languages like Farsi, Arabic, and Hebrew.

## Best solution

The open PR https://github.com/microsoft/vscode/pull/314365 introduces a configurable `editor.textDirection` setting and propagates it to chat surfaces — reviewing and landing that PR would resolve this issue along with several others.
