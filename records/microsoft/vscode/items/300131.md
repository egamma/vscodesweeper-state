# Issue #300131 — Terminal - Cannot input after git asking for ssh passphrase

[https://github.com/microsoft/vscode/issues/300131](https://github.com/microsoft/vscode/issues/300131)

## Summary

This is a duplicate of the kitty keyboard protocol regression in fish shell that was fixed by the xterm.js bump in PR #299833 (resolving #295372). The commenter's workaround confirms the same root cause. The fix shipped in 1.112.0.

## Change summary

Terminal input is blocked when programs disable echo (SSH passphrase, sudo, GPG) while using the fish shell with the kitty keyboard protocol enabled.
