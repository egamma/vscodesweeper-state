# Issue #92736 — [themes] Allow gradients along with colors

[https://github.com/microsoft/vscode/issues/92736](https://github.com/microsoft/vscode/issues/92736)

## Summary

Still a valid, unimplemented feature request. The theming API only supports flat color tokens; no gradient support exists for user-customizable theme colors. Has community interest and is already in the Backlog milestone.

## Change summary

Allow CSS gradients (not just flat colors) in theme-able backgrounds and borders.

## Best solution

Extend the color token system in `src/vs/platform/theme/common/colorRegistry.ts` to optionally accept gradient values for background tokens, or provide a separate gradient registry. This is a significant design decision affecting the entire theming surface and extension API, so it warrants explicit maintainer discussion before implementation.
