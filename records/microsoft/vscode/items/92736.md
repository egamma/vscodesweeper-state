# Issue #92736 — [themes] Allow gradients along with colors

[https://github.com/microsoft/vscode/issues/92736](https://github.com/microsoft/vscode/issues/92736)

## Summary

Valid, well-upvoted feature request (45 👍) with no implementation on main. The theming system still only supports flat colors — no gradient infrastructure exists in the color registry or theme platform code.

## Change summary

Allow CSS gradients in theme color definitions for backgrounds and borders.

## Best solution

Extend the color/theme registry (`src/vs/platform/theme/common/colorRegistry.ts` and related rendering) to accept gradient values alongside flat colors, giving theme authors a first-class way to specify gradients without resorting to unsafe CSS hacks.
