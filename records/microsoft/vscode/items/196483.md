# Issue #196483 — Relaunch terminal hides terminal after reloading window

[https://github.com/microsoft/vscode/issues/196483](https://github.com/microsoft/vscode/issues/196483)

## Summary

This is a duplicate of #228359, which was fixed by commit b625e085b74 (shipped in 1.107.0). The root cause — `attachPersistentProcess` being passed during relaunch after window reload — is now cleared before relaunching.

## Change summary

Clicking 'Relaunch Terminal' after a window reload hides the terminal panel instead of relaunching it.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/228359, which is already closed-fixed. The fix in commit b625e085b74 clears `attachPersistentProcess` before relaunch, preventing the terminal from trying to reattach to a stale process and disappearing.
