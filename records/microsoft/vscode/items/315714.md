# Issue #315714 — Branch name in title is cutoff instead of `...`

[https://github.com/microsoft/vscode/issues/315714](https://github.com/microsoft/vscode/issues/315714)

## Summary

Team-authored issue; the workspace/branch label in the Agents Window titlebar gets hard-clipped instead of showing an ellipsis when it overflows. The CSS explicitly prevents truncation on that element but the parent `overflow: hidden` causes a raw cutoff.

## Change summary

Branch name in the Agents Window titlebar is cut off abruptly instead of showing `…` when it's too long.

## Best solution

Change the `.agent-sessions-titlebar-workspace` CSS rule to allow graceful truncation: switch from `flex: 0 0 auto` to `flex: 0 1 auto` and add `overflow: hidden; text-overflow: ellipsis` so that when both the title and workspace are long, the workspace truncates with an ellipsis rather than being hard-clipped by the parent container.
