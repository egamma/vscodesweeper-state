# Issue #114115 — [html] support imports in embedded JavaScript

[https://github.com/microsoft/vscode/issues/114115](https://github.com/microsoft/vscode/issues/114115)

## Summary

Valid feature request for JS module import IntelliSense in embedded HTML script blocks. Already triaged to Backlog by a team member who confirmed the limitation still exists (Jan 2025). Related to but not fully covered by the broader #47331.

## Change summary

IntelliSense does not resolve imports or provide cross-file type information inside `<script type="module">` blocks in HTML files.

## Best solution

Extend the HTML language service's embedded JavaScript support to forward module resolution to the TypeScript language service, enabling cross-file IntelliSense in `<script type="module">` blocks. This likely lives in `extensions/html-language-features/` and the HTML language server's script region handling. Alternatively, this could be absorbed into the broader cross-file support effort tracked in https://github.com/microsoft/vscode/issues/47331.
