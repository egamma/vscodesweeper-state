# Issue #144532 — JavaScript Debug Console mangles output that includes both CSS and Object specifiers

[https://github.com/microsoft/vscode/issues/144532](https://github.com/microsoft/vscode/issues/144532)

## Summary

The format specifier handling for `%c` + `%o` lives in the js-debug debug adapter, not in VS Code core's replModel (which just displays pre-formatted output). The report is from March 2022 and js-debug has had significant updates since then; it's unclear whether this still reproduces on current builds.

## Change summary

Debug console mangles output when `console.log` combines CSS (`%c`) and object (`%o`) format specifiers, showing raw ANSI escape codes instead of styled text with a properly rendered object.

## Best solution
