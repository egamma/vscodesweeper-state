# Issue #315012 — Feature Request: Manual Refresh Button for Agent Plugin Marketplaces

[https://github.com/microsoft/vscode/issues/315012](https://github.com/microsoft/vscode/issues/315012)

## Summary

Valid feature request for a more discoverable refresh button in the Agent Plugins UI. Commands already exist via the Command Palette (`workbench.agentPlugins.checkForUpdates`, `workbench.agentPlugins.forceUpdate`), but the user wants a visible inline button next to each marketplace for faster iteration during plugin development.

## Change summary

Add a visible Refresh button in the Agent Plugins UI next to each configured marketplace to trigger an immediate update check without using the Command Palette.

## Best solution

Consider surfacing the existing `workbench.agentPlugins.checkForUpdates` command as a toolbar action in the Agent Plugins view header or as an inline button per marketplace source in `src/vs/workbench/contrib/chat/browser/agentPluginsView.ts`. Alternatively, this may already be adequately served by the existing commands — in which case closing as `*as-designed` with a pointer to the Command Palette workflow would be appropriate.
