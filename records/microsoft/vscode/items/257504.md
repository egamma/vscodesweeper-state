# Issue #257504 — NES: Word Wrap not supported

[https://github.com/microsoft/vscode/issues/257504](https://github.com/microsoft/vscode/issues/257504)

## Summary

Still valid: NES inline edit views explicitly disable word wrap (`wordWrap: 'off'`) on current main. With 12 upvotes and confirmed community impact (especially for long-line files like .md/.tex), this is a legitimate feature gap that should remain open.

## Change summary

NES inline edit suggestions do not respect the editor's word wrap setting, making suggestions invisible when they extend beyond the viewport on wrapped lines.

## Best solution

The NES side-by-side and long-distance preview editors should inherit or respect the parent editor's word wrap setting rather than hardcoding `wordWrap: 'off'`. The fix would touch `src/vs/editor/contrib/inlineCompletions/browser/view/inlineEdits/inlineEditsViews/inlineEditsSideBySideView.ts` and the long-distance preview editor.
