# Issue #289903 — Error: Missing color: inlineEdit.gutterIndicator.secondaryBackground

[https://github.com/microsoft/vscode/issues/289903](https://github.com/microsoft/vscode/issues/289903)

## Summary

The missing color was fixed 6 days after filing by rebasing `inlineEdit.gutterIndicator.secondaryBackground` onto `editorHoverBackground` (which always resolves); shipped in 1.109.0.

## Change summary

Toggling themes with the inline gutter indicator visible throws 'Missing color: inlineEdit.gutterIndicator.secondaryBackground' because the color's base (`buttonSecondaryBackground`) didn't resolve in all themes.

## Best solution

Keep the shipped fix (commit 6924e8019ad); the color now chains to `editorHoverBackground` which always resolves. The broader `observeColor` throw pattern still exists (see https://github.com/microsoft/vscode/issues/305819 for a sibling color) but that is a separate issue.
