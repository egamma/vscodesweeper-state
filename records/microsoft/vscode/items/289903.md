[https://github.com/microsoft/vscode/issues/289903](https://github.com/microsoft/vscode/issues/289903)

## Summary

Fixed on main by commit 6924e8019ad (2026-01-29), which changed the secondary gutter indicator color defaults from buttonSecondaryBackground (undefined in some themes) to editorHoverBackground (universally available). Shipped in 1.109.0.

## Change summary

Toggling through themes with the inline gutter indicator visible throws 'Missing color: inlineEdit.gutterIndicator.secondaryBackground' because some themes lack buttonSecondaryBackground in the color resolution chain.

## Best solution

Keep the shipped fix in commit 6924e8019ad (1.109.0); the color defaults now resolve to universally-available editor hover colors, preventing the error across all themes.
