# Issue #71813 — [css] highlighting not working for some transition properties

[https://github.com/microsoft/vscode/issues/71813](https://github.com/microsoft/vscode/issues/71813)

## Summary

Still reproduces on main: the CSS TextMate grammar has no pattern to highlight property names (like `background-color`) when they appear as values inside `transition:`. The related PR #71824 was never merged, and no fix has landed since.

## Change summary

CSS property names used as values in the `transition` shorthand (e.g. `transition: background-color 0.3s`) receive no syntax highlighting.

## Best solution

Add a grammar pattern in `extensions/css/syntaxes/css.tmLanguage.json` under the `property-values` repository (or a new transition-specific context) that matches known animatable property names and scopes them appropriately (e.g. `support.type.property-name.css`) when they appear in a value position for `transition` or `transition-property`.
