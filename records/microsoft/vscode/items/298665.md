# Issue #298665 — Wrong window composition on wayland

[https://github.com/microsoft/vscode/issues/298665](https://github.com/microsoft/vscode/issues/298665)

## Summary

This reports a white rectangle artifact behind the VS Code window on Wayland with native title bar during resize/move — the same class of Electron/Wayland window composition defect actively tracked by the canonical issue #247054, which covers broken window borders and decoration artifacts on Wayland. A commenter there describes this exact native-titlebar resize glitch. The fix path is the Electron upgrade (42+ brings shadows, 43+ brings rounded corners), already landing in Insiders.

## Change summary

An opaque white rectangle appears behind the VS Code window on Wayland when using native title bar style and resizing or moving the window.
