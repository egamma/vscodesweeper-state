# Issue #261493 — Chat in Editor should have same background as chat in panel

[https://github.com/microsoft/vscode/issues/261493](https://github.com/microsoft/vscode/issues/261493)

## Summary

Team-authored issue, keeping open. The root cause — chat editor using `editorBackground` for its list background instead of a panel-like color — persists on main, though partial fixes have landed for specific sub-elements (terminal output, editor overlay widgets).

## Change summary

Chat opened in an editor tab uses the editor background color instead of the panel background, causing color contrast issues and broken styling for chat sub-elements.

## Best solution

Align the chat editor's list and widget background colors with the panel chat's colors, likely in `src/vs/workbench/contrib/chat/browser/widgetHosts/editor/chatEditor.ts` (line ~132 where `listBackground: editorBackground` is set). This would fix the root cause rather than patching individual sub-elements.
