## Summary

This is an acknowledged feature request to support JSON pointer fragments (e.g. `#/$defs/inner_schema`) in schema association URLs. The maintainer confirmed it's unsupported and pointed to the relevant code in vscode-json-languageservice. No work has been done on it since. ## Change summary

Allow `json.schemas` URL associations to include JSON pointer fragments (e.g. `#/$defs/inner_schema`) to reference sub-schemas directly.
