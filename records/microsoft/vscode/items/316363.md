# Issue #316363 — Request: Hindi Language Pack for VS Code

[https://github.com/microsoft/vscode/issues/316363](https://github.com/microsoft/vscode/issues/316363)

## Summary

Language packs are extensions in VS Code's architecture — adding Hindi is a community translation effort that results in a Marketplace extension, not a core code change. This belongs in the extension/localization-community lane.

## Change summary

Add Hindi as a display language option in VS Code via a language pack.

## Best solution

A Hindi language pack should be created as a Marketplace extension through the VS Code Community Localization Program (https://aka.ms/vscodeloc). This is the established path for all language packs — they are extensions, not core changes.
