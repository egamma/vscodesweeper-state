# Issue #84486 — [themes] provide command to change theme

[https://github.com/microsoft/vscode/issues/84486](https://github.com/microsoft/vscode/issues/84486)

## Summary

The user's core need (a keyboard shortcut to switch between light and dark themes) is addressed by the `workbench.action.toggleLightDarkThemes` command, which was implemented for issue #151466 (closed as verified/shipped). That issue explicitly cites this one as related. The toggle command, combined with `workbench.preferredDarkColorTheme` and `workbench.preferredLightColorTheme` settings, lets users configure and instantly switch between their preferred themes via a keybinding.

## Change summary

Request for a command that accepts a theme name as an argument so keybindings can switch themes immediately without using the picker.

## Best solution

Close as duplicate of https://github.com/microsoft/vscode/issues/151466, which shipped the `workbench.action.toggleLightDarkThemes` command. The preferred-theme settings complete the workflow. If the user specifically needs a command accepting an arbitrary theme ID as an argument (beyond toggle), that would be a separate narrower request, but the stated use case is covered.
