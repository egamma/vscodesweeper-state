## Summary

Verified duplicate: this is the same Chromium sandbox failure when running VS Code elevated that is already tracked by #122951, an open upstream-linked issue. A team member already identified the match.

## Change summary

VS Code fails to launch as Administrator with error code 18 (sandbox failure) on Windows Server 2019 in a VM with no real GPU.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/122951 which tracks the upstream Chromium/Electron sandbox limitation when running elevated. The workaround is `--disable-chromium-sandbox`. Resolution depends on the upstream Electron fix (https://github.com/electron/electron/issues/49167).
