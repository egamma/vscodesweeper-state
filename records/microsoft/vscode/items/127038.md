# Issue #127038 — Add "Show all results" button when hitting maxResults

[https://github.com/microsoft/vscode/issues/127038](https://github.com/microsoft/vscode/issues/127038)

## Summary

Team-authored feature request (roblourens is a MEMBER) that is still unimplemented on main — the search view shows only a static warning when maxResults is hit, with no actionable button to rerun without the limit. Keeping open per policy.

## Change summary

Add a 'Show all results' button/link in the search view when the maxResults limit is hit, allowing users to rerun the search without the cap.

## Best solution

When `limitHit` is true, render an actionable link or button alongside the existing warning message in the search view (around `searchView.ts:1897`) that reruns the current query with `maxResults` unset. The perf investigation noted in the issue body and the related https://github.com/microsoft/vscode/issues/127621 should be addressed first or in parallel, since removing the limit on large result sets can cause renderer slowdowns via `setChildren`.
