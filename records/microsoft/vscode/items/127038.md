# Issue #127038 — Add "Show all results" button when hitting maxResults

[https://github.com/microsoft/vscode/issues/127038](https://github.com/microsoft/vscode/issues/127038)

## Summary

Team-authored feature request on the 'On Deck' milestone — hard keep-open. The feature (a 'Show all results' button when hitting maxResults) has not been implemented on current main; the search view still only shows a text warning.

## Change summary

Add a UI button to rerun the search without the maxResults limit when the result set is truncated.

## Best solution

Add a clickable 'Show all results' action in the search results message bar (around `searchView.ts` line 1898 where the `limitHit` warning is shown) that reruns the query with `maxResults` cleared. The perf investigation noted in the issue body and the related https://github.com/microsoft/vscode/issues/127621 should be addressed first or in parallel to avoid UI freezes on large result sets.
