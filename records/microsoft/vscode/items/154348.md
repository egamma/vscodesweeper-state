# Issue #154348 — Unable to use outline with search editor

[https://github.com/microsoft/vscode/issues/154348](https://github.com/microsoft/vscode/issues/154348)

## Summary

Fixed on main: commit 4845ea3bb0c guards outline navigation so that clicking an outline item in a search editor navigates in-place instead of trying to resolve the unresolvable `search-editor:` URI.

## Change summary

Clicking an outline item while a search editor is active opens a broken new editor instead of navigating within the search editor.

## Best solution

Keep the shipped fix in commit 4845ea3bb0c; no further change needed. The fix avoids the `openCodeEditor` path for non-openable URIs by applying the selection directly in the current editor.
