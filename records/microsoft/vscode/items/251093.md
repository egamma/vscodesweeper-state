# Issue #251093 — Keep getting popups for "An update is ready to install"

[https://github.com/microsoft/vscode/issues/251093](https://github.com/microsoft/vscode/issues/251093)

## Summary

This is a duplicate of #276982, which tracks the same macOS privileged-helper popup problem with better framing (give context into what helper is being installed). Both report the repeated 'An update is ready to install / Add Helper' dialog on macOS when the user lacks admin rights.

## Change summary

macOS repeatedly shows the 'An update is ready to install — Code is trying to add a new helper tool' dialog, with no way to suppress it for users without admin rights.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/276982, which tracks the need for better context and UX around the macOS privileged-helper dialog. The underlying behavior comes from Electron's Squirrel updater, so VS Code's lever is either suppressing repeated prompts or improving the dialog's messaging. Users can work around this by setting `update.mode` to `none` or `manual`.
