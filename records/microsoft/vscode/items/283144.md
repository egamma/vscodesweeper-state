# Issue #283144 — Quick question widget cannot be repositioned like other quick pick widgets

[https://github.com/microsoft/vscode/issues/283144](https://github.com/microsoft/vscode/issues/283144)

## Summary

Team-authored feature request to allow the quick question (Copilot quick chat) widget to be repositioned via drag, like other quick pick widgets. Requires an API for widgets to contribute drag areas. Keeping open per team-authored policy.

## Change summary

The quick question widget (Copilot quick chat) cannot be dragged/repositioned like the command palette and other quick pick widgets.

## Best solution

Extend the widget infrastructure so that non-quick-pick widgets (like quick chat) can contribute drag areas, enabling consistent repositioning behavior across all picker-like surfaces. Likely involves changes in `src/vs/workbench/contrib/chat/browser/` (quick chat widget) and the quick-input drag infrastructure.
