# Issue #166615 — SCM - Create an event for when the value of a SourceControlInputBox changes

[https://github.com/microsoft/vscode/issues/166615](https://github.com/microsoft/vscode/issues/166615)

## Summary

The proposed API `scmTextDocument` (adding `document` property to `SourceControlInputBox`) was added in commit 218b8bd8dac but remains proposed — not finalized into the stable API. The issue still tracks valid work: finalizing or evolving this API so extensions can detect SCM input changes without polling.

## Change summary

Add an event or mechanism for extensions to detect when the value of a SourceControlInputBox changes.

## Best solution

Finalize the `scmTextDocument` proposed API (exposing `SourceControlInputBox.document`) into the stable `vscode.d.ts`, which lets extensions use `workspace.onDidChangeTextDocument` to detect input changes. The area owner (scm/api) should decide whether to finalize as-is or add a dedicated `onDidChangeValue` event instead.
