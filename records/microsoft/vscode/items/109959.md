# Issue #109959 — Horizonal Scrolling through pointer on scroll bar

[https://github.com/microsoft/vscode/issues/109959](https://github.com/microsoft/vscode/issues/109959)

## Summary

Valid feature request for scrollbar-hover-aware horizontal scrolling. Not implemented on main — the scrollbar widget has no pointer-over-horizontal-bar detection to redirect wheel events. Low upvote count (2) means it hasn't passed the community threshold, but it's in the Backlog milestone so keeping open as under-discussion.

## Change summary

When the mouse pointer is hovering over the horizontal scrollbar, scroll wheel events should scroll horizontally instead of vertically.

## Best solution

Keep open as a low-traction feature request. The implementation would live in `src/vs/base/browser/ui/scrollbar/scrollableElement.ts` — detect when the pointer is over the horizontal scrollbar element and redirect deltaY to deltaX in `_onMouseWheel`. Shift+scroll already provides a workaround. Given 2 upvotes over 5+ years, this is unlikely to gain enough traction for prioritization.
