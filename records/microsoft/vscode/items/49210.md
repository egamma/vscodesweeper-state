# Issue #49210 — [html-templating] easy way to inherite advanced html features

[https://github.com/microsoft/vscode/issues/49210](https://github.com/microsoft/vscode/issues/49210)

## Summary

The requested capability was implemented via the `htmlLanguageParticipants` contribution point, merged in commit cef02dae8d2 (July 2022). Extensions can now register their language to inherit HTML language features.

## Change summary

Request for an easy way to inherit advanced HTML features (tag highlighting, auto-close) in templating language extensions.

## Best solution

Keep the shipped `htmlLanguageParticipants` contribution point (commit cef02dae8d2). Extensions like Better Jinja can add `"htmlLanguageParticipants": [{"languageId": "jinja-html"}]` to their package.json to inherit HTML language features. No further core change needed.
