Request an over-under / toggle-blink comparison mode for binary image diffs in the Git source control view, beyond the current side-by-side layout.

## Best solution

Extend `BinaryResourceDiffEditor` (or its underlying binary editor infrastructure in `src/vs/workbench/browser/parts/editor/binaryDiffEditor.ts`) to support additional comparison modes — at minimum a toggle/blink mode that swaps between before and after images in the same viewport.
