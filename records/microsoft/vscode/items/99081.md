# Issue #99081 — Settings sync - Cannot dismiss "Configure..." quick pick dialog

[https://github.com/microsoft/vscode/issues/99081](https://github.com/microsoft/vscode/issues/99081)

## Summary

Team-authored issue requesting a close button on non-dismissible (ignoreFocusOut) quick picks. Not implemented on main; the discussion has strong team consensus but no PR has landed. Keep open for maintainer action.

## Change summary

Quick picks with ignoreFocusOut=true have no mouse-accessible way to dismiss them — only Escape works.

## Best solution

Implement the proposal from the discussion: automatically show an X close button in the title bar when ignoreFocusOut is true. Likely touches `src/vs/platform/quickinput/browser/quickInput.ts` and related UI files. Multiple team members and the design lead agreed on this approach.
