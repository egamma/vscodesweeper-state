# Issue #295731 — Preserve pasted screenshots in Copilot chat across IDE restarts / session reloads

[https://github.com/microsoft/vscode/issues/295731](https://github.com/microsoft/vscode/issues/295731)

## Summary

The core request — persist pasted images across IDE restarts — has been addressed on main. Commit 8c3bcf84aa3 adds input-state image persistence, and the chat model serialization pipeline already encodes image binary data as base64 via toExport/fromExport for sent messages. The related issue #314556 confirms images are now being inlined/persisted (and tracks the resulting session-growth concern separately).

## Change summary

Pasted screenshots in Copilot chat are lost after IDE restart or extension reload, while text is preserved.

## Best solution

Keep the shipped persistence fix in commit 8c3bcf84aa3 (main-only, expected in 1.127). The session-growth side-effect of base64-inlined images is tracked separately in https://github.com/microsoft/vscode/issues/314556 — no further action needed on this issue.
