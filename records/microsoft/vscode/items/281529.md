# Issue #281529 — SCM Graph - Branch pills in scm graph could expand if they have empty space to the left

[https://github.com/microsoft/vscode/issues/281529](https://github.com/microsoft/vscode/issues/281529)

## Summary

Team-authored polish request for SCM Graph branch pills to expand into available horizontal space. Hard keep-open (MEMBER author). The constraint is still present on current main — no fix has landed.

## Change summary

Branch reference pills (badges) in the SCM Graph view are truncated with ellipsis even when ample horizontal space is available to the left.

## Best solution

Adjust the CSS layout for the history-item label/description elements in `src/vs/workbench/contrib/scm/browser/media/scm.css` (e.g. the `max-width: 100px` on `.description` around line 247, and potentially the label container's flex behavior) so that branch pills can grow into available space. The rendering logic in `scmHistoryViewPane.ts` `_renderBadge()` may also need to account for dynamic sizing based on the graph column width.
