# Issue #152517 — Expose access token expiration time from `AuthenticationSession`

[https://github.com/microsoft/vscode/issues/152517](https://github.com/microsoft/vscode/issues/152517)

## Summary

Team-authored feature request for exposing token expiration time on AuthenticationSession — not yet implemented on main, and the hard keep-open rule for MEMBER-authored issues applies.

## Change summary

Add an `expiresAt` property to the `AuthenticationSession` API so extensions can pass token expiration to client SDKs (e.g. Azure TokenCredential) without fabricating values.

## Best solution

Add an optional `expiresAt?: number` property to `AuthenticationSession` in the stable API (or start as a proposed API). The authentication providers (Microsoft account, GitHub) would need to propagate the expiration from the token response. Likely files: `src/vscode-dts/vscode.d.ts`, `src/vs/workbench/services/authentication/`, and the built-in auth extensions.
