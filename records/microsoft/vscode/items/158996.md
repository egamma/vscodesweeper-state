# Issue #158996 — "Follow link" hover is not linkified on markdown links

[https://github.com/microsoft/vscode/issues/158996](https://github.com/microsoft/vscode/issues/158996)

## Summary

Keep open for maintainer handling: this is a team-authored issue with an assigned On Deck milestone, both of which are hard keep-open conditions. The current code path still appears aligned with the reported root cause, so there is no basis for a cleanup close.

## Change summary

The editor link hover should render "Follow link" as a clickable markdown link for markdown document links whose targets are resolved lazily.

## Best solution

Resolve lazy DocumentLink targets before constructing the link hover, likely in the editor link contribution around src/vs/editor/contrib/links/browser/links.ts and src/vs/editor/contrib/links/browser/getLinks.ts, so markdown links get the same clickable hover treatment as links with eager urls.
