# Issue #94610 — Scrollbar doens't respond to touch screen dragging

[https://github.com/microsoft/vscode/issues/94610](https://github.com/microsoft/vscode/issues/94610)

## Summary

This appears to have been implemented after the report: the editor scrollbar now handles pointer events and captures pointer movement, which covers touch dragging as well as mouse dragging. The Backlog milestone does not block closing when there is a concrete shipped fix.

## Change summary

Support dragging the editor scrollbar with a Windows touch screen.

## Best solution

Treat this as completed by https://github.com/microsoft/vscode/commit/3345f477f7e70d24412098ab7783de8e70a7ba22, which moved scrollbar dragging from mouse-only handling to pointer capture.
