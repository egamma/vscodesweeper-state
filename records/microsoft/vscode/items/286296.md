# Issue #286296 — Support decorations that hide characters

[https://github.com/microsoft/vscode/issues/286296](https://github.com/microsoft/vscode/issues/286296)

## Summary

Team-authored feature request for a new editor decoration capability to hide/collapse characters inline — still unimplemented on main. Hard keep-open (MEMBER author).

## Change summary

Add editor decoration support that can visually hide characters in a line, enabling WYSIWYG-style rendering (e.g. hiding markdown syntax characters).

## Best solution

Implement a new decoration option (alongside existing `InjectedText` for adding text) that allows hiding/collapsing character ranges within a line. This lives in `src/vs/editor/common/model.ts` (decoration options) and the view/layout layer. Would unlock WYSIWYG markdown and similar rich-text-over-plaintext extension scenarios.
