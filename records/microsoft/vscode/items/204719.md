## Summary

Still present on main: the walkthrough embedded editor CSS sets the custom background on `.monaco-editor-background` and `.margin-view-overlays` but omits the `.monaco-editor` selector itself, causing a mismatch on selection. alexdima's comment identifies the fix.

## Change summary

The Interactive Playground's embedded editor shows two different background colors when text is selected, because not all required CSS selectors receive the custom background.

## Best solution
