# Issue #313034 — Github Copilot Session Error

[https://github.com/microsoft/vscode/issues/313034](https://github.com/microsoft/vscode/issues/313034)

## Summary

Fixed on main and shipped. The RangeError at JSON.stringify when serializing oversized chat sessions was addressed by PR #319183 (initial fix) and PR #322806 (hardening the deep-clone path), both now in released builds.

## Change summary

Large Copilot chat sessions fail to serialize with RangeError: Invalid string length, risking session data loss on window close.

## Best solution

Keep the shipped fixes; no further change needed. The user should update to VS Code 1.123.0+ where the RangeError-tolerant serialization is in place.
