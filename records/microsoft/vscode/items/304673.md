# Issue #304673 — Error when using Copilot Chat with Postman extension: Duplicate enum items in JSON schema

[https://github.com/microsoft/vscode/issues/304673](https://github.com/microsoft/vscode/issues/304673)

## Summary

The error originates from the Postman extension (v0.39.2) providing an MCP tool with a malformed JSON schema (duplicate enum values in the `method` property). VS Code correctly validates MCP tool schemas and reports the violation — the fix belongs in the Postman extension, not VS Code.

## Change summary

Copilot Chat fails to use the Postman MCP tool because the extension's tool schema contains duplicate enum items.

## Best solution

The reporter should file this issue with the Postman extension team — their `mcp_postman_mcp_updateCollectionRequest` tool schema has duplicate entries at positions 14 and 29 in the `method` enum. VS Code's schema validation is working correctly here.
