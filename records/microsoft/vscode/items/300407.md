# Issue #300407 — Automated Screenshot Tool Failure - Error 400 (Image dimensions exceed 2000px)

[https://github.com/microsoft/vscode/issues/300407](https://github.com/microsoft/vscode/issues/300407)

## Summary

The error originates from the Copilot extension sending oversized screenshot images to the Anthropic API. VS Code core exposed the `_chat.resizeImage` command (PR #301734, merged 4 days after this issue) specifically so the extension could resize images before sending them. The fix responsibility lies with the Copilot extension, which has since gained access to this resize API. The reporter was also on an outdated version (1.110.1).

## Change summary

Copilot's screenshot tool sends images exceeding the Anthropic API's 2000px limit on high-resolution displays, causing an Error 400 loop.

## Best solution

The Copilot extension should call the `_chat.resizeImage` command (now available since 1.111+) to downscale screenshots before sending them to the model API. The VS Code core plumbing is already in place; any remaining fix belongs in the Copilot extension.
