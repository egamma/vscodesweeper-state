## Change summary

Allow editor.wordWrap modes like 'wordWrapColumn' to define the wrap style used on Alt+Z toggle, while still opening files unwrapped by default.

## Best solution

Add a new setting or extend the existing `editor.wordWrap` enum (e.g. a 'wordWrapColumnDefaultOff' value or a separate boolean) so that column-based wrapping is the toggle-on behavior without being the default-open behavior. The implementation lives in `src/vs/editor/` around the word-wrap toggle action and editor configuration resolution.
