# Issue #237178 — Please add an option to have word wrap toggled off for every editor by default

[https://github.com/microsoft/vscode/issues/237178](https://github.com/microsoft/vscode/issues/237178)

## Summary

Keep open as a valid editor wrapping feature request in Backlog; the maintainer discussion clarified that this is not covered by `editor.wordWrap: off` because the requested toggle-on behavior must preserve `wordWrapColumn`. Current main still exposes the existing wrapping modes and internal on/off toggle overrides, not a user-facing default-off mode for column wrapping.

## Change summary

Add a way for files to open with word wrap off while Alt+Z toggles on the configured `editor.wordWrapColumn` behavior.

## Best solution

Track this in the editor wrapping backlog as a new setting or mode, likely near `src/vs/editor/common/config/editorOptions.ts` and `src/vs/workbench/contrib/codeEditor/browser/toggleWordWrap.ts`, that separates the default open state from the wrapping style used when toggled on.
