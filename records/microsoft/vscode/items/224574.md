# Issue #224574 — Unable to sign in with Plasma 6 Wayland

[https://github.com/microsoft/vscode/issues/224574](https://github.com/microsoft/vscode/issues/224574)

## Summary

Correctly triaged as an upstream Electron issue — `shell.openExternal()` fails to open URLs under Wayland on NixOS without explicit xdg-portal configuration. Already labeled `upstream` and `electron`; keeping open as the upstream tracker.

## Change summary

Signing in to Microsoft/GitHub accounts fails on Plasma 6 Wayland (NixOS) because the browser never opens — `shell.openExternal()` in Electron does not work correctly without xdg-portal configuration.

## Best solution

The root cause is in Electron's `shell.openExternal()` API not using xdg-desktop-portal on Wayland. The NixOS-specific workaround (enabling `xdg.portal.xdgOpenUsePortal`) exists, but the upstream Electron fix would make this work out of the box. Keep this open as VS Code's upstream tracker; consider filing or linking to a specific Electron issue if one doesn't already exist.
