Two separate concerns reported together without sufficient reproduction detail: (1) focus loss after REPL command execution may depend on the debug adapter's stop/continue behavior, (2) display expression persistence is a GDB/debug adapter concern (cpptools), not VS Code core. Needs a minimal repro with version info before any fix can be scoped.
