## Summary

Renderer crash (SIGBUS on shared-memory pages, exit code 135) affecting multiple Linux users on VS Code 1.129.0 across Intel, AMD, and NVIDIA GPUs on both X11 and Wayland. The crash signature (SIGBUS at page-aligned anonymous/shm fault addresses) points to an upstream Electron/Chromium compositor or shared-memory regression, likely introduced or exposed by the Electron 42.5.0 bump that landed for 1.129.0. Multiple independent confirmations (12 upvotes, several commenters) make this credible and broadly impactful. The `info-needed` label is already applied, but the report from NjengaC (comment 7) is actually quite thorough — it includes crash dumps, bisect data, and environment details — so reclassifying or routing to the Electron/upstream area would be more productive than waiting for more info.
