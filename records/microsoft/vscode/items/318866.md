# Issue #318866 — `([src/main.ts 17](src/main.ts#17))` is not properly linkified in Chat view

[https://github.com/microsoft/vscode/issues/318866](https://github.com/microsoft/vscode/issues/318866)

## Summary

Confirmed bug in the linkifier: splitting on whitespace at line 86 of linkifier.ts breaks markdown links whose display text contains spaces and whose `[` is preceded by non-whitespace (e.g. `(`). An open PR by the reporter already addresses this — keeping open pending that PR's review and merge.

## Change summary

Markdown links like `([src/main.ts 17](src/main.ts#17))` are not linkified in Chat view when link text contains spaces and `[` is not preceded by whitespace.

## Best solution

Review and land the author's open PR https://github.com/microsoft/vscode/pull/319073, which fixes the linkifier's whitespace-split logic to handle markdown links whose `[` is preceded by non-space characters and whose display text contains spaces.
