Team-authored bug (hard keep-open). The report describes kept/accepted files reverting to 'changed' state when switching conversations via history. The editing session restore path (`restoreFromSnapshot`) does serialize and restore the `ModifiedFileEntryState`, and `setContents` has a guard to re-accept already-accepted entries.
