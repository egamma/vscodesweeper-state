## Summary

The core complaint — Copilot not knowing about the terminal shell — appears addressed on current main: the runInTerminalTool detects fish via isFish() and provides fish-specific model instructions. However, a team member is actively discussing further improvements (using bash/zsh instead of fish for agent commands), so keeping open for the team's judgment.

## Change summary

Current main already provides fish-specific model descriptions and shell detection.
