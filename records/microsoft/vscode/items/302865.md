# Issue #302865 — Hooks not getting invoked from plugins installed from agentplugins marketplace

[https://github.com/microsoft/vscode/issues/302865](https://github.com/microsoft/vscode/issues/302865)

## Summary

Team-authored issue reporting that hooks defined in agentPlugins marketplace-installed plugins are not invoked because the plugin path is not whitelisted. Keeping open per policy (MEMBER author).

## Change summary

Hooks defined in agentPlugins installed from the marketplace are not invoked because the plugin's path is not included in the hooks whitelist.

## Best solution

Extend the hooks path whitelist logic to include the agentPlugins directory (e.g. `AppData/Roaming/Code - Insiders/agentPlugins/...`) so that hooks declared in a plugin's `plugin.json` are resolved and invoked the same way as `.agents/hooks` or `.github/hooks`.
