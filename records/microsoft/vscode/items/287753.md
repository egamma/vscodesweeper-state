# Issue #287753 — memory leak related to `domStylesheets`

[https://github.com/microsoft/vscode/issues/287753](https://github.com/microsoft/vscode/issues/287753)

## Summary

Fixed by the reporter's own PR (#287754), merged the same day and shipped in 1.109.0. The `WrappedStyleElement` now extends `Disposable` and passes its `_store` to `createStyleSheet`, ensuring the global stylesheet entry is cleaned up on dispose.

## Change summary

Opening/closing peek definition (or other features using `createStyleSheet2`) leaked detached HTMLStyleElements into the global stylesheet map because disposal never removed them.

## Best solution

Keep the shipped fix (commit a3dcfddb6ea); no further change needed. Close as completed.
