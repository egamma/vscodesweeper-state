## Summary

Fixed on main via xterm.js bump to 6.1.0-beta.220 which picks up the upstream fix for Kitty keyboard protocol swallowing Shift+number characters; shipped in 1.123.0.

## Change summary

Shift+number keys produce digits instead of symbols in Neovim (and other apps using Kitty keyboard protocol) inside VS Code's integrated terminal.

## Best solution

Keep the shipped xterm.js fix (commit 103a85ad836, 1.123.0); no further VS Code-side change needed. The upstream xterm.js issue (xtermjs/xterm.js#5823) tracks any remaining edge cases.
