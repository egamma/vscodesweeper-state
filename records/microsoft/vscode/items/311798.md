Team-authored issue (MEMBER), must keep open. Partial fixes landed on main (commits a21f814cede and f6bf90fde75, May 28 2026) adding `reasoning_content` field support for Minimax/DeepSeek/Kimi streams. However, the reported symptom — raw `<think>` tags in text content — may indicate the model also embeds thinking inside text, which is a distinct path with no stripping logic in the chat/agent flow.
