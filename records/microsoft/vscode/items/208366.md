## Change summary

Show file mode and encoding change information in the VS Code diff editor, not just content differences.

## Best solution

The diff editor or SCM change view could display a banner or decoration when `git diff` reports mode/encoding changes without content changes. This would likely touch the git extension's diff information provider and the diff editor's header/status area. Given the niche audience, it could also be served by an extension reading `git diff --raw` output.
