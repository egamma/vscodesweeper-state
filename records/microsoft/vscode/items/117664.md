# Issue #117664 — Diff editor scrolls horizontally when dragging selection vertically

[https://github.com/microsoft/vscode/issues/117664](https://github.com/microsoft/vscode/issues/117664)

## Summary

Keep open: this is a team-authored issue with an assigned milestone, both of which are hard keep-open conditions. The discussion also identifies a real editor behavior around selection dragging over view zones that needs maintainer judgment rather than cleanup closure.

## Change summary

Dragging a vertical selection in the diff editor can cause horizontal scrolling because the cursor maps to the end of the line above a view zone.

## Best solution

The editor rendering owner should decide whether mouse coordinates over a view zone can preserve the horizontal offset by mapping through the visible line above when possible, while keeping the existing special handling for large or non-visible view zones.
