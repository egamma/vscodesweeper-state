# Issue #87741 — Auto save & new line prevents undo from working

[https://github.com/microsoft/vscode/issues/87741](https://github.com/microsoft/vscode/issues/87741)

## Summary

Keep open as a plausible current bug: the report is specifically about the final-newline save participant adding an undo entry, and current main still applies that participant through normal edit operations without an undo-stack regression test for this path.

## Change summary

With `files.insertFinalNewline` and auto save enabled, undoing a user edit at EOF first removes the automatically inserted final newline, which is immediately reinserted on save and prevents undo from reaching the user edit.
