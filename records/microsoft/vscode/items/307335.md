# Issue #307335 — Copilot login failed (PermissiveAuthRequiredError: Permissive authentication is required)

[https://github.com/microsoft/vscode/issues/307335](https://github.com/microsoft/vscode/issues/307335)

## Summary

Self-resolved configuration issue: the reporter discovered in Comment 10 that their own `"github.copilot.advanced": { "authPermissions": "minimal" }` setting was blocking the newer Copilot auth flow. Removing the setting fixed it. This is a support question, not a VS Code bug.

## Change summary

Copilot login fails with PermissiveAuthRequiredError when the user has `authPermissions: "minimal"` in their settings, which blocks the expanded OAuth scopes newer Copilot versions require.

## Best solution

Close as a resolved support question. The fix is removing the `github.copilot.advanced.authPermissions` setting. A separate polish issue could track improving the error message to surface this misconfiguration more clearly, but that's not what this issue requests.
