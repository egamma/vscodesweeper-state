# Issue #307335 — Copilot login failed (PermissiveAuthRequiredError: Permissive authentication is required)

[https://github.com/microsoft/vscode/issues/307335](https://github.com/microsoft/vscode/issues/307335)

## Summary

Self-resolved support question — the root cause is the Copilot extension's handling of the `authPermissions: "minimal"` setting, which conflicts with newer extension versions requiring permissive scopes. Not a VS Code core issue.

## Change summary

Copilot login fails with PermissiveAuthRequiredError when the user setting `github.copilot.advanced.authPermissions` is set to "minimal".

## Best solution

Close as caused-by-extension. The reporter identified the fix (remove the `authPermissions: "minimal"` setting). Any improvement to error messaging or graceful degradation belongs in the GitHub Copilot Chat extension, not VS Code core.
