# Issue #237981 — Terminal suggest: Handle paths on git bash

[https://github.com/microsoft/vscode/issues/237981](https://github.com/microsoft/vscode/issues/237981)

## Summary

Implemented on main: Git Bash path handling in terminal suggest was added in June 2025, with dedicated helper functions for converting between Git Bash and Windows paths, and shipped in 1.102.0.

## Change summary

Terminal suggest should handle Git Bash's special path format (e.g. /c/Users/... mapping to C:\Users\...).

## Best solution

Keep the shipped implementation — `terminalGitBashHelpers.ts` provides bidirectional path conversion and the completion service uses it for Git Bash resource completions. No further work needed.
