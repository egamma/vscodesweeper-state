# Issue #103527 — Move panel to statusbar - as icon who show popup panel

[https://github.com/microsoft/vscode/issues/103527](https://github.com/microsoft/vscode/issues/103527)

## Summary

A well-supported feature request (42 upvotes, in Backlog) for popup panels triggered from statusbar icons. No existing capability covers this — the statusbar currently has no popup-panel mechanism, and no canonical issue absorbs this specific ask.

## Change summary

Allow panels to be accessed as popup overlays triggered from statusbar icons, reducing the permanent space cost of collapsed panels.

## Best solution

This remains an open design question for the workbench layout team. The request is distinct from detachable views (https://github.com/microsoft/vscode/issues/193267) and flexible layout (https://github.com/microsoft/vscode/issues/208658) — it proposes a new interaction pattern (statusbar icon → transient popup). Keep open for the layout area owner to evaluate whether this fits the evolving workbench layout direction.
