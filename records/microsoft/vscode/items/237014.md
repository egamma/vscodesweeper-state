# Issue #237014 — Some sidebar items, such as Copilot Edits, select colors from unrelated elements

[https://github.com/microsoft/vscode/issues/237014](https://github.com/microsoft/vscode/issues/237014)

## Summary

Still valid on main: chat widget colors derive backgrounds from `editorBackground` but foregrounds from the general `foreground` token, which breaks themes where sidebar and editor use contrasting color schemes. A team member (roblourens) acknowledged the difficulty in a recent comment. No fix found.

## Change summary

Chat/sidebar widgets mix color tokens from unrelated theme scopes (editor background + sidebar foreground), causing near-zero contrast in themes with differing sidebar/editor color schemes.

## Best solution

Audit chat color registrations in `src/vs/workbench/contrib/chat/common/widget/chatColors.ts` and the chat widget CSS to ensure background/foreground pairs are sourced from the same scope (either sidebar or editor tokens consistently). The fix likely touches `chatColors.ts` color defaults and possibly view-pane override styles.
