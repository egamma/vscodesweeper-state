## Summary

Still valid: the macOS Window menu is missing the native 'Cycle Through Windows' entry. This depends on Electron exposing the appropriate menu role; no fix has landed on main and the issue remains correctly tagged as upstream/electron.

## Change summary

The macOS Window menu should include a 'Cycle Through Windows' entry like other native macOS apps.

## Best solution

Upstream: Electron needs to support a menu role for 'Cycle Through Windows' (the macOS system-level window cycling). Once Electron adds it, VS Code can add a `createRoleMenuItem` call in `setMacWindowMenu()` in `src/vs/platform/menubar/electron-main/menubar.ts` alongside the existing Minimize/Zoom/Bring All to Front entries.
