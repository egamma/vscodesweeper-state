# Issue #202417 — [css] Outline doesn't show @layer 

[https://github.com/microsoft/vscode/issues/202417](https://github.com/microsoft/vscode/issues/202417)

## Summary

Valid feature request — @layer is parsed by vscode-css-languageservice (Layer node type exists) but not included in collectDocumentSymbols, so it never appears in the Outline panel. The fix belongs upstream in vscode-css-languageservice.

## Change summary

CSS Outline panel should show @layer rules alongside @media, @keyframes, etc.

## Best solution

Add a `Layer` handler in `collectDocumentSymbols` in the upstream vscode-css-languageservice repo (similar to the existing `Media` handler). The node type and parser support already exist; only the symbol collection is missing.
