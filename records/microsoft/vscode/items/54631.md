# Issue #54631 — [folding] Add keyboard shortcut to jump between #region and #endregion

[https://github.com/microsoft/vscode/issues/54631](https://github.com/microsoft/vscode/issues/54631)

## Summary

This feature request for jumping between matching #region/#endregion markers (analogous to bracket matching) has not been implemented. Existing fold navigation commands (gotoNextFold, gotoPreviousFold, gotoParentFold) move between ranges sequentially, not between start/end of a matched pair.

## Change summary

Request a keyboard shortcut to jump between the opening #region and closing #endregion of the same folding region, analogous to Jump to Bracket.

## Best solution

Extend the folding contribution (src/vs/editor/contrib/folding/browser/folding.ts) with a 'Go to Matching Fold' action that jumps between the start and end lines of the current region. Alternatively, extend jumpToBracket in bracketMatching.ts to recognize region markers. The existing foldingModel already exposes start/end line info for each range.
