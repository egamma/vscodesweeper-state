# Issue #54631 — [folding] Add keyboard shortcut to jump between #region and #endregion

[https://github.com/microsoft/vscode/issues/54631](https://github.com/microsoft/vscode/issues/54631)

## Summary

Still unimplemented on main. Existing folding navigation commands (gotoParentFold, gotoPreviousFold, gotoNextFold) navigate between folds but do not jump between matching #region/#endregion markers the way jumpToBracket does for brackets. The feature request remains valid with 7 upvotes, below the 20-upvote threshold for Backlog promotion.

## Change summary

Request a keyboard shortcut to jump between the start (#region) and end (#endregion) of a folding region, analogous to editor.action.jumpToBracket.

## Best solution

Either extend `editor.action.jumpToBracket` to recognize region markers, or add a new command (e.g. `editor.action.jumpToMatchingFoldingMarker`) in `src/vs/editor/contrib/folding/browser/folding.ts` that identifies the current region and moves the cursor to the opposing marker line.
