# Issue #322205 — Add a Directory of Free AI Coding Assistants and Alternatives

[https://github.com/microsoft/vscode/issues/322205](https://github.com/microsoft/vscode/issues/322205)

## Summary

This requests curated editorial/marketing content (a directory of free AI tools) inside VS Code, which is outside the scope of the product — VS Code is a code editor, not a recommendation platform for third-party services.

## Change summary

Add a built-in directory or marketplace category listing free AI coding assistants with pricing/plan information.

## Best solution

The VS Code Marketplace already allows users to search for AI extensions by category and popularity. Curating a hand-maintained directory of free tools with pricing info is editorial content that doesn't belong in the editor core. Users can discover extensions via the existing marketplace search and filtering.
