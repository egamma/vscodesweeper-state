# Issue #246158 — `ctrl+space` keybinding shouldn't toggle checkbox in quick pick

[https://github.com/microsoft/vscode/issues/246158](https://github.com/microsoft/vscode/issues/246158)

## Summary

This appears fixed on main: commit 334fc2cacab moved checkbox toggling to a proper keybinding command bound only to Space (no modifiers), so ctrl+space no longer triggers checkbox toggle. Keeping open because it's team-authored and needs explicit maintainer judgment to close.

## Change summary

ctrl+space in a multi-select quick pick was toggling the checkbox instead of toggling the hover.

## Best solution

Already fixed by https://github.com/microsoft/vscode/commit/334fc2cacab296fd70fd2b999204ead4405662ff (shipped in 1.108.0). The old inline keydown handler in quickInputList.ts didn't filter modifier keys; the new command-based binding fires only on bare Space. A maintainer can close this as completed.
