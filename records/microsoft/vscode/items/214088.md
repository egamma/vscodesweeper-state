# Issue #214088 — `explorer.excludeGitIgnore` should also exclude files listed in `.git/info/exclude`

[https://github.com/microsoft/vscode/issues/214088](https://github.com/microsoft/vscode/issues/214088)

## Summary

Valid feature request — the explorer's excludeGitIgnore filter only processes `.gitignore` files and does not read `.git/info/exclude` or `core.excludesFile`. The gap is confirmed on current main. Low community traction (0 upvotes), so it stays open in the community process.

## Change summary

Make `explorer.excludeGitIgnore` also respect `.git/info/exclude` and `core.excludesFile` ignore rules, not just `.gitignore` files.

## Best solution

Extend `FilesFilter` in `src/vs/workbench/contrib/files/browser/views/explorerViewer.ts` to also load `.git/info/exclude` (and optionally `core.excludesFile`) into the ignore tree when `explorer.excludeGitIgnore` is enabled, so the explorer hides the same files that `git status` considers ignored.
