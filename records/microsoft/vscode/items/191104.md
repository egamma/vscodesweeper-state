Team-authored feature request to persist quick search results across sessions. The feature has not been implemented on current main — the quick search picker has no state retention mechanism. Keeping open per team-author policy.

## Change summary

Quick search (quick text search) loses all results when dismissed; request to retain state like quick chat does.
