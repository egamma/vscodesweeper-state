# Issue #32579 — [themes] Disable Italic Option Feature Request

[https://github.com/microsoft/vscode/issues/32579](https://github.com/microsoft/vscode/issues/32579)

## Summary

Keep open as a popular Backlog feature request: it asks for a global user-level way to suppress theme-provided italic styling, and the current workaround still requires scope-specific token customization or theme changes.

## Change summary

Users want a single VS Code setting or command to disable italic token font styles contributed by themes.

## Best solution

The themes/editor owners should decide whether to add a user-level editor theming control that strips `italic` from token font styles, likely near token color customization handling, while preserving existing scope-specific overrides for users who want finer control.
