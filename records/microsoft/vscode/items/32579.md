# Issue #32579 — [themes] Disable Italic Option Feature Request

[https://github.com/microsoft/vscode/issues/32579](https://github.com/microsoft/vscode/issues/32579)

## Summary

Still a valid feature request — no global 'disable italic' setting exists on current main. The workaround (per-scope textMateRules) is tedious and incomplete. Accessibility and usability justification is strong, and a team contributor (sana-ajani) expressed support for a simpler solution.

## Change summary

Request for a single setting to globally disable italic font style across all editor token scopes, similar to Sublime Text's `no_italic` option.

## Best solution

Add a top-level boolean setting (e.g. `editor.fontStyleOverrides.italic: false`) in the theme/token resolution layer (`src/vs/workbench/services/themes/common/colorThemeData.ts`) that strips italic from all resolved token styles before rendering. This avoids requiring users to enumerate every scope manually.
