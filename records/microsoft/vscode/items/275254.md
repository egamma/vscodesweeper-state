# Issue #275254 — Handoffs are not screen reader discoverable

[https://github.com/microsoft/vscode/issues/275254](https://github.com/microsoft/vscode/issues/275254)

## Summary

Team-authored accessibility bug that remains unfixed on main. The handoff/suggest-next widget has aria-labels on buttons but no aria-status announcement when handoffs appear, and the chat accessibility help dialog does not mention shift+tab navigation to reach them.

## Change summary

Screen reader users receive no announcement when handoff actions appear in chat, and the accessibility help dialog does not document how to reach them via shift+tab.

## Best solution

Add an aria-status announcement in `chatSuggestNextWidget.ts` when handoff actions render (e.g. via `status()` or `IAccessibilitySignalService`), and update the chat accessibility help provider to document the shift+tab path to handoff actions.
