# Issue #216853 — Copy a link from integrated terminal via context menu

[https://github.com/microsoft/vscode/issues/216853](https://github.com/microsoft/vscode/issues/216853)

## Summary

Valid feature request — no 'Copy Link' action exists in the terminal link hover or context menu on current main. The capability remains unimplemented and is not tracked elsewhere as a duplicate.

## Change summary

Add a 'Copy Link' action to the terminal's link hover/context menu so users can copy a detected hyperlink to the clipboard without opening it.

## Best solution

Add a hover action or context-menu entry in the terminal link hover (terminalLinkManager.ts, around the _showHover call) that copies the link text to the clipboard. The related issue https://github.com/microsoft/vscode/issues/231446 asks for 'Open in Browser' as a separate action on the same surface — these could be addressed together as a set of link-hover actions.
