## Summary

Valid feature request with 29 upvotes, promoted to Backlog. The pinning infrastructure exists (quickPickPin.ts) but is only wired to tasks and terminal history — not yet to the command palette or 'Open Recent'. This is the canonical issue; #258202 was closed as a duplicate of it.

## Change summary

Add pin-to-top support in the Command Palette and 'File: Open Recent' quick picks, mirroring the existing pinning in Tasks.

## Best solution

Extend the existing `showWithPinnedItems` utility (src/vs/platform/quickinput/browser/quickPickPin.ts) to the command palette and 'Open Recent' quick access providers. The plumbing is already in place — it needs wiring into the relevant quick-access contributions.
