Verified duplicate of #299477, which was fixed in PR #307721 (shipped in 1.116.0) and marked as verified. This issue describes the same `/compact` button wiping chat session history.

## Change summary

Pressing the compact button in the Copilot context widget deleted the entire chat session history instead of compacting it.

## Best solution
