# Issue #258194 — Adding a screenshot to chat context fails because Blob.bytes() is unsupported in some browsers

[https://github.com/microsoft/vscode/issues/258194](https://github.com/microsoft/vscode/issues/258194)

## Summary

Still reproduces on main: `Blob.bytes()` at line 714 of `browserHostService.ts` is used without a fallback, and this API lacks support in older browser engines. The fix is a one-line change to use `blob.arrayBuffer()` instead.

## Change summary

Adding a screenshot to chat context in the web version fails because `Blob.bytes()` is not supported in all browser engines.

## Best solution

Replace `await blob.bytes()` with `new Uint8Array(await blob.arrayBuffer())` (or similar) in `src/vs/workbench/services/host/browser/browserHostService.ts` line 714 to ensure cross-browser compatibility. `Blob.arrayBuffer()` has much wider support.
