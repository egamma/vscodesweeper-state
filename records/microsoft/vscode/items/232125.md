# Issue #232125 — Shortcuts for auto indentation not working

[https://github.com/microsoft/vscode/issues/232125](https://github.com/microsoft/vscode/issues/232125)

## Summary

The reindent commands are invoked successfully (keybinding works), but produce no edits because Python's built-in language configuration lacks `indentationRules`. The `getReindentEditOperations` function returns early with an empty array when `indentRulesSupport` is null.

## Change summary

Reindent Lines / Reindent Selected Lines commands do nothing for Python because the Python language configuration provides no `indentationRules`.

## Best solution

Add `indentationRules` (at minimum `increaseIndentPattern` and `decreaseIndentPattern`) to `extensions/python/language-configuration.json`, or make the reindent commands fall back to `onEnterRules` when `indentationRules` are absent. The former is simpler and more targeted.
