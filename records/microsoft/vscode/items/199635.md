Team-authored upstream issue tracking a Chromium/platform change to disable macOS slide-back animation on cancelled drags for floating windows. Still open upstream with no resolution; must keep open per team-author rule.

## Change summary

Disable the macOS slide-back animation when a tab drag is cancelled (e.g., dropping a tab outside the app to create a floating window), removing an undesirable delay.
