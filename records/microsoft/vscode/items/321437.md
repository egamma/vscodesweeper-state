# Issue #321437 — The JSON language server's NPM package (vscode-json-languageserver) is five years out of date

[https://github.com/microsoft/vscode/issues/321437](https://github.com/microsoft/vscode/issues/321437)

## Summary

Valid request: the vscode-json-languageserver NPM package is stuck at v1.3.4 (last published ~5 years ago) while the source in the repo is actively maintained. This is a publishing/release process gap, not a bug in VS Code itself.

## Change summary

The vscode-json-languageserver NPM package has not been published for ~5 years despite active source development, preventing external consumers from using it without building from source.

## Best solution

The team should decide whether to resume publishing vscode-json-languageserver to NPM on a regular cadence (bumping the version in extensions/json-language-features/server/package.json and running npm publish), or explicitly document that the package is no longer independently published and consumers should use the bundled extension or build from source.
