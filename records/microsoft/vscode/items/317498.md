# Issue #317498 — Agents App: Allow me to check for updates from the Agents App

[https://github.com/microsoft/vscode/issues/317498](https://github.com/microsoft/vscode/issues/317498)

## Summary

The requested capability — manually checking for updates from the Agents window — already exists on current main. The account menu in the Agents window includes all update state items (Check for Updates, Download, Restart to Update, etc.) via `appendUpdateMenuItems`, and the `CheckForUpdateAction` is available in the command palette (`f1: true`). This was in place before the issue was filed.

## Change summary

Allow manually checking for VS Code updates from within the Agents window, instead of requiring switching to the normal VS Code app.

## Best solution

Close as already implemented. The Agents window's account menu already includes 'Check for Updates…' (and all related update-state entries) via `registerUpdateMenuItems(AccountMenu, '3_updates')` in `src/vs/sessions/contrib/accountMenu/browser/account.contribution.ts`. The command is also available via the command palette. If the user cannot find it, a brief reply pointing to the account menu is sufficient.
