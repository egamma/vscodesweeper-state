Valid feature request with 20 upvotes, already in Backlog. The disassembly editor has no auto-close logic when a debug session ends — confirmed by inspecting current main. No duplicate found; the request is still unimplemented.

## Change summary

Request to automatically close the disassembly editor tab when the associated debug session terminates.

## Best solution

Add a listener in `debugService.ts` (or the disassembly view contribution) for `onDidEndSession` that closes any open `DisassemblyViewInput` editor tabs, ideally behind a setting as suggested in the comments. The likely files are `src/vs/workbench/contrib/debug/browser/debugService.ts` and `src/vs/workbench/contrib/debug/common/disassemblyViewInput.ts`.
