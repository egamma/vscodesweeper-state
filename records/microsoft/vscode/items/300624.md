# Issue #300624 — vs code installation zip，setting of powershell 7 does not work in using cline

[https://github.com/microsoft/vscode/issues/300624](https://github.com/microsoft/vscode/issues/300624)

## Summary

Reporter self-resolved: the issue was a Cline extension configuration mismatch (terminal profile name), not a VS Code bug. Cline is a third-party extension.

## Change summary

PowerShell 7 terminal launched by the Cline extension used the wrong executable path, ignoring the VS Code terminal profile configuration.

## Best solution

Close as caused-by-extension. The reporter confirmed the fix was adjusting Cline's own `defaultTerminalProfile` setting to match the VS Code profile name. Any remaining issues belong in the Cline extension's tracker.
