## Summary

Partially addressed but still open: several context menu items were cleaned up (Share, Copy as, Copilot Review removed from code blocks), and peek/side variants of Go to commands are guarded by `isInEmbeddedEditor`. However, comment 10 (Sep 2025) identifies remaining questions — Go to Definition / Type Definition / Implementations still appear in code block menus (intentionally per mjbvz's design), and Command Palette appearing there is questionable. The issue still has actionable items and active team discussion.

## Change summary

The checklist in comment 3 still has unchecked items (Share, Copy as, Explain scope picker).
