Confirmed upstream Chromium bug (chromium#807406) affecting Freesync/VRR monitors on fullscreen transitions. A VS Code team member already triaged this as an upstream runtime issue and linked the Chromium bug. The issue is correctly labeled and waiting on a Chromium fix.

## Change summary

Entering fullscreen in VS Code causes Freesync monitors to drop refresh rate, requiring a reboot to recover.
