# Issue #183994 — Add jump to previous/next bracket commands

[https://github.com/microsoft/vscode/issues/183994](https://github.com/microsoft/vscode/issues/183994)

## Summary

Valid feature request for sequential bracket navigation commands (next/previous bracket in document order). Not yet implemented — only `jumpToBracket` (matching bracket) exists. Low upvote count but authored by a team contributor; keeping open in Backlog.

## Change summary

Add commands to jump to the next or previous bracket in document order, independent of bracket-pair matching.

## Best solution

Implement `editor.action.jumpToNextBracket` and `editor.action.jumpToPreviousBracket` in `src/vs/editor/contrib/bracketMatching/browser/bracketMatching.ts`. The forward case can reuse the existing `findNextBracket` API on `IBracketPairsTextModelPart`; the backward case would need a new `findPreviousBracket` method (or reverse iteration over bracket pairs). These commands would complement the existing `jumpToBracket` for fast document-order navigation.
