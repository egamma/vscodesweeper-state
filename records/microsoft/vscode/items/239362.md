# Issue #239362 — [Accessibility] Cannot read multi diff content in Source Control Graph View

[https://github.com/microsoft/vscode/issues/239362](https://github.com/microsoft/vscode/issues/239362)

## Summary

Still open: the merged PR #248135 did not resolve the focus problem. lszomoru confirmed in May 2025 that the multi-diff editor's focus path fails because the template observable can still be undefined when focus() is called. The underlying timing issue in multiDiffEditorWidgetImpl.ts remains on current main.

## Change summary

Screen readers cannot read multi-diff content opened from the SCM Graph View because keyboard focus never lands in the diff editor.

## Best solution

Fix the focus timing in `src/vs/editor/browser/widget/multiDiffEditor/multiDiffEditorWidgetImpl.ts` so that when a user activates a commit in the SCM Graph, focus reliably moves into the rendered diff editor — likely by deferring the focus call until the template observable resolves to a non-undefined value.
