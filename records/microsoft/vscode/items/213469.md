# Issue #213469 — Revert and Stage block buttons are very close together

[https://github.com/microsoft/vscode/issues/213469](https://github.com/microsoft/vscode/issues/213469)

## Summary

Valid UX concern still present on main — the stage and revert block buttons in the diff editor gutter toolbar remain adjacent with no spacing or confirmation guard. Already labeled under-discussion in Backlog; no fix has landed.

## Change summary

Stage and Revert block buttons in the diff editor gutter are too close together, risking accidental destructive reverts without confirmation.

## Best solution

Either add spacing/visual separation between the stage and revert gutter buttons in `gutterFeature.ts`, or add a lightweight confirmation (e.g. modifier-key requirement or undo toast) for the revert action. The toolbar is built in `src/vs/editor/browser/widget/diffEditor/features/gutterFeature.ts` around line 255.
