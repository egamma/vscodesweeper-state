## Change summary

Automatically fold all function/method bodies and expand/collapse them as the cursor enters or leaves each region.

## Best solution

Implement a cursor-aware folding mode in the editor folding contribution (src/vs/editor/contrib/folding/browser/folding.ts) that collapses all top-level function regions and dynamically expands/collapses based on cursor position, gated behind a new editor option. Alternatively, this could be an extension-candidate built on the existing folding API, though deep integration with cursor tracking favors core.
