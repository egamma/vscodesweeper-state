# Issue #84297 — API support for Timeline view

[https://github.com/microsoft/vscode/issues/84297](https://github.com/microsoft/vscode/issues/84297)

## Summary

The Timeline API remains proposed (in `src/vscode-dts/vscode.proposed.timeline.d.ts`) and has not been finalized into the stable API (`vscode.d.ts`). With 65 upvotes and active community interest, this should stay open as a valid backlog item awaiting finalization.

## Change summary

Finalize the proposed Timeline extension API so extensions can register timeline providers without enabling a proposed API.

## Best solution

Finalize the Timeline API into the stable `vscode.d.ts`. The blocking question from 2020 (workspace-scoped timeline events) may now be addressed by the Source Control Graph feature (https://github.com/microsoft/vscode/issues/213935). The API shape in `src/vscode-dts/vscode.proposed.timeline.d.ts` is mature and has been used by multiple extensions for years. A maintainer should evaluate whether the workspace-scope blocker is resolved and proceed with finalization.
