# Issue #228670 — Line numbers in long codes cannot be highlighted correctly.

[https://github.com/microsoft/vscode/issues/228670](https://github.com/microsoft/vscode/issues/228670)

## Summary

This appears to be fixed on current main by a targeted editor change, so it can be closed as completed rather than with a triage closing label.

## Change summary

The active line number was not highlighted when the cursor was on a wrapped continuation segment of a long logical line.

## Best solution

Treat the issue as fixed by commit 503bace89cb48c7ef8cbee8109e262889bf99907, which tracks active line-number highlighting by model line instead of wrapped view line; no further editor change is needed unless the reporter still reproduces on 1.101.0 or newer.
