# Issue #288581 — Chat session trust degradation system

[https://github.com/microsoft/vscode/issues/288581](https://github.com/microsoft/vscode/issues/288581)

## Summary

Security-labeled feature request for a trust degradation system in chat sessions — must remain open for maintainer judgment per security policy. The related internal backlog issue (closed) suggests planning work has occurred.

## Change summary

Implement a trust degradation model for chat sessions where trust level decreases as the session interacts with the internet or encounters prompt injection signals, restricting auto-approve for risky tools accordingly.

## Best solution

Design and implement a multi-level trust system for chat tool auto-approval that starts fully trusted, degrades on internet access or suspected prompt injection, and maps trust levels to per-tool approval policies. The closed internal backlog issue suggests this is already being tracked internally.
