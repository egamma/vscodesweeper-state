## Summary

Valid feature request: the git extension's revert-hunk commands (`git.revertChange`, `git.revertSelectedRanges`) only work on working-tree diffs, not on hunks from historical commits. No implementation exists on main, and the request has 0 upvotes with no community traction after 14 months — keeping open for the community process.

The work lives in `extensions/git/src/commands.ts` and the SCM diff infrastructure.
