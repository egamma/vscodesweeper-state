# Issue #177733 — Watch variable window not wrapping lines

[https://github.com/microsoft/vscode/issues/177733](https://github.com/microsoft/vscode/issues/177733)

## Summary

Legitimate feature request with 101 upvotes, promoted to Backlog. Word wrap exists for the Debug Console (`debug.console.wordWrap`) but not for the Variables or Watch views. No implementation work has been done toward this; the feature gap remains on current main.

## Change summary

Request to add word wrapping (or horizontal scrolling) for long values in the Variables and Watch debug views.

## Best solution

Extend the existing `debug.console.wordWrap` pattern to the Variables and Watch views — add a setting like `debug.variablesView.wordWrap` and apply it in `variablesView.ts` / `watchExpressionsView.ts`. The Debug Console implementation in `repl.ts` and `debug.contribution.ts` provides a proven template. Given 101 upvotes and Backlog status, this is a well-validated community request ready for implementation.
