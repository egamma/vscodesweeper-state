# Issue #317896 — Update installation hangs at Extracting Files step (very last step)

[https://github.com/microsoft/vscode/issues/317896](https://github.com/microsoft/vscode/issues/317896)

## Summary

Known issue with PowerShell calls hanging during Windows update/install. A team member (dmitrivMS) cross-linked this with #315030 and acknowledged the underlying PowerShell problem. This report describes the same root cause — PowerShell-based Appx operations freezing the installer — as #315030, which is the more recent canonical tracking issue.

## Change summary

VS Code update installation hangs at the 'Extracting Files' step on Windows because PowerShell-based Appx package operations become unresponsive.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/315030, which tracks the same PowerShell-induced update hang on Windows. The fix path, as dmitrivMS noted, is to remove all PowerShell usage from the setup scripts in `build/win32/code.iss`.
