# Issue #268829 — Copilot Chat feature request: show timestamp for messages/prompts

[https://github.com/microsoft/vscode/issues/268829](https://github.com/microsoft/vscode/issues/268829)

## Summary

Well-supported feature request (94 upvotes, in Backlog) for showing timestamps/elapsed time in chat. The model infrastructure exists but rendering is intentionally hidden on current main. PR #325061 ('chat: add timestamps and elapsed time') appears to be the implementation candidate but has not landed on main yet — keep open.

## Change summary

Show timestamps and elapsed duration on chat messages so users can tell when a prompt started and how long it ran.

## Best solution

Land PR #325061 which adds visible timestamps and elapsed time to chat messages. The model-layer plumbing (timestamp, elapsedMs, confirmationAdjustedTimestamp) and formatting utilities already exist on main (commit 83dc26b3bbe); the remaining work is un-hiding the stats rendering in ChatWorkingProgressContentPart and adding per-message timestamp display (e.g. on hover).
