# Issue #326123 — Feature Request: Add option to collapse/expand queued messages in Copilot Chat

[https://github.com/microsoft/vscode/issues/326123](https://github.com/microsoft/vscode/issues/326123)

## Summary

Valid feature request for collapsible pending/queued messages in the Copilot Chat queue. The queue infrastructure exists (drag-and-drop reordering, queue/steer actions), but there is no collapse/expand toggle for pending message rows today. No duplicate found. Routing to the chat area for triage.

## Change summary

Add a collapse/expand toggle for pending/queued messages in the Copilot Chat pane so that long prompts can be condensed to a single-line preview, making queue management and reordering easier.

## Best solution

Add an optional collapsed rendering mode for pending request rows in `chatListRenderer.ts`, showing a single-line preview with a chevron toggle. The pending drag-and-drop infrastructure (`chatPendingDragAndDrop.ts`) already renders these rows — the collapse UI would sit alongside the existing drag handle. This is a polish/UX improvement to the chat queue feature.
