# Issue #304417 — changes made in terminal

[https://github.com/microsoft/vscode/issues/304417](https://github.com/microsoft/vscode/issues/304417)

## Summary

Valid feature request for reviewing file changes made by Copilot agent via terminal commands. A team member (justschen) is actively engaging on this, asking about terminal-level change reporting — keeping open for that internal discussion.

## Change summary

When Copilot agent mode makes code changes through terminal commands rather than direct edits, the user cannot see a diff overview or approve/reject those changes the way they can with inline edits.

## Best solution

Surface a unified changed-files review (diff view with accept/reject) for all file modifications made during an agent session, regardless of whether they came from direct edits or terminal commands. This likely involves the terminal tool reporting changed files back to the chat agent editing layer. The member comment suggests this is already being explored internally between the chat-agent-editing and terminal teams.
