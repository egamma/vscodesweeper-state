# Issue #310102 — Getting "Retried with Autopilot" messages with autopilot disabled

[https://github.com/microsoft/vscode/issues/310102](https://github.com/microsoft/vscode/issues/310102)

## Summary

Fixed on main: the misleading 'Retried with Autopilot' message shown in autoApprove mode (without autopilot enabled) was corrected in two follow-up commits that differentiate the label by permission level.

## Change summary

The retry progress message incorrectly showed 'Retried with Autopilot' even when only auto-approvals were enabled and autopilot was disabled.

## Best solution

Keep the shipped fix — the current wording on main shows 'Recovering from a request error…' for autoApprove and 'Autopilot: recovering from a request error…' only when autopilot is actually active. No further change needed.
