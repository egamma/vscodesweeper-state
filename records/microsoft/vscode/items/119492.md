# Issue #119492 — Debug: current stopped line should indicate better?

[https://github.com/microsoft/vscode/issues/119492](https://github.com/microsoft/vscode/issues/119492)

## Summary

The inline debug value decorations still use the same yellow background color (#ffc80033) on current main — no styling changes have been made to make them more subtle. The issue remains a valid polish request with team interest (roblourens commented expressing the same concern). Keeping open as under-discussion.

## Change summary

Debug inline value decorations are too visually noisy with their yellow background, making it hard to identify the current stopped line.

## Best solution

Restyle the inline debug value decorations to be more subtle — reduce or remove the yellow background highlight, aiming for something closer to GitLens-style annotations. The relevant color registrations are in `src/vs/workbench/contrib/debug/browser/debugEditorContribution.ts` (lines 65–72: `editor.inlineValuesForeground` and `editor.inlineValuesBackground`). This is a design decision that needs input from the design team.
