# Issue #61298 — Show Timestamp while Debugging

[https://github.com/microsoft/vscode/issues/61298](https://github.com/microsoft/vscode/issues/61298)

## Summary

Valid feature request with strong community support (44 upvotes, well above the 20-upvote backlog threshold). Blocked on the upstream DAP protocol change (microsoft/debug-adapter-protocol#81, still open). A community extension exists as a workaround, but the proper solution requires a protocol-level timestamp attribute on OutputEvent.

## Change summary

Show timestamps on debug console output messages, similar to Chrome DevTools' timestamp option.

## Best solution

The upstream DAP issue https://github.com/microsoft/debug-adapter-protocol/issues/81 needs to land first, adding a `timestamp` attribute to `OutputEvent`. Once that ships, VS Code should render the timestamp in the debug console (likely in `src/vs/workbench/contrib/debug/browser/debugConsoleView.ts` and related rendering code), behind a user setting. Until then, the community extension (debug-console-plus) provides a partial workaround.
