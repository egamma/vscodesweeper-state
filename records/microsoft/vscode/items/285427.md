# Issue #285427 — Reference folder task in multi-root workspace

[https://github.com/microsoft/vscode/issues/285427](https://github.com/microsoft/vscode/issues/285427)

## Summary

Valid feature request that reached 20+ upvotes and was promoted to Backlog by the community process. The capability to reference folder-scoped tasks from workspace-level `dependsOn` is confirmed unimplemented on current main — keeping open as an accepted backlog item.

## Change summary

Allow workspace-level tasks to reference folder-specific tasks in `dependsOn` by adding a `folder` qualifier to the task dependency specification in multi-root workspaces.

## Best solution

Extend `ITaskDependency` and `TaskDependency.from` in `src/vs/workbench/contrib/tasks/common/taskConfiguration.ts` to accept an optional `folder` property (matching the workspace folder name), and update the JSON schema in `jsonSchema_v2.ts` to allow `{ label: "...", folder: "..." }` objects in `dependsOn`. The prior closed issues (#185084, #84800) failed the community upvote threshold but this one succeeded — it is the canonical tracker for this work.
