# Issue #229523 — Surface find icon in explorer view toolbar

[https://github.com/microsoft/vscode/issues/229523](https://github.com/microsoft/vscode/issues/229523)

## Summary

The explorer find feature exists but has no toolbar icon for discoverability. The ViewTitle menu on current main shows New File, New Folder, Refresh, and Collapse All — no find/search action. This remains a valid open request with team acknowledgment.

## Change summary

Add a find/search icon to the explorer view toolbar to make the find-in-explorer feature more discoverable.

## Best solution

Register a find action in the ViewTitle menu for the explorer view (in `explorerView.ts`) with the search icon, likely replacing or demoting the Refresh action as discussed. The ExplorerFindProvider infrastructure is already in place.
