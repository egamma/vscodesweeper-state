## Summary

Valid, team-endorsed feature request to add a find/search icon to the Explorer view toolbar. The find functionality exists but lacks a toolbar entry for discoverability. Not yet implemented on main.

## Change summary

Add a find icon to the Explorer view toolbar to make the existing find-in-explorer feature more discoverable.

## Best solution
