# Issue #188594 — [themes] support non-HEX formats in color customizations

[https://github.com/microsoft/vscode/issues/188594](https://github.com/microsoft/vscode/issues/188594)

## Summary

Valid feature request; color customizations in settings.json still only accept HEX/HEXA via `Color.fromHex()`. No support for HSL, RGB, or other CSS color formats has been added on main. Already triaged to Backlog as a feature-request.

## Change summary

Allow non-HEX color formats (HSL, RGB, named colors, etc.) in workbench.colorCustomizations settings.

## Best solution

Extend color parsing in `src/vs/workbench/services/themes/common/colorThemeData.ts` (line 422) and the underlying `Color` class in `src/vs/base/common/color.ts` to accept standard CSS color formats (rgb(), hsl(), named colors) in addition to HEX. This would also benefit token color customizations.
