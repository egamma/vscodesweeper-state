# Issue #78758 — [themes] Show extension info in Preferences: Color Theme panel

[https://github.com/microsoft/vscode/issues/78758](https://github.com/microsoft/vscode/issues/78758)

## Summary

Still a valid feature request. The theme picker now shows a gear button to navigate to the providing extension, but does not display extension name/info inline as requested. The issue is in Backlog milestone and represents a real UX gap.

## Change summary

Show the providing extension's name inline in the Color Theme quick pick so users can identify which extension to uninstall.

## Best solution

In `toEntry()` in `src/vs/workbench/contrib/themes/browser/themes.contribution.ts`, populate the `description` field with the extension display name from `theme.extensionData` for installed (non-builtin) themes, similar to how marketplace themes already show `${ext.displayName} · ${ext.publisherDisplayName}` at line 144.
