# Issue #78758 — [themes] Show extension info in Preferences: Color Theme panel

[https://github.com/microsoft/vscode/issues/78758](https://github.com/microsoft/vscode/issues/78758)

## Summary

Keep open as a valid Backlog feature request in the themes area; the current picker has some extension affordance but still does not visibly show the providing extension next to installed color themes as requested.

## Change summary

Show which extension provides each installed color theme in the Preferences: Color Theme quick pick so users can identify theme extensions to uninstall.

## Best solution

If accepted, update the color theme quick pick in `src/vs/workbench/contrib/themes/browser/themes.contribution.ts` so installed extension-provided themes expose their provider in the item text, description, or detail without losing useful theme descriptions; the existing Extensions-view workaround can remain secondary.
