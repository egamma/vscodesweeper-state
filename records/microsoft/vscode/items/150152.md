# Issue #150152 — Cannot auth if vscode.dev is in iframe

[https://github.com/microsoft/vscode/issues/150152](https://github.com/microsoft/vscode/issues/150152)

## Summary

Feature request to support GitHub authentication when vscode.dev is embedded in an iframe. Team members explicitly asked to keep this open in 2022; low community traction (1 upvote) but a real use case exists (e.g. Nuxt DevTools embedding).

## Change summary

GitHub authentication fails when vscode.dev is loaded inside an iframe because the popup-based auth flow is blocked or interrupted by the embedding context.

## Best solution

The auth flow needs to support iframe-compatible mechanisms (e.g. redirect-based or postMessage-based auth instead of popup windows). This likely belongs in the vscode-dev/authentication layer. The open issue https://github.com/microsoft/vscode-dev/issues/616 may be the canonical tracking item for the embedded auth flow work. A team member should decide whether to consolidate there or keep both.
