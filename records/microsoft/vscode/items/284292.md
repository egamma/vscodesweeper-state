# Issue #284292 — Allow language model providers to influence model ordering in the model selector

[https://github.com/microsoft/vscode/issues/284292](https://github.com/microsoft/vscode/issues/284292)

## Summary

Valid feature request — the model picker currently sorts alphabetically with no mechanism for providers to influence ordering or add tags. No implementation exists on main. Low community traction (1 upvote) so it stays open for the community process.

## Change summary

Allow language model providers to influence model ordering in the picker, or provide tags (trending/featured) for discovery.

## Best solution

Add an optional `sortHint` or `priority` field to `ILanguageModelChatMetadata` (in `src/vs/workbench/contrib/chat/common/languageModels.ts`) and use it as a secondary sort key in `chatModelPicker.ts`. Alternatively, support a `tags` array on metadata and surface them in the picker UI.
