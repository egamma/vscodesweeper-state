# Issue #210672 — Terminal Image display has low resolution

[https://github.com/microsoft/vscode/issues/210672](https://github.com/microsoft/vscode/issues/210672)

## Summary

Correctly triaged as upstream — the root cause is in xterm.js's image addon using CSS pixels instead of device pixels on high-DPI screens. The upstream issue (xtermjs/xterm.js#5035) remains open with no fix landed yet.

## Change summary

Terminal images render at half resolution on high-DPI (Retina) screens because the xterm.js image addon measures in CSS pixels rather than accounting for devicePixelRatio.

## Best solution

Wait for the fix in xterm.js (https://github.com/xtermjs/xterm.js/issues/5035) to land and then update the @xterm/addon-image dependency. No VS Code-side workaround is viable without the upstream change.
