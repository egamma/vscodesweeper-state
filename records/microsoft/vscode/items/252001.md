# Issue #252001 — Allow tasks to be run via terminal suggest

[https://github.com/microsoft/vscode/issues/252001](https://github.com/microsoft/vscode/issues/252001)

## Summary

Valid feature request from a team contributor — tasks in terminal suggest is unimplemented on current main. The associated PR #252002 was closed without merging. Keeping open in Backlog.

## Change summary

Surface configured tasks as completion items in the terminal suggest widget so users can discover and run tasks inline.

## Best solution

Implement a new completion item provider in `src/vs/workbench/contrib/terminalContrib/suggest/` that queries the task service for configured tasks and presents them with appropriate priority and the task icon. The design questions in the issue body (spaces in labels, npm script matching, new completion item kind) still need resolution.
