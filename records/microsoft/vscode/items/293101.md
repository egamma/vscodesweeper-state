# Issue #293101 — Consider `Shift Tab` to toggle between Plan/Agent modes in chat input

[https://github.com/microsoft/vscode/issues/293101](https://github.com/microsoft/vscode/issues/293101)

## Summary

Active design discussion with significant team disagreement on whether Shift+Tab should toggle chat modes. Multiple MEMBER contributors raised accessibility concerns and questioned the UX. The related PR (#293121) was closed without merging. No resolution has been reached — this needs a product decision.

## Change summary

Add a Shift+Tab keyboard shortcut to toggle between Plan and Agent modes in the chat input area.

## Best solution

This needs a product decision among the team. The core tension is discoverability of mode switching (isidorn's argument) vs. accessibility impact of stealing Shift+Tab (meganrogge/TylerLeonhardt's concern) and whether a dedicated toggle makes sense when there are many agents (roblourens's concern). The existing `cmd+.` shortcut already covers mode switching. If pursued, scoping the keybinding with `!screenReaderOptimized` is the minimum accessibility guard, but keyboard-only users would still be affected.
