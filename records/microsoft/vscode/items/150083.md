# Issue #150083 — When Debug: Inline Values is set together with Word wrap, breakpoints are not correctly focused 

[https://github.com/microsoft/vscode/issues/150083](https://github.com/microsoft/vscode/issues/150083)

## Summary

Fixed on main and shipped in 1.90.0. Commit 9aaf087e6bb explicitly preserves the cursor's vertical viewport position when inline value decorations are applied with word wrap enabled, which is exactly the behavior this issue requests.

## Change summary

Debug inline values combined with word wrap push the breakpoint line out of view because the editor reveals the line before decorations are applied.
