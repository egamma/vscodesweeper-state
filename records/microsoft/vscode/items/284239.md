# Issue #284239 — Solid frame around VSCode window

[https://github.com/microsoft/vscode/issues/284239](https://github.com/microsoft/vscode/issues/284239)

## Summary

This is a duplicate of the canonical issue tracking missing/broken window decorations (shadow, border) on Linux GNOME/Wayland with custom title bar style. The root cause is upstream in Electron; newer Electron versions (42+) are bringing improvements (shadows, and 43+ rounded corners).

## Change summary

VS Code shows a solid colored frame/border instead of proper window decorations (shadow) on Linux GNOME with Wayland.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/225073 which tracks the upstream Electron issue for missing window decorations on Wayland with custom title bar. Electron 42+ adds shadows and Electron 43+ adds rounded corners — VS Code will inherit these as it updates Electron.
