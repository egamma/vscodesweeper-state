# Issue #251315 — Support JSON schema draft 2020-12

[https://github.com/microsoft/vscode/issues/251315](https://github.com/microsoft/vscode/issues/251315)

## Summary

Active feature request on the 'On Deck' milestone with team engagement — must remain open per milestone rules. Partial progress was made in 1.106 (bundled 2020-12 meta-schema to avoid $dynamicRef warnings for simple cases), but full draft 2020-12 support remains unimplemented.

## Change summary

VS Code's JSON language service only fully supports JSON Schema draft-07; schemas declaring draft 2020-12 (especially those using $dynamicRef/$dynamicAnchor) produce validation errors and block MCP tool usage.

## Best solution

Implement full JSON Schema 2020-12 vocabulary support in vscode-json-languageservice (tracked upstream at https://github.com/microsoft/vscode-json-languageservice/issues/323 and the parent https://github.com/microsoft/vscode/issues/155379). The 'On Deck' milestone signals team intent to address this.
