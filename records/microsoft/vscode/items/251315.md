# Issue #251315 — Support JSON schema draft 2020-12

[https://github.com/microsoft/vscode/issues/251315](https://github.com/microsoft/vscode/issues/251315)

## Summary

Assigned to the 'On Deck' milestone (a scheduling decision), so this must stay open. The request for full JSON Schema 2020-12 support remains valid — partial improvements shipped in 1.106 but $dynamicRef/$dynamicAnchor are still unsupported, and MCP is adopting 2020-12 as default.

## Change summary

VS Code's JSON validator lacks full support for JSON Schema draft 2020-12, blocking MCP tools and OpenAPI schemas that use newer features like $dynamicRef.

## Best solution

Implement $dynamicRef/$dynamicAnchor support in vscode-json-languageservice (https://github.com/microsoft/vscode-json-languageservice/issues/323 tracks a failing test). The partial fix in 1.106 bundled the meta-schemas but the validator still cannot resolve dynamic references — completing that work unblocks MCP tool schemas and OpenAPI 3.1+ usage.
