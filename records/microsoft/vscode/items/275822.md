# Issue #275822 — Double follow link when hovering link in markdown file

[https://github.com/microsoft/vscode/issues/275822](https://github.com/microsoft/vscode/issues/275822)

## Summary

Still reproduces on main: hovering a link in a markdown file shows duplicate 'Follow link' entries because both the LinkDetector decoration hoverMessage and the markdown language service hover provider contribute the same content independently.

## Change summary

Hovering a link in a markdown file displays two identical 'Follow link (cmd + click)' entries in the hover tooltip.

## Best solution

Deduplicate link hover messages in the MarkdownHoverParticipant, either by detecting when a decoration hoverMessage duplicates a hover-provider result for the same range, or by having the LinkDetector skip its decoration hoverMessage when a registered HoverProvider already covers the same link range.
