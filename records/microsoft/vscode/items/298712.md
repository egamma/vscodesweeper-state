# Issue #298712 — Cannot edit messages in Claude Agent or Copilot CLI chat sessions

[https://github.com/microsoft/vscode/issues/298712](https://github.com/microsoft/vscode/issues/298712)

## Summary

Valid, team-acknowledged feature request for editing sent messages in external agent sessions (Claude Agent, Copilot CLI). Team members confirmed this is on their roadmap and actively refined the issue to separate 'edit' from 'checkpoint/undo' (tracked in #295713). Checkpoint/forking infrastructure has landed on main, but the specific 'edit a previously sent message' UX for external agent sessions remains unimplemented.

## Change summary

Allow users to edit previously sent messages in Claude Agent and Copilot CLI chat sessions.

## Best solution

Implement an edit-message action for external agent chat sessions, building on the checkpoint/forking infrastructure already landed (PR #306968). The chat actions in `src/vs/workbench/contrib/chat/browser/actions/` and the agent host session handler would likely need an edit-and-resubmit flow. Team members are already tracking this alongside the checkpoint API proposal (https://github.com/microsoft/vscode/issues/295713).
