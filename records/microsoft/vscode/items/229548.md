## Summary

Valid feature request from a team member: the drag-and-drop overlay message should be context-aware, showing 'Hold shift to drop into Chat' when hovering over inline chat or chat editors instead of the generic 'Hold shift to drop into editor'. Still unimplemented on main.

## Change summary

Dragging a file over inline chat or chat editor shows the generic 'Hold shift to drop into Editor' message instead of a chat-specific hint.
