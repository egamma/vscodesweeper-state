# Issue #229548 — Dragging a file over Inline Chat or Chat editor should show message `Hold shift to drop into Chat`

[https://github.com/microsoft/vscode/issues/229548](https://github.com/microsoft/vscode/issues/229548)

## Summary

Team-authored feature request still unimplemented on main — the drag-over prompt in editorDropTarget.ts is always 'Hold shift to drop into editor' with no special case for inline chat or chat editors. Keeping open per hard keep-open rule (MEMBER author).

## Change summary

When dragging a file over inline chat or the chat editor, the overlay should say 'Hold shift to drop into Chat' rather than 'Hold shift to drop into Editor'.

## Best solution

Extend editorDropTarget.ts (or introduce a contribution point) so the drop-into-prompt message is context-aware — when the drag target overlaps an inline chat widget or a chat editor, render 'Hold shift to drop into Chat' instead of the generic editor message.
