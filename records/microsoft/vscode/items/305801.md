# Issue #305801 — Sessions: No way to differentiate between sessions windows when you pop out editors

[https://github.com/microsoft/vscode/issues/305801](https://github.com/microsoft/vscode/issues/305801)

## Summary

Team-authored issue on the 'On Deck' milestone. The request for per-session window titles remains unimplemented — the title is still hardcoded to "Agents" on current main.

## Change summary

When sessions are popped out into separate windows, the title bar should reflect the active session's name to differentiate between them.

## Best solution

Update `src/vs/sessions/electron-browser/parts/titlebarPart.ts` (and its browser counterpart) to observe the active session and set `document.title` dynamically to include the session name. The auxiliary window title path likely also needs the same treatment.
