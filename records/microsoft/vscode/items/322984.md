# Issue #322984 — Seeing two blinking cursors

[https://github.com/microsoft/vscode/issues/322984](https://github.com/microsoft/vscode/issues/322984)

## Summary

Team-authored issue reporting a visual bug where two blinking cursors appear in the markdown diff view. The underlying cause is that custom editors cannot easily opt out of the diff view (linked to #138525). Keeping open for maintainer judgment.

## Change summary

Clicking on an edit in a markdown file's diff view shows two blinking cursors simultaneously.

## Best solution

Either allow custom editors to opt out of the diff view (the root cause tracked in https://github.com/microsoft/vscode/issues/138525), or suppress the cursor in one of the two editor panes when the markdown editor is shown in diff mode.
