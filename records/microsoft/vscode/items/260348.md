# Issue #260348 — Differentiate Icons for Copilot Chat and Chat Sessions in Sidebar

[https://github.com/microsoft/vscode/issues/260348](https://github.com/microsoft/vscode/issues/260348)

## Summary

The product has evolved significantly since this was filed. 'Chat Sessions' has been replaced by the Agents window, which is now a separate top-level window surface — not a sidebar view that would share the same activity bar. The original scenario of two identically-iconned panels sitting side-by-side in the same sidebar no longer exists in the current product. The issue can be closed as the underlying UX concern has been resolved by the product redesign.

## Change summary

Differentiate the sidebar icons for Copilot Chat and Chat Sessions so they are visually distinguishable.

## Best solution

The Agents window is now a dedicated top-level surface separate from the workbench sidebar, so the original duplicate-icon-in-sidebar problem no longer applies. If a future layout change reintroduces icon ambiguity, a new issue with updated context would be more actionable.
