# Issue #239006 — Editor GPU: The viewport rendering strategy stops working after > 20k lines

## Summary

This is a known, actively-tracked GPU renderer bug filed by the feature author. The viewport render strategy goes blank after ~20k lines. An open fix PR (#295684) exists but hasn't merged yet — the issue remains valid.

## Change summary

The GPU editor's viewport rendering strategy stops rendering (goes blank) when scrolling past approximately 20,000 lines.
