## Summary

Valid feature request still unimplemented on main. The unicode highlight banner currently only offers a 'disable highlighting' action; no 'highlight all' button has been added. Authored by a team member, so keeping open per policy.

## Change summary

Add a 'highlight all occurrences' button to the unicode character banner so users can act on the warning instead of only dismissing or disabling it.

## Best solution

Add an action to the banner in `src/vs/editor/contrib/unicodeHighlighter/browser/unicodeHighlighter.ts` (around line 120) that triggers highlighting of all flagged characters even when the count exceeds the decoration limit, giving users a path forward beyond disabling.
