# Issue #270381 — Raise or make configurable the hard 200-result cap in Copilot’s grep_search

[https://github.com/microsoft/vscode/issues/270381](https://github.com/microsoft/vscode/issues/270381)

## Summary

The requested feature — making the grep_search 200-result cap configurable — has been implemented on main. Two new settings (`github.copilot.chat.tools.grepSearch.maxResultsCap` and `github.copilot.chat.tools.grepSearch.defaultMaxResults`) were added, allowing users to raise or lower the cap.

## Change summary

Copilot Chat's grep_search tool hard-caps results at 200 with no way for users to raise the limit.

## Best solution

Keep the shipped configurable cap (commit 33dec983b5b). Users can now set `github.copilot.chat.tools.grepSearch.maxResultsCap` to raise or remove the 200-result limit. No further change needed beyond confirming on the latest Insiders build.
