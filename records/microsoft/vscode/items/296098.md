# Issue #296098 — `fetch_webpage` crashes VS code

[https://github.com/microsoft/vscode/issues/296098](https://github.com/microsoft/vscode/issues/296098)

## Summary

Confirmed crash in the Copilot Chat extension's fetch_webpage tool when processing shell script URLs; an open fix PR exists in vscode-copilot-chat — keeping open until it merges.

## Change summary

The fetch_webpage tool crashes the VS Code extension host when fetching URLs that resolve to shell scripts, due to undefined query embeddings during post-processing.

## Best solution

Land the open fix PR (https://github.com/microsoft/vscode-copilot-chat/pull/4019) which guards against undefined query embeddings for non-HTML content types, then close this issue as fixed.
