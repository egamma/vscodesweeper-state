# Issue #235620 — SCM Graph - better handle long branch names in the hover

[https://github.com/microsoft/vscode/issues/235620](https://github.com/microsoft/vscode/issues/235620)

## Summary

Team-authored issue with 'On Deck' milestone — must stay open. The hover for SCM graph branch labels still lacks max-width/overflow handling for individual label spans, so long branch names stretch the hover widget instead of truncating.

## Change summary

Long branch names in the SCM graph hover stretch the tooltip horizontally without truncation or wrapping.

## Best solution

Add a `max-width` with `overflow: hidden; text-overflow: ellipsis` (or word-break) to the label `<span>` elements inside `.history-item-hover-container` so long branch names are truncated in the hover, similar to how inline branch labels already use `max-width: 100px` at line 714 of `scm.css`.
