# Issue #58658 — [folding] go to region command

[https://github.com/microsoft/vscode/issues/58658](https://github.com/microsoft/vscode/issues/58658)

## Summary

This feature request for a 'go to region' command and/or regions in the outline remains unimplemented on main. Extensions exist that address it, but no core support has been added. It sits in Backlog with community interest.

## Change summary

Request for a command to navigate between named code regions (#region/#endregion), and optionally show them in the Outline view.

## Best solution

Either accept this as `*extension-candidate` since multiple extensions now cover it well (Region Helper, Outline Sections), or implement native region display in the outline via the document symbol provider in `src/vs/editor/contrib/folding/`. A team decision is needed on whether regions belong in core outline or remain extension territory.
