Valid upstream issue: the native title bar on Linux ignores the _GTK_THEME_VARIANT X property, remaining light even when the system uses a dark GTK theme. Already correctly labeled as upstream/electron. The related issue #130974 (GNOME dark/light switching via freedesktop portal) was fixed, but this reporter confirmed in Feb 2024 that the _GTK_THEME_VARIANT path is still broken — it's a distinct upstream Electron bug.
