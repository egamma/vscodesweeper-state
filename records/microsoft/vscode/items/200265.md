## Summary

Likely fixed by commit dd5be7a8e52 which prevented an infinite loop in `selectResource` when `item.children` is empty — the exact scenario triggered by uploading a file while a new-file input is pending. Shipped in 1.86.0; reporter was on 1.84.2.

## Change summary

VS Code freezes when uploading a file into a folder where a new-file input box is still pending in the explorer.
