# Issue #264748 — Terminal suggest descriptions are only visible in the flyout

[https://github.com/microsoft/vscode/issues/264748](https://github.com/microsoft/vscode/issues/264748)

## Summary

Team-authored polish issue; the flag descriptions (docstrings from man pages) in terminal suggest are only visible in the flyout detail panel, not inline in the suggestion row. Still unfixed on main — the data flows through `documentation` (flyout-only) rather than `detail` (visible in the row).

## Change summary

Show terminal suggest flag descriptions inline in the suggestion row, not only in the flyout detail panel.

## Best solution

Set the flag docstring as the completion's `detail` (or `label.description`) so it renders in the row's `detailsLabel` element, rather than only populating `documentation` which requires the flyout. The fix likely touches `terminalCompletionItem.ts` and/or the completion providers that populate `ITerminalCompletion` fields.
