# Issue #293826 — Github Copilot Edit mode gone after latest update?

[https://github.com/microsoft/vscode/issues/293826](https://github.com/microsoft/vscode/issues/293826)

## Summary

Active product discussion about removing Edit mode in favor of Agent mode. The team (isidorn) has engaged directly, explaining the rationale and offering workarounds (custom agents, the hidden setting). With 40 upvotes and ongoing community feedback, this stays open as under-discussion.

## Change summary

Edit mode was hidden by default in Copilot Chat, replaced by Agent mode; users want it back or a supported way to replicate its constrained editing behavior.

## Best solution

The team should decide whether to (a) keep the `chat.editMode.hidden` setting long-term as a supported path, (b) ship a built-in custom agent that replicates Edit mode's constrained behavior, or (c) accept that Agent mode fully supersedes Edit. The 40+ upvotes and concrete user scenarios (scoped single-file edits, no permission prompts, lower token usage) suggest a supported lightweight-edit persona is worth preserving even if Agent is the default.
