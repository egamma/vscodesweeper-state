# Issue #210295 — Scrolling breaking semantic highlighting 

[https://github.com/microsoft/vscode/issues/210295](https://github.com/microsoft/vscode/issues/210295)

## Summary

Intermittent viewport semantic token rendering glitch that multiple users have confirmed across different languages. Several fixes have landed in the sparse tokens infrastructure since the last reproduction (Sept 2025), but the intermittent nature makes it impossible to confirm resolution without reporter feedback.

## Change summary

Scrolling up and down causes semantic highlighting to randomly stop rendering for parts of the viewport, with the broken region shifting unpredictably.

## Best solution

Ask the reporter to verify on current Insiders (which includes the Oct 2025 SparseMultilineTokensStorage delta-line fix and the March 2025 invalid-tokens repair). If still reproducible, the next step is adding diagnostic logging to `setPartial` in `src/vs/editor/common/tokens/sparseTokensStore.ts` to capture the token data state when the glitch occurs.
