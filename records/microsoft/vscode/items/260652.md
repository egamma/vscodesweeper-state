# Issue #260652 — [FREQ]  When AI proposing block of code, include comment or pop-up explaining intent

[https://github.com/microsoft/vscode/issues/260652](https://github.com/microsoft/vscode/issues/260652)

## Summary

Feature request for AI code suggestions to include intent explanations. Thinking model support in chat partially addresses this, but the request also covers inline completions and a proposal-refine dialogue UX. Keeping open as it spans multiple surfaces and has had maintainer engagement.

## Change summary

When AI proposes a block of code (inline or in chat), include a comment or pop-up explaining what the suggested code does and why.

## Best solution

The thinking-model streaming (already shipped in chat) addresses the chat side. For inline completions, consider whether a lightweight hover or tooltip showing the model's reasoning for a suggestion is feasible — this likely belongs in the Copilot extension rather than VS Code core. The broader 'proposal-accept/reject/refine' dialogue UX described in comments is a larger design discussion best tracked separately.
