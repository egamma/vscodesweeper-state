## Summary

Team-authored accessibility improvement tracking a default-setting change for high-contrast and screen-reader modes; keeping open for the maintainer to schedule.

## Change summary

Default `accessibility.chat.showCheckmarks` to `true` when a high-contrast theme or screen reader is active.

## Best solution

In the configuration registration for `accessibility.chat.showCheckmarks`, add a default override keyed on `CONTEXT_ACCESSIBILITY_MODE_ENABLED` or a high-contrast theme detection, likely in the chat accessibility configuration area (e.g. `src/vs/workbench/contrib/chat/browser/` or `src/vs/workbench/contrib/accessibility/`).
