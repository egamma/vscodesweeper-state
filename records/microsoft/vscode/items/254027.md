# Issue #254027 — Copilot chat history should support full‑text search across all chat messages, not just conversation titles

[https://github.com/microsoft/vscode/issues/254027](https://github.com/microsoft/vscode/issues/254027)

## Summary

This is the canonical issue for full-text search across Copilot chat history — multiple duplicates (#269155, #271813) were closed pointing here, it has 24 community upvotes, and it sits in Backlog with the `chat` label. No implementation exists on current main. It should stay open as the tracking issue.

## Change summary

Chat history search only matches conversation titles; users want full-text search across all messages to find past code snippets, discussions, and answers.

## Best solution

Implement full-text indexing of chat message content in the chat session store (likely `src/vs/workbench/contrib/chat/common/model/chatSessionStore.ts` and the chat history UI) so the history search box matches against message bodies, not just titles. The open issue https://github.com/microsoft/vscode/issues/309012 (Ctrl+F within chat) substantially overlaps and should be evaluated as a potential duplicate of this canonical issue.
