# Issue #298292 — Chat input area has no independent font size setting

[https://github.com/microsoft/vscode/issues/298292](https://github.com/microsoft/vscode/issues/298292)

## Summary

Valid feature request — the chat input editor has no independent font size (or family/weight) setting on current main. The existing `IChatInputEditorOptions` only carries `backgroundColor` and `accessibilitySupport`; no `chat.inputEditor.fontSize` setting is registered. This is a real accessibility gap worth routing to the chat area.

## Change summary

Add a `chat.inputEditor.fontSize` (and optionally fontFamily/fontWeight) setting to control the font size of the chat input typing area independently from chat codeblocks and chat messages.

## Best solution

Register `chat.inputEditor.fontSize` (and family/weight) settings in `chat.shared.contribution.ts`, extend `IChatInputEditorOptions` in `chatOptions.ts` to carry the new values, and wire them into the chat input editor's Monaco options. The pattern is identical to the existing `chat.editor.fontSize` for result editors. This is a small, self-contained change with accessibility value.
