# Issue #199050 — Unexpected indentation added on new line if a string contains a '{'

[https://github.com/microsoft/vscode/issues/199050](https://github.com/microsoft/vscode/issues/199050)

## Summary

Fixed on main by commit 8d2287f6f7d which removed the naive `{` matching from the PHP increaseIndentPattern. The old pattern matched any `{` not closed on the same line, even inside strings.

## Change summary

PHP files incorrectly indent the next line when the current line contains a `{` inside a string.

## Best solution

Keep the shipped fix (commit 8d2287f6f7d); no further change needed. The `increaseIndentPattern` no longer matches bare `{` characters.
