# Issue #292832 — Github Copilot chat not working

## Summary

This is a support question about Copilot Chat timing out in WSL, self-resolved by the reporter via a WSL2 networking configuration change (`networkingMode=Mirrored`). The root cause is WSL network connectivity, not a VS Code core defect.

## Change summary

Copilot Chat fails to start ('took too long') when running VS Code in WSL with default WSL2 networking mode.

## Best solution

Close as a support question — the issue is a WSL2 network configuration problem, not a VS Code or Copilot extension bug.
