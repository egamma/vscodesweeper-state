# Issue #223918 — code moved feature for the inline view

[https://github.com/microsoft/vscode/issues/223918](https://github.com/microsoft/vscode/issues/223918)

## Summary

Still valid on main: the 'Show Moved Code Blocks' toggle is visible in inline diff view but has no effect because `showMoves` is explicitly gated on `renderSideBySide` — the menu item should be hidden or disabled when inline mode is active.

## Change summary

The 'Show Moved Code Blocks' toolbar button appears in inline diff view but does nothing since moved-block rendering requires side-by-side mode.

## Best solution

Add a `when` clause (e.g. `EditorContextKeys.diffEditorInlineMode.toNegated()`) to the menu registration at `diffEditor.contribution.ts` line 44 so the toggle is hidden in inline mode, or add a `precondition` that disables it. A more ambitious fix would be to implement moved-block indicators for inline view.
