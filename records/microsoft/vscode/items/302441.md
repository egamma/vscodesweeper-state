# Issue #302441 — LLMs not appearing in VS Code 

[https://github.com/microsoft/vscode/issues/302441](https://github.com/microsoft/vscode/issues/302441)

## Summary

The available LLM model list is fetched and displayed by the GitHub Copilot extension, not by VS Code core — this is an extension-owned behavior, so the issue belongs on the Copilot extension's tracker.

## Change summary

Certain LLM models enabled in the user's GitHub organization do not appear in the VS Code Copilot model picker, while they do appear in IntelliJ.

## Best solution

Report this to the GitHub Copilot extension team (https://github.com/community/community or the Copilot feedback channels). The model catalog is entirely managed by the Copilot extension and its backend; VS Code core has no role in which models appear. Updating to the latest Copilot extension version may also resolve it, as model support is added in extension updates.
