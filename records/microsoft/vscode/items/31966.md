# Issue #31966 — [folding] show comment first line for folded sections

[https://github.com/microsoft/vscode/issues/31966](https://github.com/microsoft/vscode/issues/31966)

## Summary

Valid feature request with 28 upvotes, in Backlog. The prerequisite (language-aware folding) is done, but the actual feature — showing the comment's first descriptive line as collapsed text — is not yet implemented on main. Keep open.

## Change summary

When folding a doc-comment block, show the first descriptive line (e.g. 'Print person name on terminal …') instead of the opening delimiter ('/** …').

## Best solution

Extend the folding decoration logic in `src/vs/editor/contrib/folding/browser/foldingDecorations.ts` to optionally extract and display the first meaningful comment line as the collapsed placeholder text, leveraging the language-aware folding model that already identifies comment ranges.
