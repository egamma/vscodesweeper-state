# Issue #228111 — the placeholder container is too narrow, it looks like it allocates space for the vertical scrollbar

[https://github.com/microsoft/vscode/issues/228111](https://github.com/microsoft/vscode/issues/228111)

## Summary

Team-authored polish issue, still present on main. The placeholder width calculation unconditionally subtracts the vertical scrollbar width even when no scrollbar is visible, making the placeholder text container too narrow.

## Change summary

Editor placeholder text container is too narrow because it reserves space for a vertical scrollbar that may not be visible.

## Best solution

In `src/vs/editor/contrib/placeholderText/browser/placeholderTextContribution.ts` line 61, the width calculation should only subtract `verticalScrollbarWidth` when the scrollbar is actually visible (i.e., when content overflows). Alternatively, use `contentWidth` directly since the placeholder only appears in empty editors where there's never a scrollbar.
