# Issue #260391 — Fonts are fuzzy after update to v103 on linux 

[https://github.com/microsoft/vscode/issues/260391](https://github.com/microsoft/vscode/issues/260391)

## Summary

This is a confirmed upstream issue in Chromium's Fontations font stack (which replaced FreeType) that lacks horizontal hinting support. The temporary workaround (disabling FontationsLinuxSystemFonts) was removed in Chromium 142/Electron 39. The issue has an assigned 'On Deck' milestone, so it should stay open awaiting the upstream fix.

## Change summary

Font rendering on Linux became fuzzy/blurry after Chromium switched from FreeType to Fontations, which does not support horizontal hinting.

## Best solution

The fix lives in the upstream Fontations library (https://github.com/googlefonts/fontations/pull/1496 / https://github.com/googlefonts/fontations/issues/1215). Once horizontal hinting lands in Fontations and ships in a Chromium/Electron update, the issue resolves. VS Code should track the upstream progress and pick up the fix when available. No VS Code-side workaround remains since the FreeType fallback was removed.
