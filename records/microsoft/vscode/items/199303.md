[https://github.com/microsoft/vscode/issues/199303](https://github.com/microsoft/vscode/issues/199303)

## Summary

Team-authored issue (hediet, MEMBER) reporting toolbar flicker in the diff editor caused by context-key timing in WorkbenchContextKeysHandler. No evidence of a targeted fix for this specific scenario on main; keeping open per hard keep-open rule for team-authored issues.

## Change summary

Diff editor toolbar flickers when switching focus between editors because context keys update asynchronously after the active editor changes.

## Best solution

Move the `isInDiffEditor` context key handling into the TextDiffEditor pane itself (as benibenj suggested), mirroring the pattern used by SettingsEditor and KeybindingsEditor, so the toolbar visibility doesn't depend on the event-driven WorkbenchContextKeysHandler timing.
