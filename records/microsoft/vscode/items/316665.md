## Change summary

Right-clicking in an unfocused auxiliary window opens the context menu in the focused window instead of the window where the click occurred.

## Best solution

Merge the open PR https://github.com/microsoft/vscode/pull/316678, which adds a `container` field to IContextMenuDelegate and a `targetWindowId` to native menu IPC so the menu anchors to the correct window.
