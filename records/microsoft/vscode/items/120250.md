## Change summary

Add a filter to the Source Control view to show only repositories with modifications, useful for workspaces with many repos.

## Best solution

Implement a filter toggle in the SCM view (likely in `src/vs/workbench/contrib/scm/browser/scmViewPane.ts` and `scmRepositoriesViewPane.ts`) that hides repositories with no pending changes. Alternatively, this could be shipped as an enhancement to the existing SCM Repositories view with a toolbar filter action.
