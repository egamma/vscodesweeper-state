Valid feature request for Copilot to automatically include recent file edits as chat context. No community upvotes after ~10 months. Current implicit context covers the active editor but not an edit-history feed. No canonical duplicate found — keeping open for the community process.

## Change summary

Request for Copilot chat to automatically incorporate recent file changes and edits across the workspace as context, reducing manual context management.

## Best solution

The existing implicit context system (chatImplicitContext.ts) could be extended to track recently edited files and supply them as additional context entries. However, this is a product-direction decision about how much automatic context to feed the model — it touches cost, relevance filtering, and the context-window budget. Best left for the chat-agent area owners to evaluate alongside their context strategy.
