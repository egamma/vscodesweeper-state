# Issue #233122 — Add support for adding `svg` as attachments

[https://github.com/microsoft/vscode/issues/233122](https://github.com/microsoft/vscode/issues/233122)

## Summary

Team-authored feature request to support SVG image attachments in chat. SVG was explicitly excluded from the attachable image MIME types (commit 59990f4108d omitted SVG from drag-and-drop), though partial SVG rendering support exists in `chatAttachmentWidgets.ts`. Keeping open per team-member policy.

## Change summary

Allow SVG files to be attached as image inputs in chat conversations.

## Best solution

Add `svg: 'image/svg+xml'` to `CHAT_ATTACHABLE_IMAGE_MIME_TYPES` in `src/vs/workbench/contrib/chat/common/model/chatModel.ts` and remove the SVG exclusion added in commit 59990f4108d (drag-and-drop filtering). The attachment widget already handles `image/svg` rendering. Security considerations around SVG content (embedded scripts, external references) should be evaluated before enabling.
