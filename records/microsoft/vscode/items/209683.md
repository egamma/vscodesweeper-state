# Issue #209683 — consider how to make `when` clauses more discoverable for keyboard users

[https://github.com/microsoft/vscode/issues/209683](https://github.com/microsoft/vscode/issues/209683)

## Summary

Team-authored accessibility feature request to make context key inspection keyboard-accessible; keeping open for maintainer judgment.

## Change summary

The 'Developer: Inspect Context Keys' command requires a mouse click, making it inaccessible to keyboard-only users who need to discover when-clause context keys.

## Best solution

Add a keyboard-friendly mode to the Inspect Context Keys command that captures context from the previously-focused element before the quick pick steals focus, enabling screen reader and keyboard users to discover when-clause values without a mouse.
