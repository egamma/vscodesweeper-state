[https://github.com/microsoft/vscode/issues/253585](https://github.com/microsoft/vscode/issues/253585)

## Summary

Valid feature request for commit-message-based filtering in the SCM Graph. The existing graph filtering only operates on branch references, not on commit message content. Only 1 upvote and limited traction — keeping open for the community process.

## Change summary

Add an option to filter/hide commits in the Source Control Graph by commit message pattern (e.g., hide merge commits matching specific text).

## Best solution

Add a text-pattern filter to the SCM history graph view (likely in `src/vs/workbench/contrib/scm/browser/scmHistoryViewPane.ts`) that hides commits whose message matches user-defined exclusion patterns. The existing `historyItemsFilter` infrastructure filters by ref, so this would be a new filtering dimension. Low traction so far — let the community process decide priority.
