## Summary

Valid feature request with 28 upvotes in the Backlog. The requested 'show bracket pair colorization only when cursor is adjacent' mode has not been implemented — current options are limited to enabled/disabled and independent color pool per bracket type.

## Change summary

Add a bracket pair colorization mode that only colorizes brackets when the cursor is adjacent to or inside a bracket pair.

## Best solution

Add a new option to `IBracketPairColorizationOptions` (e.g. `showOnlyWhenActive`) in `src/vs/editor/common/config/editorOptions.ts` that restricts colorization to the active bracket pair at the cursor position, combining the bracket-matching highlight behavior with bracket-pair colorization. This would live in the `colorizedBracketPairsDecorationProvider.ts` logic.
