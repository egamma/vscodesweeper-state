# Issue #23099 — Add dropdown for search input history

[https://github.com/microsoft/vscode/issues/23099](https://github.com/microsoft/vscode/issues/23099)

## Summary

The request for a visible dropdown of search history in Find and Search widgets remains unimplemented on current main. History navigation exists only via keyboard (Alt+Up/Down) with a placeholder hint added in PR #129324, but no visual dropdown. With 59 upvotes and Backlog milestone, this stays open.

## Change summary

Add a visual dropdown UI to the Find widget and Search view input fields that displays prior search history entries, rather than requiring blind keyboard navigation.

## Best solution

Extend HistoryInputBox (src/vs/base/browser/ui/inputbox/inputBox.ts) with an optional dropdown button that opens a list/quick-pick of history entries. Wire it into the Find widget (src/vs/editor/contrib/find/browser/findWidget.ts) and the Search view widget (src/vs/workbench/contrib/search/browser/searchWidget.ts), including the 'files to include' and 'files to exclude' pattern fields.
