## Summary

Valid feature request for the agent to retrieve earlier turns from the current session instead of relying only on the rolling summary. No duplicate found — #301174 covers cross-session history, while this targets intra-session context retrieval. Routing to the chat-summarization area for triage.

## Change summary

Allow Copilot Chat to scroll back and read earlier messages within the current session, bypassing the rolling-summary context window.

## Best solution

A 'read session history' tool or equivalent mechanism that lets the model fetch earlier turns from the active session on demand, complementing the existing summarization strategy. This likely lives in the Copilot Chat extension (`vscode-copilot-chat`) rather than VS Code core, as context management is handled by the extension's chat request pipeline.
