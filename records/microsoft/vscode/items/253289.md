# Issue #253289 — MCP Server - uninstalling server leaves input behind

[https://github.com/microsoft/vscode/issues/253289](https://github.com/microsoft/vscode/issues/253289)

## Summary

Still reproduces on main: the `removeMcpServers` method in `mcpResourceScannerService.ts` deletes the server entry but does not clean up associated `inputs` entries. Keeping open — team-authored issue.

## Change summary

Uninstalling an MCP server removes the server entry from mcp.json but leaves orphaned input entries behind.

## Best solution

Extend `removeMcpServers` in `src/vs/platform/mcp/common/mcpResourceScannerService.ts` to identify inputs referenced only by the removed server(s) and delete those inputs from the `inputs` array. Requires matching input IDs used in server config args against the inputs section.
