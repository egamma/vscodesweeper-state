## Summary

The task rerun path has received multiple fixes since this was filed — most notably 40386f82cf8 (general rerun fix, shipped 1.107.0) and 4fe05641998 (carries systemInfo forward on rerun so the platform/args-format is resolved correctly). The root cause — args passed as a string to a Unix shell on rerun — is addressed by ensuring systemInfo propagates correctly.

Terminal fails to launch on 'Rerun Last Task' because shell args are passed as a joined string instead of an array to a Unix shell.
