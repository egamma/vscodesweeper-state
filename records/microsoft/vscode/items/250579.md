Built-in extension translations (e.g. source control view) are missing when connected to a remote server in a non-English locale until the server is manually restarted.

Review open PR https://github.com/microsoft/vscode/pull/251252 to see if it addresses this, or implement the notification approach in the l10n/remote startup path.
