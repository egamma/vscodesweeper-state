# Issue #26360 — Git - Support file stat mode changes

[https://github.com/microsoft/vscode/issues/26360](https://github.com/microsoft/vscode/issues/26360)

## Summary

Still a valid gap on current main: the git extension's diff commands use `--diff-filter=ADMR` which excludes type-change (`T`) status, and there is no UI to surface file mode changes. Keeping open as an unimplemented feature request.

## Change summary

Show file permission/mode changes (e.g. chmod) in the Source Control view instead of showing an empty diff.

## Best solution

Add `T` to the `--diff-filter` flags in `extensions/git/src/git.ts` (lines 1841, 1861, 1912) and surface mode-change information in the Source Control diff view, possibly as a label or description on the change entry. The related closed issue https://github.com/microsoft/vscode/issues/92060 was closed without a fix (just a cross-reference back here), so this remains the canonical tracker.
