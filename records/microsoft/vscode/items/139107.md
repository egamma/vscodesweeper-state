# Issue #139107 — Colorize background like in CoenraadS/Bracket-Pair-Colorizer-2

[https://github.com/microsoft/vscode/issues/139107](https://github.com/microsoft/vscode/issues/139107)

## Summary

Valid, accepted feature request (Backlog + help wanted) for bracket-scope background colorization — not yet implemented on main. The existing color tokens only cover bracket foreground and guide-line backgrounds, not the region between bracket pairs. Should remain open for community contribution.

## Change summary

Add background colorization for the region enclosed by matched bracket pairs, similar to Bracket-Pair-Colorizer-2's scope highlighting.

## Best solution

Implement background decorations in `colorizedBracketPairsDecorationProvider.ts` that apply a semi-transparent background color to the lines between matched bracket pairs, gated behind a new setting and new `editorBracketHighlight.background*` color tokens. Already marked `help wanted` for community contribution.
