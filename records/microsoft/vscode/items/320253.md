# Issue #320253 — run and debug

[https://github.com/microsoft/vscode/issues/320253](https://github.com/microsoft/vscode/issues/320253)

## Summary

The Run and Debug prompt for a Plain Text file is expected because VS Code core does not provide a debugger for arbitrary plain text files; a language/debug extension must contribute that capability.

## Change summary

The reporter expected Run and Debug to work for a file currently recognized as Plain Text because they believe an extension is installed for it.

  > The Run and Debug action depends on a debugger contributed for the current file/language. If this is about a specific extension, please report it to that extension with the file type and expected debugger.
