## Summary

The SCM Graph view exists and covers basic visualization, but the specific requested enhancements (open in editor area, reset/rebase/revert/cherry-pick from graph context menu) remain unimplemented on main. Keeping open as an active feature request with maintainer engagement.

## Change summary

Enhance the built-in Source Control Graph view with git operations (reset, rebase, revert, cherry-pick) and the ability to open it in the editor area.

These belong in `src/vs/workbench/contrib/scm/browser/scmHistoryViewPane.ts` and the git extension's contributed actions. The maintainer (@lszomoru) is actively engaged — leave open for incremental delivery.
