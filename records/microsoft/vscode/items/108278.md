# Issue #108278 — SCM - Diff action buttons are too far from the gutter

[https://github.com/microsoft/vscode/issues/108278](https://github.com/microsoft/vscode/issues/108278)

## Summary

This is the canonical issue for moving SCM diff action buttons closer to the gutter. Three duplicates (https://github.com/microsoft/vscode/issues/221475, https://github.com/microsoft/vscode/issues/231638, https://github.com/microsoft/vscode/issues/250287) were closed pointing here. The feature is not yet implemented on main — the PeekViewWidget still renders the action bar on the right side of the header. Keep open as the active tracker.

## Change summary

Move the SCM dirty diff (now Quick Diff) peek widget action buttons (stage, revert, etc.) to the left side, closer to the gutter, for better ergonomics.

## Best solution

Add a left/right action-bar position option to PeekViewWidget (as joaomoreno and jrieken discussed in the comments), and default the Quick Diff widget to the left. The change would touch `src/vs/editor/contrib/peekView/browser/peekView.ts` (ActionBar placement) and `src/vs/workbench/contrib/scm/browser/quickDiffWidget.ts` (pass the option).
