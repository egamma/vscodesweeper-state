# Issue #247766 — Should ChromeOS be considered Linux (for fonts at least)?

[https://github.com/microsoft/vscode/issues/247766](https://github.com/microsoft/vscode/issues/247766)

## Summary

The font-fallback half of this issue was resolved by the merged PR #248133 (unrecognized OS now defaults to Linux fonts). The broader question — whether ChromeOS should get its own `isChromeOS` platform flag — remains under discussion, with an open PR #309097 proposing exactly that. Keeping open for that unresolved design question.

## Change summary

ChromeOS is not detected as Linux on the web, causing incorrect default font selection and raising the question of whether ChromeOS deserves its own platform flag.

## Best solution

Decide whether to add an `isChromeOS` flag in `src/vs/base/common/platform.ts` (as proposed in PR https://github.com/microsoft/vscode/pull/309097) or to fold ChromeOS into the existing `isLinux` detection by matching `CrOS` in the user agent. The font-fallback issue is already solved; the remaining work is pure platform detection policy.
