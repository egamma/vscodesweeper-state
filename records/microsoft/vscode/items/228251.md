# Issue #228251 — Find in the multi-diff editor is limited to the focused file

[https://github.com/microsoft/vscode/issues/228251](https://github.com/microsoft/vscode/issues/228251)

## Summary

The feature request for cross-file find in the multi-diff editor remains unimplemented on current main. The find widget still operates only on the focused individual diff editor. Keeping open as a valid enhancement.

## Change summary

The find widget in the multi-diff editor should search across all files shown, not just the currently focused one.

## Best solution

Implement a top-level find controller for the multi-diff editor widget (likely in `src/vs/editor/browser/widget/multiDiffEditor/`) that aggregates matches across all document diffs, similar to how notebook find searches across cells. The multi-diff editor workbench contribution (`src/vs/workbench/contrib/multiDiffEditor/browser/multiDiffEditor.ts`) would wire the widget-level find to the workbench find infrastructure.
