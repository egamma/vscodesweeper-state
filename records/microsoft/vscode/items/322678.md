# Issue #322678 — Feature request: allow per-option descriptions in snippet choice placeholders (`${1|...|}`)

[https://github.com/microsoft/vscode/issues/322678](https://github.com/microsoft/vscode/issues/322678)

## Summary

This is a coherent new snippet capability request, not a close candidate. Current snippet choice parsing and choice completion items only carry/display option values, so routing to the editor/snippet area for design discussion is appropriate.

## Change summary

Allow snippet choice placeholder options to include per-option descriptions shown in the choice dropdown.

## Best solution

Decide whether VS Code wants to extend the TextMate-style snippet choice grammar or expose a richer snippet JSON form, then update the snippet parser and snippet choice completion provider to carry an optional option description without breaking existing `${1|one,two|}` snippets.
