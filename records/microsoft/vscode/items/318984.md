# Issue #318984 — VSCode GitHub Copilot Chat Local Agent Mode crashes when MCP server uses JSON Schema draft-2020-12 features

[https://github.com/microsoft/vscode/issues/318984](https://github.com/microsoft/vscode/issues/318984)

## Summary

Valid bug in the Copilot extension's AJV-based tool input validator — plain AJV doesn't support draft-2020-12, causing crashes when MCP servers emit schemas with dynamic/ref features. Routing to copilot/MCP area owners.

## Change summary

Copilot Chat's AJV validator crashes on MCP tool schemas using JSON Schema draft-2020-12 features like $dynamicRef and unevaluatedProperties.

## Best solution

Either upgrade/configure AJV in `extensions/copilot/src/extension/tools/common/toolsService.ts` to handle draft-2020-12 schemas, or extend the error-handling path so that partially-compiled schemas never produce a crash (the try/catch at line 344 already handles full compilation failures, but partial compilation leading to `wrapper0.validate is not a function` is not caught).
