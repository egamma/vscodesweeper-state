# Issue #276111 — Missing tool embeddings

[https://github.com/microsoft/vscode/issues/276111](https://github.com/microsoft/vscode/issues/276111)

## Summary

Team-authored issue reporting 404s when fetching tool embeddings from the CDN (embeddings.vscode-cdn.net). The fetching code exists in the copilot extension's embeddings platform layer and gracefully handles the 404, but the missing CDN resources mean tool embeddings won't function. Keeping open per team-authored policy.

## Change summary

Tool embeddings fail to load because the CDN returns 404 for the tools/latest.txt and tools/core.json endpoints at the text-3-small/v1.106 path.

## Best solution

Ensure the CDN pipeline publishes tool embeddings for each release version. The fetch logic in `extensions/copilot/src/platform/embeddings/common/embeddingsIndex.ts` handles missing data gracefully, so the fix is likely in the build/publish infrastructure that populates embeddings.vscode-cdn.net rather than in source code.
