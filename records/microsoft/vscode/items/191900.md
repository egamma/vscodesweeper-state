# Issue #191900 — Git - "Create a branch from..." does not offer migrate

[https://github.com/microsoft/vscode/issues/191900](https://github.com/microsoft/vscode/issues/191900)

## Summary

Still reproduces on main: the `_branch` method in the git extension does not catch `DirtyWorkTree` errors or offer stash/migrate options, unlike the checkout flow. An open PR (#304986) already addresses this — it should be reviewed/landed rather than the issue closed.

## Change summary

Creating a branch via 'Create Branch From…' with a dirty worktree fails without offering to stash or migrate changes, unlike the checkout flow.

## Best solution

Review and land the open PR https://github.com/microsoft/vscode/pull/304986, which adds the stash/migrate handling to `_branch()` in `extensions/git/src/commands.ts`, mirroring the existing pattern in the checkout error handler (lines ~2934–2977).
