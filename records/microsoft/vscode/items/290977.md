# Issue #290977 — Easier way to clean up leftover terminals from chat sessions

[https://github.com/microsoft/vscode/issues/290977](https://github.com/microsoft/vscode/issues/290977)

## Summary

Team-authored feature request for a kill/trash button in the hidden-terminals picker to streamline bulk cleanup of chat-spawned terminals. Must stay open — author is a team member.

## Change summary

Add a trash/kill action to the hidden-terminals quick-pick so users can dispose of leftover chat terminals without opening each one individually.

## Best solution

Add a delete/kill inline button to each item in the hidden-terminals quick-pick (and possibly a 'kill all' action). The terminal quick-pick infrastructure lives around `src/vs/workbench/contrib/terminal/browser/` and the chat-terminal integration in `src/vs/workbench/contrib/chat/browser/chatTerminal*`. This is a quality-of-life improvement for the agent terminal workflow that the team should scope and implement.
