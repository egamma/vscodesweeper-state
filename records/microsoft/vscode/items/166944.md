# Issue #166944 — [css] Please add an option to switch to using line-by-line comments

[https://github.com/microsoft/vscode/issues/166944](https://github.com/microsoft/vscode/issues/166944)

## Summary

Valid feature request promoted to Backlog with 22 upvotes. No implementation found on main. The request for a CSS line-by-line comment option remains unaddressed in core, though a workaround (multi-cursor) and a third-party extension exist.

## Change summary

Request an option to toggle CSS multi-line comments as per-line block comments instead of a single wrapping block comment, for Chrome DevTools compatibility.

## Best solution

Add a setting (or leverage the existing language configuration `lineComment` field) in the CSS/LESS/SCSS language configuration to allow per-line `/* */` commenting when multiple lines are selected. Alternatively, accept the existing extension (Toggle Line Comments) as the solution and close as `*extension-candidate` — but given backlog promotion and upvote count, implementing in core seems warranted.
