# Issue #112376 — [json] Relative path wrongly resolved

[https://github.com/microsoft/vscode/issues/112376](https://github.com/microsoft/vscode/issues/112376)

## Summary

This is an upstream bug in vscode-json-languageservice's $ref resolution logic. The current bundled version (5.4.4) shows improved parentHandle passing in resolveRefs, but without a definitive fixing commit or reproduction test against current main, the issue should stay open for verification.

## Change summary

Relative $ref paths in JSON schemas are resolved against the wrong base URI when schemas reference other schemas with their own relative $ref links.
