# Issue #112376 — [json] Relative path wrongly resolved

[https://github.com/microsoft/vscode/issues/112376](https://github.com/microsoft/vscode/issues/112376)

## Summary

Confirmed bug in vscode-json-languageservice's $ref relative path resolution. Team member reproduced it and identified the root cause in the upstream package. No evidence of a fix shipped yet.

## Change summary

JSON $ref relative paths are resolved against the wrong base URI when schemas are merged, causing incorrect file lookups.

## Best solution

Fix the relative path resolution timing in `jsonSchemaService.ts` in microsoft/vscode-json-languageservice — specifically around the `merge`/`resolveRefs` interaction where merged $refs are incorrectly resolved from the root schema's URI instead of the referencing schema's URI.
