# Issue #320781 — selectable sort order for copilot chat sessions

[https://github.com/microsoft/vscode/issues/320781](https://github.com/microsoft/vscode/issues/320781)

## Summary

Valid feature request for an ascending (oldest-first) sort option in the chat sessions list. The current implementation (`AgentSessionsSorter`) hardcodes descending order (`timeB - timeA`). No existing setting or option provides ascending sort.

## Change summary

Add a setting or UI toggle to sort Copilot Chat sessions chronologically (oldest first) instead of the current fixed reverse-chronological (newest first) order.

## Best solution

Add an ascending/descending direction option to the existing `AgentSessionsSorting` mechanism in `src/vs/workbench/contrib/chat/browser/agentSessions/agentSessionsFilter.ts` and update the `AgentSessionsSorter.compare()` in `agentSessionsViewer.ts` to respect it. This is a small, localized enhancement to the existing sort infrastructure.
