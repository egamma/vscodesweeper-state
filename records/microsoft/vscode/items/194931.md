# Issue #194931 — Find and Replace in closed notebook search

[https://github.com/microsoft/vscode/issues/194931](https://github.com/microsoft/vscode/issues/194931)

## Summary

Team-authored feature request on the 'On Deck' milestone — two independent hard keep-open rules apply. The request for find-and-replace in closed (unopened) notebooks remains unimplemented and is actively being investigated by team members.

## Change summary

Enable find-and-replace to work on notebook files that are not currently open in the editor, via the global search sidebar.

## Best solution

Implement closed-notebook replace by resolving a NotebookTextModel from the full serialized text returned by the extension host, applying replacements, and saving via ext host APIs — as outlined in the author's comment. The escape-sequence/string-matching discrepancy between open and closed notebook search (noted by Yoyokrazy) should be addressed as part of or prior to this work.
