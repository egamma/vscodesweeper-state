# Issue #194931 — Find and Replace in closed notebook search

[https://github.com/microsoft/vscode/issues/194931](https://github.com/microsoft/vscode/issues/194931)

## Summary

Team-authored feature request (MEMBER) for find-and-replace in closed/unopened notebooks via the search viewlet — must keep open per policy. The feature is still unimplemented; comments outline a concrete implementation plan.

## Change summary

Enable find-and-replace (not just find) for notebook files that are not currently open in the editor, via the global search viewlet.

## Best solution

Implement replace support for closed notebooks by resolving a full NotebookTextModel from extension-host search results, performing the replacement edits on the model, and saving via ext-host APIs — as outlined in the MEMBER's implementation sketch in comment 1. The work lives in the notebook search contribution area (src/vs/workbench/contrib/search/ and src/vs/workbench/contrib/notebook/).
