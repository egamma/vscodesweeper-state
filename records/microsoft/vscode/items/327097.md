# Issue #327097 — Feature Request: Allow filtering/hiding chat sessions by specific agent/provider

[https://github.com/microsoft/vscode/issues/327097](https://github.com/microsoft/vscode/issues/327097)

## Summary

The requested capability already exists: the Agent Sessions view has a filter submenu that lets you toggle provider visibility — excluding specific providers' sessions from the merged list. It's a UI filter toggle rather than a settings.json key, but delivers the same result.

## Change summary

Allow hiding chat sessions from a specific agent/provider in the merged session list.

## Best solution

The existing filter menu in the Agent Sessions view (implemented in `agentSessionsFilter.ts`) already supports excluding sessions by provider type. The reporter can use the filter icon/submenu in the session list to deselect the unwanted provider (e.g. Codex). No code change needed.
