# Issue #299101 — Like/Dislike buttons

[https://github.com/microsoft/vscode/issues/299101](https://github.com/microsoft/vscode/issues/299101)

## Summary

Valid polish request: the chat vote buttons (thumbs up/down) use only a CSS background toggle for their active state, which is too subtle in many themes. The filled icon variants (`thumbsupFilled`/`thumbsdownFilled`) exist in the codicon library but aren't used. This is a polish item, not a bug — the buttons function correctly.

## Change summary

Chat thumbs-up/thumbs-down vote buttons lack visually distinct active states — the toggled appearance is too subtle to tell at a glance whether feedback has been given.

## Best solution

Switch the `toggled` property on both `MarkHelpfulAction` and `MarkUnhelpfulAction` from a bare `ContextKeyExpr` to a `{ condition, icon }` object, using `Codicon.thumbsupFilled` and `Codicon.thumbsdownFilled` respectively, so the toggled state swaps to a filled icon. The infrastructure for toggled icons already exists in `menuEntryActionViewItem.ts`.
