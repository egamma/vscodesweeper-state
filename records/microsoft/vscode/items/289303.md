## Summary

This is a transient Copilot service-side error (HTTP 400 with 'invalid_tool_call_format') where the model (Claude Haiku 4.5) generated malformed JSON in tool call arguments. The root cause is in the Copilot backend/model, not in VS Code core — the extension surfaced the error correctly. The Copilot extension version (0.36.1) is also significantly outdated.
