# Issue #275212 — Thinking section expand from 5 lines to infinite lines does not work

[https://github.com/microsoft/vscode/issues/275212](https://github.com/microsoft/vscode/issues/275212)

## Summary

The thinking section UX has been significantly reworked since this issue was filed. There are now three configurable display modes via `chat.agent.thinkingStyle` (collapsed, collapsedPreview, fixedScrolling), with `fixedScrolling` as the default. The original feedback about the 5-line expand behavior is largely addressed, but the issue is labeled `under-discussion` and may still have residual UX feedback value — keeping open for the team to decide if the current defaults satisfy the concern.

## Change summary

The thinking section's expand from 5 lines to infinite feels off; a 1-2 line collapsed preview with an expand-to-full toggle would be a better default.

## Best solution

A team member should evaluate whether the current `chat.agent.thinkingStyle` options (especially `collapsed` and `collapsedPreview`) fully address this feedback. If so, close as resolved. If the specific '1-2 line preview' behavior is still missing, refine the `collapsedPreview` mode or document the resolution.
