# Issue #313192 — Folding Strategy: Brackets

[https://github.com/microsoft/vscode/issues/313192](https://github.com/microsoft/vscode/issues/313192)

## Summary

This is a genuine feature request for a new 'brackets' folding strategy option. The current codebase only supports 'auto' and 'indentation' — no bracket-based folding strategy exists on main. Keeping open for area triage.

## Change summary

Add a 'brackets' option to `editor.foldingStrategy` that uses the bracket-pair colorizer to compute folding ranges.

## Best solution

Route to the editor-folding area for product evaluation. The implementation would add a 'brackets' enum value in `src/vs/editor/common/config/editorOptions.ts` and a corresponding `BracketFoldingRangeProvider` in `src/vs/editor/contrib/folding/browser/`. Needs maintainer judgment on whether this is desirable given the existing bracket-pair infrastructure.
