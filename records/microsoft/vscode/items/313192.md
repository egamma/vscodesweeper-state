# Issue #313192 — Folding Strategy: Brackets

[https://github.com/microsoft/vscode/issues/313192](https://github.com/microsoft/vscode/issues/313192)

## Summary

This is a still-open feature request for a new folding strategy, not a close candidate. Current main still exposes only `auto` and `indentation`, so it should be classified and routed to the editor area for product/implementation consideration.

## Change summary

Add an `editor.foldingStrategy` option that computes folding ranges from bracket pair parsing, possibly combined with indentation folding.

## Best solution

The editor/folding owners should decide whether bracket-pair parsing should become a first-class folding range source in core, likely around the folding controller/range provider path, or whether this should remain extension-owned via folding range providers.
