# Issue #279145 — System setup without background update errors out when restarting the OS

[https://github.com/microsoft/vscode/issues/279145](https://github.com/microsoft/vscode/issues/279145)

## Summary

Team-authored bug report about the Windows system setup erroring out when an OS shutdown is attempted while a non-background update is in progress. The issue is still open with no fix found on main; keeping open per team-member policy.

## Change summary

System setup (with background updates disabled) errors when the user triggers an OS shutdown during an update, because Inno Setup blocks the shutdown and the resulting cancellation/interruption surfaces an error dialog.

## Best solution

Handle the OS shutdown signal (WM_QUERYENDSESSION / session-ending scenario) gracefully in the Win32 update service so that an in-progress Inno Setup install either completes cleanly or is cancelled without surfacing an error dialog. The fix likely lives in `src/vs/platform/update/electron-main/updateService.win32.ts`, around the `sessionend` flag and the Inno spawn/cancel logic.
