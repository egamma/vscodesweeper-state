# Issue #267030 — Copilot opens and adds to new sidebar chat instead of attaching files to aux window chat

[https://github.com/microsoft/vscode/issues/267030](https://github.com/microsoft/vscode/issues/267030)

## Summary

Fixed on main: the 'Add File to Chat' action previously used `showChatView()` which only targeted the sidebar, ignoring aux window chat. Commit e9cb9f3de35 replaced it with `chatWidgetService.revealWidget()` which reveals the last-focused widget regardless of location (aux window, editor, quick chat). Shipped in 1.107.0.

## Change summary

'Add File to Chat' from the explorer context menu opens a new sidebar chat instead of attaching files to the currently active chat in an auxiliary window.

## Best solution

Keep the shipped fix — `revealWidget()` now properly targets whichever chat surface was last focused, including aux windows. No further change needed; users on 1.107.0+ should no longer see this.
