# Issue #71712 — [folding] Distinctly highlight corresponding `#endregion` for `#region X` comments

[https://github.com/microsoft/vscode/issues/71712](https://github.com/microsoft/vscode/issues/71712)

## Summary

Legitimate feature request for bracket-match-style highlighting between #region/#endregion pairs. Not implemented on main — the existing foldingHighlight option only highlights collapsed ranges, not the corresponding endpoint marker. Keeping open as an unimplemented enhancement in the Backlog.

## Change summary

Highlight the matching #endregion line when the cursor is on a #region marker (similar to bracket pair matching).

## Best solution

Extend the folding contribution (src/vs/editor/contrib/folding/browser/) to detect when the cursor is on a region marker line and apply a bracket-match-style decoration to the corresponding end marker. This could reuse the bracket-pair highlight infrastructure or register its own decoration provider keyed to the cursor position within a folding range of kind 'region'.
