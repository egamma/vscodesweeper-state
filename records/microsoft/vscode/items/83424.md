# Issue #83424 — Add custom QuickPick highlighting

[https://github.com/microsoft/vscode/issues/83424](https://github.com/microsoft/vscode/issues/83424)

## Summary

Still a valid, unimplemented feature request. The internal QuickInput API already has IQuickPickItemHighlights support, but it is not exposed in the public extension API (vscode.d.ts) nor as a proposed API. A team member (TylerLeonhardt) expressed willingness to consider a PR in Oct 2021.

## Change summary

Allow extensions to provide custom highlight ranges on QuickPickItem label, description, and detail fields.

## Best solution

Expose the existing internal `IQuickPickItemHighlights` pattern as a proposed extension API on `QuickPickItem` (similar to `TreeItemLabel.highlights`), letting extensions supply `[number, number][]` ranges for label/description/detail when they handle their own filtering.
