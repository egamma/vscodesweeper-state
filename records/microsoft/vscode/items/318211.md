# Issue #318211 — [Insiders] BYOK not showing reasoning tokens in chat

## Summary

Fixed on main and shipped in 1.123.0. The reporter was on 1.122.0-insider, which predated the fix that added support for `reasoning_content` and `reasoning` fields in BYOK streaming responses.

## Change summary

BYOK models streaming reasoning tokens via `delta.reasoning` or `delta.reasoning_content` did not display thinking text in Copilot Chat.

## Best solution

Keep the shipped fix (commits a21f814c + f6bf90fd + c2228335, landed in 1.123.0). No further change needed — the SSE processor now reads `reasoning_content` and `reasoning` fields from streaming deltas and surfaces them as thinking content.
