# Issue #322157 — Freeze while trying to commit file

[https://github.com/microsoft/vscode/issues/322157](https://github.com/microsoft/vscode/issues/322157)

## Summary

Verified duplicate of #321312 — the freeze is triggered by native OS dialogs (commit confirmation, delete confirmation, Help > About, etc.) on CachyOS/KDE Linux, traced to an upstream Electron bug (electron/electron#51988). Both reporters here are on CachyOS, matching the exact pattern.

## Change summary

VS Code freezes on CachyOS Linux when any native confirmation dialog is shown, including the git commit dialog.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/321312, which tracks the upstream Electron bug (electron/electron#51988). The workaround is setting `"window.dialogStyle": "custom"`. The permanent fix will come from the Electron/Chromium side.
