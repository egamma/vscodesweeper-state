# Issue #301537 — GitHub Copilot Chat activity "staging" (adding) un-versioned files to GIT - or - Copilot Chat Auto-Stages ~2000 Files on Every Interaction

[https://github.com/microsoft/vscode/issues/301537](https://github.com/microsoft/vscode/issues/301537)

## Summary

Fixed on main and shipped: commit 15499ba9afa removed the automatic `git add` call in `handleRequestCompleted` for folder sessions, directly addressing the reported auto-staging behavior.

## Change summary

Copilot Chat automatically staged all untracked files (~2000) in the workspace within seconds of any chat prompt.

## Best solution

Keep the shipped fix (1.116.0); no further change needed. The `gitService.add()` call that ran after every request completion in non-agent-sessions workspaces has been removed.
