# Issue #43026 — Provide ability to ignore all whitespace in diff editor (feature request)

[https://github.com/microsoft/vscode/issues/43026](https://github.com/microsoft/vscode/issues/43026)

## Summary

Still unimplemented on main — only `ignoreTrimWhitespace` (leading/trailing) exists; no `ignoreAllWhitespace` option has been added. With 633 upvotes and Backlog milestone, this is a well-established community request that should remain open.

## Change summary

Add a diff editor option to ignore all whitespace differences (not just leading/trailing), equivalent to `git diff -w`.

## Best solution

Implement a new `diffEditor.ignoreAllWhitespace` option in `src/vs/editor/common/config/` and wire it through the diff computation pipeline (`ILinesDiffComputerOptions`) so that lines differing only in internal whitespace are treated as equal. The diff algorithm in `src/vs/editor/common/diff/defaultLinesDiffComputer/` would need a mode that normalizes all whitespace before comparison.
