# Issue #310707 — search highlight barely visible

[https://github.com/microsoft/vscode/issues/310707](https://github.com/microsoft/vscode/issues/310707)

## Summary

The barely-visible search highlight is a known consequence of the new 2026 default dark theme (shipped in 1.116.0) which uses low-opacity blue highlight colors. This is a duplicate of #319262 which reports the same symptom specifically for the minimap (and this issue's commenter confirms the minimap aspect).

## Change summary

Editor search highlights became barely visible after the 2026 default dark theme was activated in v1.116.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/319262 which tracks the same minimap/editor search highlight visibility regression in the 2026 dark theme. The fix is to adjust `editor.findMatchHighlightBackground` and `editorOverviewRuler.findMatchForeground` in `extensions/theme-defaults/themes/2026-dark.json` for better contrast.
