# Issue #250437 — Suggested context looks like attached context

[https://github.com/microsoft/vscode/issues/250437](https://github.com/microsoft/vscode/issues/250437)

## Summary

Team-authored polish request to make suggested (implicit) context attachments visually distinct from real attachments. The dashed border + italic styling exists on main but team consensus is it needs stronger differentiation. Keeping open per team-authored rule.

## Change summary

Suggested context attachments in the chat input look too similar to actually attached context, confusing users about what is included.

## Best solution

Increase visual separation between suggested and attached context — e.g. wider dash spacing, different background, or relocating suggested items. The styling lives in `src/vs/workbench/contrib/chat/browser/widget/media/chat.css` (`.implicit.disabled` rules around line 2368) and the widget in `src/vs/workbench/contrib/chat/browser/attachments/implicitContextAttachment.ts`.
