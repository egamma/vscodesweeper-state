# Issue #256329 — GitHub Copilot Chat diff UI does not wrap deleted (red) lines, as it does added (green) lines, when word wrap is enabled

[https://github.com/microsoft/vscode/issues/256329](https://github.com/microsoft/vscode/issues/256329)

## Summary

Still reproduces on main: the inline diff editor explicitly disables word wrap on the original (left/red) side. An open PR #292094 targets this exact file but has not been merged yet.

## Change summary

In inline diff mode (used by chat agent editing), deleted (red) lines do not wrap even when word wrap is enabled, because the original-side editor explicitly sets wordWrapOverride to 'off'.

## Best solution

Merge PR https://github.com/microsoft/vscode/pull/292094 after review — it modifies `src/vs/editor/browser/widget/diffEditor/components/diffEditorEditors.ts` to allow word wrap on the original side in inline mode. The fix at line 142 should conditionally enable wrapping rather than always disabling it.
