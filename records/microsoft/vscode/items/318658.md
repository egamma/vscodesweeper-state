# Issue #318658 — Add maximize button for the code browser window and the terminal window, too

[https://github.com/microsoft/vscode/issues/318658](https://github.com/microsoft/vscode/issues/318658)

## Summary

The requested maximize functionality already exists for both the editor (tab area) and the terminal/panel. VS Code ships `workbench.action.toggleMaximizeEditorGroup` and `workbench.action.toggleMaximizedPanel` with UI buttons and keybindings — this covers the central ask.

## Change summary

Request to add maximize buttons to the editor tabs area and terminal panel, similar to the Copilot chat maximize button.

## Best solution

The capability already ships: `workbench.action.toggleMaximizeEditorGroup` maximizes an editor group, and `workbench.action.toggleMaximizedPanel` maximizes the panel (terminal/output). The reporter can bind these to a shortcut or find them in the panel/editor title context menus. No code change needed.

- **Proposed comment:**

  > Thanks for the suggestion! This functionality actually already exists in VS Code today:
  > 
  > - **Editor/tabs area**: Use the command `View: Toggle Maximize Editor Group` (`workbench.action.toggleMaximizeEditorGroup`). You can also right-click an editor tab → "Maximize Group", or double-click the tab bar (configurable).
  > - **Terminal/panel area**: Use `View: Toggle Maximized Panel` (`workbench.action.toggleMaximizedPanel`). There's also a maximize icon button in the panel's title bar.
  > 
  > Both of these can be assigned to custom keyboard shortcuts via the Keyboard Shortcuts editor. Since the requested capability is already shipped, I'm closing this as already covered by existing functionality.
