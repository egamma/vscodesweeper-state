[https://github.com/microsoft/vscode/issues/291972](https://github.com/microsoft/vscode/issues/291972)

## Summary

Team-authored accessibility feature request to add a command for copying the most recent chat response text with fewer keystrokes. Keeping open per team-authored policy.

## Change summary

Add a dedicated command to copy the most recent chat response text, reducing the current 3-keystroke flow for one-handed developers.

## Best solution

Add a registered command (e.g. in the chat contribution) that reads the last assistant response from the chat model and copies it to the clipboard, bindable to a single keybinding. The existing copy action in the chat response context view can serve as a reference implementation.
