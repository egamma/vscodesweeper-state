# Issue #292795 — "Invalid String Length" in Copilot Chat

[https://github.com/microsoft/vscode/issues/292795](https://github.com/microsoft/vscode/issues/292795)

## Summary

Fixed on main and shipped in 1.123.0. The chat session serialization now catches V8's max-string-length RangeError and retries with a truncating replacer, preserving the session instead of losing it.

## Change summary

Chat sessions are lost with 'Invalid String Length' error when serialization exceeds V8's ~512 MiB string limit.

## Best solution

Keep the shipped fix (commit eb308be4f9d, enhanced by 7cc4a4c5e49 and e266aa425b9) — sessions with oversized content are now gracefully truncated rather than lost. No further change needed.
