# Issue #165161 — [Accessibility] Open Folder dialog controls do not have accelerator keys

[https://github.com/microsoft/vscode/issues/165161](https://github.com/microsoft/vscode/issues/165161)

## Summary

Partially addressed by merged PR #272554 (Select Folder button now has a hotkey on Windows), but the remaining controls (folder name field, folder location treeview) are an upstream Electron bug (electron/electron#48631). Milestone is 'On Deck', which is a scheduling milestone — keeping open per policy.

## Change summary

The native Open Folder dialog on Windows lacks accelerator keys on its controls, unlike the Open File dialog.

## Best solution

The Select Folder button hotkey shipped via https://github.com/microsoft/vscode/commit/5156ca62bfb. The remaining missing accelerators (folder name field, treeview) depend on Electron fixing https://github.com/electron/electron/issues/48631. Keep tracking here until the upstream fix lands and VS Code picks up the new Electron version.
