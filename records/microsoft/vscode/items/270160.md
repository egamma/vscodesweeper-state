## Summary

Valid feature request in Backlog with 20+ upvotes. The code at line 307-309 of computeAutomaticInstructions.ts still prepends `**/` to patterns that don't start with `/` or `**/`, confirming the partial-match behavior remains. The fix would be in `src/vs/workbench/contrib/chat/common/promptSyntax/computeAutomaticInstructions.ts` in the `_matches` method, ensuring the pattern is matched against workspace-relative paths rather than full URI paths when the root-relative syntax is used.
