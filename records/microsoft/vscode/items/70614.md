# Issue #70614 — [html] embedded java script grammar doesn't end because of missing semicolon

[https://github.com/microsoft/vscode/issues/70614](https://github.com/microsoft/vscode/issues/70614)

## Summary

This is a duplicate of the canonical tracking issue https://github.com/microsoft/vscode/issues/20488, which covers the broader TextMate limitation where embedded JS grammars can corrupt the syntax highlighting of the rest of an HTML document. A team member explicitly linked it in comment 6.

## Change summary

Missing semicolons in inline `<script>` blocks cause the JS grammar to consume `<` from subsequent HTML tags, breaking syntax highlighting and comment toggling.
