# Issue #306812 — White flash when reloading window & using Dark theme

[https://github.com/microsoft/vscode/issues/306812](https://github.com/microsoft/vscode/issues/306812)

## Summary

Still reproduces on main. The BrowserWindow background color is set at creation time via `getBackgroundColor()` and Electron doesn't support changing it post-creation. When the system theme switches light→dark and the window reloads, the stale white background flashes. Maintainer aeschli confirmed the limitation but no fix has landed.

## Change summary

Window flashes white on reload after switching from a light to dark system theme, because the Electron BrowserWindow background color is fixed at creation time.

## Best solution

Recreate the BrowserWindow (or destroy and reopen) on theme change to pick up the new background color, or explore Electron's newer APIs (if any) for post-creation background changes. Alternatively, a workaround in the renderer (CSS transition or overlay) could mask the flash. The relevant code is in `src/vs/platform/theme/electron-main/themeMainServiceImpl.ts` (`getBackgroundColor`) and `src/vs/platform/windows/electron-main/windows.ts` (window creation options).
