# Issue #312948 — Chat title shows in all capital letter

[https://github.com/microsoft/vscode/issues/312948](https://github.com/microsoft/vscode/issues/312948)

## Summary

The chat view title is styled with `text-transform: uppercase` via intentional CSS in `chatViewTitleControl.css` and `chatViewPane.css`. This is a deliberate design choice for section-header-style view titles, consistent with other view pane headers in the workbench.

## Change summary

The auto-generated chat title in the Chat view pane header appears in all capital letters due to CSS text-transform.

## Best solution

The uppercase styling on `.chat-view-title-label` (in `src/vs/workbench/contrib/chat/browser/widgetHosts/viewPane/media/chatViewTitleControl.css:37`) and `.agent-sessions-title-container` (in `chatViewPane.css:117`) is an intentional design pattern for view pane section headers. The sessions app already made a deliberate choice to preserve casing in its tab strip (PR #311942), but the workbench chat view title uses uppercase as a design convention. If the team wants to revisit this, removing `text-transform: uppercase` from these selectors would preserve the original title casing.
