# Issue #300053 — Add "Move To -> Editor Area" to Integrated Terminal Panel Context Menu

[https://github.com/microsoft/vscode/issues/300053](https://github.com/microsoft/vscode/issues/300053)

## Summary

Valid discoverability request. "Move to Editor Area" already exists on the terminal tab context menu (TerminalTabContext) but is absent from the xterm surface context menu (TerminalInstanceContext) and the panel-level "Move To" submenu. The user acknowledged finding it on the tab menu but still requests improved discoverability.

## Change summary

Add a "Move to Editor Area" option to the terminal xterm surface context menu or the panel-level "Move To" submenu for better discoverability.

## Best solution

Add a `TerminalCommandId.MoveToEditor` entry to `MenuId.TerminalInstanceContext` in `src/vs/workbench/contrib/terminal/browser/terminalMenus.ts`, mirroring the existing entry on `MenuId.TerminalTabContext` (line 569–578). This is a small, localized menu registration change.
