## Summary

Valid UX polish request for the SCM Graph: always show a subtle background highlight on the current (HEAD) commit row, rather than only distinguishing it by an empty circle glyph. Already correctly labeled and triaged to Backlog; no duplicate found.

## Change summary

Make the current commit in the SCM Graph always have a subtle background color highlight for easier identification, rather than relying solely on the hollow circle indicator.

## Best solution

Add a persistent background highlight (or a configurable theme color) to the current history item row in the SCM Graph tree. The `kind: 'HEAD'` distinction already exists in `src/vs/workbench/contrib/scm/browser/scmHistory.ts` (line 308); the renderer could apply a CSS class for that row. This is a small polish change in the SCM history view pane renderer.
