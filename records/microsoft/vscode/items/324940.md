# Issue #324940 — No way to understand the symbol meaning

[https://github.com/microsoft/vscode/issues/324940](https://github.com/microsoft/vscode/issues/324940)

## Summary

Valid polish issue: the pre-release bookmark icon (Codicon.versions) rendered by PreReleaseBookmarkWidget has no tooltip explaining its meaning, unlike other badge icons. The icon visually resembles stacked windows, confusing users who don't know it indicates a pre-release extension.

## Change summary

The pre-release bookmark icon on extensions in the extensions view has no tooltip, and when both recommendation (star) and pre-release icons apply, only the pre-release icon is shown.

## Best solution

Add a tooltip to the PreReleaseBookmarkWidget in `src/vs/workbench/contrib/extensions/browser/extensionsWidgets.ts` (around line 454) explaining 'This extension has a pre-release version', similar to how other badges provide tooltips. Optionally allow both the recommendation star and pre-release badge to coexist or document the intended precedence.
