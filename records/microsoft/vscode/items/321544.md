# Issue #321544 — Remove built-in Copilot code review from SCM view and code selection

[https://github.com/microsoft/vscode/issues/321544](https://github.com/microsoft/vscode/issues/321544)

## Summary

Team-authored issue tracking a planned deprecation of the non-agentic Copilot Code Review (CCR) surfaces in favor of the agentic `/review` command. Must stay open for maintainer judgment.

## Change summary

Remove the built-in non-agentic Copilot code review from the SCM view (staged/unstaged/uncommitted changes) and from the editor code-selection surface, replacing both with the agentic `/review` command via the Copilot SDK.

## Best solution

The review infrastructure lives in `extensions/copilot/src/platform/review/` (service, command interfaces) and `extensions/copilot/src/extension/review/` (doReview, githubReviewAgent, PR comments provider). The work involves removing or gating the non-agentic CCR code paths and wiring the existing UI entry points to the agentic `/review` slash command instead. The cross-reference to the IntelliJ repo confirms this is a coordinated multi-IDE deprecation.
