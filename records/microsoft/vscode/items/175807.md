## Summary

Valid feature request still unimplemented. The `srcset` attribute is not recognized for path completion in the upstream vscode-html-languageservice; no commits or code changes address this on current main.

## Change summary

Path auto-completion does not trigger inside the `srcset` attribute value, unlike `src`.

## Best solution

Add `srcset` to the list of path-completable attributes in the upstream `microsoft/vscode-html-languageservice` package (likely in the path completion participant that maps tag/attribute pairs to path suggestions).
