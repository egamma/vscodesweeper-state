# Issue #73840 — New revealProblems property in task presentation options is missing from the API

[https://github.com/microsoft/vscode/issues/73840](https://github.com/microsoft/vscode/issues/73840)

## Summary

Still valid: `revealProblems` exists internally and in `tasks.json` but has never been added to the extension API's `TaskPresentationOptions` in `vscode.d.ts`. The gap remains on current main.

## Change summary

Expose the `revealProblems` property in the `TaskPresentationOptions` extension API so extensions can control problems-panel visibility when providing tasks.

## Best solution

Add `revealProblems?: TaskRevealKind` (or a new `TaskRevealProblemsKind` enum) to `TaskPresentationOptions` in `src/vscode-dts/vscode.d.ts` and wire it through the ext-host task conversion layer so the internal `RevealProblemKind` is honored when tasks are contributed via the API.
