## Summary

The core ask (native window controls with custom titlebar on Linux) is shipped via WCO support (commit d0857f8ec96, August 2024). Remaining gaps — button layout position (left/right), which buttons appear per GNOME config, and spacing/styling fidelity — depend on Electron's WCO implementation upstream. Keeping open as upstream; the issue is correctly labeled.
