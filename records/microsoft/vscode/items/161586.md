# Issue #161586 — Close, minimize and maximize buttons missing on macOS after using the restart to update option

[https://github.com/microsoft/vscode/issues/161586](https://github.com/microsoft/vscode/issues/161586)

## Summary

The exact scenario described — window controls missing after a failed fullscreen restore on macOS — was addressed by commit 1a37acbdf9c which adds a 10-second timeout to detect stalled fullscreen transitions and falls back to leaving fullscreen so window controls reappear. Team members were already unable to reproduce before the fix landed; this can be closed as completed.

## Change summary

Close/minimize/maximize buttons disappear on macOS when VS Code restores windows after an update, specifically when a fullscreen transition fails.

## Best solution

Keep the shipped fix (commit 1a37acbdf9c, 1.87.0). The timeout-based fallback ensures window controls are restored when native macOS fullscreen transition fails. No further change needed.
