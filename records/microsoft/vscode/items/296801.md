# Issue #296801 — Extensions: MCP Servers view becomes empty when applying sort

[https://github.com/microsoft/vscode/issues/296801](https://github.com/microsoft/vscode/issues/296801)

## Summary

Still reproduces on main: the `query()` method in mcpServersView.ts only strips `@mcp` but passes `@sort:…` as literal gallery search text, yielding empty results. The linked PR #304985 was closed without merging. Issue has 'On Deck' milestone so keeping open.

## Change summary

Applying a @sort qualifier in the MCP Servers view empties the list because the sort token is passed as a literal search string to the gallery query.

## Best solution

Strip the `@sort:(\w+)(-\w*)?` qualifier from the query text before calling `queryGallery` in `src/vs/workbench/contrib/mcp/browser/mcpServersView.ts` line 339, mirroring the existing pattern in `extensionsViews.ts`.
