# Issue #297338 — Ctrl+scroll wheel zoom jumps around

[https://github.com/microsoft/vscode/issues/297338](https://github.com/microsoft/vscode/issues/297338)

## Summary

Keep open: the report has concrete reproduction steps, reporter-provided screen recordings, and a second user seeing similar Logitech mouse behavior on macOS. I did not find evidence in current main that this has been fixed or superseded.

## Change summary

Ctrl/Cmd plus mouse-wheel editor font zoom can also move the editor scroll position, eventually jumping thousands of lines.

## Best solution

Reproduce with a physical Logitech-style wheel on Windows/macOS and fix the editor mouse-wheel zoom path in src/vs/editor/browser/controller/mouseHandler.ts so zoom gestures are consumed without applying scroll movement, keeping the viewport anchored near the cursor.
