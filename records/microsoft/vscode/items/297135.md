# Issue #297135 — Feature Request: Show monthly progress indicator in 'Premium requests' progress bar

[https://github.com/microsoft/vscode/issues/297135](https://github.com/microsoft/vscode/issues/297135)

## Summary

Valid feature request to show a monthly billing-period progress indicator alongside the premium requests quota bar. The current quota dashboard (chatStatusDashboard.ts) shows quota usage percentage and a reset date, but no visual indicator of how far through the billing cycle the user is. Not yet implemented on main. Low community traction (1 upvote, 144 days old); keeping open for the community process.

## Change summary

Add a visual indicator showing how far through the current billing month the user is, alongside the premium requests usage progress bar in the Copilot quota dashboard.

## Best solution

Enhance the quota indicator in `src/vs/workbench/contrib/chat/browser/chatStatus/chatStatusDashboard.ts` (the `createQuotaIndicator` method) to compute and display elapsed billing-period progress using the existing `resetDate` data. This would help users pace their premium request usage. However, this is a polish/UX enhancement with low traction — it can remain in the backlog until demand grows or a broader quota-UX redesign occurs.
