# Issue #29155 — [folding] Allow to show amount of lines folded

[https://github.com/microsoft/vscode/issues/29155](https://github.com/microsoft/vscode/issues/29155)

## Summary

This feature request to show the count of folded lines in the fold placeholder remains unimplemented on current main. The folding decoration still renders a static ellipsis character with no line count. It's in the Backlog milestone with the `feature-request` label, so it should stay open.

## Change summary

Show the number of folded lines (e.g. '12 lines') in the fold placeholder instead of just an ellipsis.

## Best solution

Add an editor option (e.g. `editor.showFoldedLineCount`) that, when enabled, renders the folded line count alongside the ellipsis in `src/vs/editor/contrib/folding/browser/foldingDecorations.ts`. This would require dynamic `after` content per fold region rather than the current static CSS `content` approach.
