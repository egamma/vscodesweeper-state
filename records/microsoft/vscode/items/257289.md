## Summary

Team-authored feature request for better task scoping and pre-flight checks in the Copilot Coding Agent workflow — must remain open for maintainer judgment.

## Change summary

CCA tasks need better pre-flight scoping: interactive back-and-forth to shape requests to appropriate size, and access/capability checks before the agent begins work.

## Best solution

Implement a pre-flight phase for CCA tasks that (1) interactively helps the user scope a request to a model-appropriate size, and (2) verifies the agent has the necessary access (e.g. remote DB connectivity) before committing to execution. This aligns with the agents window and agent host workstreams.
