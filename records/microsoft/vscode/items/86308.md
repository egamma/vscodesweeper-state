# Issue #86308 — [themes] Minimal icon theme should use icon font

[https://github.com/microsoft/vscode/issues/86308](https://github.com/microsoft/vscode/issues/86308)

## Summary

Still open: the minimal icon theme (vs_minimal-icon-theme.json) continues to use SVG iconPaths for all icons. No conversion to icon font has landed. Author is a MEMBER, so this requires maintainer judgment.

## Change summary

The built-in minimal file icon theme should use the codicon icon font instead of SVG files to get proper LCD text rendering on adjacent labels.

## Best solution

Convert the icon definitions in `extensions/theme-defaults/fileicons/vs_minimal-icon-theme.json` from `iconPath` (SVG) references to `fontCharacter`/`fontId` entries using the codicon font, once the file-icon-theme schema fully supports font references (tracked in the dependency chain noted in comments).
