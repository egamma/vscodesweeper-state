# Issue #152518 — Find/Search inputs: Allow setting input font, like in SCM.

[https://github.com/microsoft/vscode/issues/152518](https://github.com/microsoft/vscode/issues/152518)

## Summary

Valid feature request with 28 upvotes, promoted to Backlog. No implementation exists on main for search input font settings. Related but distinct from #101064 (find widget) and #519 (workbench font). Keeping open as a legitimate backlog item.

## Change summary

Allow users to configure font family and size for the search view input fields, similar to the existing `scm.inputFontFamily` setting.

## Best solution

Implement a `search.inputFontFamily` (and optionally `search.inputFontSize`) setting following the `scm.inputFontFamily` pattern in `src/vs/workbench/contrib/search/`. Coordinate with #101064 for the editor find widget, as comment #5 suggests they should be addressed together.
