# Issue #152518 — Find/Search inputs: Allow setting input font, like in SCM.

[https://github.com/microsoft/vscode/issues/152518](https://github.com/microsoft/vscode/issues/152518)

## Summary

Valid, community-promoted feature request (28 upvotes) that remains unimplemented on main. The SCM input has `scm.inputFontFamily` but no equivalent exists for search or find inputs. Team members acknowledged the request and a contributor expressed interest but no PR was submitted.

## Change summary

Allow users to configure the font family and size of search panel inputs and the editor find widget, analogous to the existing `scm.inputFontFamily` setting.

## Best solution

Add `search.inputFontFamily` (and optionally `search.inputFontSize`) settings following the `scm.inputFontFamily` pattern in `src/vs/workbench/contrib/scm/browser/scm.contribution.ts` and `scmInput.ts`. Coordinate with https://github.com/microsoft/vscode/issues/101064 for the editor find widget equivalent so both land together or in a consistent manner.
