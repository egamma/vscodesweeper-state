# Issue #300269 — expose open terminal UI even when terminal command is collapsed

[https://github.com/microsoft/vscode/issues/300269](https://github.com/microsoft/vscode/issues/300269)

## Summary

Team-authored feature request for surfacing a quick-access button to open a terminal directly from a collapsed tool-call result in chat, rather than requiring multiple expand clicks. Keeping open — author is a team member, and the request is under active discussion.

## Change summary

When an agent runs a terminal command in chat, the user must expand multiple collapsed sections to reach the 'open terminal' icon; a more prominent affordance is requested.

## Best solution

Surface a small 'Open Terminal' button or icon inline on the collapsed tool-call summary row (near the terminal command name) so users can jump to the terminal output in one click without expanding thinking steps. The chat-terminal area and the chat tool-call rendering in `src/vs/workbench/contrib/chat/` are the likely implementation surface.
