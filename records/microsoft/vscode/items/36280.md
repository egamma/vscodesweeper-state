# Issue #36280 — [html] JSON edit support inside HTML document / script tag with type="application/json"

[https://github.com/microsoft/vscode/issues/36280](https://github.com/microsoft/vscode/issues/36280)

## Summary

Still unimplemented on main — the HTML grammar does not embed JSON for script tags with type="application/json". A community extension exists, but the core feature request remains valid. Milestone is Backlog with moderate community interest.

## Change summary

Request JSON syntax highlighting and language service support inside HTML <script type="application/json"> tags.

## Best solution

Either adopt the upstream HTML grammar change (https://github.com/textmate/html.tmbundle/issues/103) to recognize application/json script tags and wire JSON language services in the HTML language server, or formally classify this as best served by the existing community extension (vscode-json-script-tag). The HTML language server in extensions/html-language-features would need mode-switching logic similar to how it handles embedded CSS/JS.
