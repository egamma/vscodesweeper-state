# Issue #170399 — Scroll speed doubled when screen scaling is enabled

[https://github.com/microsoft/vscode/issues/170399](https://github.com/microsoft/vscode/issues/170399)

## Summary

Already fixed on main: wheelDelta-based scrolling now accounts for device pixel ratio on affected Chrome/Electron versions, which matches this Linux scaling=2 report. This can be closed as completed rather than as a duplicate, because the concrete fix is identifiable in the checkout.

## Change summary

Mouse wheel scrolling in the editor moved twice as far when Linux display scaling was set to 2.

## Best solution

Treat the issue as fixed by commit 4f338c87ab68833847df18ddcb8a5f5d7ee667fc, with the follow-up guard in ef78ec4e6f6715c444bd54ff6bab576a43c8b257; no further editor-scrollbar work is needed unless it still reproduces on a current build.
