# Issue #229643 — SCM Graph - Unsure how to tell when graph is locked to a repository

[https://github.com/microsoft/vscode/issues/229643](https://github.com/microsoft/vscode/issues/229643)

## Summary

Team-authored UX polish issue — the SCM Graph repository picker does not visually indicate 'auto' mode the way the branch/ref picker does. The inconsistency is still present on current main. Keeping open per team-author rule.

## Change summary

The SCM Graph repository picker shows the repository name but has no visual indicator distinguishing 'auto' mode from a manually locked repository selection, unlike the branch picker which shows 'Auto' text.

## Best solution

Update `SCMRepositoryActionViewItem` in `src/vs/workbench/contrib/scm/browser/scmHistoryViewPane.ts` to visually indicate when the repository selection is in 'auto' mode — e.g. showing 'Auto' text or a suffix like the branch picker does, or adding a distinguishing icon/badge. The branch picker's pattern (showing 'Auto' label when `_historyItemsFilter === 'auto'`) is the natural model to follow.
