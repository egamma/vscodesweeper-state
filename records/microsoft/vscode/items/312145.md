# Issue #312145 — Remote SSH: Copilot/agent plugin discovery fails with PendingMigrationError: navigator is now a global in nodejs on VS Code 1.117.0

[https://github.com/microsoft/vscode/issues/312145](https://github.com/microsoft/vscode/issues/312145)

## Summary

Verified duplicate of the canonical tracking issue #252077, which owns the navigator-global migration work for Node.js 22+ in the extension host. This issue reports a specific symptom (Copilot plugin discovery failure) of that same underlying problem.

## Change summary

Remote SSH extension host throws PendingMigrationError when the bundled Copilot extension accesses `navigator`, breaking agent plugin discovery.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/252077, which is the team-owned tracking issue (assigned, labeled `debt`/`remote`/`ssh`) for the navigator-global migration. The fix there — either opting Copilot into `supportGlobalNavigator` or updating Copilot's code to not probe `navigator` — resolves this symptom.
