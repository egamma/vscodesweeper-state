# Issue #247014 — Feature Request: Allow extension to adjust sidebar width with vscode extensions api

[https://github.com/microsoft/vscode/issues/247014](https://github.com/microsoft/vscode/issues/247014)

## Summary

Legitimate feature request with strong community support (45 upvotes), already triaged into Backlog with correct labels. No proposed or existing extension API for controlling sidebar/layout dimensions exists on current main.

## Change summary

Allow extensions to programmatically adjust sidebar width and other layout dimensions via a new VS Code extension API.

## Best solution

Design and ship a proposed extension API surface for layout dimension control (e.g. sidebar width), likely via `vscode.window` or a new `vscode.layout` namespace, with appropriate user consent. The implementation would touch `src/vs/workbench/api/` (ext host shape + main thread relay) and `src/vs/workbench/browser/layout/` (the actual resize logic). Area owners for `api` and `layout` should co-design.
