# Issue #169081 — Git - Add the way to display the number of additions and removal of lines 

[https://github.com/microsoft/vscode/issues/169081](https://github.com/microsoft/vscode/issues/169081)

## Summary

Valid feature request to show per-file insertion/deletion counts in the SCM changed-files list. VS Code now shows diff stats in the SCM history graph hover (commit level), but not per-file in the working changes view. Only 3 upvotes and no comments in 2.5 years — low traction but a reasonable ask; keeping open for the community process.

## Change summary

Display per-file line addition and deletion counts (e.g. +10 −5) alongside the A/R/M/D status letters in the Source Control changed-files list.

## Best solution

This would require extending `ISCMResourceDecorations` or the SCM resource model to carry per-file diff statistics, then surfacing them in the SCM view pane renderers. The git extension already computes `shortStat` at the commit level (`extensions/git/src/hover.ts`); per-file stats would need a similar mechanism. Given the low community traction, this is a reasonable backlog item but not urgent — it could also be explored as an extension leveraging the SCM API.
