# Issue #265795 — Please Fix Shared Input and History Across Multiple Copilot Chat Sessions

[https://github.com/microsoft/vscode/issues/265795](https://github.com/microsoft/vscode/issues/265795)

## Summary

Both reported problems — shared input text across chat sessions and shared up/down history — have been fixed on main by scoping chat input history per session, and shipped in 1.124.0.

## Change summary

Input typed in one Copilot chat session appeared in all sessions, and arrow-key history navigation showed prompts from other sessions.

## Best solution

Keep the shipped fix (commit 1faec7cd38c, 1.124.0). The ChatWidgetHistoryService now keys history per session via a `historyKey`, and ChatHistoryNavigator filters by that key. No further change needed.
