# Issue #318632 — Agents Window - Import Session

[https://github.com/microsoft/vscode/issues/318632](https://github.com/microsoft/vscode/issues/318632)

## Summary

Team-authored feature request for importing existing chat sessions into the Agents Window. Current main has a one-time auto-migration and an 'Open in Agents Window' action scoped to CopilotCLI/AgentHostCopilot session types, but no general-purpose import mechanism. Keeping open per team-author policy.

## Change summary

Allow users to import or transfer any existing local agent chat session into the Agents Window on demand.

## Best solution

Extend the local chat sessions provider or add an explicit 'Import Session' action that lets users pick from their full local chat history and bring sessions into the Agents Window, beyond the current one-time migration and the session-type-restricted 'Open in Agents Window' command.
