# Issue #318632 — Agents Window - Import Session

[https://github.com/microsoft/vscode/issues/318632](https://github.com/microsoft/vscode/issues/318632)

## Summary

Team-authored feature request for importing existing local chat sessions into the Agents window. An 'Open in Agents Window' action exists for active sessions, but full import of historical local sessions appears to be a remaining gap. Keep open per team-member rule.

## Change summary

Allow importing existing local agent chat sessions into the Agents window so users can consolidate multi-window workflows.

## Best solution

Extend the local chat sessions provider (`src/vs/sessions/contrib/providers/localChatSessions/`) to surface historical sessions from the workbench's chat history, or add an explicit import action in the Agents window that can load persisted session resources from the workbench's chat storage.
