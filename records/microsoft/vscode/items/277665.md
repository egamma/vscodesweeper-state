# Issue #277665 — Require manual accepting of Copilot Edits

[https://github.com/microsoft/vscode/issues/277665](https://github.com/microsoft/vscode/issues/277665)

## Summary

The requested capability is now implemented on main: VS Code has both a per-session permissions picker (Default Approvals / Bypass Approvals / Autopilot) and a granular `chat.tools.edits.autoApprove` setting for pattern-based control. Users can easily switch between manual and auto-approval of edits.

## Change summary

Request for a toggle between manual and auto-acceptance of Copilot edits, similar to Claude Code CLI's behavior.

## Best solution

Keep the shipped behavior: the `chat.permissions.default` setting and per-session permissions picker provide exactly the requested workflow — easily switching between requiring manual approval and auto-accepting edits.
