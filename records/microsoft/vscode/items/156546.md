# Issue #156546 — Allow to resolve save conflicts using the merge editor

[https://github.com/microsoft/vscode/issues/156546](https://github.com/microsoft/vscode/issues/156546)

## Summary

Team-authored feature request (hard keep-open). The merge editor integration for save conflicts has not been implemented on main — the save error handler still uses the diff editor exclusively. Related issue #156915 was closed as a duplicate of this one.

## Change summary

Use the merge editor (with no base) instead of the diff editor when resolving file-system save conflicts.

## Best solution

Implement an opt-in setting (disabled by default) that routes save-conflict resolution through the merge editor in `src/vs/workbench/contrib/files/browser/editors/textFileSaveErrorHandler.ts`. hediet (comment 4) expressed willingness to implement; the approach of a setting behind which to experiment was agreed upon.
