[https://github.com/microsoft/vscode/issues/267530](https://github.com/microsoft/vscode/issues/267530)

## Summary

The root dependency is still glob@^5.0.13 (resolves to 5.0.15) and multiple transitive deps pull glob v7/v8. Recent commits (d352c0a, 57e0f40) did partial cleanup but the deprecation warnings persist — real debt remains.

## Change summary

Upgrade deprecated glob versions (v5/v7/v8) across the dependency tree to remove npm deprecation warnings.

## Best solution

Bump the root glob dependency from ^5 to a supported version (v10+ or v11), update direct consumers (rimraf, asar, mocha, gulp-json-editor, @vscode/gulp-electron) to versions that no longer pull deprecated glob, and use pnpm overrides for any remaining transitive holdouts. The copilot extension already uses glob@^11.
