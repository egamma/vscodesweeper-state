# Issue #26856 — See git diff of two files in horizontal layout

[https://github.com/microsoft/vscode/issues/26856](https://github.com/microsoft/vscode/issues/26856)

## Summary

Valid, popular feature request (81 upvotes, well above the 20-upvote Backlog threshold) for a top/bottom diff layout — not yet implemented on main. The diff editor still only supports side-by-side (left/right) or inline view; no horizontal/stacked layout exists.

## Change summary

Allow the diff editor to display the two file versions stacked vertically (top/bottom) rather than only side-by-side (left/right).

## Best solution

Implement a top/bottom layout mode in the diff editor widget (likely in `src/vs/editor/browser/widget/diffEditor/`) with a user-facing toggle, benefiting vertical-monitor users and wide-code scenarios. The `renderSideBySide` option could be extended to a tri-state or a separate orientation setting.
