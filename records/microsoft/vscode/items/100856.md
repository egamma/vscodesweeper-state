# Issue #100856 — Extension doesn't activate after install

[https://github.com/microsoft/vscode/issues/100856](https://github.com/microsoft/vscode/issues/100856)

## Summary

Team-authored feature request for better UX when extension activation fails. The current code only logs telemetry on activation error with no user-facing notification or Running Extensions view indication; the gap remains on main.

## Change summary

Show a clear UI message and Running Extensions view indication when an extension fails to activate, rather than silently failing.

## Best solution

Add a user-visible notification (e.g. via INotificationService) in `_onDidActivateExtensionError` and surface the activation-failed state in the Running Extensions view so users understand why commands from a failed extension are 'not found'.
