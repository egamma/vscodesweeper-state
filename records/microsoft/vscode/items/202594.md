Keep open as an editor clipboard bug: current code is meant to write editor selections to Linux PRIMARY, but the report describes intermittent middle-click fallback to the regular clipboard and has a later Wayland confirmation. Middle-click paste on Linux should use the last selected text from the primary selection buffer, not the regular Ctrl+C clipboard contents.
