# Issue #282578 — Tweak terminal suggest setting to be off by default for certain problematic configs

[https://github.com/microsoft/vscode/issues/282578](https://github.com/microsoft/vscode/issues/282578)

## Summary

Team-authored feature request to make terminal suggest smarter by defaulting off for problematic shell configs like powerlevel10k. Keeping open per team-authored policy.

## Change summary

Add an `auto` mode for the terminal suggest setting that disables suggestions when incompatible shell configurations (e.g. powerlevel10k) are detected.

## Best solution

Implement a three-state setting (`auto`/`on`/`off`) for terminal suggest, with `auto` detecting known-problematic configs like powerlevel10k and disabling suggest accordingly. Likely lives in `src/vs/workbench/contrib/terminalContrib/suggest/`.
