# Issue #124895 — Can't drag scroll bars anymore

[https://github.com/microsoft/vscode/issues/124895](https://github.com/microsoft/vscode/issues/124895)

## Summary

Upstream Chromium bug affecting scrollbar drag on certain Linux window managers (Fluxbox, IceWM, KDE Plasma). The issue is already labeled `upstream-issue-fixed`, VS Code now ships Electron 42 (Chromium ~134) — far past the affected Chromium 89–102 — and there have been no user reports since October 2022. This can be closed as resolved through upstream adoption.

## Change summary

Scrollbar click-and-drag on Linux with certain window managers (Fluxbox, IceWM, KDE Plasma) selects text instead of scrolling, due to an upstream Chromium bug with mouse event handling.

## Best solution

Close as resolved — the upstream Chromium fix has been adopted through successive Electron upgrades (now at Electron 42/Chromium ~134). No VS Code-side change is needed. The `upstream-issue-fixed` label already reflects the team's assessment.
