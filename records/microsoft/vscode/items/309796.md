# Issue #309796 — Terminal approval UI is missing slashes (on windows)

[https://github.com/microsoft/vscode/issues/309796](https://github.com/microsoft/vscode/issues/309796)

## Summary

Team-authored bug with an assigned milestone; the root cause remains on main — the regex at line 564 of chatToolConfirmationCarouselPart.ts still strips backslashes from Windows paths in the confirmation carousel title.

## Change summary

Terminal approval UI strips backslash path separators on Windows because the markdown-stripping regex removes all backslash characters.

## Best solution

Fix the regex `.replace(/[\\*_#>]/g, '')` in `chatToolConfirmationCarouselPart.ts` to not strip backslashes (they are path separators on Windows, not just markdown escapes). A fixture was already added (PR #315226) to validate the fix.
