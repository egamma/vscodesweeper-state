# Issue #310211 — Add an option to keep multiple remote repo accounts signed

[https://github.com/microsoft/vscode/issues/310211](https://github.com/microsoft/vscode/issues/310211)

## Summary

Verified duplicate of #256635, which requested exactly this (multiple GitHub accounts with easy switching) and has been implemented, verified, and shipped. The GitHub authentication provider already sets `supportsMultipleAccounts: true`, and account preferences allow per-extension account selection.

## Change summary

Allow users to stay signed into multiple GitHub accounts simultaneously and select which account to use per repo.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/256635 — the feature is already implemented and shipped. Users can sign into multiple GitHub accounts and use the Account Preferences UI to control which account each extension uses.
