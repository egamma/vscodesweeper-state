# Issue #191285 — [css] Support "speak" property

[https://github.com/microsoft/vscode/issues/191285](https://github.com/microsoft/vscode/issues/191285)

## Summary

Still valid: the `speak` property is not in the MDN-sourced CSS data that vscode-css-languageservice bundles, so it is still flagged as unknown. The maintainer noted this is waiting on MDN adoption. Keeping open as a tracked feature request.

## Change summary

The CSS `speak` property (CSS Speech Module Level 1) is flagged as unknown in VS Code's CSS/SCSS language support.

## Best solution

The property should be added upstream in https://github.com/microsoft/vscode-custom-data (which extracts from MDN). Once MDN lists `speak` as a recognized property with sufficient browser support, the custom-data repo will pick it up and it will flow into vscode-css-languageservice. Alternatively, the vscode-custom-data repo could manually add it ahead of MDN if the W3C spec is considered stable enough.
