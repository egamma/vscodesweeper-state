# Issue #191285 — [css] Support "speak" property

[https://github.com/microsoft/vscode/issues/191285](https://github.com/microsoft/vscode/issues/191285)

## Summary

This is an upstream data dependency — the `speak` property must first appear in MDN's CSS data before it flows into vscode-custom-data and the CSS language service. The maintainer acknowledged this in 2023 and the issue remains valid as a tracking item.

## Change summary

The CSS/SCSS language service flags the `speak` property as unknown because it is not yet in MDN's CSS property data.

## Best solution

Track until MDN adds `speak` to their CSS properties data (https://github.com/mdn/data). Once present, the vscode-custom-data repo will pick it up and the warning will resolve. In the meantime, users can suppress it via `css.lint.validProperties`.
