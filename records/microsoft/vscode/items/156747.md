# Issue #156747 — Allow Word Seperator Selection

[https://github.com/microsoft/vscode/issues/156747](https://github.com/microsoft/vscode/issues/156747)

## Summary

Implemented on current main: double-click word selection now includes separator/punctuation tokens, which addresses the reported inability to select `=` without changing `editor.wordSeparators`.

## Change summary

Allow users to double-click a word separator such as `=` and select that separator without changing normal word selection boundaries.

## Best solution

Treat this as completed by commit 6055fcf8d8b9256741e181238d0f0f8caa3f97cc; the editor should keep selecting separator tokens directly on double-click while preserving `editor.wordSeparators` for word navigation and operations.
