# Issue #281828 — Allow Secondary Side Bar to be on the same side as primary side bar

## Summary

Valid feature request with 40 upvotes, promoted to Backlog, with an open community PR (#320701) that implements the feature. A team member (bpasero) expressed interest for the chat agent window exploration. Keep open pending PR review.

## Change summary

Allow the secondary side bar to be placed on the same side as the primary side bar instead of always being on the opposite side.

## Best solution

Review and land the open PR https://github.com/microsoft/vscode/pull/320701, which adds a `workbench.secondarySideBar.location` setting with `opposite`/`left`/`right` values. The PR preserves existing behavior by defaulting to `opposite`.
