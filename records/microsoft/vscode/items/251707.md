# Issue #251707 — Cannot install VSCode Server

[https://github.com/microsoft/vscode/issues/251707](https://github.com/microsoft/vscode/issues/251707)

## Summary

The standalone VS Code Server installer (via `aka.ms/install-vscode-server/setup.sh`) was intentionally deprecated and replaced by Remote Tunnels (`code tunnel`). The blob storage URL no longer resolving is by design, not a bug.

## Change summary

The standalone VS Code Server setup script fails because the download URL `vscodeserverlauncher.blob.core.windows.net` no longer resolves.

## Best solution

Close as designed. The standalone VS Code Server (`code serve-web`) was intentionally retired in favor of Remote Tunnels. Users should use `code tunnel` or the Remote - Tunnels extension instead.
