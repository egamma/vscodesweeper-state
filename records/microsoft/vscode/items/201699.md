## Summary

Confirmed upstream Electron crash on macOS when quitting with auxiliary (floating) windows open. The crash is in Electron Framework's Objective-C messaging during app termination — not in VS Code's own code. The issue is assigned to a team member and properly labeled upstream/electron; it should stay open until Electron resolves the underlying native crash or a workaround lands.

## Change summary

VS Code crashes (SIGSEGV in objc_msgSend) on macOS when quitting via the Dock context menu while a floating/auxiliary window is open.

## Best solution
