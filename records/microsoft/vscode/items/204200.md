## Change summary

Request to make code folding/unfolding actions undoable via the standard Undo command (Cmd+Z).

## Best solution

Integrate folding state changes into the editor's undo/redo stack (or a parallel view-state undo stack) in `src/vs/editor/contrib/folding/`, gated behind a setting. This is a non-trivial design choice since folding is view-state rather than document-state, so it likely needs architectural discussion on whether to extend the existing undo stack or create a separate one.
