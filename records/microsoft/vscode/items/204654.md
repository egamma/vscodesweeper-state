# Issue #204654 — Unwanted scrolling in macOS with native trackpad and pinch zoom gesture.

[https://github.com/microsoft/vscode/issues/204654](https://github.com/microsoft/vscode/issues/204654)

## Summary

Keep open: the report has clear macOS trackpad steps, no extensions, and current code still appears to let pinch-generated wheel deltas fall through to normal scrolling when `editor.mouseWheelZoom` is disabled.

## Change summary

A macOS pinch gesture scrolls the editor vertically even when editor wheel zoom is disabled, where the expected behavior is no editor movement.

## Best solution

Decide in the editor/scrollbar input path how to identify macOS pinch-generated wheel events when `editor.mouseWheelZoom` is false, then consume or ignore those events so they do not become normal scroll input.

## Evidence

- **scroll path:** In `src/vs/base/browser/ui/scrollbar/scrollableElement.ts`, `_onMouseWheel` applies any nonzero `deltaY`/`deltaX` to the scroll position unless the browser event was already default-prevented.
