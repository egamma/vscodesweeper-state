# Issue #204654 — Unwanted scrolling in macOS with native trackpad and pinch zoom gesture.

[https://github.com/microsoft/vscode/issues/204654](https://github.com/microsoft/vscode/issues/204654)

## Summary

Still reproduces on main: the early return at line 153 of mouseHandler.ts when mouseWheelZoom is disabled does not prevent the pinch gesture event from propagating to scrollableElement, causing unwanted vertical scrolling. Two community PRs attempted fixes but neither was merged.

## Change summary

macOS pinch-to-zoom trackpad gesture causes unwanted vertical scrolling in the editor when `editor.mouseWheelZoom` is disabled.

## Best solution

Before the early return in `onMouseWheel` when `mouseWheelZoom` is disabled, detect pinch gesture events (via `hasMouseWheelZoomModifiers`) and call `preventDefault()`/`stopPropagation()` so they don't reach `scrollableElement`. The fix is ~7 lines in `src/vs/editor/browser/controller/mouseHandler.ts`.
