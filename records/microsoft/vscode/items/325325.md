## Summary

Duplicate of the canonical conpty/winpty-removal issue on unsupported Windows versions. The reporter is on Windows 10 build 16299 (version 1709), which does not support conpty and is no longer supported by VS Code; winpty was removed in v1.109.

## Change summary

Terminal fails to launch on Windows 10 version 1709 (build 16299) with 'Cannot launch conpty' after winpty removal in VS Code 1.109.

## Best solution
