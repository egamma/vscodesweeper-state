# Issue #309274 — Git commit message generation - allow path exclusion

[https://github.com/microsoft/vscode/issues/309274](https://github.com/microsoft/vscode/issues/309274)

## Summary

Valid feature request for a path-exclusion mechanism in Copilot commit message generation. No existing implementation or duplicate found; the current setting only supports text and file instructions with no glob-based exclude option.

## Change summary

Allow users to exclude specific files or folders (via glob patterns) from being considered when generating Copilot commit messages.

## Best solution

Add an exclude-paths option (glob-based) to the `github.copilot.chat.commitMessageGeneration` configuration, filtering diffs before they reach the prompt in `extensions/copilot/src/extension/prompt/node/gitCommitMessageGenerator.ts`. This keeps excluded files committed but omitted from message context.
