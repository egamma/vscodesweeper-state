# Issue #28080 — [decorations] Support hover decorations over the line numbers (i.e. gutter)

[https://github.com/microsoft/vscode/issues/28080](https://github.com/microsoft/vscode/issues/28080)

## Summary

The internal editor model supports `glyphMarginHoverMessage` and `lineNumberHoverMessage`, but the extension host API (`DecorationOptions` in vscode.d.ts) still only exposes a generic `hoverMessage`. The gap remains: extensions cannot target gutter-specific hovers.

## Change summary

Extensions should be able to provide hover decorations specifically for the gutter (glyph margin) and line numbers, separate from inline hover messages.

## Best solution

Expose `glyphMarginHoverMessage` and/or `lineNumberHoverMessage` on `DecorationOptions` in the extension API (vscode.d.ts), wired through the extension host decoration adapter in `src/vs/workbench/api/common/extHostTypes.ts` and related marshalling code. A proposed API stage would be appropriate.
