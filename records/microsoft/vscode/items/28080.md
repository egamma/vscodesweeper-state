# Issue #28080 — [decorations] Support hover decorations over the line numbers (i.e. gutter)

[https://github.com/microsoft/vscode/issues/28080](https://github.com/microsoft/vscode/issues/28080)

## Summary

Still valid: the internal editor model supports `glyphMarginHoverMessage` but it is not exposed through the extension host API (`vscode.d.ts`). The feature has 72 upvotes, well above the backlog threshold, and remains unfulfilled.

## Change summary

Allow extensions to provide hover content specifically for gutter/glyph-margin decorations via the `DecorationRenderOptions` API.

## Best solution

Expose `glyphMarginHoverMessage` (already present on the internal `IModelDecorationOptions`) through the extension API by adding a corresponding property to `DecorationRenderOptions` in `vscode.d.ts` and wiring it in the decoration adapter in `src/vs/workbench/api/common/extHostTypes.ts`.
