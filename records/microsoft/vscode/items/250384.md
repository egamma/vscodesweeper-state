## Change summary

Chat response focus outline has square corners (mismatched with rounded request border) and extends into empty space below content.

## Best solution

Align the response focus outline with the request's rounded-corner treatment and clip it to actual content bounds. The discussion in https://github.com/microsoft/vscode/issues/273933 suggests either adopting the request-style focus outline for responses or using tree `paddingBottom` to avoid including empty space. Design sign-off from the UX team is needed before implementation.
