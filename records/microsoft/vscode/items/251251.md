# Issue #251251 — Feature Request: Add Temperature for Custom Agent Profiles

[https://github.com/microsoft/vscode/issues/251251](https://github.com/microsoft/vscode/issues/251251)

## Summary

Community-promoted feature request (20+ upvotes, in Backlog) requesting temperature control for chat modes/BYOK models. Maintainers expressed reluctance but did not formally close; active community discussion continues with a related external PR.

## Change summary

Allow users to configure LLM temperature (and potentially other model parameters) in chat modes or BYOK model settings.

## Best solution

Keep in Backlog as under-discussion. The narrower ask — letting BYOK users omit or customize temperature so models like o4-mini/o3-mini don't fail with hardcoded values — is a pragmatic subset that could be addressed without exposing temperature in chat mode files. The broader temperature-per-mode request needs a product decision on how much LLM knob exposure is appropriate.
