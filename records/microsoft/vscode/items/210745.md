# Issue #210745 — Diff Editor Unstage buttons

[https://github.com/microsoft/vscode/issues/210745](https://github.com/microsoft/vscode/issues/210745)

## Summary

Valid feature request; confirmed unimplemented on current main. The git extension contributes `git.diff.stageHunk` to `diffEditor/gutter/hunk` for unstaged changes, but there is no analogous unstage button for the staged-changes diff view.

## Change summary

Add unstage block/selection buttons to the diff editor gutter when viewing staged changes.

## Best solution

Add `git.diff.unstageHunk` and `git.diff.unstageSelection` commands in the git extension (mirroring `git.diff.stageHunk`/`git.diff.stageSelection`) and contribute them to the `diffEditor/gutter/hunk` and `diffEditor/gutter/selection` menu points with a `when` clause matching staged-file URIs (the `%22ref%22%3A%22%22%7D` index pattern). The implementation can reuse the existing `unstageSelectedRanges` logic in `extensions/git/src/commands.ts`.
