# Issue #304839 — Agent file problems

[https://github.com/microsoft/vscode/issues/304839](https://github.com/microsoft/vscode/issues/304839)

## Summary

Verified duplicate of #302880, which tracked the same prompts-diagnostics-provider warnings for built-in agent .md files. The issue was fixed on two fronts: PR #304872 restricted diagnostics to files open in an editor, and Copilot Chat extension 0.41.2 removed the unknown tool references. Both fixes shipped in VS Code 1.114.0.

## Change summary

Built-in agent markdown files (Ask, Explore, Plan) surface spurious 'Unknown tool' warnings in the Problems panel for tools from optional extensions.

## Best solution

Already resolved — the fix shipped in VS Code 1.114.0 via commit 6648ece5 (diagnostics scoped to open editors) and Copilot Chat 0.41.2 (agent files no longer reference tools from optional extensions). Updating VS Code and the Copilot Chat extension resolves the issue.
