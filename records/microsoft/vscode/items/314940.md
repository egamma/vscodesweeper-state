## Change summary

Add a dependency manager for agent plugins that resolves semver constraints, installs transitive dependencies from marketplaces, and prunes orphaned plugins.

## Best solution

This is a significant architectural addition to the agent plugin system. Best path: let this mature via the community process and revisit if plugin ecosystem adoption grows and a real dependency-resolution need emerges from concrete user workflows.
