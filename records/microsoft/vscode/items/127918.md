# Issue #127918 — Make minimap slider visible when scrolling 

[https://github.com/microsoft/vscode/issues/127918](https://github.com/microsoft/vscode/issues/127918)

## Summary

Keep open: this remains a valid minimap feature request, and current main does not appear to implement showing the minimap slider during normal editor scrolling when `editor.minimap.showSlider` is `mouseover`. The Backlog milestone is a holding bin and does not force closure, but there is not strong evidence for a duplicate, completed fix, or out-of-scope verdict.

## Change summary

Make the minimap slider become visible while the editor is being scrolled, even when `editor.minimap.showSlider` is configured as `mouseover`.

## Best solution

If accepted, the editor minimap implementation should toggle slider visibility from the minimap scroll path, likely near `src/vs/editor/browser/viewParts/minimap/minimap.ts` `onScrollChanged`, while keeping the existing `always` and `mouseover` setting semantics clear.

## Evidence

- **current labels:** The issue is already classified as `feature-request` and `editor-minimap`, with milestone `Backlog`.
- **configuration still lacks requested mode:** `src/vs/editor/common/config/editorOptions.ts` defines `editor.minimap.showSlider` with only `always` and `mouseover`; no `scroll` or combined mouseover/scroll option exists on current main.
- **current render behavior:** `src/vs/editor/browser/viewParts/minimap/minimap.css` shows the slider for `.slider-mouseover:hover .minimap-slider` and `.slider-mouseover .minimap-slider.active`, but `onScrollChanged` in `src/vs/editor/browser/viewParts/minimap/minimap.ts` only toggles the minimap DOM node active for `autohide === 'scroll'`, not the slider active class for normal scrolling.
- **related implementation is adjacent but not equivalent:** Commit `fa92e4854f833973fb3e6247742bd1278b50c415` added `editor.minimap.autohide` with a `scroll` mode for showing the minimap itself, referencing https://github.com/microsoft/vscode/issues/107116, but it did not add a scroll-triggered slider visibility mode.
