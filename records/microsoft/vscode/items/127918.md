# Issue #127918 — Make minimap slider visible when scrolling 

[https://github.com/microsoft/vscode/issues/127918](https://github.com/microsoft/vscode/issues/127918)

## Summary

Valid feature request still unimplemented on main. The `editor.minimap.showSlider` setting only supports `'always'` and `'mouseover'` — no scroll-triggered visibility exists. With only 7 upvotes and in Backlog Candidates, it stays open for the community process.

## Change summary

Show the minimap slider when scrolling, not just on mouseover — a third option between 'always' and 'mouseover'.

## Best solution

Add a third enum value (e.g. `'scroll'`) to `editor.minimap.showSlider` that reveals the slider during scroll events and hides it after a short timeout. Implementation would touch `src/vs/editor/common/config/editorOptions.ts` (enum + validation) and `src/vs/editor/browser/viewParts/minimap/minimap.ts` (CSS class toggling on scroll events). Low traction so far — leave for the community process.
