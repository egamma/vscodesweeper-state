# Issue #319383 — add project manager

[https://github.com/microsoft/vscode/issues/319383](https://github.com/microsoft/vscode/issues/319383)

## Summary

Project scaffolding with language/framework templates is squarely extension territory — several Marketplace extensions already provide this, and it cuts against the 'lightweight text editor out of the box' value.

## Change summary

Add built-in project creation with templates for C/C++, web frameworks, Electron, and Tauri.

## Best solution

Ship this as a Marketplace extension (or use existing ones like Yeoman scaffolding, Project Manager, or framework-specific generators). VS Code core deliberately stays framework-agnostic; bundling opinionated project templates would bloat the default product for a niche audience.
