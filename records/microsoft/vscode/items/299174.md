## Summary

Legitimate performance bug reporting excessive CSS style recalculation during Copilot agent activity. Several partial fixes have landed since the report (thinking layout thrashing, forced reflow avoidance), but the core problem — agent activity triggering broad style invalidations via codicon/decoration selectors — likely persists. Routing to the chat area for investigation with the attached trace.

## Change summary

Investigate the attached trace to identify which CSS selectors and DOM mutations during agent activity trigger the broad style invalidations. Partial fixes have landed (commits 719c62b60e8, f48dceacafb, 0fc534223c4) but likely don't fully address the codicon selector breadth problem described here.
