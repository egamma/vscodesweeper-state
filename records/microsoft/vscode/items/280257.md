## Summary

The setting `chat.promptFilesRecommendations` is registered on current main with `ConfigurationScope.RESOURCE` scope, which should support both user and workspace settings via automatic object-key merging. The report's reproduction steps are too vague to verify — we don't know what specific prompt names were configured at each level or what was expected. The second commenter's 'unknown configuration setting' is a separate version-availability issue. More details are needed to diagnose.
