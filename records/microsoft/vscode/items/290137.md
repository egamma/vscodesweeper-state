[https://github.com/microsoft/vscode/issues/290137](https://github.com/microsoft/vscode/issues/290137)

## Summary

The 'GitHub Copilot: Toggle (Enable/Disable)' command is owned by the Copilot extension, not VS Code core. Requests to change its behavior should be filed against the extension's repository.

## Change summary

When toggling Copilot enable/disable via keybinding, also dismiss or show any visible inline suggestion.

## Best solution

File this as a feature request on https://github.com/microsoft/vscode-copilot-release or the Copilot extension's feedback channel. The toggle command is registered by the extension and its behavior (including dismissing inline completions on disable) is the extension's responsibility.
