# Issue #290715 — Zen Mode is not using space well while center mode.

[https://github.com/microsoft/vscode/issues/290715](https://github.com/microsoft/vscode/issues/290715)

## Summary

Feature request for better zen mode + center layout interaction: a quick toggle for center layout while in zen mode, and having the sidebar utilize the empty centered space rather than pushing it aside. The existing `zenMode.centerLayout` setting only controls whether center layout activates with zen mode — it doesn't provide the dynamic toggle or sidebar space behavior requested.

## Change summary

Request for a quick-access toggle to disable center layout while in zen mode, and for the sidebar to occupy the centered empty space instead of displacing it.

## Best solution

Consider adding a visible toggle in zen mode to dynamically enable/disable center layout, and evaluate whether the sidebar could expand into the centered empty space when opened. The work likely belongs in `src/vs/workbench/browser/layout.ts` (centered editor layout handling) and potentially a new zen mode toolbar action.
