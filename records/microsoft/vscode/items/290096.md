# Issue #290096 — Allow --ours and --theirs options for multiple files in large merge conflicts

[https://github.com/microsoft/vscode/issues/290096](https://github.com/microsoft/vscode/issues/290096)

## Summary

Valid feature request for bulk merge conflict resolution (accept incoming/outgoing for multiple files at once) in the Source Control view. No existing implementation found on main. Routing to the git area.

## Change summary

Add context menu options on the merge-conflicts group header in the SCM view to accept incoming or outgoing changes for all conflicted files at once.

## Best solution

Add group-level context menu actions ('Accept All Incoming', 'Accept All Current') on the merge-conflicts resource group in the SCM view, backed by git checkout --theirs/--ours for the selected files. Implementation would live in extensions/git/src/commands.ts and extensions/git/package.json (menu contributions).
