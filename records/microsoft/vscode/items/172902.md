# Issue #172902 — Extra selection getting added when you Alt + click on a multiple selection to remove the selection

[https://github.com/microsoft/vscode/issues/172902](https://github.com/microsoft/vscode/issues/172902)

## Summary

Keep open: the current editor mouse/multicursor code still appears to allow the reported two-step behavior, and there is no fixing commit or duplicate evidence in the issue context.

## Change summary

Alt-clicking an existing multicursor selection should remove that selection without immediately extending another selection from the last cursor.

## Best solution

Handle the multicursor removal gesture in the editor mouse path so a pure Alt-click inside an existing selection toggles that cursor off and does not enter the later selection-extension path. The relevant code is in src/vs/editor/browser/view/viewController.ts and src/vs/editor/browser/coreCommands.ts.
