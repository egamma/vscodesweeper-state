# Issue #176282 — Auto-attach remote session

[https://github.com/microsoft/vscode/issues/176282](https://github.com/microsoft/vscode/issues/176282)

## Summary

Valid feature request to pass pattern-captured values (IP/port) from serverReadyAction into the launched debug configuration. The capability is still missing on current main — the startDebugging action launches a static named config without substitution of captured groups.

## Change summary

Allow serverReadyAction to pass regex capture groups as variables into the attached debug session configuration.

## Best solution

Extend the `startDebugging` path in `extensions/debug-server-ready/src/extension.ts` to perform variable substitution (e.g. `${1}`, `${2}`) from the regex capture groups into the inline `config` object. This keeps it in the existing built-in extension without needing core changes. Low upvote count suggests this should remain in Backlog until more demand emerges.
