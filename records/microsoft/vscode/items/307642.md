# Issue #307642 — VS Code Copilot Fails to access codespaces terminal on liveshare

[https://github.com/microsoft/vscode/issues/307642](https://github.com/microsoft/vscode/issues/307642)

## Summary

The reported failure is an interaction between two extensions (Copilot Chat and Live Share) — Copilot agent mode tries to access a terminal/filesystem via Live Share's vsls virtual path, which is entirely mediated by those extensions rather than VS Code core. This belongs in the Copilot or Live Share extension issue trackers.

## Change summary

Copilot agent mode fails to use a shared terminal when connected to a codespace via Live Share, accessing a missing vsls folder instead of the remote terminal.

## Best solution

File against the Copilot Chat extension (https://github.com/microsoft/vscode-copilot-release) or Live Share extension (https://github.com/MicrosoftDocs/live-share) — the terminal routing through Live Share's virtual filesystem is owned by those extensions, not VS Code core.
