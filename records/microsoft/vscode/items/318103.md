# Issue #318103 — Agents Window: support extension development workflow (extensionDevelopmentPath + debugger)

[https://github.com/microsoft/vscode/issues/318103](https://github.com/microsoft/vscode/issues/318103)

## Summary

Keep open as a valid Backlog feature request: current main still treats extension development and Agents Window launch paths as separate flows, and there is no concrete fixing commit for the requested combined workflow.

## Change summary

Support launching/debugging an extension development host inside the Agents Window with `--agents` and `--extensionDevelopmentPath`.

## Best solution

Route jointly to `extension-host` and `agents-window` for a design that either makes the `extensionHost` debug launch path honor `--agents`, or makes Agents Window creation inherit the extension development paths and debug arguments from the originating Extension Development Host.
