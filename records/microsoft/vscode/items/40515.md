# Issue #40515 — Tasks with target = TaskScope.Global as the scope do not show up in task picker UI

[https://github.com/microsoft/vscode/issues/40515](https://github.com/microsoft/vscode/issues/40515)

## Summary

Global tasks (`TaskScope.Global`) are still explicitly documented as unsupported in the API (`vscode.d.ts` line 9103: "Global tasks are currently not supported."). The 2019 fixes (commits bfa3db2f, 2d96b9b6) addressed execution failures but did not implement full picker/UI support. The issue remains a valid, unresolved feature request on the On Deck milestone.

## Change summary

Tasks provided via the extension API with `TaskScope.Global` scope do not appear in the task picker UI.

## Best solution

Implement full lifecycle support for `TaskScope.Global` tasks: show them in the task picker, handle variable resolution gracefully (leave workspace-dependent variables as literals), and exclude them from 'Configure Task' flows that require a tasks.json. Update the API doc in `vscode.d.ts` to remove the 'currently not supported' caveat once shipped.
