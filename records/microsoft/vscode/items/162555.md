Valid feature request for quick-open to match paths extending above the workspace root (e.g. git-repo-relative paths). Not yet implemented on main; 6 upvotes, below the 20-upvote promotion threshold. Keeping open for community traction.

## Change summary

Allow the file quick-open picker (Cmd+P) to find files when the typed path extends above the open workspace folder, e.g. using a path relative to the git repository root.
