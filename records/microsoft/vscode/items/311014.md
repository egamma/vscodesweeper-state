# Issue #311014 — Make `diffEditor.ignoreTrimWhitespace` false by default

[https://github.com/microsoft/vscode/issues/311014](https://github.com/microsoft/vscode/issues/311014)

## Summary

Feature request to change a long-standing default. No upvotes yet and changing defaults carries risk of disrupting existing users; keeping open for the community process to gauge demand.

## Change summary

Change the default value of `diffEditor.ignoreTrimWhitespace` from `true` to `false` so whitespace-only changes are shown in diffs by default.

## Best solution

Leave the default as-is for now. The YAML extension already overrides this per-language (in `extensions/yaml/package.json`). If demand grows, the team could consider flipping the default or adding per-language overrides for more languages. Users who want this behavior today can set the option in their settings.
