# Issue #222404 — Add `code --help -a` 

[https://github.com/microsoft/vscode/issues/222404](https://github.com/microsoft/vscode/issues/222404)

## Summary

Valid feature request from a core team member (Tyriar/Daniel Imms) for a verbose help mode that exposes all CLI options. Not yet implemented — options without a `cat` field remain hidden from `--help` output, and no `--help -a` equivalent exists on current main.

## Change summary

Add a `code --help -a` (or equivalent) flag that shows all CLI options including internal/hidden ones, similar to `git help -a`.

## Best solution

Add a `--help-all` or `-a` flag to `buildHelpMessage` in `src/vs/platform/environment/node/argv.ts` that includes options lacking a `cat` field. Low complexity — the filter logic is already in `buildHelpMessage`; the change would add an alternate path that skips the `o.cat` gate.
