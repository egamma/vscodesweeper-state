# Issue #203793 — Muti-file diff editor - collapse icon missing from the right-hand side

[https://github.com/microsoft/vscode/issues/203793](https://github.com/microsoft/vscode/issues/203793)

## Summary

Team-authored issue (MEMBER). The collapse icon in the multi-diff editor header is rendered only once on the left side; a matching icon on the right-hand side of the header (for the modified-file pane) is absent. Recent commits added header click-to-collapse functionality but did not add a second chevron icon on the right.

## Change summary

The multi-file diff editor header shows a collapse/expand chevron only on the left side; it is missing from the right-hand side.

## Best solution

Add a mirrored collapse-button element on the right side of the header in `src/vs/editor/browser/widget/multiDiffEditor/diffEditorItemTemplate.ts`, wired to the same collapsed observable, so users can collapse from either side of the diff.
