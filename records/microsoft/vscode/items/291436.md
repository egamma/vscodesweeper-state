Team-authored polish issue requesting smoother animation when transitioning to the maximized session view in the Agents window. Hard keep-open rule applies (MEMBER author).

## Change summary

Clicking a session in the Agents window triggers a jumpy/jarring transition to the maximized chat/sessions view.

## Best solution

Smooth the layout transition in the single-pane layout controller and/or the sessions part CSS. The relevant animation/transition code lives in `src/vs/sessions/browser/media/style.css` (lines ~806–883, the part-reveal transitions) and the layout orchestration in `src/vs/sessions/contrib/layout/browser/singlePane/`. Adding or tuning CSS transitions on the container resize (opacity, transform, or height easing) and coordinating the layout reflow in the controller should eliminate the jumpiness.
