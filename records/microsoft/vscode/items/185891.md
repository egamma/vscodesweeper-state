# Issue #185891 — Smooth Scrolling: Difference Between VS Code / Chrome

[https://github.com/microsoft/vscode/issues/185891](https://github.com/microsoft/vscode/issues/185891)

## Summary

Team-authored discussion issue comparing smooth scrolling implementations between Monaco/VS Code and Chrome's native scrolling. Still under discussion with no resolution; hard keep-open applies (MEMBER author).

## Change summary

Investigate whether the editor's smooth scrolling animation should adopt Chrome-like easing/behavior, and whether free-spin mouse wheel (pixel-level) scrolling should be better supported.

## Best solution

The issue captures two distinct concerns: (1) the easing curve of the animated smooth scroll vs Chrome's, and (2) proper pixel-level smooth scrolling for free-spin mouse wheels. A decision is needed on whether to change the default animation curve and whether to improve free-spin wheel support in the editor scrollbar layer (`src/vs/editor/browser/viewParts/editorScrollbar/`).
