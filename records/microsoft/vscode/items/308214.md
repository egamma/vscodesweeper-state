# Issue #308214 — Chat input btn flicker as I type

[https://github.com/microsoft/vscode/issues/308214](https://github.com/microsoft/vscode/issues/308214)

## Summary

The reported chat input button flicker while typing remains unfixed on current main — no commit addressing this specific visual regression was found. The author (jrieken) is a core VS Code team member; keeping open for maintainer attention.

## Change summary

Actions below the chat input box flicker/re-render as the user types a prompt.

## Best solution

Investigate the toolbar relayout triggers in `chatInputPart.ts` — the `_toolbarRelayoutScheduler`, `onDidChangeMenuItems` handlers, and context-key-driven `when` clause re-evaluations on the secondary/execute toolbars likely cause unnecessary DOM updates on each keystroke. A debounce or guard against no-op relayouts would fix the flicker.
