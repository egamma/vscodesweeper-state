## Change summary

The vscode.dev redirect page after a Copilot plan upgrade displays a browser error in Edge, even though it still triggers the VS Code protocol handler prompt.

The fix likely lives in the vscode.dev redirect page itself (server-side/web), not in the VS Code client.
