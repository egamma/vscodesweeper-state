## Change summary

CSS color picker does not appear when a CSS variable is used inside a color function like `hsl(var(--hue-color), 0%, 0%)`.

## Best solution

Enhance the color detection logic in vscode-css-languageservice (`cssNavigation.ts`) to resolve CSS custom property values and produce color decorations/pickers when the resolved composite value forms a valid color. The change lives in the upstream language service repo, not in this repo directly.
