# Issue #123979 — [folding]  highlight folding range when hovering over the collapse button

[https://github.com/microsoft/vscode/issues/123979](https://github.com/microsoft/vscode/issues/123979)

## Summary

This is a valid feature request for hover-preview highlighting of the folding range before collapsing. The existing `editor.foldingHighlight` setting only highlights already-collapsed ranges; no code on main previews the range on gutter-icon hover.

## Change summary

Highlight the folding range background when hovering over the fold gutter icon (before collapsing), similar to Visual Studio.

## Best solution

Add a mousemove/hover listener on the folding gutter icons in the folding contribution (`src/vs/editor/contrib/folding/browser/folding.ts`) that applies a transient decoration over the target folding range while hovered, gated behind an opt-in or the existing `editor.foldingHighlight` setting.
