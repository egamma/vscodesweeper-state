# Issue #123979 — [folding]  highlight folding range when hovering over the collapse button

[https://github.com/microsoft/vscode/issues/123979](https://github.com/microsoft/vscode/issues/123979)

## Summary

This feature request for highlighting the folding range on hover over the collapse gutter icon is not implemented on current main. The existing `foldingHighlight` setting only highlights already-collapsed ranges; there is no hover-preview for expanded ranges. Low community traction (0 upvotes) but a valid, well-described request in Backlog.

## Change summary

Request to highlight the folding range (background color on the lines about to be collapsed) when hovering over the fold/collapse gutter icon, similar to Visual Studio behavior.

## Best solution

Implement a mouseMove listener on the folding gutter icons that temporarily applies a background decoration to the full folding range while the user hovers, controlled by the existing `editor.foldingHighlight` setting or a new dedicated setting. The work would live in `src/vs/editor/contrib/folding/browser/folding.ts` (mouse handling) and `foldingDecorations.ts` (a new hover-preview decoration).
