Team-authored issue; the commented-out cleanup code for extension auth usage remains unresolved on current main. The MCP cleanup contribution is also still commented out. Keeping open per policy.

## Change summary

Auth provider account usage data is not cleaned up when extensions or MCP servers are uninstalled, leaving stale access records.

## Best solution
