# Issue #326021 — Link in plan widget is a different color

[https://github.com/microsoft/vscode/issues/326021](https://github.com/microsoft/vscode/issues/326021)

## Summary

Team-authored visual bug — links in the plan review widget don't use the standard `--vscode-textLink-foreground` token, causing low-contrast link colors across themes. An open PR (https://github.com/microsoft/vscode/pull/326188) already addresses this with a small CSS fix. Keeping open for the PR to land.

## Change summary

Links rendered inside the plan review widget use a non-standard color instead of the `--vscode-textLink-foreground` design token, producing low-contrast text in many themes.

## Best solution

Land https://github.com/microsoft/vscode/pull/326188, which adds `color: var(--vscode-textLink-foreground)` and matching hover/active rules to `chatPlanReview.css`. The fix is small and localized.
