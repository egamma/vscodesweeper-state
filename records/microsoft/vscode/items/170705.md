# Issue #170705 — PHP formatting on typing

[https://github.com/microsoft/vscode/issues/170705](https://github.com/microsoft/vscode/issues/170705)

## Summary

Keep open: this has a maintainer diagnosis and the current auto-indent implementation still appears to use the model-level language configuration when inheriting indentation, which matches the reported PHP-with-embedded-HTML failure mode.

## Change summary

Typing an HTML closing tag inside a PHP file deindents the tag to column 1 instead of preserving the surrounding HTML indentation.

## Best solution

Fix the core editor auto-indent path so inherited indentation for embedded HTML in PHP uses the relevant scoped/token language rules for the preceding lines, then add a regression test for typing `</ul>` inside an indented HTML block in a `.php` model.

## Evidence

- **maintainer diagnosis:** roblourens identified that PHP files inherit HTML indentation rules for the typed line, but inherited indentation was computed from the PHP/model language instead; see https://github.com/microsoft/vscode/issues/170705#issuecomment-1374714589 and https://github.com/microsoft/vscode/issues/170705#issuecomment-1374715874.
- **current code:** In src/vs/editor/common/languages/autoIndent.ts, getIndentActionForType uses model.getLanguageIdAtPosition(range.startLineNumber, range.startColumn) for the typed line, but getInheritIndentForLine still initializes indentRulesSupport from model.tokenization.getLanguageId(), which is consistent with the member's concern for embedded PHP/HTML.
- **no concrete fix found:** A git log search for issue 170705 and 'PHP formatting' found no fixing commit, and recent autoIndent.ts commits such as d309e11579e3bfe877014834384e9f502e4996ad and 8ca1b8c195bcef349aa0ab8ca6fb440cd82933ff address other indentation cases rather than this embedded-language inheritance bug.
- **workaround only:** The only maintainer-provided mitigation was lowering editor.autoIndent below Full, which avoids the bad behavior but also disables desired automatic HTML indentation.
