## Summary

Team-authored feature request exploring how MCP servers could provide drop-in replacements for built-in code search tools (grep-based and semantic vector search). Keeping open — authored by a team member and parked in Backlog for future design work.

## Change summary

Allow MCP servers or LSPs with code indexing to replace built-in grep-based file search and semantic vector search with faster, more token-efficient alternatives.
