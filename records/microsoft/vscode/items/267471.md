# Issue #267471 — Git Clone - local folder with space. Problem with path. Escapes "space" to %20

[https://github.com/microsoft/vscode/issues/267471](https://github.com/microsoft/vscode/issues/267471)

## Summary

Confirmed bug still present on main — `encodeURI(url)` on line 480 of `extensions/git/src/git.ts` encodes spaces to `%20` for local paths, breaking git clone. Milestone is 'On Deck' and an open PR (#297212) already targets this fix; keeping open.

## Change summary

Git clone fails for local repository paths containing spaces because VS Code encodes them with `%20` via `encodeURI`.

## Best solution

Land the open PR https://github.com/microsoft/vscode/pull/297212, which should distinguish local file paths from remote URLs and avoid `encodeURI` encoding on local paths. The fix is localized to `extensions/git/src/git.ts` line 480.
