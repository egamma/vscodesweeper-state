## Summary

Valid backlog feature request with 24 community upvotes — no implementation exists on current main. The command palette still renders without any overlay or backdrop dimming effect.

## Change summary

Add an optional overlay/dimming backdrop behind the command palette (quick open) to improve visual focus when it is open.

## Best solution

Implement an opt-in setting (e.g. under quickInput or commandPalette) that renders a semi-transparent overlay behind the quick pick widget. The implementation would live in `src/vs/platform/quickinput/browser/quickInputController.ts` and its associated CSS. Given the 'simple, uniform UI' value, this should be a user-opt-in setting rather than default-on.
