# Issue #192664 — Feature Request: Typing enter to close an openning bracket

[https://github.com/microsoft/vscode/issues/192664](https://github.com/microsoft/vscode/issues/192664)

## Summary

Valid feature request with minimal community traction (1 upvote in ~3 years). The request is for VS Code to auto-close an existing opening bracket when Enter is pressed, similar to JetBrains behavior. No duplicate found; keeping open in the community process.

## Change summary

Auto-close an existing unmatched opening bracket (e.g. `{`) when the user presses Enter after it, rather than requiring manual typing of the closing bracket.

## Best solution

This would be a new behavior mode in `src/vs/editor/contrib/bracketMatching/` or the auto-closing logic in `src/vs/editor/common/cursor/cursorTypeOperations.ts`, gated behind a setting (e.g. extending `editor.autoClosingBrackets`). With only 1 upvote and no further discussion in 3 years, it lacks the traction to prioritize but remains a valid enhancement request for the community process.
