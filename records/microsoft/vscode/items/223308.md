The paste auto-indentation bug for Allman-style braces in TypeScript is still present in the algorithm on main. The new `autoIndentOnPaste` setting (default: false) mitigates exposure, and the `tree-sitter-autoindent-candidate` label correctly tracks it for a proper fix via tree-sitter indentation.

## Change summary

Pasting an Allman-style if-block in TypeScript incorrectly adds extra indentation to the opening brace and body.
