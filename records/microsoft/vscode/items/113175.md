# Issue #113175 — Linux: native context menu runs action under mouse when opened

[https://github.com/microsoft/vscode/issues/113175](https://github.com/microsoft/vscode/issues/113175)

## Summary

This upstream Chromium/Electron bug was fixed in the Chromium runtime and confirmed working in Electron 35+. VS Code moved to Electron 37 in v1.103.0 (July 2025) and is now on Electron 42, so the fix has been shipping for about a year.

## Change summary

On Linux, native context menus appeared under the mouse cursor, causing the right-click release to accidentally trigger a menu action — especially with non-zero zoom levels or high DPI.

## Best solution

The upstream Chromium fix has been shipping in VS Code since v1.103.0 (Electron 37). No VS Code-side change is needed. Close as completed via the upstream fix.
