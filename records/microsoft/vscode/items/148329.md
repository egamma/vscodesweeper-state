This request for per-level indent guide colorization was implemented via PR #178112 (merged July 2023, shipped in 1.81.0). Users can now set `editorIndentGuide.background1` through `background6` to colorize indent guides by depth level.

## Change summary

Colorize indent guides per indentation level, similar to bracket pair colorization, for indentation-based languages.
