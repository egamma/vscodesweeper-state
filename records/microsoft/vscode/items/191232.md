# Issue #191232 — Strange behavior on html indent line color

[https://github.com/microsoft/vscode/issues/191232](https://github.com/microsoft/vscode/issues/191232)

## Summary

The behavior is still present on current main — bracket pair guides suppress active indent guide coloring by design (line 216 of indentGuides.ts). A team member acknowledged this as a known limitation with 'at the moment' phrasing, and the issue is already labeled under-discussion. Keeping open for a potential future design change.

## Change summary

Indent guide coloring is suppressed on lines where bracket pair guides are displayed in HTML, making some indent lines appear uncolored.

## Best solution

Decide whether `highlightActiveIndentation: 'always'` is sufficient documentation of the workaround, or whether the exclusivity between bracket and indent guides should be relaxed so both render with their respective colors simultaneously. The relevant code is in `src/vs/editor/browser/viewParts/indentGuides/indentGuides.ts` around line 215–216.
