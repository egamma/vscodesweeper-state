# Issue #156178 — Git - Enable AI extensions to suggest a branch name (given the commit message)

[https://github.com/microsoft/vscode/issues/156178](https://github.com/microsoft/vscode/issues/156178)

## Summary

Team-authored feature request (hediet is MEMBER) — must keep open for maintainer judgment. No extension API for branch name suggestion exists on current main; the agent host has an internal branch name generator but nothing is exposed via the extension API or the SCM branch picker UI.

## Change summary

Allow AI extensions (e.g. Copilot) to suggest a branch name in the Git branch picker, derived from the last commit message.

## Best solution

Add a proposed extension API (e.g. a `BranchNameProvider` registration point or an extension-contributed default value in the branch-creation quick pick) so that extensions like Copilot can supply suggested branch names. The agent host already has `copilotBranchNameGenerator.ts` for its own sessions — the general-purpose equivalent would live in the SCM/git extension API surface (`src/vscode-dts/` + `extensions/git/src/commands.ts` branch-creation flow).
