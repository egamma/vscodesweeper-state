# Issue #140647 — Activity bar activity indicator is `1px` but configured as `2px`

[https://github.com/microsoft/vscode/issues/140647](https://github.com/microsoft/vscode/issues/140647)

## Summary

Still reproduces on current main: the activity bar indicator is styled as `border-left: 2px solid` but renders visually as 1px. Confirmed by a team member (bpasero) as recently as Dec 2024, reproducing even on web and with native title bar.

## Change summary

The activity bar's active indicator renders at 1px despite being configured as 2px in CSS, likely a sub-pixel rendering issue.

## Best solution

Investigate the sub-pixel rendering cause — possibly the indicator's container or parent has a fractional offset that clips the border. The fix likely lives in `src/vs/workbench/browser/parts/activitybar/media/activityaction.css` (the `:before` pseudo-element at line 87) or the layout logic that positions the activity bar relative to the window frame.
