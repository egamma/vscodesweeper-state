Confirmed regression still present on main — staging a file in the SCM view drops focus instead of advancing to the next resource. An open PR (#324682) is actively addressing it; keep open until that PR lands.

Using a keybinding to stage a file in the SCM changes list loses focus from the file list, breaking the keyboard-driven staging workflow.
