# Issue #273893 — Unable to create launch configuration file

[https://github.com/microsoft/vscode/issues/273893](https://github.com/microsoft/vscode/issues/273893)

## Summary

Fixed in 1.106.0: the "create a launch.json file" link broke in non-English locales because the markdown/URI syntax wasn't locked from translation after a refactor to createCommandUri. The fix locks those strings properly.

## Change summary

Clicking "create a launch.json file" in the Run and Debug welcome view did nothing in non-English locales.

## Best solution

Keep the shipped fix in commit 329b9861c51 (1.106.0); no further change needed. The reporter's Italian locale triggered the mislocalization that corrupted the command link.
