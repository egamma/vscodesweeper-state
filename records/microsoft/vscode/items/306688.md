# Issue #306688 — KDE Global App Menu not working (regression)

## Summary

Verified duplicate of #283522, which already tracks this exact regression (KDE global menu broken after the Electron 39 update) and is labeled as an upstream issue with linked upstream bugs.

## Change summary

KDE's global app menu does not display VS Code menu items on KDE Plasma (Wayland), a regression introduced with the Electron 39 update.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/283522, which owns the upstream tracking for this Electron/KDE/Wayland global-menu regression. The root cause is upstream in KDE (global menu support for non-Qt apps on Wayland) and Electron — see KDE bug 424485 and the linked Wayland protocols MR.
