# Issue #256177 — IHostService.restart() method causes “nosuid” problem on Linux

[https://github.com/microsoft/vscode/issues/256177](https://github.com/microsoft/vscode/issues/256177)

## Summary

Verified duplicate of #253204, which tracks the same root cause (Electron's app.relaunch() sets PR_SET_NO_NEW_PRIVS on Linux, breaking sudo in terminals). An open draft PR (#296711) already addresses it against the canonical issue.

## Change summary

After calling IHostService.restart() on Linux, sudo stops working in the integrated terminal due to the PR_SET_NO_NEW_PRIVS flag set by Electron's relaunch mechanism.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/253204 — the open draft PR https://github.com/microsoft/vscode/pull/296711 replaces Electron's app.relaunch() with a Node.js spawn on Linux to avoid the flag. Landing that PR resolves both issues.
