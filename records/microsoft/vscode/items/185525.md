# Issue #185525 — f is not iterable

[https://github.com/microsoft/vscode/issues/185525](https://github.com/microsoft/vscode/issues/185525)

## Summary

Team-authored telemetry error, kept open. The crash path is still present on main: `indents[lineIndex]` at line 130 of `indentGuides.ts` has no null guard, so if a race condition produces fewer entries than the visible range expects, `for (const guide of indent)` throws "f is not iterable".

## Change summary

Indent guides overlay crashes with "f is not iterable" when `indents[lineIndex]` is undefined due to a timing mismatch between the visible range and the guides array length.

## Best solution

Add a defensive guard at line 130 in `src/vs/editor/browser/viewParts/indentGuides/indentGuides.ts` — e.g. `if (!indent) { output[lineIndex] = ''; continue; }` — to prevent the crash when the array is shorter than expected due to concurrent model changes.
