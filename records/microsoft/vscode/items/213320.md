# Issue #213320 — Make Tabs Multiselect Keyboard Accessible

[https://github.com/microsoft/vscode/issues/213320](https://github.com/microsoft/vscode/issues/213320)

## Summary

Team-authored accessibility feature request (hard keep-open). Tab multiselect exists via mouse (Ctrl+click) but no keyboard-only mechanism to toggle tab selection was found on current main.

## Change summary

Allow users to select/unselect editor tabs using only the keyboard, without requiring mouse interaction.

## Best solution

Add a keybinding (e.g. Ctrl+Space or similar) that toggles the selection state of the focused tab, analogous to how list/tree widgets support keyboard multiselect. The implementation would likely touch `src/vs/workbench/browser/parts/editor/editorCommands.ts` and `src/vs/workbench/browser/parts/editor/editorGroupView.ts`.
