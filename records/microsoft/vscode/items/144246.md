# Issue #144246 — Indicate if Task Exited for both dependency tasks

[https://github.com/microsoft/vscode/issues/144246](https://github.com/microsoft/vscode/issues/144246)

## Summary

Team-authored feature request asking for better visual indication in the task list when dependency tasks have exited (currently the checkmark/error icon persists even after exit). Keeping open per team-author policy.

## Change summary

The task list sidebar does not visually distinguish running tasks from exited tasks when using dependency (background) tasks like deemon.

## Best solution

Update the terminal/task list rendering to reflect exited state for dependency tasks — likely in the task terminal status contribution area. The task status icons should update when the underlying process exits, so users can tell at a glance whether a watched task is still alive.
