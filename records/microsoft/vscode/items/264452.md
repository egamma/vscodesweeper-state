# Issue #264452 — please announce  suggestion number along with the text in editor.

[https://github.com/microsoft/vscode/issues/264452](https://github.com/microsoft/vscode/issues/264452)

## Summary

Still a gap on main: the screen reader announcement in `_provideScreenReaderUpdate` only reads the suggestion text and an optional accessible-view hint, but never includes the suggestion index or total count — even though this data is available in the hints widget (`currentSuggestionIdx + 1 / suggestionCount`). Milestone is 'On Deck', so keeping open.

## Change summary

Screen reader does not announce inline completion suggestion number (e.g., '2 of 5') when cycling through suggestions.

## Best solution

Pass the current suggestion index and total count into `_provideScreenReaderUpdate` (in `inlineCompletionsController.ts`) so the `alert()` call includes something like 'suggestion 2 of 5'. The count is already available via the model's `_suggestionCount` observable used by the hints widget.
