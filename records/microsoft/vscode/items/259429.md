## Change summary

## Best solution

The reporter needs to provide diagnostic data — a CPU profile captured during the freeze (Help > Report Issue > Performance Issue), a Process Explorer screenshot (Help > Open Process Explorer), and confirmation of whether the issue reproduces with extensions disabled (`--disable-extensions`). The unresponsive detection is Electron's built-in renderer-heartbeat mechanism (`src/vs/platform/windows/electron-main/windowImpl.ts` line 810); root causes range from extension host blocking to GPU/driver issues to antivirus interference.
