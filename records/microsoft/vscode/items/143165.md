# Issue #143165 — Git - Support for Git in Msys2

[https://github.com/microsoft/vscode/issues/143165](https://github.com/microsoft/vscode/issues/143165)

## Summary

Still a valid, community-backed feature request (27 upvotes, in Backlog). MSYS2 git path translation is not implemented in the git extension on current main, and an open PR (#175338) exists but has not merged.

## Change summary

VS Code's built-in git extension does not correctly invoke or parse output from MSYS2's git, requiring users to install a separate Windows git.

## Best solution

Land the path-translation work in `extensions/git/src/` (the open PR https://github.com/microsoft/vscode/pull/175338 is one approach). A maintainer should review that PR or decide on an alternative strategy for MSYS2/Cygwin path handling.
