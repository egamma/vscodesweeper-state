## Summary

The report likely stems from bash's login-shell behavior: VS Code spawns the user's shell with `-i -l -c`, which for bash means `~/.bashrc` is only sourced if `~/.profile` or `~/.bash_profile` explicitly sources it. Without knowing the reporter's shell init file contents, cannot confirm a VS Code bug.
