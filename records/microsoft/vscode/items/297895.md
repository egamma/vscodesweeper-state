# Issue #297895 — Show progress indication for downloading update if it's taking a long time?

[https://github.com/microsoft/vscode/issues/297895](https://github.com/microsoft/vscode/issues/297895)

## Summary

Team-authored feature request for download progress indication during updates on macOS. Requires Electron-level work per maintainer comment. Keeping open — team member issue needing explicit maintainer judgment.

## Change summary

Show download progress (percentage or time estimate) in the status bar when an update download takes more than a minute, instead of an indefinite 'Downloading update...' message.

## Best solution

Expose download progress from Electron's autoUpdater (which provides `download-progress` events on some platforms) and surface it in the update status bar item. This is an Electron/platform integration task — the macOS `autoUpdater` module may need additional work to emit granular progress. The relevant code likely lives in `src/vs/platform/update/` and the Electron main-process update service.
