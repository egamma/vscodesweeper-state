# Issue #288838 — [Feature Request] Add frontmatter option to open .prompt.md files in a new chat session

[https://github.com/microsoft/vscode/issues/288838](https://github.com/microsoft/vscode/issues/288838)

## Summary

Valid feature request promoted to Backlog with 23 upvotes. A 'run in new chat' action exists via keyboard modifier, but no declarative frontmatter option (e.g. `newChat: true`) is implemented on current main.

## Change summary

Request a frontmatter option in .prompt.md files to declaratively open them in a new chat session instead of the current one.

## Best solution

Add a `newChat: true` (or similar) frontmatter key to the prompt file schema in `src/vs/workbench/contrib/chat/common/promptSyntax/`, and wire it into `runPromptAction.ts` so it triggers the existing 'run-in-new-chat' code path automatically. This would complement the existing Ctrl+click behavior with a discoverable, file-level declaration.
