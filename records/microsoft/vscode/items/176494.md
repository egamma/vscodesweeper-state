# Issue #176494 — Markdown: Don't highlight unmatched brackets

[https://github.com/microsoft/vscode/issues/176494](https://github.com/microsoft/vscode/issues/176494)

## Summary

Team-authored issue (hediet is MEMBER) — must remain open for explicit maintainer judgment. The issue tracks unwanted bracket highlighting in Markdown for unmatched brackets (e.g. in math equations). A workaround exists but disables bracket selection-highlight too.

## Change summary

Unmatched brackets in Markdown (e.g. in math equations) are incorrectly highlighted by the bracket matching feature.

## Best solution

The editor's bracket matching should be aware of language-specific contexts where brackets are not syntactically meaningful (like Markdown prose or math blocks), either by improving the bracket-matching heuristic or by providing a finer-grained setting that suppresses unmatched-bracket highlighting without also disabling bracket selection-highlight.
