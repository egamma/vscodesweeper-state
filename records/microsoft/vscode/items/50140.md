[https://github.com/microsoft/vscode/issues/50140](https://github.com/microsoft/vscode/issues/50140)

## Summary

Keep open: this is a highly upvoted Backlog feature request, and current main only shows limited experimental Tree-sitter tokenization rather than general first-class support for Tree-sitter grammars. The existing `feature-request`, `languages-basic`, and `tokenization` labels are appropriate.

## Change summary

Support Tree-sitter grammars for syntax highlighting in addition to TextMate grammars.

## Best solution

Continue tracking this in the tokenization/languages-basic backlog until VS Code either provides a stable first-class Tree-sitter grammar contribution/API path or explicitly scopes the experiment to a smaller supported set of built-in languages.
