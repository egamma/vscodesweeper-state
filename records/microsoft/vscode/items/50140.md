# Issue #50140 — Support syntax highlighting with tree-sitter

[https://github.com/microsoft/vscode/issues/50140](https://github.com/microsoft/vscode/issues/50140)

## Summary

Tree-sitter tokenization is actively being developed on main behind experimental flags (editor.experimental.preferTreeSitter.*) but is not yet the default for any language — the feature request remains valid as a tracking issue for full tree-sitter syntax highlighting support.

## Change summary

Support tree-sitter grammars for syntax highlighting as an alternative to TextMate grammars.

## Best solution

Continue the existing experimental tree-sitter tokenization work (src/vs/editor/common/model/tokens/treeSitter/) toward graduating it from experimental to stable, language by language. This issue serves as the canonical community tracking issue for that effort.
