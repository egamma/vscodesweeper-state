# Issue #50140 — Support syntax highlighting with tree-sitter

[https://github.com/microsoft/vscode/issues/50140](https://github.com/microsoft/vscode/issues/50140)

## Summary

Tree-sitter tokenization is actively being implemented on main as an experimental feature (editor.experimental.preferTreeSitter for TypeScript, CSS, ini, regex), but it is not yet the default or complete for all languages. The issue remains valid as a tracking request for full tree-sitter support.

## Change summary

Support tree-sitter grammars for syntax highlighting as an alternative to TextMate grammars.

## Best solution

Continue the experimental tree-sitter tokenization work already underway in src/vs/editor/common/model/tokens/treeSitter/ and src/vs/workbench/services/treeSitter/. Once stable and covering more languages, graduate from experimental to default. This issue serves as a reasonable canonical tracker for that effort.
