## Summary

Implemented on main: a new setting `editor.hover.showLongLineWarning` (default true) was added in commit c3e06383df1, allowing users to suppress the 'Tokenization is skipped...' hover without changing tokenization behavior.

## Change summary

Request for a way to disable the 'Tokenization is skipped for long lines' hover popup that blocks mouse interaction on large files.

## Best solution

Keep the shipped fix — users can set `"editor.hover.showLongLineWarning": false` to suppress the warning. No further change needed.
