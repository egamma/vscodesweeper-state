# Issue #290109 — Terminal Quick Suggestions popup window overflows with the text editor

[https://github.com/microsoft/vscode/issues/290109](https://github.com/microsoft/vscode/issues/290109)

## Summary

Still reproduces on main: the terminal suggest widget container is resolved to the panel element, creating a stacking context that clips the popup below the editor. An open community PR (#295968) exists but has not merged.

## Change summary

Terminal quick suggestions popup is clipped behind the editor when the terminal panel is small, rendering it unusable without expanding the terminal.

## Best solution

Review and land https://github.com/microsoft/vscode/pull/295968 (or an equivalent fix) that resolves the suggest addon container to the workbench root container instead of the panel, avoiding the stacking context clip. The fix is in `src/vs/workbench/contrib/terminalContrib/suggest/browser/terminal.suggest.contribution.ts` at `_resolveAddonContainer`.
