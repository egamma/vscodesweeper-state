  > Thanks for reaching out. Word suggestions are a configurable feature — you can disable them via settings:
  > 
  > 1. Open Settings (Ctrl+,)
  > 2. Search for `editor.wordBasedSuggestions` and set it to `off`
  > 3. Optionally, search for `editor.quickSuggestions` and set it to `off` for all contexts if you want to disable all automatic suggestions
