# Issue #307576 — Keep/Undo widget sits over code text

[https://github.com/microsoft/vscode/issues/307576](https://github.com/microsoft/vscode/issues/307576)

## Summary

Still reproduces on main: the Keep/Undo overlay widget is positioned with `position: absolute; bottom: 24px; right: 24px` in the editor group container, which can obscure the code changes it refers to. No fix has been applied for this specific overlap concern.

## Change summary

The floating Keep/Undo accept/reject overlay widget for AI edits can obscure the changed code lines it is meant to help review.

## Best solution

Adjust the overlay positioning logic in `chatEditingEditorOverlay.ts` to avoid overlapping visible diff regions — e.g., shift the widget when it would cover modified lines, or integrate it into the editor's scroll/layout-aware widget system rather than a fixed absolute position.
