[https://github.com/microsoft/vscode/issues/188757](https://github.com/microsoft/vscode/issues/188757)

## Summary

Team-authored upstream Electron issue about native title bar menu behavior on Windows. The experimental setting `nativeContextMenuLocation` has since been removed/renamed, but the native menu infrastructure persists and this may still affect users with native title bar style. Hard keep-open: author is MEMBER.

## Change summary

Native title bar menus on Windows cannot be opened to the right of the cursor, only to the left.

## Best solution

This is an Electron-level menu positioning issue. Track whether the current Electron version (and the native context menu service in `src/vs/workbench/services/contextmenu/electron-browser/contextmenuService.ts`) has resolved the positioning problem. If the native title bar menu experience remains broken, file or link the upstream Electron bug.
