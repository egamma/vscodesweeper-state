# Issue #62751 — Git - Provide task progress

[https://github.com/microsoft/vscode/issues/62751](https://github.com/microsoft/vscode/issues/62751)

## Summary

The request for detailed git progress (percentages, object counts during push/pull/fetch) remains unimplemented on main. Only clone parses --progress output; push/pull/fetch show an indeterminate spinner with no granular feedback. With 23 upvotes (near the 20-upvote backlog threshold) and a valid use case, this should stay open.

## Change summary

Show detailed progress information (percentages, object counts) from git stderr during push, pull, and fetch operations, similar to what clone already does.

## Best solution

Extend push/pull/fetch in `extensions/git/src/git.ts` to pass `--progress` and parse git's stderr progress lines (like clone already does), then surface the parsed percentage/stage info through the existing `withProgress` notification UI in `extensions/git/src/repository.ts`.
