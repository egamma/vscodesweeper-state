# Issue #294633 — Missing folder from Windows Explorer

[https://github.com/microsoft/vscode/issues/294633](https://github.com/microsoft/vscode/issues/294633)

## Summary

Verified duplicate of #293182. The Windows Explorer folder context menu was lost after the Windows 11 context menu migration in 1.109; fixed by PR #294661 (1.110) and further hardened by PR #300752 (1.112) which auto-repairs the folder entry during background updates.

## Change summary

Windows Explorer 'Open with Code' context menu entry for folders disappeared after an auto-update on Windows.
