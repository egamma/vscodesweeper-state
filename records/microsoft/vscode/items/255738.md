# Issue #255738 — Build: Universal build flakes at "Set hardened entitlements"

[https://github.com/microsoft/vscode/issues/255738](https://github.com/microsoft/vscode/issues/255738)

## Summary

Active, unresolved CI flake in the macOS universal build's entitlement signing step. Multiple team members are investigating and multiple mitigations have been tried (logging, retries, osx-sign update) but the flake continues to recur. The issue remains necessary.

## Change summary

The macOS universal build intermittently fails at the 'Set hardened entitlements' codesign step, requiring manual job restarts.

## Best solution

Continue investigating the root cause — likely a transient keychain access race on the Azure Pipelines macOS agents. The retry mechanism (PR #270324) hasn't fully resolved it. A more robust fix might involve killing stale codesign processes before retries or coordinating keychain access across parallel signing tasks in the build pipeline.
