# Issue #185770 — Cursor should snap to nearest sash for easy resize

[https://github.com/microsoft/vscode/issues/185770](https://github.com/microsoft/vscode/issues/185770)

## Summary

Valid feature request for cursor snap-to-sash behavior; partially addressed by `workbench.sash.size` but the snap UX remains unimplemented. A maintainer experimented with a prototype, indicating interest. Low upvotes but in Backlog — keeping open per community process.

## Change summary

Add a keyboard+mouse gesture that snaps the cursor to the nearest sash for easier resize, similar to i3 window manager behavior.

## Best solution

If pursued, implement a modifier+click gesture in the sash widget layer (`src/vs/base/browser/ui/sash/`) that warps the pointer to the nearest sash edge. The maintainer's prototype (comment 3) suggests feasibility. Alternatively, this could remain addressed by the existing `workbench.sash.size` setting for most users, with the snap feature deferred until more community demand materializes.
