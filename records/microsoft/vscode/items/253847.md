# Issue #253847 — Side loaded extensions should not sync metadata from MP

[https://github.com/microsoft/vscode/issues/253847](https://github.com/microsoft/vscode/issues/253847)

## Summary

Valid feature request to improve security posture of side-loaded extensions by preventing metadata sync from Marketplace and clarifying the update button. No implementation found on main; keeping open in Backlog.

## Change summary

Side-loaded extensions should not sync metadata from the Marketplace, and their update button should read 'Update to Marketplace Version' to distinguish from normal updates.

## Best solution

Implement the two changes in the extension management layer: (1) guard metadata sync in extensionManagementService to skip side-loaded extensions until explicitly updated, and (2) change the update button label in extensionsWorkbenchService when the source is a side-loaded extension being promoted to a Marketplace version.
