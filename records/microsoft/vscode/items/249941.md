# Issue #249941 — Git - git blame tool

[https://github.com/microsoft/vscode/issues/249941](https://github.com/microsoft/vscode/issues/249941)

## Summary

Team-authored feature request for a git blame agent-mode tool. The existing git blame infrastructure in extensions/git (inline decorations, status bar, hover) does not expose a language-model tool for agent use. Keeping open per team-member policy.

## Change summary

Add a git blame language-model tool for agent mode so the agent can cross-reference commit/PR messages when writing documentation.

## Best solution

Register a new `LanguageModelTool` in extensions/git (alongside the existing blame.ts infrastructure) that accepts a file path and line range, runs `git blame`, and returns structured commit/PR metadata. The existing `GitBlameInformationManager` in extensions/git/src/blame.ts already has the plumbing to call `git blame` and parse results — the tool would wrap that for agent consumption.
