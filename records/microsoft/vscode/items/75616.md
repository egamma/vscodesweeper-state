Team-authored issue discussing whether proxy settings should be machine-scoped. The discussion between members shows an unresolved design question about remote vs. local machine network contexts. Keeping open per team-authored policy.

## Change summary

Tag proxy settings (http.proxy, http.proxyStrictSSL, etc.) as machine-scoped so they don't roam across machines.
