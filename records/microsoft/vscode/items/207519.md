# Issue #207519 — Multi Diff doesn't seem to have Inline View toggle

[https://github.com/microsoft/vscode/issues/207519](https://github.com/microsoft/vscode/issues/207519)

## Summary

The request for a Toggle Inline View action in the workbench multi-diff editor is still valid. The recent commit 0e125587d34 added this toggle only to the Agents window Changes editor, not to the general multi-diff editor used in the main workbench (SCM 'View All Changes'). Issue #237200 was closed as a duplicate of this one, confirming this is the canonical tracker.

## Change summary

Add an Inline View toggle to the multi-diff editor's toolbar/context menu in the main workbench, matching what single-file diffs already offer.

## Best solution

Register a ToggleInlineView action in `src/vs/workbench/contrib/multiDiffEditor/browser/actions.ts` (or contribution) that calls `multiDiffEditorWidget.toggleRenderSideBySide()` — the underlying widget API already exists (added in 0e125587d34). The action just needs to be wired into the EditorTitle menu with the `multiDiffEditorActiveId` context key, mirroring how it was done for the Agents window in `src/vs/sessions/contrib/changes/browser/changesViewActions.ts`.
