# Issue #293877 — Time tracking for AI Chat interactions

[https://github.com/microsoft/vscode/issues/293877](https://github.com/microsoft/vscode/issues/293877)

## Summary

Time tracking for AI chat interactions is a specialized project-management capability that fits cleanly in an extension rather than core VS Code. The request — timestamps, durations, session aggregation, CSV/JSON export — is layered on top of chat and doesn't require core changes to deliver.

## Change summary

Add per-interaction time tracking metadata (timestamps, response durations, session aggregates, CSV/JSON export) to the AI Chat experience.

## Best solution

Ship this as a Marketplace extension that hooks into chat events via the extension API. The chat extension API already exposes turn-level events that an extension could timestamp and aggregate. If specific API surface is missing to make this possible, a narrower issue requesting that API gap would be appropriate for core.
