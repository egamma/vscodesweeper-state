# Issue #281526 — Cropped text should have ellipises

[https://github.com/microsoft/vscode/issues/281526](https://github.com/microsoft/vscode/issues/281526)

## Summary

The long distance hint widget has ellipsis styles on some inner elements (breadcrumb items, outline container), but the specific narrow-resize scenario reported may still produce cropped text without visible ellipsis. No commit references this issue, so it likely remains open work.

## Change summary

When the NES long jump widget is resized narrow, text is cropped without showing ellipsis.

## Best solution

Ensure the top-level bar container and all text-bearing children properly propagate overflow constraints so that `text-overflow: ellipsis` fires when the widget is narrowed. The fix belongs in `src/vs/editor/contrib/inlineCompletions/browser/view/inlineEdits/inlineEditsViews/longDistanceHint/inlineEditsLongDistanceHint.ts`.
