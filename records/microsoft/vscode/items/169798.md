# Issue #169798 — When in debug REPL, tab key changes focus instead of indents

[https://github.com/microsoft/vscode/issues/169798](https://github.com/microsoft/vscode/issues/169798)

## Summary

Still unimplemented on main: the Debug REPL input editor has no tab-trapping or mode-toggle logic — tab still moves focus out. Team members discussed an approach (Chrome DevTools-style behavior or Ctrl+M toggle) but no implementation landed. Keeping open as an accepted backlog item with accessibility implications.

## Change summary

Pressing Tab in the Debug REPL input should insert indentation instead of moving focus away.

## Best solution

Implement Chrome DevTools-style tab behavior in the REPL input (tab inserts indent except on the first line or at end of content) or adopt the editor's Ctrl+M tab-focus toggle, ideally as a shared behavior across all SimpleEditorWidgets as isidorn suggested. The relevant code is in `src/vs/workbench/contrib/debug/browser/repl.ts` and `src/vs/workbench/contrib/codeEditor/browser/simpleEditorOptions.ts`.
