# Issue #260732 — Weird UI issues with 1.103 on a Lenovo ThinkPad X13s Gen 1 (arm) with Ubuntu 25.04

[https://github.com/microsoft/vscode/issues/260732](https://github.com/microsoft/vscode/issues/260732)

## Summary

Upstream Chromium/mesa GPU rendering bug on ARM64 (Qualcomm Adreno) that has been resolved by platform updates — Chromium 140+ and Ubuntu 25.10 ship the fix. Multiple commenters confirm the issue is gone; no VS Code-side change was needed or made.

## Change summary

GPU rendering corruption on Lenovo ThinkPad X13s (ARM64) with Ubuntu 25.04, starting in VS Code 1.103 (Electron upgrade).

## Best solution

Close as resolved upstream. The root cause was a Chromium GPU rasterization bug affecting ARM64 Adreno GPUs (tracked at https://issues.chromium.org/issues/441317241), fixed in newer Chromium and mesa versions shipped with Ubuntu 25.10. Users on older Ubuntu can use `--disable-gpu` or update their OS.
