# Issue #165219 — [json] Support json-schema draft 2020-12

[https://github.com/microsoft/vscode/issues/165219](https://github.com/microsoft/vscode/issues/165219)

## Summary

Still an open gap — the bundled vscode-json-languageservice explicitly flags $dynamicRef/$dynamicAnchor as unsupported. Active upstream work (microsoft/vscode-json-languageservice#308) is in progress but not yet merged or consumed.

## Change summary

JSON Schema draft 2020-12 keywords ($dynamicRef, $dynamicAnchor, $vocabulary) are not supported for IntelliSense/validation in nested schema objects.

## Best solution

Merge the upstream PR at https://github.com/microsoft/vscode-json-languageservice/pull/308, then bump the dependency in extensions/json-language-features/server/package.json. The feature lives entirely in the language service library, not in VS Code core.
