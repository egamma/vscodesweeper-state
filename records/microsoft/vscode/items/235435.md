# Issue #235435 — GPU rendering: workbench.colorCustomizations for brackets only applies after window reload

[https://github.com/microsoft/vscode/issues/235435](https://github.com/microsoft/vscode/issues/235435)

## Summary

Fixed on main by commit e10604b8373 which clears the GPU decoration CSS cache on theme change, so bracket color customizations now apply immediately without a window reload. Shipped in 1.109.0.

## Change summary

GPU-rendered bracket pair colors from workbench.colorCustomizations only applied after a window reload, not immediately on settings change.

## Best solution

Keep the shipped fix (commit e10604b8373, VS Code 1.109.0). The GPU renderer now listens to onDidColorThemeChange and invalidates its CSS rule cache and texture atlas, causing bracket colors to refresh on the next frame.
