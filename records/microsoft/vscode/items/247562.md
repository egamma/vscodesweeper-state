# Issue #247562 — Refine alt text: Code action changes more than one image

[https://github.com/microsoft/vscode/issues/247562](https://github.com/microsoft/vscode/issues/247562)

## Summary

Still reproduces on main: the 'Refine alt text' code action passes `vscode.window.activeTextEditor?.selection` as `initialRange` to inline chat instead of the specific image's range, so inline chat edits a broader context that includes other images. Team-authored issue — keeping open.

## Change summary

The 'Refine alt text' code action modifies alt text on multiple images in the document instead of only the one the cursor is on.

## Best solution

Pass the image's line range from `provideAltTextQuickFix` through the command arguments and use it as `initialRange` in `doGenerateAltText`, so inline chat is scoped to only the target image line. The fix touches `inlineChatCodeActions.ts` (add range to `ImageCodeAction` args) and `inlineChatCommands.ts` (use that range instead of `activeTextEditor?.selection`).
