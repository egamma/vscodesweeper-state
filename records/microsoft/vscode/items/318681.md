# Issue #318681 — Add support for local-only tracked changes in Source Control

[https://github.com/microsoft/vscode/issues/318681](https://github.com/microsoft/vscode/issues/318681)

## Summary

Valid feature request with an open PR from the same author; closely related to #313007 which asks for the same capability. Keeping open because the linked PR is an active implementation candidate.

## Change summary

Add a 'Local Only' SCM resource group so tracked files with local modifications can be excluded from commits without modifying .gitignore or using raw git plumbing.

## Best solution

Evaluate whether this belongs in core or as an extension — the SCM extension API already supports custom resource groups, so this could be built as a Marketplace extension wrapping skip-worktree. If kept in core, consolidate with https://github.com/microsoft/vscode/issues/313007 which asks for the same thing. Review the open PR https://github.com/microsoft/vscode/pull/318881 to decide.
