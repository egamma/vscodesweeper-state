# Issue #285762 — Sticky scroll showing in git diff editor

[https://github.com/microsoft/vscode/issues/285762](https://github.com/microsoft/vscode/issues/285762)

## Summary

Fixed on main: the terminal sticky scroll overlay state was never actually updated due to a missing assignment, so switching to the alt buffer didn't hide it. Commit 401b925cbdb adds the missing `this._state = state` line.

## Change summary

Terminal sticky scroll overlay incorrectly remains visible when switching to the alternate buffer (e.g. vim as git commit editor).

## Best solution

Keep the shipped fix in commit 401b925cbdb; no further change needed. The `_setState` method now correctly updates internal state before acting on it, so entering the alt buffer hides the overlay as intended.
