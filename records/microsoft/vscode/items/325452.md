# Issue #325452 — Terminal quick fix icon overlaps left side bar

[https://github.com/microsoft/vscode/issues/325452](https://github.com/microsoft/vscode/issues/325452)

## Summary

Still reproduces on main — team-authored issue, keeping open. The quick fix decoration icon uses a negative margin (`MarginLeft = -17`) that can push the icon outside the terminal container and into the sidebar when the terminal is positioned next to it.

## Change summary

The terminal quick fix lightbulb/sparkle icon overlaps the left sidebar because the decoration's negative margin is not clipped to the terminal bounds.

## Best solution

Either clip overflow on the terminal decoration container, or adjust the decoration positioning logic in `src/vs/workbench/contrib/terminal/browser/xterm/decorationStyles.ts` so that `MarginLeft = -17` does not push the icon outside the terminal's visible area. A CSS `overflow: hidden` on the terminal container or a boundary check before applying the margin would resolve this.
