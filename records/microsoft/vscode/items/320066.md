# Issue #320066 — GitHub Copilot Chat view cannot be scrolled using Page Up/Page Down keys, only mouse wheel works

[https://github.com/microsoft/vscode/issues/320066](https://github.com/microsoft/vscode/issues/320066)

## Summary

This is a duplicate of the open canonical issue #226194 ('Scroll lists by page'), which tracks exactly this gap: PageUp/PageDown in list-based views (especially chat) only moves focus between items rather than scrolling by a page, so it fails for tall items like long chat responses.

## Change summary

PageUp/PageDown keys do not scroll the Copilot Chat panel; only mouse wheel and scrollbar work.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/226194, which is in the Backlog and tracks adding a true page-scroll command for list/tree widgets — implementing that delivers exactly the behavior requested here.
