[https://github.com/microsoft/vscode/issues/321598](https://github.com/microsoft/vscode/issues/321598)

## Summary

Team-authored bug in the Agents window: the Changes view does not refresh when files are modified externally (via file system events). Keeping open per team-member policy.

## Change summary

The Agents window's Changes view does not react to file system change events, so externally modified files don't appear until a manual refresh.

## Best solution
