# Issue #323937 — can't select text in the ask questions widget to copy.

[https://github.com/microsoft/vscode/issues/323937](https://github.com/microsoft/vscode/issues/323937)

## Summary

Team-authored papercut bug — text in the ask-questions widget (chat question carousel) is not selectable for copying. The CSS shows `user-select: none` on `.chat-question-list-item` which may suppress selection on question text that should be copyable. Keeping open per team-member policy.

## Change summary

Users cannot select and copy text from the agent's ask-questions widget in the chat view.

## Best solution

Audit `user-select` rules in `chatQuestionCarousel.css` — the `.chat-question-list-item` at line 224 uses `user-select: none` which likely suppresses text selection on the question text. Add `user-select: text` on the question label/description elements within the list item so that the question text is selectable while keeping the interactive list-item click behavior intact.
