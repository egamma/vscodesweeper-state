## Summary

Still reproduces on main. The open PR #286895 has not been merged. When multiple cursors have empty selections and copy is performed, `pasteOnNewLine` causes `_distributePasteToCursors` to bail out (line 673), and the full text is pasted identically at each cursor via `_simplePaste` instead of distributing one line per cursor above each position.

## Change summary
