[https://github.com/microsoft/vscode/issues/303371](https://github.com/microsoft/vscode/issues/303371)

## Summary

This is the same Node.js system certificate crash on macOS that was fixed in November 2025 by defaulting `http.systemCertificatesNode` to `false`. The reporter's workaround confirms the root cause. The canonical issue #277064 is closed and verified; ongoing tracking lives in #287739.

## Change summary

Extension host crashes with exit code 6 on macOS 26.4 due to Node.js system certificate loading.

## Best solution

Close as duplicate of the canonical issue https://github.com/microsoft/vscode/issues/277064. The reporter likely had `http.systemCertificatesNode` set to `true` explicitly; the default has been `false` since 1.106 (commit 16cd8a75cc7). Ongoing work to make the setting safe to re-enable is tracked by https://github.com/microsoft/vscode/issues/287739.
