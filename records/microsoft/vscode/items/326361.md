# Issue #326361 — Model picker no longer offers 200k context option for Claude Sonnet — only 1M

[https://github.com/microsoft/vscode/issues/326361](https://github.com/microsoft/vscode/issues/326361)

## Summary

The available context window variants in the model picker are determined by the GitHub Copilot service, not VS Code core. Since Anthropic charges the same price regardless of context window size, offering only the 1M option is an intentional simplification — there is no cost-saving benefit from a smaller window.

## Change summary

Request to restore the 200k context window option for Claude Sonnet models in the Copilot Chat model picker.

## Best solution

Current behavior is intentional: Anthropic's uniform pricing means the 200k variant offered no cost advantage, so it was removed to simplify the model picker. Users who want to limit context sent per request can manage this via prompt length and attached context rather than a context window toggle.
