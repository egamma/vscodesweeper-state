# Issue #325709 — Can't select text in questions widget's question

[https://github.com/microsoft/vscode/issues/325709](https://github.com/microsoft/vscode/issues/325709)

## Summary

Team-authored issue, keeping open. The bug is confirmed in current source: the `.chat-question-title` element lacks the `user-select: text` override that its siblings (`.chat-question-description`, `.chat-question-carousel-message`) already have.

## Change summary

Text in the question carousel's title/question area cannot be selected or copied because `user-select: text` is not set on `.chat-question-title`.

## Best solution

Add `user-select: text; -webkit-user-select: text;` to the `.chat-question-title` rule in `chatQuestionCarousel.css` (around line 89), matching the existing pattern used for `.chat-question-description` and `.chat-question-carousel-message` at lines 480–501.
