# Issue #244780 — SCM Graph - Show current branch

[https://github.com/microsoft/vscode/issues/244780](https://github.com/microsoft/vscode/issues/244780)

## Summary

Valid feature request: the user wants a graph filter mode that shows only the current branch (and optionally its remote), excluding the merge base. Current `Auto` mode always includes all three refs (local, remote, merge base) — confirmed in current source. The request has low community traction (2 upvotes) but a clear use case from a contributor who frequently switches branches for PR review.

## Change summary

Add an SCM graph filter option that tracks only the currently checked-out branch without including the merge base reference.

## Best solution

Add a new filter mode (e.g. `Current Branch`) in `_resolveHistoryItemFilter()` in `src/vs/workbench/contrib/scm/browser/scmHistoryViewPane.ts` that includes only `historyItemRef` and optionally `historyItemRemoteRef`, omitting `historyItemBaseRef`. This keeps the graph focused on the checked-out branch for users who don't want merge-base context.
