# Issue #302369 — Agent vs Plan modes mix up on multiple chats

[https://github.com/microsoft/vscode/issues/302369](https://github.com/microsoft/vscode/issues/302369)

## Change summary

Running two concurrent chat sessions with different modes (Agent and Plan) causes mode state to leak — the Agent session loses edit tool permissions.
