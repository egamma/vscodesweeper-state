# Issue #76545 — Hyper modifier is unknown

[https://github.com/microsoft/vscode/issues/76545](https://github.com/microsoft/vscode/issues/76545)

## Summary

This should remain open because it is a Backlog feature request with an assigned milestone, and the hard keep-open rule applies regardless of age. The request is still a concrete keybinding capability gap: recognizing Hyper/Mod3 as a modifier in the keybinding recorder and shortcut dispatch.

## Change summary

Allow Linux users who map Caps Lock to Hyper/Mod3 to bind and use Hyper-based VS Code keyboard shortcuts.

## Best solution

Keep this tracked in the keybindings area until maintainers decide whether VS Code's keyboard event normalization should expose Hyper/Mod3 as a supported modifier on Linux, likely in the keyboard/keybinding handling layer rather than as an extension.
