- **Proposed comment:**

  > Thanks for the detailed report! The setting you're looking for already exists:
  > 
  > 1. Open Settings (Ctrl+,)
  > 2. Search for `workbench.fontAliasing`
  > 3. Set it to **`none`** — this disables font smoothing entirely, giving you sharp pixel edges (no ClearType/antialiasing)
  > 
  > You can also try `antialiased` for pixel-level (rather than sub-pixel) smoothing if `none` is too harsh.
  > 
  > Additionally, you can set `"disable-lcd-text": true` in your `argv.json` (run the command **Preferences: Configure Runtime Arguments**) to disable LCD text rendering at the Chromium level.
  > 
  > Since this is already available, I'm closing this as answered. Please open a new issue if the existing settings don't meet your needs.
