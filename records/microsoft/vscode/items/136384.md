# Issue #136384 — Expose preRestartTask

[https://github.com/microsoft/vscode/issues/136384](https://github.com/microsoft/vscode/issues/136384)

## Summary

Valid feature request: `preRestartTask` and `postRestartTask` are implemented in the debug service code and defined in the IConfig interface, but never exposed in the launch.json JSON schema — users cannot configure them. Keeping open as a real gap.

## Change summary

Expose preRestartTask (and postRestartTask) in launch.json so users can configure a build task to run on debug restart without stopping/starting.

## Best solution

Add `preRestartTask` and `postRestartTask` properties to the launch configuration JSON schema in `src/vs/workbench/contrib/debug/common/debugSchemas.ts`, mirroring the existing `preLaunchTask`/`postDebugTask` schema entries. The runtime code already handles them.
