## Change summary

Copilot CLI sessions do not appear in the Agents window session list, creating a fragmented experience between CLI and the agent-focused UI.

## Best solution

Unify session discovery so the Agents window lists sessions from all agent harnesses (CLI, Chat view, cloud).
