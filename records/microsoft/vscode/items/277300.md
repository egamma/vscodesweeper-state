# Issue #277300 — Google Authentication stopped working in 1.106

[https://github.com/microsoft/vscode/issues/277300](https://github.com/microsoft/vscode/issues/277300)

## Summary

Fixed on main and shipped in 1.107.0: the `http.systemCertificatesNode` setting was enabled by default in 1.106, which broke SSL certificate verification for some users (including Google OAuth). It was reverted to `false` by default three days after this report.

## Change summary

Google Authentication (and other HTTPS connections) broke in 1.106 because the new `http.systemCertificatesNode` setting was enabled by default, causing 'unable to get issuer certificate' errors.

## Best solution

Keep the shipped fix — `systemCertificatesNode` defaults to `false` since 1.107.0. No further change needed; the regression is resolved.
