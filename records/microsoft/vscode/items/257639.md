# Issue #257639 — right-to-left display in search mode

[https://github.com/microsoft/vscode/issues/257639](https://github.com/microsoft/vscode/issues/257639)

## Summary

The find-highlight decoration disrupts RTL text layout because the editor's find decorations have no RTL awareness — this is a specific manifestation of the broad RTL support gap tracked by the canonical issue #11770, which absorbs this work.

## Change summary

Searching in RTL text causes words on the matched line to shift position; exiting search restores them.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/11770, which tracks comprehensive RTL language support including editor rendering and decorations. The find-decoration RTL fix is part of that broader effort.
