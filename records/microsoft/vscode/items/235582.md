Valid feature request for caps lock detection in password input prompts. No implementation exists on current main; the quick input and input box components have no caps lock awareness. Low upvote count (1) means it stays in Backlog Candidates under the community process.

## Change summary

Show a caps lock warning indicator when typing into password input prompts (quick pick password boxes).

Add a KeyboardEvent listener for caps lock state in the quick input password mode (src/vs/platform/quickinput/browser/) and display a warning message when detected, similar to how browsers and OS login dialogs handle this. The feature is small and self-contained within the quick input layer.
