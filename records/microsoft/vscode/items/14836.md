# Issue #14836 — Add "find all occurences" in the current file feature

[https://github.com/microsoft/vscode/issues/14836](https://github.com/microsoft/vscode/issues/14836)

## Summary

Highly upvoted (231 👍) feature request for a one-click 'Find All in Current File' that lists all matches in a panel, similar to Visual Studio. VS Code offers workarounds (Search sidebar with filesToInclude, Search Editor, Select All Occurrences) but none delivers the seamless single-action experience requested. Two community PRs remain open. Well above the 20-upvote Backlog threshold — should stay open.

## Change summary

Add a dedicated 'Find All Occurrences in Current File' action that lists all matches in a results panel, accessible directly from the editor find widget.

## Best solution

Implement a dedicated command (e.g. in the find widget or context menu) that opens the Search sidebar or Search Editor pre-scoped to the active file with the current find query. This could build on the existing `findInFilesCommand` with `filesToInclude` set to `${relativeFile}`, exposed as a single action. Review the open community PRs (https://github.com/microsoft/vscode/pull/128915, https://github.com/microsoft/vscode/pull/7559) for prior implementation attempts.
