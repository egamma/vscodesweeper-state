# Issue #296895 — Bracket pair colourization pairs improperly for curly braces

[https://github.com/microsoft/vscode/issues/296895](https://github.com/microsoft/vscode/issues/296895)

## Summary

Not a colorization bug — font ligatures visually merge curly-brace glyphs, making correctly-colored brackets appear mismatched. Contributor RedCMD identified the cause and linked numerous prior duplicates; this is the same confusion tracked by the canonical issue.

## Change summary

Curly-brace bracket pair colorization appears mismatched when font ligatures are enabled.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/155554 which tracks the font-ligatures-vs-bracket-colorization confusion. The fix for the user is `"editor.fontLigatures": false` or using a font that doesn't ligate braces.
