Team-authored issue (MEMBER) reporting that Claude Opus 4.5 incorrectly displays a '(preview)' suffix in the model picker. The model name shown in the picker comes from the `name` field in `ILanguageModelChatMetadata` provided by the Copilot extension at registration time — the VS Code core model picker (`chatModelPicker.ts`) renders whatever name the extension supplies. Keeping open for the team to address, likely on the extension side.
