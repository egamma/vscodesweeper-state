# Issue #124393 — [json] show number of children in JSON arrays

[https://github.com/microsoft/vscode/issues/124393](https://github.com/microsoft/vscode/issues/124393)

## Summary

Valid feature request in Backlog with 25 upvotes; not yet implemented on main. The JSON hover provider does not show array/object child counts.

## Change summary

Request to show the number of children in JSON arrays/objects, either on hover or inline like debug variable previews.

## Best solution

Enhance the JSON language server's hover provider (in `vscode-json-languageservice`) to include element count for arrays and property count for objects when hovering over `[` or `{`. This likely lives upstream in the `vscode-json-languageservice` package rather than in the built-in extension alone.
