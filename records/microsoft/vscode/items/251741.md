# Issue #251741 — Add C++ Debugging to MCP development mode

[https://github.com/microsoft/vscode/issues/251741](https://github.com/microsoft/vscode/issues/251741)

## Summary

Valid feature request from a team member to extend MCP dev mode debugging beyond Node.js and Python to support C++/gdb. The hard keep-open rule applies (MEMBER author). The current implementation in mcpPlatformTypes.ts explicitly limits the debug type union to 'node' | 'debugpy'.

## Change summary

Add C++ (gdb/cppdbg) debugging support to the MCP development mode debug property, which currently only supports Node.js and Python/debugpy.

## Best solution

Extend the `IMcpDevModeConfig.debug` discriminated union in `src/vs/platform/mcp/common/mcpPlatformTypes.ts` with a `{ type: 'cppdbg' }` variant (or similar), add a corresponding case in `McpDevModeDebugging.transform()` in `src/vs/workbench/contrib/mcp/common/mcpDevMode.ts` that launches a cppdbg attach session, and update the configuration schema to advertise the new type.
