# Issue #184679 — Fuzzy search not working well in Recent Folders quickpick in 1.79.0

[https://github.com/microsoft/vscode/issues/184679](https://github.com/microsoft/vscode/issues/184679)

## Summary

The issue is triaged (Backlog, labeled quick-pick) and the root cause was identified by a team member. Related issue #194448 covers broader fuzzy matching inconsistencies. Keep open as an accepted backlog bug.

## Change summary

Fuzzy search in the Open Recent quickpick fails to match items when the query spans both the folder name and the remote host name.

## Best solution

Improve the quickpick filter in `src/vs/platform/quickinput/browser/quickInputList.ts` to concatenate label+description when matching, or ensure remote host info is always in the label where fuzzy matching can find it. May be resolved together with https://github.com/microsoft/vscode/issues/194448.
