# Issue #316242 — Copilot CLI agent panel: cannot attach screenshots or images when a multiple-choice ask_user prompt is active

[https://github.com/microsoft/vscode/issues/316242](https://github.com/microsoft/vscode/issues/316242)

## Summary

Feature request to allow image/screenshot attachments when a multiple-choice elicitation prompt is active in the Copilot agent panel. The current question carousel and elicitation UI intentionally presents a structured choice interface without freeform image input. No duplicate found.

## Change summary

Allow pasting or dragging images into the multiple-choice ask_user prompt in the Copilot agent panel, so users can provide visual context alongside or instead of selecting a predefined choice.

## Best solution

The question carousel part (`chatQuestionCarouselPart.ts`) and elicitation content part (`chatElicitationContentPart.ts`) would need to support image attachment alongside structured choices. This could mean adding a drop/paste target to the 'other' freeform input or surfacing a secondary input area. Worth evaluating whether the agent harness protocols (Copilot SDK, Claude, Codex) can even receive image data in an elicitation response before investing in the UI work.
