## Summary

Team-authored polish issue that remains unfixed on main. The variable decoration colors in the chat input editor (`chatSlashCommandForeground`) appear to have insufficient contrast against the background, particularly on hover. No changes to the relevant color definitions or decoration types address this since the issue was filed.
