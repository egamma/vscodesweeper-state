## Summary

The downgrade prevention is intentional VS Code behavior (line 1079–1080 of extensionManagementService.ts). The root cause is the github.copilot extension declaring a dependency on an older copilot-chat version than what ships built-in — this is a transient packaging issue on the extension side, not a VS Code core bug.
