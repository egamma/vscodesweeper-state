# Issue #52103 — [folding] unfold when pressing enter on last line

[https://github.com/microsoft/vscode/issues/52103](https://github.com/microsoft/vscode/issues/52103)

## Summary

This feature request — automatically unfolding a region when pressing Enter on its last visible line — has not been implemented on main. The folding contribution handles content changes only by re-computing ranges, with no special Enter-on-last-line unfold logic. A team member (aeschli) acknowledged the request in 2018 and it sits in the Backlog.

## Change summary

Pressing Enter on the last visible line of a folded region should unfold that region so the new line is visible.

## Best solution

Add logic in the folding contribution (src/vs/editor/contrib/folding/browser/folding.ts) to detect when a newline is inserted at the end of a folded region's start line and automatically unfold the region, similar to how `unfoldOnClickAfterEndOfLine` handles clicks.
