# Issue #289679 — Partial Completion doesnt work with cmd+right

[https://github.com/microsoft/vscode/issues/289679](https://github.com/microsoft/vscode/issues/289679)

## Summary

The partial accept (Cmd+Right) for Next Edit Suggestions (inline edits) was intentionally removed in PRs #241721 and #244968. The keybinding still works for traditional inline suggestions (ghost text completions), but not for the newer inline edit predictions, which is by design.

## Change summary

Cmd+Right does not partially accept next edit predictions word-by-word.

## Best solution

Close as designed — the team deliberately removed the partial accept flow for inline edits. Cmd+Right still works for traditional inline suggestions. If partial accept for NES is desired as a new feature, a separate feature request could track that.
