[https://github.com/microsoft/vscode/issues/303498](https://github.com/microsoft/vscode/issues/303498)

## Summary

This is a js-debug source map resolution issue that belongs in the microsoft/vscode-js-debug repository, not the main VS Code repo. The sourceMapPathOverrides and absSourceRoot logic live entirely in vscode-js-debug.

## Change summary

Breakpoints in TypeScript are not hit when Angular's servePath option prepends a path segment, because js-debug's absSourceRoot computation bakes the servePath into the filesystem path where sourceMapPathOverrides cannot intercept it.

## Best solution
