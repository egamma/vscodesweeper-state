# Issue #180056 — Would it be possible to create some way to identify that the zen mode is active?

[https://github.com/microsoft/vscode/issues/180056](https://github.com/microsoft/vscode/issues/180056)

## Summary

Valid accessibility feature request with an assigned 'On Deck' milestone and an open PR (#305701) already implementing it — must stay open.

## Change summary

Screen reader users cannot tell whether zen mode is active because the status bar (where such info would live) is hidden by default in zen mode.

## Best solution

Land the open PR https://github.com/microsoft/vscode/pull/305701 which adds a zen mode indicator to the window title, making it discoverable by screen readers regardless of status bar visibility.
