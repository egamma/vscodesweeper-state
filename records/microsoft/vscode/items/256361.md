# Issue #256361 — Allow to keep/discard changes/files while the agent runs

[https://github.com/microsoft/vscode/issues/256361](https://github.com/microsoft/vscode/issues/256361)

## Summary

Team-authored issue (MEMBER); must keep open. The request is still valid — current code gates keep/discard buttons behind `ctxHasRequestInProgress.negate()`, preventing interaction during agent runs.

## Change summary

Allow users to keep or discard individual file changes while an agent session is still running, rather than waiting for completion.

## Best solution

Relax the `ctxHasRequestInProgress.negate()` guard in `chatEditingEditorActions.ts` (lines ~206 and ~374) so the keep/discard/review buttons remain available during agent runs, while ensuring the underlying accept/discard logic handles concurrent modifications safely.
