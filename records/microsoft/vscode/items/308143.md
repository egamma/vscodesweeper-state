# Issue #308143 — Why No Hyperlink to Open a workspace on the Welcome page?

[https://github.com/microsoft/vscode/issues/308143](https://github.com/microsoft/vscode/issues/308143)

## Summary

The Welcome page already displays recently opened folders and workspaces in the "Recent" section, which provides clickable links to reopen them. The "Start" section is for creating/opening new items. Opening a folder IS opening a project — the warning about replacing a workspace is standard behavior when switching contexts.

## Change summary

User wants a dedicated "Open Workspace" link in the Welcome page's Start section to reopen existing projects.

## Best solution

This is working as designed: the Welcome page's Recent section (visible on the right or below the Start section) already lists previously opened folders and workspaces as clickable links. The File > Open Recent menu also provides this. No change needed.
