# Issue #277736 — Support Right-to-Left (RTL) Text for Persian in Copilot Chat

[https://github.com/microsoft/vscode/issues/277736](https://github.com/microsoft/vscode/issues/277736)

## Summary

Valid feature request in the Backlog with 22 upvotes and an open PR (#314365) that directly addresses it. The PR is a substantial community contribution covering bidirectional text support across editor and chat surfaces — keep open and let the PR review process proceed.

## Change summary

Persian and other RTL text in Copilot Chat renders left-to-right, making mixed RTL/LTR content hard to read.

## Best solution

Review and land (or iterate on) the open community PR https://github.com/microsoft/vscode/pull/314365, which adds a configurable `editor.textDirection` setting and applies `dir="auto"` / `unicode-bidi: plaintext` handling across chat markdown rendering, editor lines, and workbench labels — directly solving this request along with several related RTL issues.
