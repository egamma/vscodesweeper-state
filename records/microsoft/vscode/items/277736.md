# Issue #277736 — Support Right-to-Left (RTL) Text for Persian in Copilot Chat

[https://github.com/microsoft/vscode/issues/277736](https://github.com/microsoft/vscode/issues/277736)

## Summary

Active PR #314365 explicitly lists this issue in its 'Fixes' list and introduces a comprehensive bidirectional text direction model for the editor and chat surfaces. The PR is open and not yet merged — keep the issue open until it lands.

## Change summary

RTL text in Copilot Chat renders left-to-right, making Persian/Arabic/Hebrew mixed with English unreadable.

## Best solution

Land PR https://github.com/microsoft/vscode/pull/314365 which introduces `editor.textDirection` with contextual/auto/auto-follow modes and propagates direction to chat markdown rendering. This issue will be closed automatically when that PR merges.
