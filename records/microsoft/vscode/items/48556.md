# Issue #48556 — [folding] fold all but comments

[https://github.com/microsoft/vscode/issues/48556](https://github.com/microsoft/vscode/issues/48556)

## Summary

Valid feature request for a 'Fold All Except Comments' command. Not yet implemented on main — the existing commands are `Fold All Block Comments` (folds comments) and `Fold All Except Selected` (cursor-based), neither of which satisfies this request. Has 6 upvotes and 3 closed duplicates pointing back here, so it's the canonical issue for this ask.

## Change summary

Request a command that folds all code regions but leaves comment blocks unfolded, for quick file overview.

## Best solution

Add a `FoldAllExceptCommentsAction` in `src/vs/editor/contrib/folding/browser/folding.ts` that calls `setCollapseStateLevelsDown` for all regions then `setCollapseStateForType(foldingModel, FoldingRangeKind.Comment.value, false)` to unfold comment ranges — mirroring the existing `FoldAllBlockCommentsAction` pattern but inverted. Should be registered as `editor.foldAllExceptComments`.
