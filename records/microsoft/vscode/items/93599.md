# Issue #93599 — Can't drag files from a remote vscode window to a local window

[https://github.com/microsoft/vscode/issues/93599](https://github.com/microsoft/vscode/issues/93599)

## Summary

Team-authored feature request with 40 upvotes on On Deck; the cross-window remote-to-local file DnD capability remains unimplemented on current main. Must keep open per MEMBER authorship rule.

## Change summary

Allow dragging files from a remote VS Code window's explorer into a local VS Code window's explorer to copy them.

## Best solution

Implement cross-window remote-to-local file transfer on drop, likely via IPC between windows (as sketched in comment 11 by bpasero): the local window broadcasts to the remote window to resolve the file contents. The earlier WIP PR #98817 was closed without merging; a fresh implementation in the explorer DnD and workbench DnD layers is needed.
