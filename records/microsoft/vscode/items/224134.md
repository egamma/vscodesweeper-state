# Issue #224134 — OnClick and hover events on gutter icons

[https://github.com/microsoft/vscode/issues/224134](https://github.com/microsoft/vscode/issues/224134)

## Summary

Valid feature request with 50 upvotes (above Backlog threshold). The shipped `editor/lineNumber/context` API only covers right-click context menus; direct onClick and hover events on gutter decorations remain unimplemented in the extension API.

## Change summary

Expose onClick and onHover events for gutter decoration icons to extensions, enabling click-to-toggle and hover-to-preview patterns similar to breakpoints.

## Best solution

Design and finalize extension API events for gutter decoration click and hover interactions (e.g. `onDidClickGlyphMarginDecoration`, hover provider for glyph margin). This builds on the finalized `editor/lineNumber/context` menu (https://github.com/microsoft/vscode/issues/175945) but requires new event surface in the editor API layer. The 50 upvotes place it above the Backlog promotion threshold.
