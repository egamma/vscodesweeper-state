# Issue #314723 — [Feature Request] Display specific Model ID (e.g., Codex 5.3) in the Context Window tooltip

[https://github.com/microsoft/vscode/issues/314723](https://github.com/microsoft/vscode/issues/314723)

## Summary

Valid feature request to show the active model name in the context-window tooltip during a chat request. The current ChatContextUsageDetails widget shows token counts and cost but not the model identity. No duplicate found; routing to the chat area for triage.

## Change summary

Display the active model ID (e.g. 'Codex 5.3') in the context-window usage tooltip while a chat request is in progress.

## Best solution

Add a model-name row to ChatContextUsageDetails (src/vs/workbench/contrib/chat/browser/widgetHosts/viewPane/chatContextUsageDetails.ts) fed from the resolved model on the current request/response. The data is already available via the language-models service; it just needs to be surfaced in the details UI.
