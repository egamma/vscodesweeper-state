## Summary

Valid feature request for dynamic JSON schema association API. A team member acknowledged it as a long-standing gap. No proposed or finalized API exists on main; the issue remains relevant.

## Change summary

Expose the JSON schema registry to extensions so they can dynamically associate schemas with files without static glob patterns.

## Best solution

Add a proposed extension API (e.g., `vscode.languages.registerJsonSchemaAssociation`) that wraps `IJSONContributionRegistry.registerSchemaAssociation`, allowing extensions to dynamically bind schemas to specific documents. This would also resolve https://github.com/microsoft/vscode/issues/230136.
