Still open on main — the `FinishedCallback` interface in the Copilot extension still returns `Promise<number | undefined>` without any documentation clarifying whether the promise is awaited by callers. jrieken is a core team member; this is a targeted tech-debt cleanup they identified after a real bug (https://github.com/microsoft/vscode/issues/286731).

## Change summary

## Best solution
