# Issue #232488 — Hovering warning icon in terminal tab is difficult to action

[https://github.com/microsoft/vscode/issues/232488](https://github.com/microsoft/vscode/issues/232488)

## Summary

Still reproduces on main: the terminal tab decoration icon (warning) uses the list decoration infrastructure which doesn't share the label's hover groupId, so the hover vanishes when moving the mouse toward it. No fix has landed.

## Change summary

Hovering the warning icon on a terminal tab produces a hover that disappears before the user can interact with it.

## Best solution

Give the decoration icon its own hover with the same `groupId: 'terminal-tabs-list'` so hovering transitions seamlessly from the icon to the hover content, or extend the list decoration infrastructure to support hover groups.
