# Issue #212973 — 'Folders' menu item is disabled in the explorer

## Summary

Still reproduces on main: the Folders view descriptor sets `canToggleVisibility: false`, so the menu item is always greyed out. Author is a MEMBER — keeping open for maintainer judgment.

## Change summary

The 'Folders' entry in the Explorer ellipsis menu is always disabled because its view descriptor disallows visibility toggling.

## Best solution

Needs a maintainer decision on whether hiding the core file tree should be permitted or if there are edge cases to handle.
