# Issue #175499 — Git - Please add "View diff with Working File" command to timeline context menu

[https://github.com/microsoft/vscode/issues/175499](https://github.com/microsoft/vscode/issues/175499)

## Summary

Valid feature request: the timeline context menu currently only offers 'Open Changes' (diff with previous revision) but not a diff against the working file. An open PR (https://github.com/microsoft/vscode/pull/307255) already targets this issue — keep open and let that PR land or be closed before further action.

## Change summary

Add a 'View diff with Working File' command to the Git timeline context menu so users can compare any past revision against the current working copy.

## Best solution

Review and land https://github.com/microsoft/vscode/pull/307255, which adds the 'Open Changes with Working File' command to the timeline context menu in the git extension. The implementation would live in `extensions/git/src/commands.ts` (new command) and `extensions/git/package.json` (menu contribution under `timeline/item/context`).
