# Issue #174359 — Add expand/collapse audio cues

[https://github.com/microsoft/vscode/issues/174359](https://github.com/microsoft/vscode/issues/174359)

## Summary

Team-authored accessibility feature request on the 'On Deck' milestone — hard keep-open. A `foldedArea` signal exists but no general expand/collapse audio cues for breadcrumbs or tree widgets have been implemented.

## Change summary

Add audio cues that fire when expanding/collapsing UI components (breadcrumbs, trees) and differentiated pitches for heading levels in markdown.

## Best solution

Register new `AccessibilitySignal` entries for expand/collapse actions in tree/breadcrumb widgets, and consider a pitch-differentiated signal for heading levels (possibly keyed off Outline depth). Work belongs in `src/vs/platform/accessibilitySignal/` for signal registration and the relevant widget contributions for firing them.
