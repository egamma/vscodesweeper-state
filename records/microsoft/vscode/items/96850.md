# Issue #96850 — Git - "Revert selected ranges" doesn't work for staged chunks

[https://github.com/microsoft/vscode/issues/96850](https://github.com/microsoft/vscode/issues/96850)

## Summary

Valid feature request: "Revert Selected Ranges" does not work in the staged (Index) diff editor. The command's `when` clause requires `resourceScheme == file` (excluding the Index diff view), and internally it only fetches working-tree diff information. The maintainer (lszomoru) confirmed in Dec 2024 that the necessary data is now available to implement this. Keeping open in Backlog.

## Change summary

Allow the "Revert Selected Ranges" command to work when viewing staged (Index) diffs, so users can revert individual staged hunks without first unstaging the entire file.

## Best solution

Extend `revertSelectedRanges` in `extensions/git/src/commands.ts` to detect when the active editor is an Index diff view, fetch staged diff information via `getIndexDiffInformation()`, and apply the revert against the staged content. The `when` clause in `extensions/git/package.json` also needs updating to allow the command in Index diff editors (removing or relaxing `resourceScheme == file`). The staging helpers in `extensions/git/src/staging.ts` already have the necessary `getIndexDiffInformation` function.
