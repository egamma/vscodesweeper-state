# Issue #320250 — Side Scroll speed control / recent increase if side scroll speed.

[https://github.com/microsoft/vscode/issues/320250](https://github.com/microsoft/vscode/issues/320250)

## Summary

Feature request to split `editor.mouseWheelScrollSensitivity` into separate horizontal and vertical settings. No existing setting covers horizontal-only sensitivity, and no recent change explains the reported speed increase — likely a mouse driver or OS update changed the deltaX values sent to the app.

## Change summary

Add a separate horizontal scroll sensitivity setting so users can independently control side-scroll speed.

## Best solution

Add an `editor.mouseWheelHorizontalScrollSensitivity` option (similar to the existing vertical one) in `src/vs/editor/common/config/editorOptions.ts` and wire it through `editorScrollbar.ts` / `scrollableElement.ts`. Alternatively, respect the Windows per-character side-scroll system setting. Either way this is a small, well-scoped editor feature.
