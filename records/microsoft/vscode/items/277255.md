# Issue #277255 — Git - Add option to compare against base branch

[https://github.com/microsoft/vscode/issues/277255](https://github.com/microsoft/vscode/issues/277255)

## Summary

Team-authored feature request (MEMBER) — must keep open. The requested `comparisonMode` parameter for base-branch comparison has not been implemented on current main; the SCM changes tool still only supports working-tree states (`unstaged`/`staged`/`merge-conflicts`).

## Change summary

Add a comparison mode to the agent's SCM changes tool so it can diff HEAD against a base branch (e.g. origin/main) for PR-like workflows.

## Best solution

Extend `IGetScmChangesToolParams` in `extensions/copilot/src/extension/tools/node/scmChangesTool.ts` with a `comparisonMode` (e.g. `'working-tree' | 'against-base'`) and an optional `baseRef` parameter, then use the git extension API's diff capabilities (`repository.diff(baseRef)` or equivalent) to produce the base-branch diff. This aligns with agent-session PR review flows in the Agents window.
