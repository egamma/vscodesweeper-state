# Issue #266643 — Explain Changes with Copilot

[https://github.com/microsoft/vscode/issues/266643](https://github.com/microsoft/vscode/issues/266643)

## Summary

Valid feature request — an 'Explain Changes' context menu action exists for committed history items (SCMHistoryItemContext) but not for uncommitted/staged files (SCMResourceContext). The gap is real; keeping open as a labeled backlog item.

## Change summary

Add an 'Explain Changes with Copilot' context menu option for uncommitted files in the Source Control panel.

## Best solution

Extend the chat-integration pattern from `scmHistoryChatContext.ts` to register an 'Explain Changes' action on `MenuId.SCMResourceContext` that attaches the file's working-tree diff to the chat widget and sends a summarization prompt. The existing `summarizeHistoryItem` action (line 280–304 of `scmHistoryChatContext.ts`) is a close template. This is a small, localized addition in the SCM contribution area.
