# Issue #227119 — Editor GPU: Support custom glyphs

[https://github.com/microsoft/vscode/issues/227119](https://github.com/microsoft/vscode/issues/227119)

## Summary

Valid feature request: the GPU editor renderer has no custom glyph rendering for box-drawing/block characters. No implementation exists on current main; authored by a team contributor, so it stays open.

## Change summary

Render box-drawing and block element Unicode characters pixel-perfectly in the GPU editor renderer, similar to the terminal's custom glyph support.

## Best solution

Implement a custom glyph rasterizer in `src/vs/editor/browser/gpu/raster/` that draws box-drawing (U+2500–U+257F) and block element (U+2580–U+259F) characters programmatically, bypassing the font, so they connect perfectly regardless of line-height/letter-spacing settings.
