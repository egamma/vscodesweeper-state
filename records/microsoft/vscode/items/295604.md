# Issue #295604 — Show skipped questions

[https://github.com/microsoft/vscode/issues/295604](https://github.com/microsoft/vscode/issues/295604)

## Summary

Still valid on current main: when questions are skipped, `renderSkippedMessage()` only shows a bare 'Skipped' label without revealing the question content. The existing `renderSummary()` has the Q&A rendering logic but short-circuits to `renderSkippedMessage()` when no answers exist. The request to show the actual question text alongside the skip state remains unimplemented.

## Change summary

When Copilot Chat questions are skipped (or the window reloads), the user cannot see what the questions were — only a 'Skipped' label is shown.

## Best solution

Modify `renderSkippedMessage()` in `chatQuestionCarouselPart.ts` to render the question titles/messages (similar to how `renderSummary()` does for answered questions) alongside the 'Skipped' status, so users can see what was asked even when the carousel was dismissed.
