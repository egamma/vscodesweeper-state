# Issue #214389 — Integrated Terminal Distorts Text Upon Restart After Being Terminated

[https://github.com/microsoft/vscode/issues/214389](https://github.com/microsoft/vscode/issues/214389)

## Summary

Confirmed bug still present on main: terminal gets stuck at a restricted size when killed and recreated while the panel is split between Problems/Debug Console/Terminal. A team member confirmed this in December 2025.

## Change summary

Terminal renders at a distorted/restricted size when killed and restarted in a panel split with Problems and Debug Console.

## Best solution

Fix the terminal layout recalculation in the terminal group/tabbed view when a new terminal instance is created in an existing split panel — likely the terminal's dimensions aren't being recomputed against the current panel allocation in `terminalTabbedView.ts` or `terminalGroup.ts`.
