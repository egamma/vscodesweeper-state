[https://github.com/microsoft/vscode/issues/208978](https://github.com/microsoft/vscode/issues/208978)

## Summary

Partially addressed: next/previous change navigation was added to the multi-diff editor in commit a8907697a59 (Oct 2025), but `hideUnchangedRegions` remains hardcoded to `true` in the multi-diff editor rather than respecting the user's `diffEditor.hideUnchangedRegions.enabled` setting. ## Change summary

The git stash view (multi-diff editor) should respect the user's `diffEditor.hideUnchangedRegions.enabled` setting and support next/previous change navigation.

This likely belongs in the `multi-diff-editor` area.
