# Issue #185537 — Cannot select between a sequence of unicode flags

[https://github.com/microsoft/vscode/issues/185537](https://github.com/microsoft/vscode/issues/185537)

## Summary

Keep open as a real editor-core grapheme navigation bug. Current main still treats adjacent Regional Indicator code points as unbreakable without the state needed for Unicode GB12/GB13 pairing, which matches the reported inability to place the cursor between consecutive flags.

## Change summary

Cursor movement and selection cannot stop between consecutive Unicode flag emoji such as 🇩🇲🇩🇴.

## Best solution

Implement stateful Regional Indicator grapheme segmentation in src/vs/base/common/strings.ts so long flag sequences break into flag pairs, then add focused cursor/selection tests for consecutive flags.
