# Issue #315962 — Add ability to write custom commit message in VSCode Agents App

[https://github.com/microsoft/vscode/issues/315962](https://github.com/microsoft/vscode/issues/315962)

## Summary

Valid feature request — the agents window commit skill currently only generates AI commit messages with no UI for the user to supply a custom one. No duplicate found; keeping open for the area owner.

## Change summary

Allow users to write or edit the commit message in the VS Code Agents window instead of relying solely on AI-generated messages.

## Best solution

Add an editable commit-message input (or a confirmation step that lets the user override the generated message) to the agents-window commit flow. The commit skill (`src/vs/sessions/skills/commit/SKILL.md`) could instruct the agent to present the draft message for user editing before running `git commit`.
