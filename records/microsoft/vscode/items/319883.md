# Issue #319883 — GPT-4.1 does not work in Agents App

[https://github.com/microsoft/vscode/issues/319883](https://github.com/microsoft/vscode/issues/319883)

## Summary

A team member (lramos15) already triaged this, confirming GPT-4.1 was deprecated server-side on June 2, 2026, and acknowledging the UI incorrectly rendered the dead model. The immediate error is resolved by the server-side deprecation, but the UI-side handling of deprecated models remains the real bug to track.

## Change summary

GPT-4.1 appeared in the model picker and returned 400 errors after being deprecated by GitHub on June 2, 2026.

## Best solution

Ensure the model picker gracefully handles models that become deprecated or unavailable — either by filtering them out when CAPI no longer serves them, or by showing a clear deprecation state instead of letting users select a dead model. The member's comment suggests this is partly a server-side fix (CAPI stops listing the model) and partly a UI resilience issue in the model picker.
