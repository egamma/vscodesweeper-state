# Issue #303634 — Sessions: Want to be able to create new branch from branch picker

[https://github.com/microsoft/vscode/issues/303634](https://github.com/microsoft/vscode/issues/303634)

## Summary

Valid feature request from a team member: the Agents window branch picker currently only lists existing branches and has no 'create new branch' option. Keeping open — team-authored issue requires maintainer judgment.

## Change summary

Add the ability to create a new branch directly from the Agents window branch picker when starting a new session.

## Best solution

Add a 'Create New Branch…' action item to the branch picker in `src/vs/sessions/contrib/providers/copilotChatSessions/browser/branchPicker.ts`. The `showPicker` method builds an `IActionListItem[]` from existing branches — append a special item that, on select, prompts for a branch name and calls the appropriate git service to create it before setting it as the session branch.
