# Issue #308449 — The dialog cannot be closed

[https://github.com/microsoft/vscode/issues/308449](https://github.com/microsoft/vscode/issues/308449)

## Summary

The parent repository notification can reappear after dismissal because `showParentRepositoryNotification` has no guard against being called again. An open PR already targets this fix.

## Change summary

The parent repository notification dialog reappears after being dismissed, making it effectively uncloseable.

## Best solution

Land (or close) the open PR https://github.com/microsoft/vscode/pull/309256, which adds a session-scoped flag to prevent the notification from reappearing after dismissal.
