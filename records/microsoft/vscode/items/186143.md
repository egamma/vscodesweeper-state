# Issue #186143 — File quick open should allow to trigger "Go to line"

[https://github.com/microsoft/vscode/issues/186143](https://github.com/microsoft/vscode/issues/186143)

## Summary

Feature request from a core maintainer to add a dedicated 'Go to Line' drilling mode (with help text and preview) in file quick open, analogous to the existing `@` symbol drilling. The `<file>:<line>` navigation works but lacks the UX polish of a dedicated mode. The attempted PR #186657 was closed without merging; the gap remains on main.

## Change summary

Add a 'Go to Line' drilling mode in file quick open (triggered by `:`) with helper text and preview, similar to the `@` symbol drilling mode.

## Best solution

Implement a goto-line drilling UX in `src/vs/workbench/contrib/search/browser/anythingQuickAccess.ts` that, when the user types `<file>:`, switches into a dedicated line-picker mode with helper text and editor preview — mirroring how `@` enters symbol search. The closed PR #186657 provides a starting reference for the approach.
