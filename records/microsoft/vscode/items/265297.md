# Issue #265297 — Expose command and context key to close Dirty Diff inline view

[https://github.com/microsoft/vscode/issues/265297](https://github.com/microsoft/vscode/issues/265297)

## Summary

The original ask (command + context key to close dirty diff) already exists as `closeQuickDiff` with `dirtyDiffVisible`, confirmed by commenters. However, the issue has evolved into a broader request — focusing the widget and opening diff at cursor — with an open PR (#305691) addressing part of it. Keeping open for those remaining asks.

## Change summary

Expose a keyboard-accessible command and context key to close (and focus) the Dirty Diff inline widget.

## Best solution

The close command (`closeQuickDiff` + Escape keybinding + `dirtyDiffVisible` context key) is already shipped. The remaining value is a focus command (`editor.action.dirtydiff.focus`) and the open PR https://github.com/microsoft/vscode/pull/305691 for showing diff at cursor. Land or close that PR, and consider adding a focus command in `src/vs/workbench/contrib/scm/browser/quickDiffWidget.ts`.
