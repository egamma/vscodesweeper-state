# Issue #316919 — I want to attach file diff to AI Chat

[https://github.com/microsoft/vscode/issues/316919](https://github.com/microsoft/vscode/issues/316919)

## Summary

The core capability to attach SCM history item changes to chat already exists (shipped in 1.105.0), but the user's request for clearer UX distinction between 'file content at revision' vs 'file diff' and for diff-editor gutter integration are valid refinements that aren't fully addressed.

## Change summary

Add distinct context menu options to attach file content vs. file diff from SCM Graph, and allow attaching individual hunks from the diff editor gutter to chat.

## Best solution

The existing `addHistoryItemChangeToChat` action attaches the file change entry, but its menu label ('Add to Chat') doesn't distinguish content-at-revision from diff. Two improvements: (1) clarify the SCM Graph context menu to offer both options explicitly, and (2) add an 'Add to Chat' action on the diff editor gutter (revert-change arrows) for individual hunks. The SCM/chat area owners should evaluate whether the current `scmHistoryItemChange` resolution already provides diff context to the model or only the file snapshot.
