## Summary

Valid feature request asking for pull/push/fetch/sync icon buttons in the SCM title bar. Currently only Commit and Refresh are in the navigation group; the rest live in the overflow menu. A team member (lszomoru) engaged in Dec 2024 to gather specifics, and the reporter elaborated. Low upvote count (1) so not yet eligible for Backlog promotion — keeping open under discussion.

## Change summary

Add icon buttons for git pull, push, fetch, and sync to the Source Control view's title toolbar instead of burying them in the overflow menu.

## Best solution

A UX design decision for the SCM area owner: promote some subset of pull/push/fetch/sync from the `1_header` overflow group to the `navigation` group in the `scm/title` menu contributions in `extensions/git/package.json`. The tradeoff is toolbar width vs. discoverability — the team member already asked for specifics, so this is waiting on a design call.
