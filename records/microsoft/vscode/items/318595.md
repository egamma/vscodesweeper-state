# Issue #318595 — Copilot Chat extension causes extension host to constantly restart

[https://github.com/microsoft/vscode/issues/318595](https://github.com/microsoft/vscode/issues/318595)

## Summary

Active investigation by team members (TylerLeonhardt, DonJayamanne) with a confirmed reproduction path: large ~/.claude session data causes the Copilot Chat extension to hang and crash-loop the extension host. The reporter's latest comment narrows the root cause; the agent host migration may resolve it. Keep open for the team to act on.

## Change summary

Copilot Chat extension hangs and crash-loops the extension host when loading large Claude Code session data from ~/.claude.

## Best solution

The Copilot Chat extension needs to handle large Claude session directories gracefully — either by paginating/lazily loading session data, or by completing the agent host migration so this work no longer runs in the extension host. Team members are already investigating both paths.
