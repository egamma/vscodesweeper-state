# Issue #309039 — [Unhandled Error] Cannot read properties of null (reading 'getLineContent')

[https://github.com/microsoft/vscode/issues/309039](https://github.com/microsoft/vscode/issues/309039)

## Summary

Fixed on main: commit 526381b4801 adds a null guard for `this._editor.model` in the word replacement view, preventing the crash when the editor model is null during tab close.

## Change summary

Closing an editor tab while an inline edit suggestion is visible crashes with a null dereference on the editor model's `getLineContent`.

## Best solution

Keep the shipped fix in commit 526381b4801 — the unsafe `this._editor.model.get()!` is now `this._editor.model.read(undefined)` with an early return on null. No further work needed.
