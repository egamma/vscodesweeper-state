# Issue #189059 — Left click on Ghost text should show hover and pass focus

[https://github.com/microsoft/vscode/issues/189059](https://github.com/microsoft/vscode/issues/189059)

## Summary

Team-authored feature request for improving ghost text (inline completions) discoverability via click interaction — kept open per policy for team-authored issues.

## Change summary

Left-clicking on inline completion ghost text should show the hover widget and pass focus; right-clicking should open the standard editor context menu instead of hiding the ghost text.

## Best solution

Implement click handlers on the inline completion decoration so that left-click triggers the inline completions hover (showing accept/dismiss actions) and right-click opens the normal editor context menu. The work lives in the inline completions controller area (`src/vs/editor/contrib/inlineCompletions/browser/`).
