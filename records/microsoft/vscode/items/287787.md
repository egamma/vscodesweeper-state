# Issue #287787 — Model selected for custom agents when editing previous chat messages is unintuitive

[https://github.com/microsoft/vscode/issues/287787](https://github.com/microsoft/vscode/issues/287787)

## Summary

Fixed on main: commit 4293b057ca2 seeds the inline edit input from the edited request's modelId, so editing a previous message now preserves the model that was originally used rather than reverting to the agent's default.

## Change summary

Editing a previously sent chat message reverts the model picker to the custom agent's default model instead of preserving the model the user actually selected.

## Best solution

Keep the shipped fix in commit 4293b057ca2 (1.123.0); no further change needed.
