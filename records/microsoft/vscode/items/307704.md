# Issue #307704 — Cannot navigate to full list of deprecated extensions via Tab key

[https://github.com/microsoft/vscode/issues/307704](https://github.com/microsoft/vscode/issues/307704)

## Summary

Team-authored accessibility bug about Tab navigation in deprecated extensions list; must keep open per policy. A team member suggested arrow keys may be the expected navigation pattern here, but the issue remains open for resolution.

## Change summary

Tab key only focuses the first deprecated extension in the list; subsequent Tab presses skip the remaining deprecated extensions.

## Best solution

Clarify whether the deprecated extensions list should support Tab navigation (like individual focusable items) or arrow-key navigation (like a list widget). If it's a list widget, ensure arrow keys work correctly and the accessibility role communicates this; if individual items should be Tab-focusable, fix the tab order. The extensions view contributions likely live in src/vs/workbench/contrib/extensions/.
