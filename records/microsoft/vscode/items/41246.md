# Issue #41246 — Git - add of file with unicode name fails in Git panel

[https://github.com/microsoft/vscode/issues/41246](https://github.com/microsoft/vscode/issues/41246)

## Summary

Fixed on main and shipped in 1.99.0: the git extension now uses relative paths for git commands, eliminating the NFC/NFD mismatch in absolute pathspecs that caused unicode folder/file names to fail. A contributor confirmed the fix in comment #30.

## Change summary

Git operations (stage, etc.) fail when file or folder names contain unicode characters, especially on macOS where Finder uses NFD decomposition.

## Best solution

Keep the shipped fix from commits 04260cb (#242889) and 82839a3 (#245001); no further change needed. Close as completed.
