# Issue #131203 — Configurable scroll padding

[https://github.com/microsoft/vscode/issues/131203](https://github.com/microsoft/vscode/issues/131203)

## Summary

The requested functionality already exists via `editor.cursorSurroundingLines` (vertical padding) and `editor.scrollBeyondLastColumn` (horizontal padding), both of which predate this issue. This can be closed as already covered.

## Change summary

Allow configuring vertical and horizontal scroll padding around the cursor.

## Best solution

Close as as-designed — the existing settings `editor.cursorSurroundingLines` (vertical, since 2019) and `editor.scrollBeyondLastColumn` (horizontal) already provide this configurability.

  > Thanks for the suggestion! This is actually already possible with existing settings:
  > 
  > - **`editor.cursorSurroundingLines`** — controls the minimum number of visible lines above and below the cursor (vertical padding). Known as `scrollOff` in Vim.
  > - **`editor.scrollBeyondLastColumn`** — controls extra horizontal scroll padding.
  > 
  > These have been available since 2019 and should cover the use case described here. Closing as the functionality is already provided.
