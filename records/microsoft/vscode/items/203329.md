## Summary

Team-authored feature request for synchronized sash positioning across all embedded diff editors in the multi diff editor. Keeping open per policy (MEMBER author).

## Change summary

Moving the diff sash in one editor within the multi diff editor should synchronize that position across all other editors in the view.

## Best solution

Add a shared/synchronized sash position mode to the multi diff editor so that adjusting the split in one embedded diff editor applies the same ratio to all others. Likely lives in `src/vs/editor/browser/widget/multiDiffEditor/`.
