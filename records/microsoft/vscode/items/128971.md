# Issue #128971 — [json] create links for values of format `uri-reference`

[https://github.com/microsoft/vscode/issues/128971](https://github.com/microsoft/vscode/issues/128971)

## Summary

Still unimplemented on main: the JSON language service's `findLinks` only handles `$ref` JSON pointer links, not values with `format: uri-reference` in their schema. The feature request remains valid and is labeled `help wanted`.

## Change summary

JSON values declared as `format: uri-reference` in their schema should be rendered as clickable document links.

## Best solution

Extend `findLinks` in `vscode-json-languageservice` (upstream dependency) to detect string nodes whose schema declares `format: uri-reference`, resolve them relative to the document URI, and emit a DocumentLink. The VS Code side (`extensions/json-language-features/server/src/jsonServer.ts`) already calls `languageService.findLinks` so no additional wiring is needed there.
