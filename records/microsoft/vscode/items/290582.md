# Issue #290582 — Unsure how to get back to new welcome without reloading

[https://github.com/microsoft/vscode/issues/290582](https://github.com/microsoft/vscode/issues/290582)

## Summary

Team-authored issue (MEMBER) requesting a way to return to the Agents welcome view after navigating away. Back/forward session navigation was added in commit 0bd3be62b2f (May 2026) but it handles session history, not necessarily the welcome view re-entry. Related issue #293270 (back behavior from welcome) is still open. Keeping open per hard keep-open rule for team-authored issues.

## Change summary

After clicking 'View All Sessions' from the Agents welcome view, there is no way to return to the welcome without reloading the editor.

## Best solution

Ensure the welcome view is part of the session navigation history (back/forward) so users can return to it after navigating to all-sessions or a chat session. The SessionsNavigation class in `src/vs/sessions/services/sessions/browser/sessionNavigation.ts` may need to track the welcome view as a navigable state. Alternatively, keep the welcome as a persistent tab or add an explicit re-entry command. Related open issue https://github.com/microsoft/vscode/issues/293270 tracks a similar concern.
