Team-authored issue; the second checkbox (CLI recognition of --github-auth for serve-web) remains unimplemented on current main — the Rust CLI has no github_auth argument in serve_web.rs, though the Node server side already supports it. Keeping open.

## Change summary

Add --github-auth as a recognized CLI parameter for the `serve-web` command so users can pass a GitHub PAT to authenticate automatically on first load.
