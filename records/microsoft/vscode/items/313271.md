# Issue #313271 — AI auto completion disables dropdown menu

[https://github.com/microsoft/vscode/issues/313271](https://github.com/microsoft/vscode/issues/313271)

## Summary

Current main intentionally suppresses automatic quick-suggest dropdowns while inline completions are showing, so the reported AI/dropdown interaction matches the configured default rather than an untracked regression.

## Change summary

AI inline completions prevent the automatic IntelliSense dropdown from appearing unless AI/inline suggestions are disabled.

## Best solution

Keep the current default interaction, and direct users who prefer the old behavior to set `editor.quickSuggestions.other` to `on` so quick suggestions continue to show in the suggest widget.

- **Triage action:** propose-close

- **Proposed comment:**

  > Thanks for the report. This matches the current editor behavior: the default `editor.quickSuggestions.other` value is `offWhenInlineCompletions`, so automatic quick suggestions are suppressed while inline completions are showing. Users who prefer the previous dropdown behavior can set `editor.quickSuggestions.other` to `on`.

## Evidence

- **issue context:** The reporter says the issue does not occur with all extensions disabled and that disabling AI makes the dropdown work, pointing to the inline-completion/suggest-widget interaction.
- **current default:** src/vs/editor/common/config/editorOptions.ts:3766 sets `editor.quickSuggestions.other` to `offWhenInlineCompletions`, and the schema text at :3775-3776 describes that value as disabling quick suggestions when inline completions are showing.
- **current behavior:** src/vs/editor/contrib/suggest/browser/suggestModel.ts:433-440 waits for inline completions for `offWhenInlineCompletions` and then returns without triggering quick suggestions when inline suggestions are visible/suppressed.
- **maintainer decision:** Commit f08b0cab721efb62b881aafccbadc031366fdcb6 reverted `fb72b9be76b694b456c5b15903e1f63457848ec8`, restoring the `offWhenInlineCompletions` default on current main.
