[https://github.com/microsoft/vscode/issues/243751](https://github.com/microsoft/vscode/issues/243751)

## Summary

Valid feature request with 26 upvotes, promoted to Backlog. The Run and Debug view title bar currently only shows the 'Start Debugging' button; the 'Start Without Debugging' command exists but is not surfaced as a toolbar button in the view container title. No duplicate found.

## Change summary

Add a 'Run Without Debugging' button alongside the existing 'Start Debugging' button in the Run and Debug Activity Bar view title.

## Best solution

Add a `MenuRegistry.appendMenuItem(MenuId.ViewContainerTitle, ...)` entry for `DEBUG_RUN_COMMAND_ID` in `src/vs/workbench/contrib/debug/browser/debugViewlet.ts`, mirroring the existing `DEBUG_START_COMMAND_ID` entry (lines 171–192) with a distinct icon and appropriate order. The debug area owner should decide exact icon and placement.
