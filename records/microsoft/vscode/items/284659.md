# Issue #284659 — File Compare from different directory

[https://github.com/microsoft/vscode/issues/284659](https://github.com/microsoft/vscode/issues/284659)

## Summary

Verified duplicate of #260470, which was closed via the community upvote process. The explorer already supports cross-directory file comparison via Select for Compare / Compare with Selected, and the tab context menu supports comparing two multi-selected tabs via Compare Selected (added in #232841). The single-tab 'Select for Compare' workflow on the tab bar was declined for insufficient community interest.

## Change summary

Request to add 'Select for Compare' and 'Compare with Selected' to the editor tab context menu, and to compare files from different directories.

## Best solution

Close as duplicate of https://github.com/microsoft/vscode/issues/260470. Cross-directory comparison already works in the explorer; tab-level comparison is available via multi-select (Ctrl+click two tabs → Compare Selected). The single-tab select-for-compare workflow on tabs was community-processed and closed.

  > - **Cross-directory comparison already works** in the Explorer sidebar: right-click any file → "Select for Compare", then right-click a second file anywhere in the tree → "Compare with Selected". The files don't need to be in the same folder.
