## Summary

VoiceOver stops reading the code editor after jumping to end of file on macOS. Confirmed still reproducing on 1.119.1 (May 2026). No fix found on main. The root cause may involve a macOS/Chromium accessibility API change but nonetheless manifests as a VS Code bug.
