# Issue #279433 — Mode selection targets the last active chat box.

[https://github.com/microsoft/vscode/issues/279433](https://github.com/microsoft/vscode/issues/279433)

## Summary

This bug was fixed on main in commit fd9cf8e3dd1e (shipped in 1.107.0), which changed the ToggleChatModeAction to look up the correct widget by sessionResource instead of using the last-focused widget. Keeping open because the author is a team member — explicit maintainer judgment needed to close.

## Change summary

Chat mode selection from one editor chat instance incorrectly applies the mode change to whichever chat instance last had focus, instead of the instance whose dropdown was clicked.

## Best solution

Already fixed: commit fd9cf8e3dd1e added sessionResource-based widget lookup to ToggleChatModeAction, shipping in 1.107.0. A maintainer can verify and close as completed.
