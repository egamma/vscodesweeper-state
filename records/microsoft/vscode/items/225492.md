# Issue #225492 — Bracket guide not colored/incorrect with long wrapped line

[https://github.com/microsoft/vscode/issues/225492](https://github.com/microsoft/vscode/issues/225492)

## Summary

Still reproduces on main: the bracket guide rendering in `indentGuides.ts` does not correctly handle lines that wrap beyond the viewport width — the guide position is computed via `visibleRangeForPosition` which can return incorrect/fallback coordinates for off-screen wrapped segments, and the color falls back to the default inactive style.

## Change summary

Bracket pair guides display at wrong column and lose their color when a line wraps beyond the visible viewport width.

## Best solution

Fix the bracket guide position and color computation in `src/vs/editor/browser/viewParts/indentGuides/indentGuides.ts` to account for lines whose content extends beyond the wrapping column — likely by skipping guide rendering for view lines that are continuation wraps of a model line, or by correctly mapping the guide column through the line-wrapping coordinate transform.
