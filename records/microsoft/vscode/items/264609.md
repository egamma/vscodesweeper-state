## Change summary

When a chat model fails due to high demand, offer a 'Try Again with Auto' button instead of just 'Try Again', so the user can seamlessly retry with the Auto model picker.

## Best solution

The error text already distinguishes `isAuto` vs non-auto and suggests switching — the missing piece is an actionable button.
