# Issue #296348 — Missing tooltip for Chat Editing Session line counts

[https://github.com/microsoft/vscode/issues/296348](https://github.com/microsoft/vscode/issues/296348)

## Summary

Valid polish request — the line-count badges in the chat editing session UI lack tooltips. An open PR (#296349) from the same author already attempts to address this; keeping open for maintainer review of that PR.

## Change summary

Add tooltips to the +added / -removed line-count badges in the Chat Editing Session UI so the information is accessible on hover.

## Best solution

Review and land (or close) the author's open PR https://github.com/microsoft/vscode/pull/296349, which adds the requested tooltips. The relevant rendering code lives in `src/vs/workbench/contrib/chat/browser/widget/input/chatInputPart.ts` (overview badge) and `src/vs/workbench/contrib/chat/browser/widget/chatContentParts/chatReferencesContentPart.ts` (per-file badges). Per VS Code conventions, tooltips should use IHoverService rather than native title attributes.
