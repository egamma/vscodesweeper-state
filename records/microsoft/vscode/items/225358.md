# Issue #225358 — Multiline selection in linux

[https://github.com/microsoft/vscode/issues/225358](https://github.com/microsoft/vscode/issues/225358)

## Summary

By design: on Linux, middle-click is reserved for X11 primary clipboard paste (via `editor.selectionClipboard`, default `true`), so middle-click column selection is intentionally disabled. Disabling that setting restores column-select behavior.

## Change summary

Middle-click drag does not produce column selection on Linux, unlike Windows.

## Best solution

This is as-designed. On Linux, middle-click serves the X11 primary selection clipboard by default. Users who want middle-click column selection can set `"editor.selectionClipboard": false`. The commenter's Column Selection Mode toggle also works.
