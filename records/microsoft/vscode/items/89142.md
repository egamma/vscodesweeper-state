# Issue #89142 — [css] text-underline-position value completion is wrong

[https://github.com/microsoft/vscode/issues/89142](https://github.com/microsoft/vscode/issues/89142)

## Summary

Still reproduces on current main: the `under` value is absent from the `text-underline-position` values array in the bundled `vscode-css-languageservice` (v6.3.4). The syntax string correctly lists `under` but it's not offered as a completion. This is an upstream data issue in `vscode-css-languageservice`.

## Change summary

CSS `text-underline-position` completions are missing the `under` value despite it being a standard value.

## Best solution

Fix the CSS property data upstream in `microsoft/vscode-css-languageservice` (the `webCustomData` dataset) to include `under`, `from-font`, `left`, and `right` in the values array for `text-underline-position`, then bump the dependency in VS Code.
