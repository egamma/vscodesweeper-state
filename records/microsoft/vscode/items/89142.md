# Issue #89142 — [css] text-underline-position value completion is wrong

[https://github.com/microsoft/vscode/issues/89142](https://github.com/microsoft/vscode/issues/89142)

## Summary

Still reproduces: the `values` array for `text-underline-position` in `vscode-css-languageservice` (both 6.3.10 and 7.0.0-next.1) is missing `under`, `from-font`, `left`, and `right`. The syntax field is correct but completions only offer `above`, `auto`, `below`. This is an upstream data issue in `@vscode/web-custom-data`.
