## Summary

Confirmed upstream Electron bug (electron/electron#19747) affecting French keyboard layout display in native menus on macOS. VS Code is passing correct key information but Electron misinterprets it. Already correctly labeled as upstream with the issue linked. VS Code now ships Electron 42.5.0 but there's no evidence the upstream bug has been resolved.

## Change summary

French keyboard backtick symbol is displayed as the 'lower than' sign in native macOS menus.

## Best solution

Keep tracking until the upstream Electron fix lands. The root cause is in Electron's native menu accelerator rendering for non-US keyboard layouts (electron/electron#19747). No VS Code-side workaround is practical without dropping native menu accelerator labels.
