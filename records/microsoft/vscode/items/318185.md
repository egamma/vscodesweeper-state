# Issue #318185 — Add a chat.fontWeight setting

[https://github.com/microsoft/vscode/issues/318185](https://github.com/microsoft/vscode/issues/318185)

## Summary

Valid feature request: chat messages support fontFamily and fontSize but not fontWeight, while chat codeblocks do have all three. The gap is confirmed on current main — no chat.fontWeight setting exists.

## Change summary

Add a chat.fontWeight setting to control font weight in chat message text, matching the existing chat.fontFamily and chat.fontSize settings.

## Best solution

Add a `chat.fontWeight` configuration property in `src/vs/workbench/contrib/chat/browser/chat.shared.contribution.ts` alongside the existing `chat.fontSize`/`chat.fontFamily` entries, wire it through `chatLayoutService.ts`, and apply it in the chat widget CSS — mirroring how `chat.editor.fontWeight` is handled for codeblocks.
