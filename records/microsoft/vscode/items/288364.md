# Issue #288364 — Created files should show +x -y diff annotations

[https://github.com/microsoft/vscode/issues/288364](https://github.com/microsoft/vscode/issues/288364)

## Summary

Valid polish request: the SCM changes view already shows +x -y diff stat annotations on modified files but omits them on newly created (untracked/added) files. No fix found on current main — the gap remains.

## Change summary

Newly created files in the SCM changes view should display +x -y diff stat annotations like modified files do.

## Best solution

Extend the SCM resource diff stat rendering in the source control view pane to compute and display insertions for newly created files (all lines are insertions, zero deletions). The logic likely lives in the SCM view pane resource renderer or wherever diff annotations are attached to `ISCMResource` entries.
