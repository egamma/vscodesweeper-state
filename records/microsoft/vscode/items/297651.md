# Issue #297651 — [FEATURE REQUEST]: Include Copilot Chat Tools in Settings Sync

[https://github.com/microsoft/vscode/issues/297651](https://github.com/microsoft/vscode/issues/297651)

## Summary

Legitimate feature request to add chat tool sets as a dedicated Settings Sync category. MCP servers are already synced (SyncResource.Mcp), but user-configured chat tool sets are not yet a separate sync resource.

## Change summary

User wants Copilot Chat tool configurations (tool sets) to be included as a dedicated category in Settings Sync.

## Best solution

Add a new SyncResource entry for chat tool sets (or extend the existing MCP sync) so that user-configured tools sync across machines. The implementation would live in src/vs/platform/userDataSync/common/ alongside mcpSync.ts, and the Settings Sync UI would expose the new category.
