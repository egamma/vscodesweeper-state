# Issue #307493 — Stitch MCP Error

[https://github.com/microsoft/vscode/issues/307493](https://github.com/microsoft/vscode/issues/307493)

## Summary

The error originates from the third-party Stitch MCP server returning images with a mismatched MIME type (declared image/png but actually image/jpeg). VS Code and the Copilot extension correctly pass through the MCP server's declared mimeType; the fix belongs in the Stitch MCP extension, not in VS Code core.

## Change summary

Using the Stitch MCP server with Copilot Chat fails with a 400 error because the MCP server declares images as image/png when they are actually image/jpeg.

## Best solution

Report this to the Stitch MCP extension maintainer — the MCP server needs to declare the correct MIME type for images it returns in tool results. VS Code's MCP client correctly relays the mimeType from the MCP protocol's ImageContent; sniffing or correcting MIME types would violate the protocol contract.
