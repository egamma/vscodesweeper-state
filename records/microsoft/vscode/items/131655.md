# Issue #131655 — [UI/UX] problem with disassembly page.

[https://github.com/microsoft/vscode/issues/131655](https://github.com/microsoft/vscode/issues/131655)

## Summary

Team-authored bug (MEMBER) — must keep open. The disassembly view's current-instruction highlight still only colors the `.instruction` element, not the full row container, leaving a visible gap at the bottom of the row. The defect remains on current main.

## Change summary

The yellow background highlight for the current instruction in the disassembly view doesn't fill the full row height, leaving a gap at the bottom.

## Best solution

In `InstructionRenderer.rerenderBackground()` (src/vs/workbench/contrib/debug/browser/disassemblyView.ts), apply the background color to the row container rather than just the `.instruction` element, so the highlight covers the full row height including any padding/gap.
