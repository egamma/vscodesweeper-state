# Issue #242187 — Notebook find filters icon is always "enabled"

[https://github.com/microsoft/vscode/issues/242187](https://github.com/microsoft/vscode/issues/242187)

## Summary

Still present on main: the notebook find filter button uses a filter/funnel icon with toggle-style styling that makes it appear perpetually 'enabled', even though it's just a dropdown trigger. The reporter suggests switching to an ellipsis icon and possibly merging the two overflow menus. No fix has been committed.

## Change summary

The notebook find widget's filter icon visually appears always toggled/active despite being a plain dropdown action, creating a misleading UI signal.

## Best solution

Change the filter button in `notebookFindReplaceWidget.ts` to use an ellipsis icon instead of the filter/funnel icon, remove the toggle-style active decoration (the `applyStyles` method's active-option border/background), and consider merging the two overflow menus into one. This is a localized polish change in the notebook find widget.
