# Issue #261695 — SCM - Interactive Timeline Scrubber for File History

[https://github.com/microsoft/vscode/issues/261695](https://github.com/microsoft/vscode/issues/261695)

## Summary

A video-player-style timeline scrubber for browsing file git history is a specialized UI capability that fits naturally as an extension built on the existing Timeline API and SCM provider APIs, rather than core functionality — extensions like GitLens already deliver rich interactive git-history exploration in this space.

## Change summary

Add an interactive timeline scrubber component to the editor for scrubbing through a file's git history in real-time.

## Best solution

Build this as a Marketplace extension leveraging the Timeline API (`vscode.workspace.registerTimelineProvider`) and SCM APIs. Extensions like GitLens already provide rich file-history exploration; a scrubber-style UI component would be a natural addition there or in a new dedicated extension, not in VS Code core.

- **Fit:** rejected
