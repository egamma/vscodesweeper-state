# Issue #269250 — [json] show a notification when retrying to load a schema fails

[https://github.com/microsoft/vscode/issues/269250](https://github.com/microsoft/vscode/issues/269250)

## Summary

Valid feature request: when clicking the schema resolution error status bar item and the retry fails, no notification or error detail is shown to the user. The current code only toggles the icon briefly. Still unimplemented on main.

## Change summary

Show a notification with error details when retrying a JSON schema download fails, rather than silently reverting to the warning icon.

## Best solution

In `extensions/json-language-features/client/src/jsonClient.ts`, after `triggerValidation` completes, check if the schema error persists and show a `window.showWarningMessage` with the schema URL and failure reason extracted from the diagnostics in `fileSchemaErrors`.
