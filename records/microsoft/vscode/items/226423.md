# Issue #226423 — Allow more than 9 items in the title bar

[https://github.com/microsoft/vscode/issues/226423](https://github.com/microsoft/vscode/issues/226423)

## Summary

Valid feature request — the hard-coded `maxItems: 9` limit for editor title bar actions is still present on current main. A team member acknowledged it and suggested a width-based approach. Keeping open for implementation.

## Change summary

Editor title bar toolbar is hard-coded to show at most 9 action items, overflowing the rest regardless of available space.

## Best solution

Replace the fixed `maxItems: 9` in `editorTabsControl.ts` with a width-based overflow strategy (as benibenj suggested) that fills available toolbar space dynamically, similar to how the main titlebar already uses `maxItems: 12` but ideally responsive to actual container width.
