## Summary

Team-authored design discussion about whether Claude Agent should receive instruction files from the Prompt service. The latest comment (April 2026) reopens the question after skills/plugins were brought in via PR #312082, suggesting instructions should follow — but with an opt-out mechanism. Keeping open for the team to resolve.

## Change summary

Decide whether the Claude Agent harness should receive VS Code instruction files (copilot-instructions) and allow users to opt out of injecting customizations.

## Best solution

The team should decide the policy: bring instructions into the Claude Agent prompt (consistent with the skills/plugins approach from PR #312082) while providing a user-facing toggle to disable customization injection for users who prefer the vanilla Claude harness experience.
