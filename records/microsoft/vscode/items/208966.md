# Issue #208966 — macOS: state of native window tabs is not restored after restart

[https://github.com/microsoft/vscode/issues/208966](https://github.com/microsoft/vscode/issues/208966)

## Summary

This is a known upstream issue blocked on Electron (issues electron/electron#18138 and electron/electron#40529). A VS Code team member (deepak1556) actively triaged it as recently as January 2026, merging the duplicate #283098 into this one. It remains a valid tracked upstream blocker.

## Change summary

macOS native window tabs (merged windows) are not restored after VS Code restarts, losing the user's tab arrangement.

## Best solution

Continue tracking as an upstream Electron dependency. The fix requires Electron to expose APIs for saving and restoring native tab state. Once the upstream Electron issues are resolved, VS Code can implement tab-state persistence in the window management layer (src/vs/code/electron-main/app.ts, src/vs/platform/windows/electron-main/windowImpl.ts).
