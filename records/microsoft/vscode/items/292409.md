# Issue #292409 — show workspace / repo more clearly in sessions preview from welcome page

[https://github.com/microsoft/vscode/issues/292409](https://github.com/microsoft/vscode/issues/292409)

## Summary

Team-authored polish request to surface workspace/repo context more prominently in the welcome page's recent sessions list. The sessions viewer already tracks repository names (via `getRepositoryName` in `agentSessionsViewer.ts`) and renders them as badges/section labels, but the welcome page's compact session grid may not expose this clearly enough. Keeping open per team-member policy.

## Change summary

Make the workspace or repository associated with each recent session more visible in the welcome page's sessions preview.

## Best solution

Enhance the welcome page's `AgentSessionsControl` rendering (sourced from `agentSessionsViewer.ts`) to display the repository/workspace name more prominently — e.g. as a visible subtitle or badge on each session card, leveraging the existing `getRepositoryName` helper. The relevant code lives in `src/vs/workbench/contrib/chat/browser/agentSessions/agentSessionsViewer.ts` (renderer) and `src/vs/workbench/contrib/welcomeAgentSessions/browser/agentSessionsWelcome.ts` (welcome page integration).
