# Issue #202251 — Compare active file with saved resulting editor missing some features

[https://github.com/microsoft/vscode/issues/202251](https://github.com/microsoft/vscode/issues/202251)

## Summary

Team-authored accessibility feature request for the diff editor: diff signs (+/-) should be announced and outline navigation should work in compare-with-saved and similar diff views. Keeping open per team-author policy.

## Change summary

The diff editor opened via 'Compare active file with saved' (and similar compare commands) lacks screen-reader announcements for +/- diff signs and does not provide outline information for navigation.

## Best solution

Add an outline provider for diff editors (showing changed hunks or the underlying document structure) and ensure diff decorations (+/-) are announced via accessibility signals. The diff-editor contribution area owns both gaps.
