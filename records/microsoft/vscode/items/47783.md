# Issue #47783 — Command - Undo Replace All

[https://github.com/microsoft/vscode/issues/47783](https://github.com/microsoft/vscode/issues/47783)

## Summary

Valid feature request with strong community support (66 upvotes). The bulk edit undo mechanism exists but requires a changed file to be active — the request for a dedicated 'Undo Replace All' button in the Search panel that works regardless of open editors remains unimplemented.

## Change summary

Add an 'Undo Replace All' button to the Search panel that undoes all replacements across files without requiring a changed file to be the active editor.

## Best solution

Add an 'Undo Replace All' action to the Search panel that tracks the last Replace All operation's UndoRedoGroup and triggers undo across all affected files, opening them if needed. The replace service (`src/vs/workbench/contrib/search/browser/replaceService.ts`) already uses `bulkEditorService.apply()` which groups edits — the missing piece is a UI surface and logic to invoke undo on that group without requiring an active editor from the affected set.
