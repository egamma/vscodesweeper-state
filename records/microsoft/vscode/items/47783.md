# Issue #47783 — Command - Undo Replace All

[https://github.com/microsoft/vscode/issues/47783](https://github.com/microsoft/vscode/issues/47783)

## Summary

Valid, well-supported feature request (66 upvotes, well above the 20-upvote Backlog threshold) for a dedicated 'Undo Replace All' command that reliably undoes a cross-file replace regardless of which files are open. The current undo mechanism exists but has UX limitations (requires an affected file to be the active tab). No fix or implementation has landed on main; keeping open in Backlog.

## Change summary

Add a dedicated 'Undo Replace All' command/button in the Search panel that undoes all changes from the last Replace All across all affected files, regardless of which files are currently open.

## Best solution

Implement grouped undo for cross-file Replace All using UndoRedoGroup in the replace service (src/vs/workbench/contrib/search/browser/replaceService.ts) so that a single undo operation reverts all affected files atomically. Surface a button in the Search widget and/or a command. Issue #157533 was already closed as a duplicate of this one, confirming this is the canonical tracker.
