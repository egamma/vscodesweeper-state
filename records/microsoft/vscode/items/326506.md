# Issue #326506 — Add "Scroll By Page" choice for terminal pane

[https://github.com/microsoft/vscode/issues/326506](https://github.com/microsoft/vscode/issues/326506)

## Summary

Legitimate feature request for a terminal-specific 'scroll by page' setting, analogous to the editor and list versions. No duplicate found, and the terminal uses xterm.js's own scrollbar which doesn't inherit VS Code's scrollByPage option.

## Change summary

Add a 'Scroll By Page' setting for the terminal scrollbar, matching the existing editor and list scroll-by-page options.

## Best solution

Expose a `terminal.integrated.scrollByPage` setting that forwards to xterm.js's scrollbar configuration so clicking the scrollbar track scrolls by page rather than jumping to the click position. This would likely involve passing the option through the terminal's xterm configuration in `src/vs/workbench/contrib/terminal/browser/xterm/xtermTerminal.ts`.
