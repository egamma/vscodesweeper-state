# Issue #84899 — SCM - Make registerDiffInformationCommand public

[https://github.com/microsoft/vscode/issues/84899](https://github.com/microsoft/vscode/issues/84899)

## Summary

Still a valid open request: both `registerDiffInformationCommand` (proposed `diffCommand`) and the newer `textEditorDiffInformation` API remain behind `checkProposedApiEnabled` on current main and have not been finalized into the stable extension API. The issue tracks real demand from SCM extension authors (svn-scm, perforce, etc.).

## Change summary

Make `registerDiffInformationCommand` (or its successor `textEditorDiffInformation` API) available as a stable extension API so non-git SCM providers can implement revert/stage selected ranges.

## Best solution

Finalize the `textEditorDiffInformation` proposed API (which supersedes the older `diffCommand` proposal) into the stable `vscode.d.ts`, enabling third-party SCM extensions to implement revert-change and stage-selected-ranges features. The relevant files are `src/vscode-dts/vscode.proposed.textEditorDiffInformation.d.ts` and the guards in `src/vs/workbench/api/common/extHost.api.impl.ts`.
