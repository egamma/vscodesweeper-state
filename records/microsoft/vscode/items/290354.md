# Issue #290354 — Git - single line diff in the inline git blame box

[https://github.com/microsoft/vscode/issues/290354](https://github.com/microsoft/vscode/issues/290354)

## Summary

Legitimate feature request to show the line-level diff (hunk) directly in the inline git blame hover, and optionally link to the single-file diff for the commit. No duplicate found — the built-in blame annotation currently shows commit metadata but not the associated diff for the line.

## Change summary

Show the diff hunk for the current line and a link to the single-file diff in the inline git blame hover/decoration.

## Best solution

Enhance the blame hover in `extensions/git/src/blame.ts` (and `extensions/git/src/hover.ts`) to include the diff hunk for the blamed line's commit, and add a command link to open the single-file diff. This is a reasonable enhancement to the built-in git blame feature that shipped in VS Code — the infrastructure for retrieving commit diffs already exists in the git extension.
