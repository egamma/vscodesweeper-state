# Issue #263856 — Option to Auto-Accept or Bypass "Keep"/"Undo" for Copilot Chat Edits

[https://github.com/microsoft/vscode/issues/263856](https://github.com/microsoft/vscode/issues/263856)

## Summary

The requested auto-accept capability is already implemented on main via the `chat.editing.autoAcceptDelay` setting (shipped in 1.97.0). Setting it to a low value (e.g. 1 second) effectively bypasses the Keep/Undo prompt, which is the core ask.

## Change summary

Add a setting to auto-accept or bypass the Keep/Undo confirmation after Copilot Chat edits.

## Best solution

Keep the shipped `chat.editing.autoAcceptDelay` setting (commit a5d836544d2, 1.97.0). Users who want to bypass Keep/Undo can set it to a small delay. No further change is needed.
