# Issue #270650 — Copilot Agent prompts fail when chat.useNestedAgentsMdFiles is enabled with SSH FS folders

[https://github.com/microsoft/vscode/issues/270650](https://github.com/microsoft/vscode/issues/270650)

## Summary

The hang when submitting prompts with SSH FS folders present was fixed on main by commit 19b8cf5f91d, which added a fallback to the file service when no FileSearchProvider is available for a scheme. The prompt submission no longer blocks.

## Change summary

Copilot Agent prompt submission hangs when chat.useNestedAgentsMdFiles is enabled and workspace contains folders with schemes lacking a FileSearchProvider (e.g. SSH FS).

## Best solution

Keep the shipped fix (commit 19b8cf5f91d). The fallback to recursive file-service traversal resolves the blocking hang. The feature request for auto-disabling the setting or adding a status-bar toggle (comment 2) should be filed as a separate feature request if still desired.
