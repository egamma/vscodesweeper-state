## Summary

The reported untitled→real session URI reconciliation race has been fixed on main by two commits: one that keeps editors open across the replacement (52f9d128b0d) and a comprehensive session resource aliasing implementation (ec311c898de). This can be closed as completed.

## Change summary

"Mark as Done" fails to close the tab because the editor cache retains a placeholder untitled URI that never gets reconciled with the real CLI session UUID.

## Best solution

Keep the shipped fixes on main (commits 52f9d128b0d and ec311c898de); no further change needed. The session resource aliasing ensures untitled sessions are properly re-keyed when materialized.
