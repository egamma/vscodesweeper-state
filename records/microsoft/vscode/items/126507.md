# Issue #126507 — [scss] map value override => identifier expected

[https://github.com/microsoft/vscode/issues/126507](https://github.com/microsoft/vscode/issues/126507)

## Summary

Valid SCSS namespace-qualified variable assignment syntax (`namespace.$var: value`) is incorrectly flagged as an error by the built-in SCSS language service. The issue carries the 'On Deck' milestone, so it must stay open.

## Change summary

SCSS parser reports 'identifier expected' for valid namespace-qualified variable assignments (e.g. `test.$test: (...)`).

## Best solution

Fix the SCSS parser in the upstream `vscode-css-languageservice` package to recognize namespace-qualified variable assignments as valid syntax, then bump the dependency in `extensions/css-language-features/server/package.json`.
