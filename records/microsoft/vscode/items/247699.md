# Issue #247699 — agent mode says simple browser is already open but it isn't

[https://github.com/microsoft/vscode/issues/247699](https://github.com/microsoft/vscode/issues/247699)

## Summary

The simple browser tool in agent mode has been entirely replaced by the integrated browser architecture. The old `openSimpleBrowser` tool no longer exists on desktop — it was renamed to `openIntegratedBrowser` and reimplemented with proper page-existence tracking via `findExistingPagesByHost` in `browserView`. The original bug path is gone.

## Change summary

Agent mode's simple browser tool incorrectly reported the browser was already open when it was not visible.

## Best solution

The integrated browser replacement (commits 68703b58908 and b4018e320b8) eliminated this class of bug by replacing the entire tool and its state-tracking mechanism. No further change needed; if the "already open" detection misbehaves in the new integrated browser, a new issue should be filed against that system.
