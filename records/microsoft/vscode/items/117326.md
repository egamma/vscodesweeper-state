# Issue #117326 — Allow moving the floating debug.toolbar everywhere

[https://github.com/microsoft/vscode/issues/117326](https://github.com/microsoft/vscode/issues/117326)

## Summary

The core pain point (toolbar blocking tabs) has been largely addressed by the `commandCenter` and `docked` toolbar options added since this was filed, but the specific ask — freely repositioning the floating toolbar to any screen location — remains unimplemented. This is the canonical issue that others were duped to, and it has 12 upvotes (below the 20 threshold). Keeping open for the community process.

## Change summary

Allow the floating debug toolbar to be dragged and positioned anywhere on screen, not just horizontally along the top edge.

## Best solution

The `commandCenter` option (commit cef7b787450) already solves the primary frustration of the toolbar blocking tabs. The remaining ask — free vertical+horizontal positioning — was previously deemed out-of-scope in https://github.com/microsoft/vscode/issues/61336. A maintainer should decide whether the `commandCenter` alternative is sufficient to close this as addressed, or whether free-floating positioning is still worth tracking.
