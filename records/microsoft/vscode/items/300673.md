# Issue #300673 — Claude models existed on Windows, but disappeared on WSL Ubuntu 20.04 

[https://github.com/microsoft/vscode/issues/300673](https://github.com/microsoft/vscode/issues/300673)

## Summary

The available model list in Copilot Chat is determined entirely by the GitHub Copilot Chat extension and its backend service, not by VS Code core. The comments reveal that the extension version on Linux/WSL consistently lags behind Windows (e.g. 0.42.3 vs 0.43.0, 0.36.2 vs 0.40.1), and one commenter points to regional restrictions. Both causes are extension/service-side, not VS Code core.

## Change summary

Claude models appear in the Copilot Chat model picker on Windows but are missing when connected to WSL Ubuntu, despite the same account.

## Best solution

This belongs with the GitHub Copilot Chat extension team. The model picker population and extension version parity across platforms are owned by that extension and its backend entitlement service. Reporters should file at https://github.com/microsoft/vscode-copilot-release or check extension version parity manually.
