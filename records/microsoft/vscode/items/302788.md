## Summary

Feature request for clearer per-pane file identification in the diff editor. A very similar request (https://github.com/microsoft/vscode/issues/244031) was closed via the community upvote process without reaching threshold. This is a fresh instance of the same idea — keeping open to let the community process run.

## Change summary

Add explicit per-pane file identification (header/label) in the diff editor so users can immediately tell which file is on which side.

## Best solution

Route to the diff-editor area for consideration. The diff editor tab already shows `file1 ↔ file2` but when filenames are long or identical (different paths), dedicated per-pane breadcrumbs or headers would help. A prior request (https://github.com/microsoft/vscode/issues/244031) died in the community process; if this one gains traction it could be revisited.
