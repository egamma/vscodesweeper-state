# Issue #262657 — SCM unstage gui button

[https://github.com/microsoft/vscode/issues/262657](https://github.com/microsoft/vscode/issues/262657)

## Summary

Valid feature request: the unstage gutter action (`git.unstageChange`) is registered in `scm/change/title` but explicitly disabled with `"when": "false"`. The command implementation exists but the inline GUI button is not surfaced for staged files. Keeping open as a reasonable enhancement.

## Change summary

Add a GUI button in the diff editor gutter to unstage individual ranges/changes in staged files, symmetrical with the existing stage and revert buttons for unstaged files.

## Best solution

Enable the `git.unstageChange` entry in `scm/change/title` (extensions/git/package.json line 2902) with an appropriate `when` clause for staged file diffs, mirroring the pattern used for `git.stageChange`. The command implementation already exists.
