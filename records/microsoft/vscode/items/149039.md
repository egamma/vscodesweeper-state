# Issue #149039 — Bracket pair colorization: allow colorize parentheses, brackets and braces independently

[https://github.com/microsoft/vscode/issues/149039](https://github.com/microsoft/vscode/issues/149039)

## Summary

Valid feature request in the Backlog with 24 upvotes. The existing `independentColorPoolPerBracketType` setting gives each bracket type its own nesting counter but still uses the shared 6-color palette; users want distinct color schemes per bracket type. Not yet implemented on main.

## Change summary

Allow configuring separate color palettes for parentheses, brackets, and braces in bracket pair colorization.

## Best solution

Extend the bracket-pair colorization theming in `colorizedBracketPairsDecorationProvider.ts` to support per-bracket-type color tokens (e.g. `editorBracketPairColorization.parenthesisForeground1`), so themes and `workbench.colorCustomizations` can assign distinct palettes to each bracket type independently.
