# Issue #238656 — Allow customization of editor tab height and editor tab bar scrollbar height

[https://github.com/microsoft/vscode/issues/238656](https://github.com/microsoft/vscode/issues/238656)

## Summary

A low-traction feature request for arbitrary pixel-level tab height and tab-bar scrollbar height customization. The existing setting (`window.density.editorTabHeight`) only offers `default`/`compact`. With 1 upvote and no duplicates found, it stays open in Backlog Candidates for the community process.

## Change summary

Allow users to set an arbitrary pixel value for editor tab height and tab-bar scrollbar height, beyond the current default/compact toggle.

## Best solution

If ever prioritized, extend `window.density.editorTabHeight` to accept a numeric pixel value (or add a third preset like `large`) and expose a scrollbar-thickness override for the tab bar. However, with only 1 upvote and the vision favoring uniform UI over per-component tweaks, this is unlikely to graduate from the backlog without significantly more community demand.
