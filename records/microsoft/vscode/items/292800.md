# Issue #292800 — Chat: reduce size of editor scrollbars when embedded in chat

[https://github.com/microsoft/vscode/issues/292800](https://github.com/microsoft/vscode/issues/292800)

## Summary

Still open on main. The horizontal scrollbar size for chat-embedded editors is not explicitly reduced; only height-capped code blocks got a smaller vertical scrollbar (7px) in a later PR. The original request for generally smaller scrollbars in chat editors remains unaddressed.

## Change summary

Reduce the size of editor scrollbars (particularly horizontal) when editors are embedded inside the chat view, so they don't visually dominate the compact space.

## Best solution

Set `horizontalScrollbarSize` (and possibly `verticalScrollbarSize` for all cases, not just height-capped blocks) to `defaultChatScrollbarSize` (7) in the editor options at `src/vs/workbench/contrib/chat/browser/widget/chatContentParts/codeBlockPart.ts` lines ~195 and ~766. Note meganrogge's pushback in the comments about the terminal case — the editor case may warrant a different decision since editors are denser.
