# Issue #233354 — Support file icons similar to treeview item icon path in terminal icon path

[https://github.com/microsoft/vscode/issues/233354](https://github.com/microsoft/vscode/issues/233354)

## Summary

Team-authored feature request for file-icon support in terminal icon paths — keeping open per hard keep-open rule for MEMBER authors. The capability gap remains on current main.

## Change summary

Allow extensions to use file icons (via resourceUri + ThemeIcon.File) for terminal tab icons, matching the existing TreeView iconPath behavior.

## Best solution

Extend the terminal icon API to accept `ThemeIcon.File` with a `resourceUri` so extensions like the Python extension can show file-type icons (e.g. .py icon) for activated terminals. Implementation likely touches the terminal profile/icon resolution in `src/vs/workbench/contrib/terminal/` and the extension host API surface.
