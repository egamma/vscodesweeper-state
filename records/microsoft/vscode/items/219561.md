## Summary

Still reproducible on main: `extensions/json/language-configuration.json` has no folding markers for `#region`/`#endregion`, so minimap section headers don't appear for JSONC region comments. PR #318515 is open but unmerged.

## Change summary

Minimap section headers do not render for `//#region` comments in JSONC files because the JSON language configuration lacks folding marker definitions.

## Best solution

Merge https://github.com/microsoft/vscode/pull/318515 which adds `#region`/`#endregion` folding markers to `extensions/json/language-configuration.json`, aligning the language-configuration with what `vscode-json-languageservice` already supports.
