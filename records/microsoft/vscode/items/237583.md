# Issue #237583 — Editor GPU: Double rendering after undo

[https://github.com/microsoft/vscode/issues/237583](https://github.com/microsoft/vscode/issues/237583)

## Summary

Still an open, valid bug in the GPU renderer — no fixing commit found on main. The viewport render strategy uploads the full viewport each frame but appears to have a stale-content issue after undo operations that change line count.

## Change summary

GPU-rendered editor shows duplicated/stale text after undoing a multi-line deletion.

## Best solution

Fix the invalidation/re-render logic in `viewportRenderStrategy.ts` so that undo operations that restore lines produce a clean frame without ghost content from the prior state.
