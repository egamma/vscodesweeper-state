# Issue #174959 — [folding] Allow us to know if a region is folded or not

[https://github.com/microsoft/vscode/issues/174959](https://github.com/microsoft/vscode/issues/174959)

## Summary

Valid feature request for extension API to query fold state. No such API exists on current main (stable or proposed). Low community traction (1 upvote) keeps it in Backlog Candidates territory.

## Change summary

Extensions cannot query whether a given folding region is currently collapsed or expanded.

## Best solution

Add a read-only API surface (e.g. `TextEditor.foldedRanges` or an `isFolded(line)` helper) and optionally an `onDidChangeFoldState` event, gated behind a proposed API flag initially. Work belongs in the `editor-folding` / `api` area.
