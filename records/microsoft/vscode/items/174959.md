# Issue #174959 — [folding] Allow us to know if a region is folded or not

[https://github.com/microsoft/vscode/issues/174959](https://github.com/microsoft/vscode/issues/174959)

## Summary

This is a valid, unimplemented feature request for an extension API to query folding state. No proposed or stable API exists on current main for this capability. The issue is in the Backlog milestone with appropriate labels already applied.

## Change summary

Extensions cannot query whether a specific line or folding range is currently folded or unfolded in the editor.

## Best solution

Add a proposed extension API (e.g. `textEditor.isLineFolded(line)` or an event for fold state changes) in the `editor-folding` / `api` area. This would live in `src/vscode-dts/vscode.proposed.*.d.ts` with implementation bridging to the internal `FoldingModel` in `src/vs/editor/contrib/folding/`.

## Evidence
