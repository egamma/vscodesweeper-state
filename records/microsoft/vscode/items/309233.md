# Issue #309233 — Agent App session change/diff view should have branch changes option as well

[https://github.com/microsoft/vscode/issues/309233](https://github.com/microsoft/vscode/issues/309233)

## Summary

Team-authored feature request for Agents window Changes view to support a full branch diff mode (not just session-scoped changes), especially for CLI-originated sessions. Hard keep-open: MEMBER author.

## Change summary

Add a 'Full Branch Diff' mode to the Agents app Changes view so users can review all branch changes (git diff baseBranch...HEAD), including for sessions started in Copilot CLI.

## Best solution

Extend the Changes view in `src/vs/sessions/contrib/changes/` to support a branch-level diff mode that computes `baseBranch...HEAD` from the workspace git context, independent of session checkpoint tracking. This would also fix the empty Changes view for CLI-originated sessions that lack checkpoint refs.
