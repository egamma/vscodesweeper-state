# Issue #293742 — The "Open on GitHub" button does not appear for Git URLs in the format `ssh://git@github.com/...`

[https://github.com/microsoft/vscode/issues/293742](https://github.com/microsoft/vscode/issues/293742)

## Summary

Confirmed on current main: `getRepositoryFromUrl` in `extensions/github/src/util.ts` only matches `https://` and `git@…:` URL formats, missing the valid `ssh://git@github.com/…` format. An open PR (#293743) from the reporter addresses this.

## Change summary

The 'Open on GitHub' button does not appear for repositories cloned with the `ssh://git@github.com/…` URL format because that pattern is not recognized by the GitHub extension's URL parser.

## Best solution

Review and land https://github.com/microsoft/vscode/pull/293743, which adds the missing `ssh://` regex branch to `getRepositoryFromUrl` in `extensions/github/src/util.ts`.
