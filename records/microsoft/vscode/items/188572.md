# Issue #188572 — Open selected tab via extension API

[https://github.com/microsoft/vscode/issues/188572](https://github.com/microsoft/vscode/issues/188572)

## Summary

Valid, community-promoted feature request (30 upvotes, in Backlog) for a tab select/activate API on TabGroups. Not yet implemented — the current `TabGroups` interface only exposes `close`; no `select`, `activate`, or `open` method exists in the stable or proposed API surface.

## Change summary

Add an extension API method to select/activate an existing tab, analogous to the existing `tabGroups.close()` method.

## Best solution

Add a `select(tab: Tab, preserveFocus?: boolean): Thenable<void>` method to `TabGroups` in `src/vscode-dts/vscode.d.ts`, backed by a `$selectTab` proxy call in `src/vs/workbench/api/common/extHostEditorTabs.ts` and a corresponding handler in the main-thread tab service. The discussion already clarified scope: activate an existing tab only, not open arbitrary URIs.
