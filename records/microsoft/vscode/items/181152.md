# Issue #181152 — Can't Highlight Text By Draggin Mouse

[https://github.com/microsoft/vscode/issues/181152](https://github.com/microsoft/vscode/issues/181152)

## Summary

The reported inability to drag-select appears to be triggered by an external HID/OS/device bug that repeatedly sends MediaTrackPrevious, but there is still a product question about whether VS Code should let that media key clear editor selection. This should stay open for editor input ownership rather than be closed as an extension or not-reproducible issue.

## Change summary

Dragging to select text in the editor is interrupted because repeated MediaTrackPrevious input clears the current selection.

## Best solution

Editor input maintainers should decide whether media key commands should affect editor selection state at all, and if appropriate adjust the editor/keybinding handling so repeated MediaTrackPrevious events do not deselect text while dragging.
