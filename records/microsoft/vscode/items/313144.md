## Summary

The noisy 'Unknown content_block type text' warning was a known logging issue fixed in commit 4253b366bc0 (April 21, 2026), before this issue was filed. The warning did not cause the agent to stop — it was harmless log noise from the streaming path not recognizing a 'text' content_block_start event. The agent stopping early is normal model behavior (finish_reason: stop), not a VS Code bug. Updating to the latest Copilot extension version resolves the warning.
