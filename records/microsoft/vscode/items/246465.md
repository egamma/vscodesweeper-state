# Issue #246465 — Cannot resize or move window with Custom Title bar on Linux via MobaXterm

[https://github.com/microsoft/vscode/issues/246465](https://github.com/microsoft/vscode/issues/246465)

## Summary

This is a duplicate of #65101 — custom title bar does not support window drag/resize over X11 forwarding, which was closed as *as-designed with the workaround of using native title bar. The reporter already found this workaround.

## Change summary

Custom title bar cannot be moved or resized when VS Code runs over X11 forwarding (SSH + MobaXterm).

## Best solution

Close as duplicate of the established *as-designed decision in #65101. X11 forwarding does not support the frameless window drag regions used by the custom title bar; `"window.titleBarStyle": "native"` is the documented workaround.
