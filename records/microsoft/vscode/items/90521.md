# Issue #90521 — Support custom QuickPick filter logic

[https://github.com/microsoft/vscode/issues/90521](https://github.com/microsoft/vscode/issues/90521)

## Summary

The request for custom QuickPick filter logic remains unimplemented on main. The related `sortByLabel` is still a proposed API, `matchOnLabel` toggle doesn't exist, and no custom filter callback has been added. With 25 upvotes approaching the backlog threshold, this is a valid open feature request.

## Change summary

Allow extensions to provide custom filter callbacks for QuickPick, overriding the built-in matching logic.

## Best solution

Add a custom filter callback (or at minimum a `matchOnLabel: false` toggle to fully disable built-in filtering so extensions can pre-filter) to the QuickPick API. The related pieces — `sortByLabel` (proposed), custom highlighting (#83424) — form a coherent set of QuickPick extensibility improvements that could land together.
