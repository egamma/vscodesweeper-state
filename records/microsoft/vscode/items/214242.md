# Issue #214242 — Add context menu item to relaunch terminals with extra environment info

[https://github.com/microsoft/vscode/issues/214242](https://github.com/microsoft/vscode/issues/214242)

## Summary

Team-authored feature request; the relaunch command exists but is not exposed in the terminal tab/instance context menus. Still a valid gap — keeping open.

## Change summary

Add a 'Relaunch Terminal' item to the terminal right-click context menu so users can relaunch terminals with updated environment without using the hover link or killing/recreating.

## Best solution

Register `TerminalCommandId.Relaunch` in `MenuId.TerminalInstanceContext` (and optionally `TerminalEditorInstanceContext`) in `src/vs/workbench/contrib/terminal/browser/terminalMenus.ts`, gated on a context key indicating stale environment. A 'Relaunch All Terminals' command could also be added.
