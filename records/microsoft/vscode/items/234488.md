# Issue #234488 — Git vscode commands on toggled files in the scm view

[https://github.com/microsoft/vscode/issues/234488](https://github.com/microsoft/vscode/issues/234488)

## Summary

Verified duplicate of the open canonical issue #211519, which tracks exactly this: SCM commands like stage/unstage/clean should operate on highlighted/selected files in the SCM view, not only the active editor file. Proposing a duplicate close to consolidate discussion.

## Change summary

git.stage, git.unstage, and git.clean commands ignore selected files in the SCM view when invoked via keyboard; they only act on the file open in the active editor.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/211519, which is assigned to @lszomoru and already tracks this exact gap — making SCM actions operate on highlighted/selected files rather than requiring the file to be opened in the editor.
