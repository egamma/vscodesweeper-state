# Issue #240882 — Indentation on pasting with multiple lines of code not working as expected

[https://github.com/microsoft/vscode/issues/240882](https://github.com/microsoft/vscode/issues/240882)

## Summary

Still present on main — the auto-indent-on-paste logic has an early return (line 464 of indentation.ts) that skips adjusting subsequent lines when the first line's indentation already matches the expected indent. Team member engaged; tracked under editor-autoindent.

## Change summary

Pasting multiple lines with `autoIndentOnPaste` enabled only corrects the first line's indentation; subsequent lines retain their original indentation when the cursor is at the block indentation column.

## Best solution

Fix the early-return path in `AutoIndentOnPaste.trigger()` (around line 464 of `src/vs/editor/contrib/indentation/browser/indentation.ts`) so that even when the first line's indentation is already correct, the subsequent-lines adjustment logic (lines 481–524) still executes.
