## Change summary

Long terminal tab titles set via OSC sequence overflow into panel action buttons in single-tab mode because the single-terminal-tab element lacks overflow constraints.

## Best solution

The fix is a CSS addition on `.monaco-action-bar .action-item .single-terminal-tab` to add `overflow: hidden; text-overflow: ellipsis; white-space: nowrap; min-width: 0;` (or equivalent flex constraint). The existing *duplicate label indicates a maintainer already linked this to a canonical issue.
