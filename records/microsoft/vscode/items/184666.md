# Issue #184666 — Toggle line comment doesn't work for multiple lines in CSS

[https://github.com/microsoft/vscode/issues/184666](https://github.com/microsoft/vscode/issues/184666)

## Summary

This is a duplicate of #166944 — both report that multi-line toggle-line-comment in CSS wraps the entire selection in one block comment rather than per-line, which corrupts code containing existing comments. The feature request (#166944) is open and tracks the same underlying work.

## Change summary

Toggle Line Comment on multiple CSS lines wraps everything in a single `/* */` block, corrupting code that already contains comments.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/166944 — implementing per-line block comment wrapping there solves both the corruption bug and the feature request.
