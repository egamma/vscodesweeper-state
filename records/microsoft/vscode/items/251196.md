## Summary

Team-authored UX request. The mode picker on current main already has toolbar actions for custom modes — a pencil icon (Codicon.edit) to open the mode file for editing, and a file icon (Codicon.file) for read-only extension-contributed modes. The original request for a ⚙ gear icon may be considered addressed by this pencil-icon approach, but the author and design team should confirm.
