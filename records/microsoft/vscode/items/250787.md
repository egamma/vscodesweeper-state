# Issue #250787 — Avoid individual indent guides in trees

[https://github.com/microsoft/vscode/issues/250787](https://github.com/microsoft/vscode/issues/250787)

## Summary

Team-authored performance debt issue requesting a smarter approach to rendering indent guides in tree views — must remain open for maintainer judgment.

## Change summary

Tree views render hundreds of individual indent guide elements per row, causing long-task warnings and sluggishness.

## Best solution

Redesign tree indent guides to use a more efficient rendering strategy (e.g., a single CSS-based approach or shared background patterns) instead of individual DOM elements per guide per row. Likely lives in `src/vs/base/browser/ui/tree/` and the abstract tree rendering code.
