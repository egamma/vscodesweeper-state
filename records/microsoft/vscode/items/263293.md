[https://github.com/microsoft/vscode/issues/263293](https://github.com/microsoft/vscode/issues/263293)

## Summary

Team-authored feature request for faster model switching via keybindings or cycling — not yet implemented on main, and the MEMBER authorship requires explicit maintainer judgment. Keeping open.

## Change summary

Allow switching between chat models via a dedicated hotkey (bind a shortcut to a model, or cycle through models with a single keystroke).

## Best solution

Add a 'Cycle Model' or 'Previous Model' command (alongside the existing `openModelPicker`) in `src/vs/workbench/contrib/chat/browser/actions/chatExecuteActions.ts`, with a default keybinding, so users can toggle between recently-used models without opening the picker UI. Alternatively, allow keybindings to select a specific model by ID.
