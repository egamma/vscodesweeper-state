# Issue #172569 — High contrast panel section header outlines don't include problem count

[https://github.com/microsoft/vscode/issues/172569](https://github.com/microsoft/vscode/issues/172569)

## Summary

Still present on main — the HC outline is drawn on `action-label` but the badge count lives in the parent `action-item`, so the outline doesn't encompass the badge. Milestone 'On Deck' also requires keeping open.

## Change summary

In high contrast themes, the panel section header outline does not wrap the problem count badge because the outline targets `action-label` rather than the parent `action-item`.

## Best solution

Move the HC outline from `action-label` to `action-item` (or adjust the DOM/CSS structure) in the pane header action bar so the badge is visually enclosed. The fix likely lives in paneview.css or the view pane container styling.
