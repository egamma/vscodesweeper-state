# Issue #172569 — High contrast panel section header outlines don't include problem count

[https://github.com/microsoft/vscode/issues/172569](https://github.com/microsoft/vscode/issues/172569)

## Summary

Still reproduces on main: the HC outline targets `.action-label` while the badge/count is a sibling in `.action-item`, so the outline doesn't encompass the problem count. On Deck milestone requires maintainer judgment.

## Change summary

In high contrast themes, the panel section header outline excludes the problem count badge because the outline is drawn on the inner `.action-label` rather than the parent `.action-item`.

## Best solution

Move the HC outline from `.action-label` to `.action-item` (or wrap badge+label in the outlined element) in the pane header action bar styling so the count badge is visually enclosed. Likely in the actionbar or paneview CSS where HC borders are applied.
