## Change summary

Align file diff/stats display across local, background (workspace/worktree), and cloud agent sessions so the UI is consistent and accurate.

Implement the unified working-set UI agreed upon in the Zurich discussion (comment 8): a single working set docked above chat input for all session types, with Keep/Undo for local only, apply-to-workspace for background/cloud, and diff stats in the session card matching the working set exactly.
