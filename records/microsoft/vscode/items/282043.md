# Issue #282043 — Agent Sessions: File diff/stats alignment

[https://github.com/microsoft/vscode/issues/282043](https://github.com/microsoft/vscode/issues/282043)

## Summary

Team-authored design-alignment issue for file diff/stats across agent session types (local, background, cloud). Active discussion among team members with evolving requirements — must stay open for maintainer judgment.

## Change summary

Align how and when chat sessions show file diff stats across local, background (workspace/worktree), and cloud agent sessions.

## Best solution

Finalize the agreed-upon design from the Zurich discussion (comment 8): a single working set UI docked above the chat input for all three session kinds, with Keep/Undo only for local, apply-to-workspace for background/cloud, and removal of the in-response change component. Remaining open question is whether session card diff info should match the working set exactly (comment 11 consensus: yes).
