# Issue #315756 — agent sessions window always switches to the changed files

[https://github.com/microsoft/vscode/issues/315756](https://github.com/microsoft/vscode/issues/315756)

## Summary

Team-authored bug report: the agents window sidebar resets from the 'files' view back to the 'changes' view when switching tabs, losing the user's view selection. Keeping open per team-member policy.

## Change summary

The agents sessions window sidebar does not persist the user's selected view (files vs. changes), reverting to the changes view on tab switch.

## Best solution

Persist the user's sidebar view selection (files vs. changes) across tab switches in the agents window, likely in the session/chat widget state. For repos without git, default to the 'files' view as the initial landing.
