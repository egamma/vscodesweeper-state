# Issue #318382 — Ability to delete individual messages from chat session history

[https://github.com/microsoft/vscode/issues/318382](https://github.com/microsoft/vscode/issues/318382)

## Summary

Genuine feature request for selective message deletion in chat sessions. The internal `removeRequest` API exists in the chat model but is only wired for checkpoint rollback, not exposed as a per-message user action. No canonical duplicate found; keeping open for the community upvote process (7 days remain).

## Change summary

Allow users to delete individual messages from a chat session without rolling back to a checkpoint.

## Best solution

Expose a context-menu action on individual chat messages that calls the existing `ChatModel.removeRequest` (in `src/vs/workbench/contrib/chat/common/model/chatModel.ts`) for the selected turn. The plumbing exists; the missing piece is a UI surface (right-click menu item or inline button) and a `ChatServiceImpl.removeRequest` call wired to it, plus ensuring serialization/persistence handles the gap correctly.
