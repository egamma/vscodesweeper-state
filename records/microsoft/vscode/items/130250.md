## Change summary

## Best solution

Modify `src/vs/editor/contrib/folding/browser/foldingDecorations.ts` to use injected text (via `InjectedTextOptions`) showing the line count for each collapsed range, gated behind a new editor option (e.g., `editor.showFoldedLineCount`).
