# Issue #324962 — Copilot Chat side-panel input font is too small and cannot be independently resized

[https://github.com/microsoft/vscode/issues/324962](https://github.com/microsoft/vscode/issues/324962)

## Summary

Confirmed in source: the chat side-panel input editor hardcodes `fontSize = 13` and does not read from the `chat.fontSize` setting. The chat message container does respect `chat.fontSize` via `ChatLayoutService`, but the input Monaco editor ignores it. This is a real, small bug.

## Change summary

The Copilot Chat side-panel input box ignores the `chat.fontSize` setting because its Monaco editor instance hardcodes `fontSize = 13`.

## Best solution

In `src/vs/workbench/contrib/chat/browser/widget/input/chatInputPart.ts`, replace the hardcoded `options.fontSize = 13` with a read from `IChatLayoutService.fontSize` (or directly from the `chat.fontSize` configuration), and subscribe to changes so the input editor updates dynamically when the setting changes.
