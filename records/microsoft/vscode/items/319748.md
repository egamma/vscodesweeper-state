# Issue #319748 — Agents window should show total credits used by a session somewhere

[https://github.com/microsoft/vscode/issues/319748](https://github.com/microsoft/vscode/issues/319748)

## Summary

Valid feature request to surface total session credit cost in the Agents window. The Chat view pane already has session cost tracking via ChatContextUsageWidget/ChatContextUsageDetails, but the Agents window (src/vs/sessions/) does not expose this information. Routes to the agents-window area.

## Change summary

Show total credits consumed by a session somewhere in the Agents window UI, not just per-turn costs.

## Best solution

Extend the Agents window session views (src/vs/sessions/contrib/sessions/) to surface the existing sessionCost data already tracked by ChatModel. The Chat view pane's ChatContextUsageDetails already renders session cost in credits — the Agents window needs an analogous display, likely in the session list or session header.
