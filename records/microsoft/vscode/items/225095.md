## Summary

Team-authored issue; the bug is still present on main ‚Äî the inline diff renderer ignores `wrappedTextIndentLength` from `ModelLineProjectionData` when rendering wrapped continuation lines in the deleted-code view zone.

## Change summary

Inline diff wrapped lines are not indented according to the wrapping indent setting, making deleted code hard to read in narrow editors.

## Best solution
