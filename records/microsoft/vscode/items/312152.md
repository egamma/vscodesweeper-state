# Issue #312152 — The pined VS Code shortcut icon in the Windows 10 Start menu does not display.

[https://github.com/microsoft/vscode/issues/312152](https://github.com/microsoft/vscode/issues/312152)

## Summary

Still-live Windows 10 Start menu icon disappearance after updates, confirmed by multiple users through 1.124.2 despite the earlier fix in d872b794fbe (1.112.0). The original fix prevents shortcut overwriting but doesn't address icon cache invalidation on Windows 10.

## Change summary

The pinned VS Code shortcut icon in the Windows 10 Start menu disappears after every update, requiring manual shortcut property edit to restore it.

## Best solution

The `ShouldUpdateShortcut` logic in `build/win32/code.iss` prevents overwriting but doesn't refresh the Windows icon cache. The installer likely needs a `SHChangeNotify(SHCNE_ASSOCCHANGED, …)` call or equivalent icon cache flush after update to force Windows 10 to re-read the shortcut icon. This belongs in the Windows setup/installer area.
