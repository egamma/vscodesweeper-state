## Change summary

When stopping a Copilot Chat response, re-populate the input textbox with the original prompt so the user can edit and re-send without copy-pasting.

## Best solution

Implement in the chat widget's cancel/stop flow: after cancellation, set the input editor value to the user's last-sent message. A UX decision is needed on whether this should always happen on stop or be opt-in, and how it interacts with the existing edit-previous-message flow.
