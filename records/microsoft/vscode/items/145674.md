# Issue #145674 — Limit length of InputBox validation message?

[https://github.com/microsoft/vscode/issues/145674](https://github.com/microsoft/vscode/issues/145674)

## Summary

Still valid on main: the InputBox validation message has no max-height or length limit, so a long message can extend off-screen. The CSS at inputBox.css:78 only sets overflow:hidden and word-wrap but no vertical constraint.

## Change summary

InputBox validation messages have no length or height limit and can overflow the visible area.

## Best solution

Add a max-height (or max number of lines with text-overflow) to the `.monaco-inputbox-container .monaco-inputbox-message` CSS rule in `src/vs/base/browser/ui/inputbox/inputBox.css`, potentially with overflow-y:auto so the user can still scroll within the message.
