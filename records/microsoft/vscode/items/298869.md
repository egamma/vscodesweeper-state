# Issue #298869 — Add pan, node selection, and export to Mermaid diagram preview

[https://github.com/microsoft/vscode/issues/298869](https://github.com/microsoft/vscode/issues/298869)

## Summary

Pan/drag is now implemented on main (the built-in mermaid extension has a full pan-mode toggle and drag-to-pan). Node selection and diagram export (SVG/PNG) remain unimplemented and are valid feature requests worth keeping open.

## Change summary

Request to add pan/drag navigation, node selection, and image export (SVG/PNG/PDF) to the Mermaid diagram preview.

## Best solution

The pan request is already delivered via the built-in mermaid extension's `DiagramElement` class (pan-mode button, Alt+drag, scroll zoom). The remaining asks — node selection with highlighting and export as SVG/PNG — are distinct enhancements that could each be tracked separately; export is likely best handled in `extensions/mermaid-markdown-features` by adding a webview context-menu command that serializes the rendered SVG or rasterizes it to a canvas. Node selection is more speculative and may warrant an `under-discussion` label.
