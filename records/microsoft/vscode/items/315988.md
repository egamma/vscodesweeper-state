# Issue #315988 — Be able to paste text + image content into copilot chat at once

[https://github.com/microsoft/vscode/issues/315988](https://github.com/microsoft/vscode/issues/315988)

## Summary

Valid feature request: the chat paste architecture registers separate providers for images and text, but the editor's paste system picks a single edit from one provider per paste operation, so mixed clipboard content (text + image) can only attach one type. No duplicate found.

## Change summary

Allow pasting clipboard content that contains both text and an image into Copilot chat so both are attached simultaneously.

## Best solution

The chat paste infrastructure in `chatPasteProviders.ts` needs a provider (or the existing providers need coordination) that handles mixed clipboard content — detecting when both `image/*` and `text/plain` items are present and producing a single `DocumentPasteEdit` that attaches the image and inserts the text. Alternatively, the editor's `CopyPasteController` could be extended to compose edits from multiple providers.
