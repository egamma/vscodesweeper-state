# Issue #214544 — Crash when folding code

[https://github.com/microsoft/vscode/issues/214544](https://github.com/microsoft/vscode/issues/214544)

## Summary

This is a snap-package-specific crash on VS Code 1.90.0 (Linux), which matches the same root cause as #212494 — a GPU/rendering regression in the snap build fixed in 1.90.2. The reporter and other commenters confirmed switching to the .deb package resolved it, and #214493 (identical hover-crash on snap) was also confirmed fixed by 1.90.2.

## Change summary

VS Code snap package on Linux crashes (turns white and disappears) when folding code or hovering over fold indicators, starting with v1.90.0.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/212494, which tracked the snap 1.90 crash and was fixed in the 1.90.2 recovery release. No further action needed.
