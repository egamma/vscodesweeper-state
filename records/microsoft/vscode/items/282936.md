# Issue #282936 — Make the context selection easier in chatbox

[https://github.com/microsoft/vscode/issues/282936](https://github.com/microsoft/vscode/issues/282936)

## Summary

Valid polish request: the chat context picker (quick access via 'Add Context...') still closes after each selection, requiring repeated invocations to attach multiple files. The request is for multi-select or a persistent picker. No duplicate found, and a VS Code product lead reportedly encouraged filing it.

## Change summary

The 'Add Context...' chat picker closes after each file selection, forcing users to reopen it repeatedly to attach multiple items.

## Best solution

Allow multi-select in the chat context quick access picker (or keep it open after selection) so users can attach several files in one pass. The implementation lives in `AttachContextAction._show()` in `src/vs/workbench/contrib/chat/browser/actions/chatContextActions.ts`, which calls `quickInputService.quickAccess.show()` — supporting background-accept or multi-pick there would address this.
