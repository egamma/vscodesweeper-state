# Issue #243352 — Show 'Files are Identical' popup / hint  when there are no changes in two files being compared.

[https://github.com/microsoft/vscode/issues/243352](https://github.com/microsoft/vscode/issues/243352)

## Summary

Valid feature request: the diff editor already tracks the `identical` state internally but doesn't surface a user-visible message when two compared files are fully identical. A team member (hediet) engaged with the issue and noted it should connect to the existing whitespace-only-equal mechanism. Keeping open as an accepted backlog item.

## Change summary

Show a visible indication (popup, hint, or status bar item) when two files being compared in the diff editor are identical.

## Best solution

Extend the existing diff editor identical-state detection (already in `DiffState.identical` in `diffEditorViewModel.ts`) to render a user-facing message in the diff editor area or status bar. The related PR #252055 proposes a status bar approach; the maintainer's comment suggests it should also integrate with the existing whitespace-equal overlay mechanism.
