# Issue #273251 — "Copy All" from the chat window does not include the outputs from MCP / extension tools

[https://github.com/microsoft/vscode/issues/273251](https://github.com/microsoft/vscode/issues/273251)

## Summary

Team-authored bug, still reproduces on main. The `getToolInvocationText` method in `chatModel.ts` includes the tool's invocation message and input but does not serialize the tool's result/output content into the copy text.

## Change summary

"Copy All" from chat omits the actual output/result content returned by MCP and extension tool invocations.

## Best solution

Extend `getToolInvocationText` in `src/vs/workbench/contrib/chat/common/model/chatModel.ts` to also serialize `resultDetails.content` (or the relevant result text) for completed tool invocations, so the copied text includes tool outputs alongside the invocation headers.
