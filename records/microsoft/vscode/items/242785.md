# Issue #242785 — Multi diff editor: file title overlapping "Open File" icon

[https://github.com/microsoft/vscode/issues/242785](https://github.com/microsoft/vscode/issues/242785)

## Summary

Fixed on main today: commit af32fc9c27a moves text-overflow/ellipsis handling to the `.title` class itself, ensuring the file title properly truncates instead of overlapping the action icons.

## Change summary

In the multi-diff editor, long file titles overflow and visually overlap the 'Open File' action icon in the header.

## Best solution

Keep the shipped fix in commit af32fc9c27a (PR #324819) — the `.title` element now has `min-width: 0; overflow: hidden; text-overflow: ellipsis;` directly, preventing overlap with the actions toolbar.
