# Issue #291662 — Export/Save Full Copilot Chat History

[https://github.com/microsoft/vscode/issues/291662](https://github.com/microsoft/vscode/issues/291662)

## Summary

VS Code already has a per-session JSON export command (`workbench.action.chat.export`), but the request asks for bulk multi-session export to markdown with rich formatting. A marketplace extension was suggested in the comments. The feature goes beyond what's currently available, but the core export infrastructure exists and could be extended. No clear canonical duplicate absorbs all the asks here.

## Change summary

Add the ability to export the full Copilot Chat history across all sessions to a markdown file with formatting, code blocks, and diagrams preserved.

## Best solution

The existing `workbench.action.chat.export` in `src/vs/workbench/contrib/chat/browser/actions/chatImportExport.ts` covers single-session JSON export. Multi-session bulk export with markdown rendering is a richer feature that may fit better as an extension (the marketplace extension `copilot-chat-to-markdown` already addresses this). Consider whether the built-in export should gain a markdown format option, or whether this is best left to extensions leveraging the existing session data.
