# Issue #318553 — Run and debug view feels kind of broken on small views

[https://github.com/microsoft/vscode/issues/318553](https://github.com/microsoft/vscode/issues/318553)

## Summary

Team-authored issue (MEMBER) reporting that the Run and Debug view's launch configuration dropdown becomes hidden on narrow views, making it hard to discover. Keeping open per policy; the UX concern is valid and reproducible.

## Change summary

The Run and Debug view hides the launch configuration dropdown when the view is narrow, making it difficult to find launch tasks.

## Best solution

Adjust the responsive layout of the debug start view toolbar so that on narrow widths, the gear icon is hidden before the launch configuration dropdown — the dropdown is more critical for discoverability. Related issues https://github.com/microsoft/vscode/issues/321077 and https://github.com/microsoft/vscode/issues/323060 track adjacent narrow-view UX concerns in the same area.
