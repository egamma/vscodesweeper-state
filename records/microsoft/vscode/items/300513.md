# Issue #300513 — CoPilot Chat 0.39.0 is incompatible with latest VSCode 1.111.0

[https://github.com/microsoft/vscode/issues/300513](https://github.com/microsoft/vscode/issues/300513)

## Summary

The 'Chat failed to load' error is caused by the Copilot Chat extension's WSL installation state becoming stale after a VS Code update — reinstalling the extension resolves it. This is an extension-side issue, not a VS Code core bug.

## Change summary

Copilot Chat extension fails to load in WSL after upgrading VS Code, requiring manual reinstall.

## Best solution

This belongs in the Copilot Chat extension's issue tracker (https://github.com/microsoft/vscode-copilot-release). The workaround (reinstall the extension in WSL) confirms the root cause is extension installation state, not VS Code core.
