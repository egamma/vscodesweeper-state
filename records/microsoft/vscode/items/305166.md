# Issue #305166 — Support for native, context-aware File and Folder Icons (JetBrains-style)

[https://github.com/microsoft/vscode/issues/305166](https://github.com/microsoft/vscode/issues/305166)

## Summary

Feature request for a built-in JetBrains-style icon theme; a team member (aeschli) explicitly stated there are no plans for new built-in themes, and VS Code's icon theme extension point exists precisely for this purpose.

## Change summary

Request for a native, context-aware icon theme with semantic folder icons, minimalist colors, and state indicators built into VS Code core.

## Best solution

This belongs as a Marketplace icon theme extension. VS Code's icon theme contribution point already supports all the requested capabilities (custom folder icons, file icons, language-specific icons). The requester or community can build an extension matching the JetBrains aesthetic without core changes.
