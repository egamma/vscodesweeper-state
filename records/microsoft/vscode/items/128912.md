# Issue #128912 — [folding] Allow to fold all method bodies

[https://github.com/microsoft/vscode/issues/128912](https://github.com/microsoft/vscode/issues/128912)

## Summary

Still unimplemented on main — FoldingRangeKind only defines Comment, Imports, and Region; no Body/Method kind exists. With 81 upvotes and Backlog milestone, this is a valid, well-supported feature request that should remain open.

## Change summary

Request to add a 'body' FoldingRangeKind so language servers can mark method/function bodies, enabling a 'Fold All Bodies' command.

## Best solution

Add a new `FoldingRangeKind.Body` (or `Function`) to the folding API in `src/vs/editor/common/languages.ts` and a corresponding 'Fold All Method Bodies' command in `src/vs/editor/contrib/folding/browser/folding.ts`, mirroring how 'Fold All Block Comments' uses the Comment kind.
