# Issue #125120 — VS Code snap for arm64 Linux

[https://github.com/microsoft/vscode/issues/125120](https://github.com/microsoft/vscode/issues/125120)

## Summary

Still valid: the snap build pipeline is x64-only (parameter labeled "Linux x64 Snap", passed only to the x64 build template). Two community PRs are open attempting to add arm64 snap support. This is the canonical issue and should remain open.

## Change summary

Provide an arm64 snap package for VS Code on Linux, alongside the existing x64 snap.

## Best solution

Review and land the open PRs (https://github.com/microsoft/vscode/pull/298237 and/or https://github.com/microsoft/vscode/pull/271464) that add arm64 snap support, which requires adding an arm64 snap build parameter and wiring it into the Linux arm64 build stage in `build/azure-pipelines/product-build.yml` and the snap build steps. The duplicate issue https://github.com/microsoft/vscode/issues/269552 should be closed as a duplicate of this one.
