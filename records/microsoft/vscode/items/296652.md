# Issue #296652 — Feedback on tips in chat

[https://github.com/microsoft/vscode/issues/296652](https://github.com/microsoft/vscode/issues/296652)

## Summary

Team-authored UX feedback on chat tips placement, information hierarchy, and dismissal patterns — must stay open for maintainer judgment.

## Change summary

Tips in chat sessions are too prominent, feel endless, and lack clear dismissal semantics; suggestions to reposition them, tie them to the user journey, and simplify interaction.

## Best solution

Work with the design team to reposition tips (e.g. top of agent response rather than input box), map tips to the new-user journey with contextual timing, and add explicit dismiss/opt-out affordances. The related issue https://github.com/microsoft/vscode/issues/297018 about weighting getting-started vs QOL tips is part of the same effort.
