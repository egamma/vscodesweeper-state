# Issue #162810 — API for using  file quick open

[https://github.com/microsoft/vscode/issues/162810](https://github.com/microsoft/vscode/issues/162810)

## Summary

Valid feature request for an extension API to show the file quick open picker and return a URI. Not yet implemented on main — no such API or command exists. Multiple contributors have expressed interest and proposed concrete approaches.

## Change summary

Expose an extension API (or a command that returns a result) to show the file quick open UI and return the selected file URI.

## Best solution

Add a `vscode.quickFileOpen` command (or a new API method) that shows the file quick open picker and returns the selected URI. Implementation would live in `src/vs/workbench/api/` bridging to the existing quick-open infrastructure in `src/vs/workbench/contrib/quickaccess/`. The approach of a returnable command (as eamodio suggested) is lightweight and avoids a full new API surface.
