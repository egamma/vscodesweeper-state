# Issue #269134 — Tool references in prompts: support `extensionId/*`

[https://github.com/microsoft/vscode/issues/269134](https://github.com/microsoft/vscode/issues/269134)

## Summary

Team-authored feature request for treating extension tools like MCP server tools (supporting `extensionId/*` wildcard references). A member (aeschli) acknowledged the gap — extension tools aren't registered under a tool set yet. Keeping open per team-authorship policy.

## Change summary

Extension tools should support wildcard references (`extensionId/*`) in prompts, similar to how MCP servers support `serverName/*` when all tools are selected.

## Best solution

Register extension-contributed tools under a tool set (analogous to MCP server tool sets) so that `extensionId/*` references resolve correctly in prompt tool references. The work likely lives in the chat-tools contribution area.
