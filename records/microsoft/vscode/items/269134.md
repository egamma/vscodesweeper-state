# Issue #269134 — Tool references in prompts: support `extensionId/*`

[https://github.com/microsoft/vscode/issues/269134](https://github.com/microsoft/vscode/issues/269134)

## Summary

Team-authored feature request to support `extensionId/*` wildcard syntax for tool references in prompts, analogous to existing `serverName/*` for MCP servers. Kept open per policy (MEMBER author).

## Change summary

When all tools from an extension are selected in a prompt, the extension should get a compact `extensionId/*` reference like MCP servers get with `serverName/*`.

## Best solution

Register extension tools under a tool set (as aeschli noted in the comment), then apply the same wildcard collapse logic that MCP server tools already use. Likely touches the chat tool-set registration and prompt serialization code.
