# Issue #255196 — Copilot instructions in multi-root should apply per folder

[https://github.com/microsoft/vscode/issues/255196](https://github.com/microsoft/vscode/issues/255196)

## Summary

Still valid on main: agent instructions (copilot-instructions.md) are added unconditionally from all workspace folders ('always added') without per-folder scoping. The feature has active team discussion but no implementation yet.

## Change summary

In multi-root workspaces, Copilot instructions from all folders are included in every session rather than being scoped to the folder being worked in.

## Best solution

Implement per-folder scoping for agent instructions in `computeAutomaticInstructions.ts` so that copilot-instructions.md files are matched against the active folder context, similar to the existing `applyTo` pattern used for other instruction files.
