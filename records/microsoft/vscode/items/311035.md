# Issue #311035 — We need more settings for modifying SystemPrompts and ReminderPrompts.

[https://github.com/microsoft/vscode/issues/311035](https://github.com/microsoft/vscode/issues/311035)

## Summary

Feature request for user-configurable overrides of Copilot's built-in system prompts and reminder instructions, particularly when using custom agents. No duplicates found; routing to the prompts and agent area owners for product evaluation.

## Change summary

Allow users to modify or disable Copilot's built-in system prompt components (e.g. reminderInstructions) so custom agent instructions aren't contradicted by hardcoded prompt text.

## Best solution

The Copilot/chat-prompts team should evaluate whether custom agents (defined via agent.md or the customization UI) can opt out of or override specific built-in prompt sections like reminder instructions. A settings-based or per-agent frontmatter approach could let authors control which built-in prompt components apply, without exposing the full internal prompt structure.
