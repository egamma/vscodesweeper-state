# Issue #240657 — Git - Reveal in Explorer View not working for diff file view

[https://github.com/microsoft/vscode/issues/240657](https://github.com/microsoft/vscode/issues/240657)

## Summary

Valid bug with milestone 'On Deck' and an open PR (#307590) already addressing it — keeping open for the PR to land.

## Change summary

"Reveal in Explorer View" does nothing when invoked on a diff editor tab opened from Source Control, because the git: scheme URI fails the workspace-containment check.

## Best solution

Review and land PR https://github.com/microsoft/vscode/pull/307590, which converts non-file-scheme URIs to their underlying file: URI before the workspace check in `src/vs/workbench/contrib/files/browser/fileCommands.ts`.
