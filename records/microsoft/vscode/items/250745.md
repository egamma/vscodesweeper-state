## Summary

Confirmed upstream Chromium/Skia crash when the DevTools Layers tool allocates pixels for a picture snapshot. The root cause is in Skia's SkBitmap allocation (third_party/skia/src/core/SkBitmap.cpp); no VS Code-side fix is possible. Keeping open to track the upstream dependency.
