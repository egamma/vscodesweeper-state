# Issue #153910 — Click on titlebar to close command center.

[https://github.com/microsoft/vscode/issues/153910](https://github.com/microsoft/vscode/issues/153910)

## Summary

With the custom title bar now the default on all desktop platforms, the quick pick already dismisses when clicking the title bar area (focus tracking triggers a blur/hide). The original issue only manifests with the native title bar, where the OS captures mouse events for window dragging — that's a platform constraint, not a VS Code bug.

## Change summary

Clicking on the title bar area does not close the command center / quick pick when using the native title bar.

## Best solution

Close as as-designed: the native title bar is an OS-managed surface that swallows mouse events for window dragging. The default custom title bar already handles this correctly via DOM focus tracking. Users who opt into native title bar accept this platform limitation.
