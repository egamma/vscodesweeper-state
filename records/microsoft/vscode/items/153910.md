# Issue #153910 — Click on titlebar to close command center.

## Summary

This is the canonical issue for clicking the title bar not dismissing the quick pick/command center. It's a real platform constraint (Windows native draggable title bar swallows mouse events). Issue #154541 was closed as a duplicate of this one. The bug remains valid on current main — no fix found.

## Change summary

Clicking the title bar area does not dismiss the quick pick or command center because the draggable region swallows mouse events.

## Best solution

The title bar part would need to detect focus loss or overlay something to catch clicks when the quick pick is open. The fix likely involves `src/vs/workbench/browser/parts/titlebar/titlebarPart.ts` and the quick input dismiss logic, potentially toggling drag regions while the command center/quick pick is active.
