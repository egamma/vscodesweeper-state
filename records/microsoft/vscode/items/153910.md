# Issue #153910 — Click on titlebar to close command center.

[https://github.com/microsoft/vscode/issues/153910](https://github.com/microsoft/vscode/issues/153910)

## Summary

Still valid on main: the quick pick cannot dismiss on title bar clicks because the native draggable title bar region intercepts mouse events before VS Code can detect focus loss. No fix commit found.

## Change summary

Quick pick / command center does not close when clicking on the draggable title bar area.

## Best solution

Either overlay the title bar's drag region while the quick pick is open (so mouse events reach VS Code), or detect window-level focus changes that bypass the drag region. The fix likely touches src/vs/workbench/browser/parts/titlebar/ and src/vs/platform/quickinput/browser/. Alternatively, accept as a platform limitation and close as *as-designed* — a maintainer should decide.
