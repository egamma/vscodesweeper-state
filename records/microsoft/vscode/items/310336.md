# Issue #310336 — Ability to save diff views as a diff file

[https://github.com/microsoft/vscode/issues/310336](https://github.com/microsoft/vscode/issues/310336)

## Summary

Valid feature request to save/export diff editor content as a .diff/.patch file. No implementation exists on current main; the referenced predecessor (#139987) was closed by stale-bot, not by a fix. Keeping open for triage routing.

## Change summary

Add a command to save or export any diff view as a unified .diff/.patch file.

## Best solution

Implement a 'Save Diff As…' action in the diff editor contribution (likely in src/vs/editor/browser/widget/diffEditor/) that serializes the current diff hunks into unified-diff format and saves to a user-chosen file. Alternatively, this could be an extension-candidate since it doesn't require privileged APIs — a marketplace extension can read both sides via the document API and produce a patch. A product decision on core-vs-extension is needed.
