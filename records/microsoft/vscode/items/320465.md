# Issue #320465 — Goal is cut off, hovering doesn't show it all

[https://github.com/microsoft/vscode/issues/320465](https://github.com/microsoft/vscode/issues/320465)

## Summary

Team-authored issue reporting that the autopilot goal banner truncates long text and hovering doesn't reveal the full goal. The banner uses CSS ellipsis truncation and sets a native `title` on the parent element, but the hover experience is inadequate. Keeping open for the feature author to address.

## Change summary

Long autopilot goal text is truncated in the goal banner and cannot be fully read via hover or any other mechanism.

## Best solution

Replace the native `title` attribute with an `IHoverService`-based custom hover on `_textEl` (where the truncation occurs) so the full goal text is reliably shown on hover. Alternatively, allow the banner to expand or wrap on click/hover. The fix belongs in `src/vs/workbench/contrib/chat/browser/widget/input/chatGoalBannerWidget.ts`.
