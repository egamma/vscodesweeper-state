# Issue #296917 — GitHub Enterprise Subscription Not Recognized in VSCode Private Marketplace Login

[https://github.com/microsoft/vscode/issues/296917](https://github.com/microsoft/vscode/issues/296917)

## Summary

Real bug confirmed by team (isidorn): personal accounts with GitHub Enterprise Cloud subscriptions are not recognized because the enterprise flag logic only checks for GHE Server auth providers or EMU accounts (underscore in label), not Enterprise Cloud org membership. Team member sandy081 is assigned.

## Change summary

VS Code's Private Marketplace login fails for personal accounts that are members of GitHub Enterprise Cloud organizations because `account.enterprise` is never set to true for them.

## Best solution

The `enterprise` flag computation in `src/vs/workbench/services/accounts/browser/defaultAccount.ts` (line 666) needs to also detect GitHub Enterprise Cloud org membership — likely by checking the entitlements data or querying the GitHub API for the user's enterprise association. sandy081 is assigned and aware.
