# Issue #264752 — Git commit suggestions in terminal aren't very useful without expansion

[https://github.com/microsoft/vscode/issues/264752](https://github.com/microsoft/vscode/issues/264752)

## Summary

Valid feature request to show commit message previews inline in terminal suggest list. No implementation found on main; the terminal suggest completion items for commits currently only show the hash as the label with the title available in the details pane.

## Change summary

Show part of the git commit title as inline preview text in the terminal suggestion list, so users don't need to expand details to identify commits.

## Best solution

Populate the `detail` or a preview label field on `ITerminalCompletion` items of kind `Commit` with the commit message subject. The change would likely live in the completion provider that generates git commit completions (an extension-side provider sending completions to the terminal suggest widget) and/or in `terminalSuggestAddon.ts` where completions are rendered.
