## Change summary

Allow dragging terminal tabs onto each other to group them, and dragging grouped tabs out to ungroup them.

## Best solution

Implement terminal tab grouping/ungrouping via drag-and-drop in `src/vs/workbench/contrib/terminal/browser/terminalTabsList.ts`, leveraging the now-available list widget DnD-between-items feature (https://github.com/microsoft/vscode/issues/200745, closed as done).
