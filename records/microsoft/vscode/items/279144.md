# Issue #279144 — Show progress window when attempting to restart vscode during a background update

[https://github.com/microsoft/vscode/issues/279144](https://github.com/microsoft/vscode/issues/279144)

## Summary

Team-authored issue describing a missing progress UI when users try to launch VS Code while a background update is still applying on Windows. Hard keep-open: author is a team member.

## Change summary

When a user tries to launch VS Code on Windows while a background Inno Setup update is still running, nothing visible happens — no progress indicator or explanation.

## Best solution

Implement the proposed `--update-in-progress` flag in `inno_updater.exe` to show a modal progress dialog while the setup mutex is held, then auto-launch VS Code once the update completes. The work lives in the native Inno updater code and the VS Code launcher logic on Windows.
