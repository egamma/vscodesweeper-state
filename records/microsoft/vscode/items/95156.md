# Issue #95156 — [folding] mode to have region based folding only

[https://github.com/microsoft/vscode/issues/95156](https://github.com/microsoft/vscode/issues/95156)

## Summary

Valid feature request for a 'regions-only' folding strategy. The current `editor.foldingStrategy` setting only supports 'auto' and 'indentation' — no option exists to restrict folding to explicit region markers (#region/#endregion). Still unimplemented on main.

## Change summary

Request a folding strategy mode that shows only explicit region-based folding (e.g. #region/#endregion) and suppresses language/indentation-based ranges.

## Best solution

Add a third value (e.g. 'regions') to `editor.foldingStrategy` in `src/vs/editor/common/config/editorOptions.ts` and wire it through `src/vs/editor/contrib/folding/browser/folding.ts` so only `IndentRangeProvider` region markers (or a dedicated region-only provider) are used.
