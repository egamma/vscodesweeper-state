# Issue #313478 — Column Guide at Word Wrap Column

[https://github.com/microsoft/vscode/issues/313478](https://github.com/microsoft/vscode/issues/313478)

## Summary

The requested column guide is already covered by the existing `editor.rulers` setting, and the reporter confirmed that this is exactly what they needed. This can be closed as existing/as-designed behavior rather than tracked as a new editor feature.

## Change summary

Add a toggleable visual guide at the configured wrapping column so users can see where long lines will wrap.

## Best solution

Keep using the existing editor ruler support for this scenario: users can set `editor.rulers` to the desired column, including the same value they use for `editor.wordWrapColumn` or formatting limits.
