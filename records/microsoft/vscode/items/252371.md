# Issue #252371 — Copy and paste not working

[https://github.com/microsoft/vscode/issues/252371](https://github.com/microsoft/vscode/issues/252371)

## Summary

Intermittent copy/paste failure tied to EditContext API — partially addressed by PR #280300 (Dec 2025) and follow-up refactoring (Jan 2026), but commenters still report occurrences. Already labeled `important` with 107 upvotes; should remain open.

## Change summary

Copy/paste keyboard shortcuts intermittently stop working in the editor, producing a 'Contributed command editor.action.clipboardCopyAction does not exist' error, requiring a window reload.

## Best solution

Continue investigating the race condition in EditContext clipboard handling. The workaround in commit 884364539d1 and the abstraction in 91eeae05dd2 address some paths but the intermittent failure persists for some users. The fix likely needs further hardening in `src/vs/editor/browser/controller/editContext/clipboardUtils.ts` and the CopyPasteController lifecycle.
