# Issue #189624 — Indent guide transparency affects Active indent guide

[https://github.com/microsoft/vscode/issues/189624](https://github.com/microsoft/vscode/issues/189624)

## Summary

Still reproduces on main: the theming participant in indentGuides.ts discards the entire color pair when the inactive indent color is transparent, which also hides the active guide even if it has a valid opaque color.

## Change summary

Setting `editorIndentGuide.background1` to a transparent color causes the active indent guide (with an opaque `activeBackground1`) to also become invisible.

## Best solution

In `src/vs/editor/browser/viewParts/indentGuides/indentGuides.ts` (lines 302-319), decouple the active/inactive color pair so that a transparent inactive color does not suppress the active color's CSS rule. The inactive color can fall back to transparent (no box-shadow) while the active color is still emitted.
