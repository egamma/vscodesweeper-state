# Issue #320413 — Option to project from "open recent" in new window

[https://github.com/microsoft/vscode/issues/320413](https://github.com/microsoft/vscode/issues/320413)

## Summary

Valid feature request for better discoverability of 'open in new window' in the Open Recent menu. The existing `window.openFoldersInNewWindow` setting partially addresses the global-toggle alternative, but the core ask — an inline discoverable action in the menu — is not currently provided. Keeping open for the community process (11/20 upvotes).

## Change summary

Request for an explicit, discoverable UI affordance to open recent projects in a new window directly from the File > Open Recent menu.

## Best solution

The existing `window.openFoldersInNewWindow: "on"` setting covers the 'always new window' case, and Ctrl/Cmd+Click is the per-item workaround. The remaining gap is discoverability — a possible improvement would be adding an inline 'open in new window' action in the recent-items Quick Pick (similar to other Quick Pick item buttons). This belongs in the workbench-workspace area.
