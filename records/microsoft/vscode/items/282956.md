# Issue #282956 — Maximized VS Code shows white bar at bottom

[https://github.com/microsoft/vscode/issues/282956](https://github.com/microsoft/vscode/issues/282956)

## Summary

This is a duplicate of #283533, which reports the same white line/bar at the bottom of a maximized VS Code window when the Windows taskbar is set to auto-hide. Both issues point to the same upstream Chromium/Electron rendering bug on Windows — commenters on #283533 confirm it also affects Edge and Chrome, and switching the ANGLE backend to OpenGL avoids it. Consolidating on the newer issue which has more discussion and reproduction details.
