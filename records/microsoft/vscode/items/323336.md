# Issue #323336 — open with code 

[https://github.com/microsoft/vscode/issues/323336](https://github.com/microsoft/vscode/issues/323336)

## Summary

This is a support question about the Windows Explorer shell context menu integration ("Open with Code"), not a bug in VS Code's source code. The context menu entry is controlled by the VS Code installer's shell integration option, not by the editor itself.

## Change summary

The 'Open with Code' option is missing from the Windows Explorer right-click context menu on folders.

## Best solution

The reporter should reinstall VS Code and ensure the 'Add "Open with Code" action to Windows Explorer directory context menu' checkbox is selected during installation. On Windows 11+, the entry may appear under 'Show more options' in the context menu. This is an installer/setup question, not a code defect.
