# Issue #308256 — Agents App: When I mark a session as done, I can't find it anymore

[https://github.com/microsoft/vscode/issues/308256](https://github.com/microsoft/vscode/issues/308256)

## Summary

Team-authored polish request about the Agents window 'Done' sessions UX — the separate Done group at the bottom is unintuitive; users expect done sessions to remain in-place with visual differentiation. Keeping open for maintainer judgment.

## Change summary

Sessions marked as 'done' disappear into a separate group at the bottom of the list, making them hard to find; users expect them to stay in their original location with visual differentiation.

## Best solution

Show done sessions in their original workspace group with a visual 'done' indicator (strikethrough, opacity, checkmark) rather than moving them to a separate bottom section. Related issues #323450 and #321324 track overlapping problems with done/archived session visibility and should be considered together.
