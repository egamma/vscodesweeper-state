# Issue #268456 — Copy selection from Debug console

[https://github.com/microsoft/vscode/issues/268456](https://github.com/microsoft/vscode/issues/268456)

## Summary

This is a duplicate of the closed issue #2163 (multi-page selection in debug console), which was declined as *out-of-scope because the debug console uses a virtualized tree — only rendered DOM nodes are selectable, so window.getSelection() inherently cannot capture off-screen text.

## Change summary

Copying a text selection from the Debug Console via context menu only copies the visible/rendered portion when the selection spans beyond the viewport.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/2163 — the debug console's WorkbenchAsyncDataTree virtualizes rows, so native selection cannot span beyond rendered DOM. The prior decision was *out-of-scope; if this should be revisited, reopen #2163 rather than tracking it twice.
