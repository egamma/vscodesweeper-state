# Issue #303653 — After Keep still all edits are shown

[https://github.com/microsoft/vscode/issues/303653](https://github.com/microsoft/vscode/issues/303653)

## Summary

Team-authored bug: after pressing 'Keep' in an agent session, 'View All Edits' still shows edits from before the Keep, not just subsequent ones. Keeping open per team-member policy.

## Change summary

"View All Edits" in agent sessions shows all accumulated edits including those already kept, instead of only edits produced after the last Keep action.

## Best solution

The edit tracking state should be reset (or a baseline recorded) when the user presses 'Keep', so that 'View All Edits' only surfaces changes produced after that point. Likely in the agent session or chat edit-tracking contribution under `src/vs/workbench/contrib/chat/`.
