# Issue #306686 — No way to see model incident message in model picker

[https://github.com/microsoft/vscode/issues/306686](https://github.com/microsoft/vscode/issues/306686)

## Summary

Real UX regression with an agreed-upon solution direction (use the tips/chat upgrade widget above chat input for degradation messages). Milestone is 'On Deck' — a scheduling decision that blocks any close proposal.

## Change summary

After the model picker hover was removed, there is no way to surface CAPI model incident/degradation messages to users.

## Best solution

Implement the agreed approach from the discussion: leverage the tips/chat upgrade widget above the chat input to show the degradation message when a degraded model is selected, keeping it non-dismissible in that state.
