# Issue #208199 — Focused search buttons border color update

[https://github.com/microsoft/vscode/issues/208199](https://github.com/microsoft/vscode/issues/208199)

## Summary

Still reproduces on main: the toggle component (used for Match Case, Match Whole Word, etc.) has no `:focus` or `:focus-visible` CSS rule in toggle.css, so focused toggles have no visible focus indicator distinct from their checked state. A team member confirmed this should be fixed.

## Change summary

Focused search toggle buttons (Match Case, Match Whole Word) lack a distinct focus indicator — the border/outline on focus blends with the checked background color.

## Best solution

Add a `:focus-visible` border or outline style to `.monaco-custom-toggle` in `src/vs/base/browser/ui/toggle/toggle.css` that uses `var(--vscode-focusBorder)` to provide a visible focus ring, distinguishable from the checked state's `inputActiveOptionBorder`/`inputActiveOptionBackground` colors.
