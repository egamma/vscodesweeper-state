## Summary

Confirmed bug still present on main: the stash parsing regex requires the `on <branch>: <description>` format and silently drops stashes with custom messages (e.g. from lint-staged or autostash). The issue has an 'On Deck' milestone and an open fix PR, so it stays open.

## Change summary
