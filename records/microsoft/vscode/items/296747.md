# Issue #296747 — Git: Create a fixup commit from the graph view

[https://github.com/microsoft/vscode/issues/296747](https://github.com/microsoft/vscode/issues/296747)

## Summary

Valid feature request for a fixup-commit action in the SCM graph context menu. No implementation exists on main, the prerequisite API (issue #201997) is still open, and no canonical issue absorbs this work. Keeping open in the Backlog for the community process.

## Change summary

Add a 'Create fixup commit' context menu action in the SCM graph view to combine commit selection and `git commit --fixup=<sha>` into one operation.

## Best solution

Once the SCM Graph history-item context menu API (https://github.com/microsoft/vscode/issues/201997) is finalized, this could be implemented either as a built-in command in `extensions/git/` or as an extension leveraging that API. The built-in git extension already has commit infrastructure; adding a `--fixup` variant would be small if the graph API exposes the target commit SHA.
