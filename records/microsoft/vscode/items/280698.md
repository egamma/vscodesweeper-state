# Issue #280698 — Not clear which chat items are expandable

[https://github.com/microsoft/vscode/issues/280698](https://github.com/microsoft/vscode/issues/280698)

## Summary

Team-authored feature request for clearer visual affordances on expandable chat progress items. No work has landed on main addressing this — the gap remains. Keeping open for the team to design a solution.

## Change summary

Chat progress items (tool invocations, thinking sections) lack visual indicators showing which are expandable, requiring hover to discover.

## Best solution

Add a collapsed caret/chevron to expandable progress items (thinking sections, tool invocations with detail) so users can tell at a glance which items contain more information. Consider varying the status icon by tool type (pencil for edits, flask for tests) instead of uniform checkmarks. Work likely lives in the `chatSimpleToolProgressPart.ts` and related progress-part renderers under `src/vs/workbench/contrib/chat/browser/widget/chatContentParts/toolInvocationParts/`.
