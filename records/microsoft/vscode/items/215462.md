# Issue #215462 — Tree Sticky Scroll does not support Touch

[https://github.com/microsoft/vscode/issues/215462](https://github.com/microsoft/vscode/issues/215462)

## Summary

Still valid on current main: the StickyScrollFocus class only registers a mouseDown handler — no touch/tap event handling exists. A team member acknowledged the gap and expressed intent to fix it.

## Change summary

Tree sticky scroll elements in the Explorer (and other tree views) do not respond to touchscreen taps for expand/collapse.

## Best solution

Add touch event handling (via Gesture/tap events) in the StickyScrollFocus class in src/vs/base/browser/ui/tree/abstractTree.ts, mirroring the existing onMouseDown logic so sticky scroll items respond to touch input.
