# Issue #199701 — Editor title context menu actions missing from PR multidiff editor

[https://github.com/microsoft/vscode/issues/199701](https://github.com/microsoft/vscode/issues/199701)

## Summary

Still a valid gap: the multi-diff editor contributes to MenuId.EditorTitle (toolbar) but the EditorTitleContext (tab right-click) actions like Copy Path and Reveal in Finder are not tailored for multi-diff editors. No fix found on main.

## Change summary

Editor title context menu actions (Copy Path, Reveal in Finder, etc.) are missing when right-clicking a multi-diff editor tab.

## Best solution

Ensure the multi-diff editor properly exposes its resource so that generic EditorTitleContext actions (Copy Path, Copy Relative Path, Reveal in Finder) work, or contribute multi-diff-specific variants to that menu. Likely involves src/vs/workbench/contrib/multiDiffEditor/browser/actions.ts and possibly the editor input's resource getter.
