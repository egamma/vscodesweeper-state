# Issue #313211 — Merge Editor is slow or fails to load files

Verified duplicate of #291982: both report that large diffs in SCM/merge editor cause the editor to hang or brick, requiring a restart. The underlying cause is the same — diff computation on large files blocks the UI without a working timeout safeguard.
