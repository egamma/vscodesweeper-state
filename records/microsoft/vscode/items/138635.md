# Issue #138635 — Codelens breaks bracket guides

[https://github.com/microsoft/vscode/issues/138635](https://github.com/microsoft/vscode/issues/138635)

## Summary

Still reproduces on main: bracket pair guides are rendered as DynamicViewOverlay per-line elements, so they cannot span across CodeLens view zones, causing visual breaks in the vertical guide lines.

## Change summary

Bracket pair guides display discontinuously (with gaps) where CodeLens view zones are inserted between lines.

## Best solution

The indent/bracket guide overlay in `src/vs/editor/browser/viewParts/indentGuides/indentGuides.ts` would need to either extend guide rendering into view zone space or use an alternative rendering strategy (e.g., a continuous element spanning multiple lines). This is the same class of problem as https://github.com/microsoft/vscode/issues/9604 for indent guides. A fix likely requires changes to how `DynamicViewOverlay` interacts with view zones.
