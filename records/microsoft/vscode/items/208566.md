# Issue #208566 — Support to file issue from the extensions page

[https://github.com/microsoft/vscode/issues/208566](https://github.com/microsoft/vscode/issues/208566)

## Summary

Still a valid, unimplemented feature request: the extensions editor page does not offer a way to continue filing an issue through VS Code's built-in issue reporter. The request remains relevant and is from a core team member testing related work.

## Change summary

Allow filing an issue for an extension directly from that extension's editor page, integrating with VS Code's built-in issue reporter.

## Best solution

Add a 'Report Issue' action to the extension editor page (extensionEditor.ts) that invokes the issue reporter pre-filled with the extension's metadata, similar to ReportExtensionIssueAction already used in the runtime extensions editor.
