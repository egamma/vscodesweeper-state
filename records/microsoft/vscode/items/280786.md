## Summary

Team-authored bug report about edited chat requests not appearing in the per-session input history when switching between sessions. No evidence of a fix on current main; the issue remains valid.

## Change summary

When a user edits and resends a chat request in one session, the edited message does not appear in the input history when navigating history in another session (or the same session).

## Best solution
