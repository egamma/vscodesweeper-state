# Issue #286980 — Thinking block summary should linkify appropriately

[https://github.com/microsoft/vscode/issues/286980](https://github.com/microsoft/vscode/issues/286980)

## Summary

Still reproduces on current main: `setFinalizedTitle` renders the thinking summary as plain text (via `textContent`), losing all markdown/link rendering that the streaming `setTitle` path provides. The bug is localized and clear.

## Change summary

When a thinking block completes, code references like `_maxLineWidth` in the collapsed summary are rendered as plain text instead of linkified symbols.
