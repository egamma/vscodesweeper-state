## Summary

This is a support question, not a bug. VS Code is a lightweight text editor out of the box and does not ship a built-in 'start' button to run programs — that capability comes from language-specific extensions (e.g. Python, C/C++, Code Runner) or the built-in terminal. The reporter has no extensions installed, which explains the missing button.

## Change summary

User cannot find a 'start' button to run programs in VS Code with no extensions installed.

## Best solution

Direct the user to install the appropriate language extension for their programming language, which will provide run/debug buttons, or to use the integrated terminal (`Ctrl+``) to run programs manually.
