  > Thanks for reaching out. Git worktree support is part of VS Code's built-in Git extension and does **not** require a Copilot subscription. You can find worktree commands in the Source Control view — right-click the repository header (or use the `…` menu) and look for the "Worktrees" submenu.
  >
