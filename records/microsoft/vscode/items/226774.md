# Issue #226774 — Show raw json sync file doesn't open the file

[https://github.com/microsoft/vscode/issues/226774](https://github.com/microsoft/vscode/issues/226774)

## Summary

Still broken on main: the 'Show raw JSON sync data' action parses the tree item handle as `{ resource }` but `sync-resource-*` items store `{ syncResourceHandle, syncResource }` — the resource field is undefined, so the editor open fails silently.

## Change summary

The 'Show raw JSON sync data' context menu action on sync activity items fails to open the file because it reads a nonexistent `resource` property from the serialized handle.

## Best solution

Fix the action at line ~240 in `userDataSyncViews.ts` to parse the handle as `{ syncResourceHandle, syncResource }` and use `URI.revive(syncResourceHandle.uri)` to open the editor, matching how the 'Replace' action (line 287) already correctly parses the same handle format.
