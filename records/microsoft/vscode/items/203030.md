Enable clicking on intermediate nodes within a compressed (compact folders) sticky scroll element to navigate or interact with them individually.

## Best solution

Implement mouse interaction support for intermediate compressed nodes in the sticky scroll widget, likely in `src/vs/base/browser/ui/tree/` or `src/vs/workbench/browser/parts/views/` sticky scroll code.
