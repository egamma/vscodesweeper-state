# Issue #221595 — Support selecting debug configuration providers using ‘when’ clause

[https://github.com/microsoft/vscode/issues/221595](https://github.com/microsoft/vscode/issues/221595)

## Summary

Valid feature request from a team member to extend debug configuration provider selection with `when` clauses. No implementation exists on current main; `providesConfigurationWhen` is not referenced anywhere in the codebase. Hard keep-open: author is MEMBER.

## Change summary

Allow debug configuration providers to declare a `when` clause so VS Code can select the right debugger based on file extension, file name, or context variables — not just language ID.

## Best solution

Extend `contributes.debuggers` with an optional `providesConfigurationWhen` clause and update `guessDebugger` in `src/vs/workbench/contrib/debug/browser/debugAdapterManager.ts` to evaluate it as a fallback or override when selecting a debug adapter. The C# extension team (author) has concrete use cases; coordinate with them on the exact semantics before implementing.
