# Issue #219821 — Indent using spaces function does not convert tabs to spaces in mixed indentation text content

[https://github.com/microsoft/vscode/issues/219821](https://github.com/microsoft/vscode/issues/219821)

## Summary

The reporter expected 'Indent Using Spaces' to convert existing tabs to spaces, but that command only sets the indentation mode for future edits. The separate 'Convert Indentation to Spaces' command performs the conversion. The reporter acknowledged this in comment #3.

## Change summary

User expected 'Indent Using Spaces' to also convert existing tab characters to spaces.

## Best solution

Keep current behavior — 'Indent Using Spaces' sets the mode, 'Convert Indentation to Spaces' performs the conversion. These are intentionally separate commands with distinct responsibilities.
