# Issue #318008 — [Error] unhandlederror-Uncaught Error: Illegal value for lineNumber

[https://github.com/microsoft/vscode/issues/318008](https://github.com/microsoft/vscode/issues/318008)

## Summary

Keep open and route to editor hover/model owners; this is a new stable error telemetry regression with a concrete stack and non-trivial user impact, not a close candidate.

## Change summary

Hover computation can call `TextModel.getLineMaxColumn` with an invalid `anchor.range.startLineNumber`, throwing `Illegal value for lineNumber`.

## Best solution

Investigate the editor hover anchor lifecycle and either prevent stale/invalid range anchors from reaching hover participants or validate the line number before synchronous hover computation in `src/vs/editor/contrib/hover/browser/markdownHoverParticipant.ts`.
