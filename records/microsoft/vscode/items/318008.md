# Issue #318008 — [Error] unhandlederror-Uncaught Error: Illegal value for lineNumber

[https://github.com/microsoft/vscode/issues/318008](https://github.com/microsoft/vscode/issues/318008)

## Summary

Confirmed regression on current main: the hover debouncer can deliver a stale anchor after lines are deleted, causing an out-of-bounds lineNumber in getLineMaxColumn. A fix PR exists (#318018) but is not yet merged.

## Change summary

Hovering after lines are deleted can throw 'Illegal value for lineNumber' because the stale anchor's lineNumber exceeds the model's line count.

## Best solution

Merge PR https://github.com/microsoft/vscode/pull/318018 which adds a bounds guard in contentHoverComputer.computeSync before passing the anchor to participants. The fix is small and localized.
