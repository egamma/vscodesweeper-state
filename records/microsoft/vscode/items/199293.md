## Summary

## Change summary

Allow clicking on the file name in the multi-diff editor header to expand/collapse the diff, not just the chevron button.

## Best solution

The gate is in `src/vs/workbench/contrib/multiDiffEditor/browser/multiDiffEditor.ts` line 233 — change the condition or make it unconditionally true. The click handler already exists in `diffEditorItemTemplate.ts`.
