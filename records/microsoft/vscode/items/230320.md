## Summary

Confirmed bug with an open fix PR (#313487) still in progress. The cursor position after pasting large content in JSON is incorrect because the edit range is not correctly tracked in AutoIndentOnPasteCommand's computeCursorState. Keep open pending the fix PR merge.

## Change summary

Pasting large content in a JSON file leaves the cursor in the middle of the pasted text instead of at the end.

## Best solution

Merge PR #313487 which uses tracked edit operations in AutoIndentOnPasteCommand to correctly compute the final cursor position after indentation adjustments are applied.
