# Issue #127438 — StatusBarItem.tooltip2 right border doesn't show when window is narrow

[https://github.com/microsoft/vscode/issues/127438](https://github.com/microsoft/vscode/issues/127438)

## Summary

Still present on main: the `left: 1px` CSS rule on `.right-aligned` hovers (hover.css:137) causes right-border clipping when the window is narrow. An open PR (#305052) targets this exact issue.

## Change summary

StatusBar hover right border is clipped when the window is narrow and the hover is right-aligned.

## Best solution

Review and land https://github.com/microsoft/vscode/pull/305052, which removes the `left: 1px` offset on right-aligned hovers to prevent border clipping. The fix is in `src/vs/platform/hover/browser/hover.css`.
