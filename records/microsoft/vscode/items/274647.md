# Issue #274647 — Chat: Delegate to agent ignores attached plan document

[https://github.com/microsoft/vscode/issues/274647](https://github.com/microsoft/vscode/issues/274647)

## Summary

Team-authored bug report about 'Delegate to Agent' not forwarding the content of attached plan documents to the agent session, resulting in the agent lacking context to implement the requested work. Keeping open for maintainer judgment.

## Change summary

When delegating a chat prompt to an agent with an attached markdown plan document, the agent receives only a reference to the doc rather than its content, making it unable to act on the plan details.

## Best solution

Ensure the delegate-to-agent flow serializes attached document content (not just a file reference) into the agent's initial prompt context, so the agent can act on the plan without requiring re-submission of the attachment.
