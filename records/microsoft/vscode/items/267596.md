# Issue #267596 — Feature: Toggle custom Copilot instructions per chat session

[https://github.com/microsoft/vscode/issues/267596](https://github.com/microsoft/vscode/issues/267596)

## Summary

The related issue #282422 was implemented with global settings (`chat.includeApplyingInstructions`, `chat.includeReferencedInstructions`), which partially addresses this. However, this issue specifically requests a per-session UI toggle — a distinct ask that remains unimplemented. Valid feature request, low traction (1 upvote), in Backlog Candidates territory.

## Change summary

Request for a per-chat-session toggle to temporarily disable automatic attachment of custom .instructions.md files without changing the global setting.

## Best solution

Add a per-session UI affordance (e.g., a toolbar toggle or right-click option on the send button) that overrides the global `chat.includeApplyingInstructions` setting for that session only. The global settings from PR #289731 cover the static case; this would complement them with dynamic, session-scoped control.
