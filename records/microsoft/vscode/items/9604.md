# Issue #9604 — Indent Guides Have Breaks Where CodeLens UI is Rendered

[https://github.com/microsoft/vscode/issues/9604](https://github.com/microsoft/vscode/issues/9604)

## Summary

Still unfixed on main: indent guides are rendered per visible line in `IndentGuidesOverlay` without any logic to extend through view zones (CodeLens). The issue has 40 upvotes (exceeds the 20-upvote promotion threshold) and sits in the Backlog milestone — it should remain open.

## Change summary

Indent guides break visually wherever CodeLens view zones are rendered, creating discontinuous vertical lines.

## Best solution

Extend the indent-guide renderer (`src/vs/editor/browser/viewParts/indentGuides/indentGuides.ts`) to paint guides through view zones that opt in (like CodeLens), possibly using a flag on `IViewZone` such as `shouldRenderIndentGuides`. Related open issue https://github.com/microsoft/vscode/issues/138635 covers bracket guides and should be addressed together.
