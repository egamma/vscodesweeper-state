## Summary

Team-authored polish request about simplifying the chat steering/queueing button states. Several incremental refinements have landed (button state logic, Alt-key toggle, toolbar responsiveness), but the core proposal — showing only one action at a time based on typing state — remains an open design discussion with team members involved.

## Change summary

Reduce visual clutter in chat input by showing only the stop button when idle or only the send button when the user has typed text, instead of all buttons simultaneously.

## Best solution
