# Issue #276614 — auto-indenting and line spacing issues

[https://github.com/microsoft/vscode/issues/276614](https://github.com/microsoft/vscode/issues/276614)

## Summary

Legitimate auto-indent bug for C#: the built-in C# language configuration lacks `indentationRules`, so the editor cannot maintain indentation for multi-line argument lists. A team member (aiday-mar) is already engaged and has the reproduction details needed.

## Change summary

Pressing Enter inside indented C# multi-line method calls loses indentation and inserts an extra blank line instead of continuing at the same indent level.

## Best solution

Add proper `indentationRules` (increaseIndentPattern / decreaseIndentPattern) to `extensions/csharp/language-configuration.json` to handle open-paren continuation indentation, or coordinate with the C# DevKit extension team if they own these rules. The `editor-autoindent` area owner should decide ownership.
