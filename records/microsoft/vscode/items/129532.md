# Issue #129532 — Disassembly view: not possible to set instruction breakpoint with keyboard

[https://github.com/microsoft/vscode/issues/129532](https://github.com/microsoft/vscode/issues/129532)

## Summary

Fixed on main and shipped in 1.83.0. The ToggleBreakpointAction was converted to Action2 with CONTEXT_DISASSEMBLY_VIEW_FOCUS added to its keybinding, and the handler now checks for DisassemblyView to toggle instruction breakpoints via F9.

## Change summary

Pressing F9 in the disassembly view did nothing — instruction breakpoints could not be set via keyboard.

## Best solution

Keep the shipped fix from commit 530f01336e2 (1.83.0). F9 now toggles instruction breakpoints in the disassembly view. No further change needed.
