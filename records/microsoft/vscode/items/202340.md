# Issue #202340 — Feature: Debug console filters in settings.json

[https://github.com/microsoft/vscode/issues/202340](https://github.com/microsoft/vscode/issues/202340)

## Summary

Valid feature request with 23 upvotes, promoted to Backlog. The debug console filter is currently UI-only (search bar); no settings-based filter configuration exists on current main. The request remains unimplemented and worth keeping open.

## Change summary

Allow debug console filters to be configured via workspace settings (settings.json) so teams can share default filter rules.

## Best solution

Add a `debug.console.filters` setting (string array) in `src/vs/workbench/contrib/debug/browser/repl.ts` and `replFilter.ts` that pre-populates the filter query from workspace settings, complementing the existing UI filter bar. This lets teams commit shared filter rules in `.vscode/settings.json`.
