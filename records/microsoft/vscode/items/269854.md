Team-authored feature request with an active design discussion (view zone vs. NES rendering conflict). No resolution yet; keeping open for the team to decide the UX approach.

## Change summary

NES (next edit suggestions) should render when the user jumps to the next problem via F8, even though a peek/problem view zone is currently displayed.
