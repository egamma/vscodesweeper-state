# Issue #207848 — [Accessibility] Cannot clear the SCM input history

[https://github.com/microsoft/vscode/issues/207848](https://github.com/microsoft/vscode/issues/207848)

## Summary

Valid request — no mechanism exists on current main to clear the SCM input commit message history. The feature is still missing; keeping open for the area owner to evaluate.

## Change summary

Add a way to clear the SCM commit message input history so users (especially screen reader users) are not confused by stale entries when navigating with Up/Down arrows.

## Best solution

Add a 'Clear SCM Input History' command (and possibly a context menu action) that resets the history array in `SCMInput` (src/vs/workbench/contrib/scm/common/scmService.ts). The accessibility help provider in scmAccessibilityHelp.ts should mention the command. This is small and localized to the SCM contribution.
