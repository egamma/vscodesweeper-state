# Issue #138546 — [json] open definition in JSON document to navigate to the schema definition

[https://github.com/microsoft/vscode/issues/138546](https://github.com/microsoft/vscode/issues/138546)

## Summary

Legitimate feature request acknowledged by a team member (aeschli). The JSON extension still has no definition provider for navigating from document properties to their schema definitions on current main. Parked in Backlog with low community traction.

## Change summary

Request Go to Definition support in JSON documents to navigate from a property to its corresponding definition in the associated JSON schema file.

## Best solution

Implement a DefinitionProvider in the JSON language server (extensions/json-language-features/server/src/jsonServer.ts) that resolves the schema location for the property under the cursor and returns a Location pointing into the schema file. This would complement the existing $ref link support.
