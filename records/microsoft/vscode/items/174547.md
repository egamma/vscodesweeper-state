# Issue #174547 — Explorer hover text can become sticky when primary side bar is positioned on the right

[https://github.com/microsoft/vscode/issues/174547](https://github.com/microsoft/vscode/issues/174547)

## Summary

The experimental explorer hover feature that caused this bug was entirely removed in commit ed83e0e84f5 (May 2024, shipped in 1.90.0). The setting `explorer.experimental.hover` no longer exists, so the sticky-hover behavior cannot occur.

## Change summary

Explorer hover text became sticky and mispositioned when the sidebar was on the right, because the experimental hover relocated left but was no longer under the mouse.

## Best solution

Close as completed — the experimental hover feature was removed, eliminating this class of bugs entirely. No further action needed.
