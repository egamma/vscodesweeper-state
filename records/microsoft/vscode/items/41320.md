# Issue #41320 — [xml] add on enter rules

[https://github.com/microsoft/vscode/issues/41320](https://github.com/microsoft/vscode/issues/41320)

## Summary

The XML built-in extension still lacks onEnterRules and indentationRules on current main. The HTML extension has these (with void-element exclusions), making automatic indentation work for HTML but not XML. This is a valid, unfulfilled feature request.

## Change summary

XML files lack automatic indentation on Enter (onEnterRules) that HTML already provides.

## Best solution

Add `onEnterRules` and `indentationRules` to `extensions/xml/xml.language-configuration.json`, modeled on the HTML rules but without void-element exclusions (XML has no void elements). This is a small, self-contained change to the built-in XML extension.
