## Change summary

Cutting/pasting or dragging a collapsed code block should preserve its folded state at the destination.

## Best solution

Implement fold-state preservation across cut/paste and drag-drop in `src/vs/editor/contrib/folding/`. The folding model would need to serialize collapsed ranges on cut/drag, then restore them relative to the new insertion offset on paste/drop. Likely touches `foldingModel.ts` and the editor's paste/drop handling.
