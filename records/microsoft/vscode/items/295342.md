# Issue #295342 — Word Wrap broken with suggested completion

[https://github.com/microsoft/vscode/issues/295342](https://github.com/microsoft/vscode/issues/295342)

## Summary

Team-authored issue; still reproduces on main — the inline edits preview editors explicitly set `wordWrap: 'off'`, so NES suggestions in word-wrapped files truncate visually.

## Change summary

Inline completion (NES) suggestions ignore the editor's word-wrap setting, making long-line changes unreadable.

## Best solution

The side-by-side and long-distance preview editors in `src/vs/editor/contrib/inlineCompletions/browser/view/inlineEdits/inlineEditsViews/` should inherit or respect the parent editor's `wordWrap` setting rather than forcing it `'off'`.
