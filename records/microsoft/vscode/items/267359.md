# Issue #267359 — Surface terminal progress sequence results in first class UI

[https://github.com/microsoft/vscode/issues/267359](https://github.com/microsoft/vscode/issues/267359)

## Summary

The issue requests a visual progress bar UI (at the top of the terminal or on the tab) for OSC 9;4 progress sequences. While progress state tracking exists since Jan 2025, no first-class progress bar widget has been implemented — the request remains valid.

## Change summary

Display terminal progress sequences (OSC 9;4) as a visual progress bar in the terminal UI, similar to the progress bar used elsewhere in VS Code.

## Best solution

Add a ProgressBar widget to the terminal pane (likely at the top, analogous to the editor's progress bar) driven by the existing `progressState` from `@xterm/addon-progress`. The terminal tab could also show a visual indicator. Implementation would live in `src/vs/workbench/contrib/terminal/browser/` using the existing `ProgressBar` from `src/vs/base/browser/ui/progressbar/`.
