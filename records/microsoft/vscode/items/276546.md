# Issue #276546 — Keep system awake during agent actions

[https://github.com/microsoft/vscode/issues/276546](https://github.com/microsoft/vscode/issues/276546)

## Summary

This feature has been implemented: the Copilot extension now acquires a power save blocker during active chat/agent requests, preventing the system from sleeping. The blocker is reference-counted and released 2 minutes after the last request completes.

## Change summary

Request VS Code to prevent the system from sleeping while a Copilot Chat agent is actively processing a task.

## Best solution

Keep the shipped implementation in commit de441dc1784 — the PowerService in the Copilot extension acquires Electron's powerSaveBlocker during chat requests and releases it after a 2-minute grace period. No further change needed.
