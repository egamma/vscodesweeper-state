# Issue #186425 — Keyboard shortcuts for showing/hiding unchanged regions

[https://github.com/microsoft/vscode/issues/186425](https://github.com/microsoft/vscode/issues/186425)

## Summary

The specific keyboard navigation UX described (arrow-key focus on collapsed-region indicators, enter to expand) remains unimplemented on main. Bulk show/collapse commands exist in the command palette but lack keybindings, and there is no individual-region keyboard interaction. Keeping open as a valid accessibility-adjacent feature request.

## Change summary

Add keyboard shortcuts for focusing and expanding/collapsing individual unchanged region indicators in the diff editor.

## Best solution

Implement keyboard navigation for unchanged-region indicators in `src/vs/editor/browser/widget/diffEditor/features/hideUnchangedRegionsFeature.ts`: when the cursor is on the line above a collapsed region, down-arrow focuses the indicator; a second down-arrow skips past it; enter/space expands it. This improves keyboard-only and screen-reader access to the diff editor.
