# Issue #253295 — Context menu on chat message should have an Edit entry

[https://github.com/microsoft/vscode/issues/253295](https://github.com/microsoft/vscode/issues/253295)

## Summary

Still valid: the right-click context menu on chat messages (MenuId.ChatContext) does not include an 'Edit' or 'Undo Request' entry. These actions exist only in the toolbar (MenuId.ChatMessageTitle). The request to surface them in the context menu for discoverability remains unimplemented on current main.

## Change summary

Add 'Edit' and 'Undo Request' actions to the chat message right-click context menu for better discoverability.

## Best solution

Register the existing edit-message and undo-request actions (from chatExecuteActions.ts / chatQueueActions.ts) into MenuId.ChatContext alongside their existing ChatMessageTitle toolbar registrations, gated on the same when-clauses. The work lives in src/vs/workbench/contrib/chat/browser/actions/.
