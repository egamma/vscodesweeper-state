# Issue #142208 — Dark Editor Mouse Pointer - Again

[https://github.com/microsoft/vscode/issues/142208](https://github.com/microsoft/vscode/issues/142208)

## Summary

This is as-designed: the custom cursor was intentionally removed in 1.64.0 (commit 56718a15f02) to use the native OS text cursor, which since macOS Catalina has a white border suitable for dark themes. The team member explained the rationale and the reporter accepted the explanation.

## Change summary

The native OS text cursor (I-beam) appears with a black center and white outline on dark themes instead of the previous custom inverted cursor.

## Best solution

Keep the current behavior — the native OS cursor is the correct choice for accessibility (respects system cursor customization) and platform consistency. Users who want a different cursor appearance should adjust their OS pointer settings.
