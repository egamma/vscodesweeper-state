# Issue #181935 — [html] Allow renaming images using F2

[https://github.com/microsoft/vscode/issues/181935](https://github.com/microsoft/vscode/issues/181935)

## Summary

This feature request for F2 file-rename support on path attributes in HTML (analogous to markdown's existing link rename) has not been implemented. The html-language-features extension has no file-rename provider; rename only works for JS symbols in <script> tags.

## Change summary

Allow F2 rename on image/link paths in HTML files to rename both the attribute value and the referenced file on disk.

## Best solution

Implement a rename provider in `extensions/html-language-features` that detects when the cursor is on a path attribute value (src, href, etc.) and performs a workspace edit renaming both the string in the document and the file on disk, mirroring `extensions/markdown-language-features/src/languageFeatures/linkUpdater.ts`.
