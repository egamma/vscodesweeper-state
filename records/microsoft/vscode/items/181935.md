# Issue #181935 — [html] Allow renaming images using F2

[https://github.com/microsoft/vscode/issues/181935](https://github.com/microsoft/vscode/issues/181935)

## Summary

Feature request remains unimplemented on main. The HTML extension's rename provider only handles JavaScript variable renames within <script> blocks; there is no file-path rename support for attributes like `src`.

## Change summary

Allow F2 on an image path in an HTML `src` attribute to rename both the attribute value and the referenced file on disk, matching existing markdown behavior.

## Best solution

Implement a path-aware rename provider in `extensions/html-language-features` (likely a new mode or an addition to the HTML mode) that detects when the cursor is on a path-like attribute value, and performs a workspace edit renaming both the attribute text and the file. The markdown extension's `linkUpdater.ts` is a reference implementation.
