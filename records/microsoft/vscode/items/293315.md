# Issue #293315 — Git diff colors in scrollbar render incorrectly in VS Code Dark (Experimental Dark) theme

[https://github.com/microsoft/vscode/issues/293315](https://github.com/microsoft/vscode/issues/293315)

## Summary

Still reproduces on main: the 2026-dark theme inherits a blue gutter-modified color (#0078D4) from dark_modern but explicitly sets the overview ruler modified color to green (#6ab890), creating a mismatch between the gutter and scrollbar indicators for modified lines.

## Change summary

Git diff 'modified' color in the overview ruler (scrollbar) is green while the gutter indicator is blue in the Dark 2026 (formerly Experimental Dark) theme.

## Best solution

In `extensions/theme-defaults/themes/2026-dark.json`, either add an explicit `editorGutter.modifiedBackground` matching the green color scheme used for the overview ruler, or remove the explicit `editorOverviewRuler.modifiedForeground` so it derives from the gutter color as intended by the default calculation (`transparent(editorGutterModifiedBackground, 0.6)`).
