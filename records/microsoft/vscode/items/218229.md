# Issue #218229 — SCM Graph - high contract line types

[https://github.com/microsoft/vscode/issues/218229](https://github.com/microsoft/vscode/issues/218229)

## Summary

Team-authored accessibility feature request. The SCM graph lines are still drawn as uniform solid strokes differentiated only by color — no dashed/dotted line styles per branch have been implemented. Keeping open per hard keep-open rule (MEMBER author) and because the accessibility gap remains real.

## Change summary

SCM graph branch lines use color as the sole visual differentiator; request dashed/dotted/solid line styles per branch for accessibility (WCAG: color must not be the only means of conveying information).

## Best solution

Add stroke-dasharray variations to `drawVerticalLine` and the curve-drawing paths in `src/vs/workbench/contrib/scm/browser/scmHistory.ts`, cycling through solid/dashed/dotted per branch index, at minimum in high-contrast themes. Also consider exposing branch name in the hover tooltip as a complementary non-color cue.
