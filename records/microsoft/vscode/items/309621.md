# Issue #309621 — Find session flow is really not accessible

[https://github.com/microsoft/vscode/issues/309621](https://github.com/microsoft/vscode/issues/309621)

## Summary

Team-authored accessibility bug for the find/search session flow in the chat/agents area — needs an accessibility pass. Keeping open per policy (MEMBER author).

## Change summary

The find session flow lacks proper screen reader announcements (no result count, unclear search context) and keyboard navigation (Ctrl+F not working in the list).

## Best solution

Perform a full accessibility pass on the find session flow: add ARIA labels clarifying what is being searched, announce result counts on input, and ensure Ctrl+F or an equivalent keybinding works for keyboard-only users within the list.
