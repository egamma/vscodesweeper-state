[https://github.com/microsoft/vscode/issues/237314](https://github.com/microsoft/vscode/issues/237314)

## Summary

Still reproduces on main: the inline test error message (TestErrorContentWidget with allowEditorOverflow=false, and TestMessageDecoration using after-content) is clipped when appended to a long line that extends past the viewport. Assigned to connor4312 — keeping open for maintainer judgment.

## Change summary

Inline test error messages are not visible when they appear at the end of lines long enough to require horizontal scrolling.

## Best solution
