## Summary

Valid polish request — the Customize Layout dropdown includes sidebar position options but not panel position options. A team member (benibenj) already engaged and asked for UX input; the gap remains on current main.

## Change summary

Add panel position (top/bottom/left/right) controls to the Customize Layout quick pick, similar to how Primary Side Bar Position is already exposed there.

## Best solution

Add a 'Panel Position' option group to the Customize Layout quick pick in `src/vs/workbench/browser/actions/layoutActions.ts`, using the existing `PositionPanelActionId` actions from `panelActions.ts`.
