# Issue #288575 — Icons in thinking blocks are blurry

[https://github.com/microsoft/vscode/issues/288575](https://github.com/microsoft/vscode/issues/288575)

## Summary

Still open on main: thinking-block icons remain 12×12px (chatThinkingContent.css lines 312–316), which causes blurriness on standard-DPI monitors. Author is a CONTRIBUTOR, so hard keep-open applies regardless.

## Change summary

Icons in chat thinking blocks are rendered at 12×12px, causing blurry appearance on standard DPI monitors; should be 16×16px (or use crisp smaller variants).

## Best solution

Either increase the thinking-icon size to 16×16px and adjust surrounding padding in `src/vs/workbench/contrib/chat/browser/widget/chatContentParts/media/chatThinkingContent.css`, or create pixel-perfect 12px icon variants. The first approach is simpler and was recommended by the reporter.
