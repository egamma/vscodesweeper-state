# Issue #288752 — CCR should support skills

[https://github.com/microsoft/vscode/issues/288752](https://github.com/microsoft/vscode/issues/288752)

## Summary

Valid feature request with 26 upvotes asking Copilot Code Review to consume skills (SKILL.md-based customization). CCR currently loads instructions from instruction files and settings via ICustomInstructionsService but does not resolve skill directories. Already assigned to roblourens; needs area labels.

## Change summary

Copilot Code Review should load and apply user-defined skills (SKILL.md files) alongside existing instruction files and settings.

## Best solution

Extend the CCR review pipeline (in `extensions/copilot/src/extension/review/node/githubReviewAgent.ts`, specifically `loadCustomInstructions`) to also resolve skill locations via `resolveSkillConfigLocations` or the `IPromptsService` skills API, similar to how the Claude agent session already loads skills via `ClaudePluginService`. This would ensure that users who organize their coding guidelines as skills get them applied during code review.
