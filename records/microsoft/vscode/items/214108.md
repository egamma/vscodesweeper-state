# Issue #214108 — Option to default editor view to "Zen Mode"

[https://github.com/microsoft/vscode/issues/214108](https://github.com/microsoft/vscode/issues/214108)

## Summary

Valid niche feature request for an 'always start in Zen Mode' setting. The existing `zenMode.restore` (default true) partially covers this — it re-enters Zen Mode on restart if the window was closed in Zen Mode — but doesn't cover fresh workspaces. Low community signal (0 upvotes) after 13 months; keeping open for the community process.

## Change summary

Add a setting to always open new VS Code windows/workspaces in Zen Mode by default.

## Best solution

The workaround is entering Zen Mode once per workspace (it persists via `zenMode.restore`). A dedicated `zenMode.alwaysOn` or `window.defaultZenMode` setting in `src/vs/workbench/browser/workbench.zenMode.contribution.ts` would close the gap for fresh workspaces. Given the niche audience and existing workaround, this is a reasonable backlog candidate but unlikely to be prioritized without more community demand.
