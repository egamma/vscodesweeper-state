# Issue #325819 — Allow code to set the window title from the Terminal

[https://github.com/microsoft/vscode/issues/325819](https://github.com/microsoft/vscode/issues/325819)

## Summary

Valid feature request for a CLI flag to set a custom editor/window title when piping stdin to VS Code. No existing capability covers this; the `window.title` setting uses template variables but cannot be overridden per-invocation from the command line.

## Change summary

Add a CLI flag (e.g. `--title`) to set a custom tab/window title when opening files or piping stdin to VS Code.

## Best solution

If pursued, add a `--title` argument to the CLI (src/vs/platform/environment/node/argv.ts) that feeds into the title bar service for the opened editor tab. However, this is a niche workflow and could also be addressed by an extension using the `vscode.window` API — worth discussing whether it belongs in core.
