# Issue #319547 — Model token cost is badly translated in French

## Summary

The abbreviated localize strings "In: {0}" and "Out: {0}" in chatModelsWidget.ts lack sufficient context for translators, causing French to mistranslate "In" as "dans" (inside) instead of abbreviating "Entrée" (input).
