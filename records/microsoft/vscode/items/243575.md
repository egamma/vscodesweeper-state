# Issue #243575 — Dialogues such that "Open File" do not open at all on Kubuntu

[https://github.com/microsoft/vscode/issues/243575](https://github.com/microsoft/vscode/issues/243575)

## Summary

Verified duplicate of the open canonical issue tracking KDE file dialog failures on snap. The root cause is a shared-mime-info infinite loop in KDE's kdialog, combined with an Electron runtime detection bug — both tracked in #230679 with upstream links.

## Change summary

Native file dialogs (Open File, Save As) freeze or fail to appear on Kubuntu 24.04 with the VS Code snap package.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/230679, which already tracks the upstream Electron and KDE fixes. The workaround is `"files.simpleDialog.enable": true` or upgrading to Kubuntu 24.10+.
