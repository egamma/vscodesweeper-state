# Issue #324774 — Copilot Chat (agent mode): final assistant message intermittently dropped/not rendered after tool calls

[https://github.com/microsoft/vscode/issues/324774](https://github.com/microsoft/vscode/issues/324774)

## Summary

Genuine rendering bug where the final assistant markdown after tool calls is dropped due to a content-array length mismatch in the progressive vs. final rendering paths. An open PR (https://github.com/microsoft/vscode/pull/326497) already targets the fix — keep open until that PR lands.

## Change summary

Copilot Chat intermittently fails to render the assistant's final text response after tool calls complete, even though the backend work succeeds.

## Best solution

Land https://github.com/microsoft/vscode/pull/326497, which introduces `IChatTailPlaceholder` sentinels to align content-array lengths between progressive and final rendering in `ChatListItemRenderer`, fixing the `diff()` index mismatch that drops the final markdown slot.
