# Issue #291338 — subagent description runoff

[https://github.com/microsoft/vscode/issues/291338](https://github.com/microsoft/vscode/issues/291338)

## Summary

Team-authored visual bug — subagent description text overflows its container. The label element in the collapsible header lacks CSS text-overflow/ellipsis handling, so long descriptions run off. Hard keep-open (MEMBER author).

## Change summary

Subagent description text overflows its header container instead of truncating with an ellipsis.

## Best solution

Add CSS `overflow: hidden; text-overflow: ellipsis; white-space: nowrap;` (or `max-width` + ellipsis) to the subagent header label in `chatThinkingContent.css` or `chatSubagentContent.css`, complementing the existing `rcut` character truncation. The label element is `.chat-used-context-label .monaco-button.monaco-icon-button` inside `.chat-subagent-part`.
