# Issue #16998 — Add SmartIndent capability

[https://github.com/microsoft/vscode/issues/16998](https://github.com/microsoft/vscode/issues/16998)

## Summary

Feature request for a programmatic SmartIndent API (akin to Visual Studio's ISmartIndent) remains unimplemented on main. The existing indentation model is rule-based only; no provider API or virtual-space support has been added. With 14 upvotes and active interest (latest comment June 2025), it should stay open in Backlog.

## Change summary

Request an extension API that lets language services programmatically determine indentation level for a given line, enabling smart caret placement on blank lines.

## Best solution

Design a proposed `IndentationProvider` extension API (in `src/vscode-dts/`) that language extensions can implement to return indentation for a given line context, complementing the existing declarative `indentationRules`/`onEnterRules`. Virtual space (issue #13960) would be a natural companion but could be decoupled.
