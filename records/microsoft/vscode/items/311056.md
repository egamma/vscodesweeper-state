# Issue #311056 — When running tests, the pill icon background is rotating along with it's glyph

[https://github.com/microsoft/vscode/issues/311056](https://github.com/microsoft/vscode/issues/311056)

## Summary

Confirmed visual bug still present on main: the codicon-spin animation targets the entire `.codicon-loading` element rather than just the `::before` pseudo-element, causing the badge pill background to rotate along with the glyph when tests are running.

## Change summary

When tests are running, the activity bar badge pill background rotates along with the spinner glyph instead of only the glyph spinning.

## Best solution

Fix the animation target in `src/vs/base/browser/ui/codicons/codicon/codicon-modifiers.css` so that `.codicon-loading` applies the spin animation only to its `::before` pseudo-element (which contains the icon glyph), not the element itself. This prevents the badge background from rotating while preserving the spinning icon effect.
