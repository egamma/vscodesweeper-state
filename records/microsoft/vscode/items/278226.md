However, reproduction details are thin (no specific file/line example), and the issue already carries area labels — keeping open.

## Change summary

Horizontal scrollbar does not extend to cover long lines in the deleted-code (red) view zones during Copilot agent inline edit review.

## Best solution

The inline diff view zones that render deleted lines (likely in `src/vs/editor/browser/widget/diffEditor/components/diffEditorViewZones/`) need to contribute their content width to the editor's scroll width calculation so the horizontal scrollbar appears when deleted lines overflow. Alternatively, the chat editing overlay should sync scroll or ensure the deleted-line container is scrollable.
