# Issue #321148 — SCM sidebar Commit & Push keyboard shortcut

[https://github.com/microsoft/vscode/issues/321148](https://github.com/microsoft/vscode/issues/321148)

## Summary

Valid feature request: the Git extension's 'Commit & Push' is only available as an action button dropdown item, not as a standalone command bindable to a keyboard shortcut. Routing to the git area for consideration.

## Change summary

Expose a standalone 'Git: Commit & Push' command (or allow the SCM accept-input keybinding to target a specific post-commit action) so users can bind a keyboard shortcut to commit-and-push without changing their default commit behavior.

## Best solution

The Git extension should expose a dedicated `git.commitAndPush` command (analogous to how `git.push` and `git.commit` already exist) that commits staged changes and then pushes, independently of the `git.postCommitCommand` setting. This would make it bindable in keybindings.json with an optional when-clause for SCM input focus. The likely implementation lives in `extensions/git/src/commands.ts` and `extensions/git/package.json`.
