# Issue #192968 — Debug hover sometimes positioned incorrectly

[https://github.com/microsoft/vscode/issues/192968](https://github.com/microsoft/vscode/issues/192968)

## Summary

The reported behavior — debug hover overlaying its anchor when the tree is large — was addressed by a fix that constrains the tree's max height to avoid overlapping the hovered line. Shipped in 1.86.0.

## Change summary

Debug hover with many properties overlays the hovered symbol when the breakpoint line is near the top of the viewport.

## Best solution

Keep the shipped fix (maxHeightToAvoidCursorOverlay constraint in layoutTree); no further change needed. If the transient initial-positioning flicker persists on current builds, a narrower follow-up issue should be filed.
