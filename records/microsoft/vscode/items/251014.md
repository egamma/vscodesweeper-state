# Issue #251014 — code --prof-startup fails to profile renderer and exthost

## Summary

Fixed on main and shipped: the root cause was that --inspect-brk, --remote-debugging-port, and --inspect-brk-extensions were passed host:port instead of just port, which Electron/Node doesn't accept, causing the 'failed to connect' error.
