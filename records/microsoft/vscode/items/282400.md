## Summary

## Change summary

After completing a multi-line ternary expression with `;`, pressing Enter does not dedent back to the base indentation level.

## Best solution

Properly solving this requires tree-sitter-based auto-indent (as the existing label indicates). A regex-based `onEnterRule` for this pattern would be fragile and overly broad. Keep open until tree-sitter autoindent support for C++ lands.
