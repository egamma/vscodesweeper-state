## Summary

Valid feature request for rounded-corner carets, still under discussion with team member engagement. No implementation exists on main; the cursor rendering in viewCursors.css has no border-radius option. Keeping open as under-discussion given team interest.

## Change summary

Add an option to render the editor cursor/caret with rounded corners.

## Best solution

The implementation would be small — a new editor option threaded into the cursor render path.
