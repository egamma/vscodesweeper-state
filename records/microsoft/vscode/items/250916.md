# Issue #250916 — Terminal is extremely sluggish when using fish shell with `fish-async-prompt` extension

[https://github.com/microsoft/vscode/issues/250916](https://github.com/microsoft/vscode/issues/250916)

## Summary

The root cause is the third-party fish-async-prompt plugin flooding the terminal with SIGUSR1-driven prompt repaints, which triggers repeated CSI focus-in sequences (`^[[I`). The reporter themselves identified fish-async-prompt as the culprit and filed a workaround PR in that repo. The issue has been confirmation-pending for over a year with no additional reports or upvotes — requesting confirmation on whether the fish-async-prompt workaround resolved it, or whether a VS Code-side issue remains.
