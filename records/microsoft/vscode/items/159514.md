# Issue #159514 — [unicode highlight] Should highlight em dash `—` as it is easily confusable with minus `-`.

[https://github.com/microsoft/vscode/issues/159514](https://github.com/microsoft/vscode/issues/159514)

## Summary

Valid feature request: the em dash (U+2014) is not currently in the ambiguous-characters confusable data, so the unicode highlighter does not flag it. This remains an open gap.

## Change summary

Add em dash (U+2014) to the ambiguous characters confusable set so it is highlighted when used where a hyphen-minus is expected.

## Best solution

Add 8212,45 (em dash → hyphen-minus) to the `_common` confusables list in `src/vs/base/common/strings.ts`, alongside the existing en-dash/figure-dash/hyphen entries. This is a one-line data addition. Alternatively, consider adding a user-facing blocklist setting so users can flag additional characters without modifying the built-in data.
