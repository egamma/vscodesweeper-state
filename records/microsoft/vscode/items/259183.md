# Issue #259183 — Enable workspace-scoped tool sets

[https://github.com/microsoft/vscode/issues/259183](https://github.com/microsoft/vscode/issues/259183)

## Summary

Team-authored feature request for workspace-scoped tool sets. A team member (aeschli) commented that user-defined tool sets can no longer be referenced in prompt/agent files since they aren't shared across repos, which reinforces the need. Must keep open per team-author policy.

## Change summary

Tool sets are profile-scoped but referenced in workspace-scoped prompt files and chat modes, causing them to be missing for other workspace users.

## Best solution

Support defining tool sets at the workspace level (e.g. in `.vscode/` or a workspace config file), analogous to how instructions, prompt files, and chat modes already support workspace-level definitions.
