# Issue #302688 — Working View not working anymore for SVG

[https://github.com/microsoft/vscode/issues/302688](https://github.com/microsoft/vscode/issues/302688)

## Summary

Fixed on main and shipped in 1.120.0. Commit b6a00bc1d17 corrected the webview `localResourceRoots` for `git:` scheme files by stripping query/fragment, which was preventing the image preview (including SVGs) from loading the git-HEAD version of images in the diff view.

## Change summary

SVG images cannot be compared in the Source Control working view — the left (HEAD) pane shows 'An error occurred while loading the image' instead of the previous version.

## Best solution

Keep the shipped fix in commit b6a00bc1d17 (1.120.0). No further change is needed — the reporter should update to VS Code 1.120.0 or later.
