# Issue #268330 — Add ability to create Chatmode/Prompt from current chat

[https://github.com/microsoft/vscode/issues/268330](https://github.com/microsoft/vscode/issues/268330)

## Summary

Valid feature request already triaged into Backlog with appropriate labels. It asks for a workflow shortcut to save the current chat configuration as a reusable Chat Mode or Prompt file — a reasonable quality-of-life enhancement for the chat/prompts area.

## Change summary

Allow users to create a Chat Mode or Prompt file pre-populated from their current chat session's tools, instructions, and configuration.

## Best solution

Add a command (e.g. "Save as Chat Mode" / "Save as Prompt") accessible from the chat widget that serializes the active session's tool selections, instructions, and description into a new `.chatmode.md` or `.prompt.md` file, pre-filling the YAML front-matter. Likely lives in `src/vs/workbench/contrib/chat/browser/` near the existing prompt/chatmode creation flows.
