# Issue #292389 — Allow users to close out of walkthrough widget on welcome page

[https://github.com/microsoft/vscode/issues/292389](https://github.com/microsoft/vscode/issues/292389)

## Summary

Team-authored polish request to add a dismiss/close affordance to the walkthrough card on the Agents Sessions welcome page. The walkthrough card currently has no dismiss button (unlike the TOS/privacy notice card which already has one). Keeping open per team-member policy.

## Change summary

Allow users to dismiss the walkthrough widget on the welcome page to reduce clutter, rather than waiting for sessions to load and replace it.

## Best solution

Add a dismiss button (and persist the dismissal in storage) to the walkthrough card in `AgentSessionsWelcomePage.buildWalkthroughs()` in `src/vs/workbench/contrib/welcomeAgentSessions/browser/agentSessionsWelcome.ts`, mirroring the pattern already used for the TOS/privacy notice card at line ~700–757 of the same file.
