## Change summary

Add visible file name headers above each pane in the compare/diff editor so users can quickly tell which file is left vs. right.

## Best solution

The diff editor widget could render an optional inline header bar above each side showing the file name or relative path. This would live in the diff editor contribution area (`src/vs/editor/browser/widget/diffEditor/`). Alternatively, improved breadcrumb disambiguation for similar-named files could address the underlying need with less UI cost.
