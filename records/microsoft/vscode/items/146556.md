# Issue #146556 — SCM - Ability to change background of rows that are new or changed

[https://github.com/microsoft/vscode/issues/146556](https://github.com/microsoft/vscode/issues/146556)

## Summary

Valid feature request with 24 upvotes, promoted to Backlog. The SCM view currently only applies foreground color decorations to changed/added files — no background highlighting exists on current main. The capability gap remains.

## Change summary

Allow background color highlighting of SCM rows for new/modified files, complementing the existing gutter indicators and foreground text decorations.

## Best solution

Add customizable background color tokens (e.g. via `registerColor`) for SCM resource rows based on their git status (modified, added, untracked, etc.), likely in the SCM tree renderer and the git decoration provider. This would pair well with the existing `gitDecoration.*Foreground` color tokens by adding corresponding background tokens.
