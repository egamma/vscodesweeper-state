# Issue #318700 — Focus is lost after interacting with "Auto Update" checkbox using keyboard:A11y_VisualStudioCode_Extensions_Keyboard

[https://github.com/microsoft/vscode/issues/318700](https://github.com/microsoft/vscode/issues/318700)

## Summary

Still reproduces on main: toggling the Auto Update checkbox triggers an onDidChange event that resets focusability on the action bar (setFocusable(false)/setFocusable(true) cycle), causing focus to be lost from the checkbox. Team-authored, keeping open.

## Change summary

Focus is lost from the Auto Update checkbox in the extension editor after toggling it via keyboard.

## Best solution

In `src/vs/workbench/contrib/extensions/browser/extensionEditor.ts` lines 390-393, the setFocusable reset should preserve focus on the currently active element — either skip the reset when the focused element is still valid, or restore focus after the setFocusable cycle.
