# Issue #119919 — SCM - API: Finalize `SourceControl.selected`

[https://github.com/microsoft/vscode/issues/119919](https://github.com/microsoft/vscode/issues/119919)

## Summary

The `SourceControl.selected` API remains a proposed (unfinalised) API on current main — still present in `vscode.proposed.scmSelectedProvider.d.ts` and consumed only by the built-in git extension. The second API mentioned (input validation) is tracked separately in #131007. The original request to finalize these proposals is still valid, though the 2022 discussion suggests the team considered deprecating `selected`. Keeping open for the API owner to decide.

## Change summary

Finalize or deprecate the long-standing proposed `SourceControl.selected` and `onDidChangeSelection` SCM extension API.

## Best solution

The SCM/API area owner should decide whether to finalize `scmSelectedProvider` into the stable `vscode.d.ts` or formally deprecate it. The input-validation half is already tracked by https://github.com/microsoft/vscode/issues/131007 and can be removed from this issue's scope. If the team still wants to deprecate `selected` (as discussed in the Dec 2022 comment), the proposal file should be removed and the git extension updated.
