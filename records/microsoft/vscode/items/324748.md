# Issue #324748 — `ELECTRON_GET_USE_PROXY` excluded from integrated terminal environments

[https://github.com/microsoft/vscode/issues/324748](https://github.com/microsoft/vscode/issues/324748)

## Summary

Valid bug: `sanitizeProcessEnvironment` in `src/vs/base/common/processes.ts` strips all `ELECTRON_*` environment variables from the integrated terminal, including `ELECTRON_GET_USE_PROXY` which is unrelated to the running Electron instance and is used by `electron/get` for proxy configuration. An open PR (#324749) from the reporter already addresses it.

## Change summary

The `ELECTRON_GET_USE_PROXY` environment variable (used by `electron/get` for proxy routing) is incorrectly stripped from the integrated terminal environment by the blanket `ELECTRON_*` exclusion.

## Best solution

Review and land https://github.com/microsoft/vscode/pull/324749, which adds a negative lookahead for `ELECTRON_GET_USE_PROXY` to the regex in `sanitizeProcessEnvironment` (`src/vs/base/common/processes.ts`). The change is small, localized, and includes a test update.
