## Change summary

After double-clicking a word, Shift+Alt+Left should extend selection leftward like native macOS text selection instead of collapsing the selected word.

## Best solution

The editor-wordnav owner should decide whether macOS word selection should preserve a right-side anchor after double-click word selection, or whether this needs a separate setting/command. If accepted, cover the interaction around `WordSelect` plus `cursorWordLeftSelect` with a focused regression test in the editor word-operation/cursor tests.
