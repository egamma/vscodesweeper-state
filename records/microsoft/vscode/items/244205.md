# Issue #244205 — Add workspace and user settings option to disable "Other Recommendations" from being recommended

[https://github.com/microsoft/vscode/issues/244205](https://github.com/microsoft/vscode/issues/244205)

## Summary

Valid, well-supported feature request (45 upvotes, promoted to Backlog) with active maintainer engagement. Not yet implemented on main — the only existing setting (`extensions.ignoreRecommendations`) suppresses all recommendations including workspace ones, which doesn't meet the need.

## Change summary

Add a setting to disable non-workspace ('Other') extension recommendations while preserving workspace recommendations.

## Best solution

Add an `extensions.ignoreOtherRecommendations` setting (or similar) that suppresses non-workspace recommendations in both the Extensions sidebar and the notification dialog, without affecting workspace recommendations. The notification service in `src/vs/workbench/contrib/extensions/browser/extensionRecommendationNotificationService.ts` and the extensions contribution registration in `extensions.contribution.ts` would need changes. The author has offered to contribute.
