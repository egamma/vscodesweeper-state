[https://github.com/microsoft/vscode/issues/313051](https://github.com/microsoft/vscode/issues/313051)

## Summary

Team-authored feature request for a documented extension API to programmatically commit & push through the GitHub Repositories SCM provider in vscode.dev. A team member (osortega) confirmed no such API exists today and proposed exposing one. Keeping open — team member issue requiring maintainer judgment.

## Change summary

Extensions running in vscode.dev have no supported API to programmatically commit and push through the GitHub Repositories provider while keeping the SCM panel in sync.

## Best solution

Expose a documented API (or command) on the GitHub Repositories SCM provider that accepts a commit message, performs commit & push, and returns a completion signal — mirroring what the existing UI button does. Likely lives in the `remotehub` / `github-repositories` extension code and would need coordination with the SCM API surface. @lszomoru was tagged for input.
