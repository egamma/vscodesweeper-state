# Issue #232604 — After update, app is completely unusable

[https://github.com/microsoft/vscode/issues/232604](https://github.com/microsoft/vscode/issues/232604)

## Summary

Resolved in 1.96.0 per the reporter's own confirmation (comment 20). The root cause was a crash in the NVIDIA Vulkan driver (`libnvidia-vksc-core.so`) triggered by Chromium's GPU process; the Electron 33 upgrade shipped with 1.96 avoided the driver crash path.

## Change summary

VS Code 1.95 on Ubuntu with certain NVIDIA drivers rendered a gray, unresponsive window due to a GPU process crash.

## Best solution

Keep the shipped Electron 33 upgrade (1.96.0+); no further VS Code change is needed. Users on older versions can work around it with `--disable-gpu`. The underlying NVIDIA driver bug is outside VS Code's control.
