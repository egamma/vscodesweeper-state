# Issue #163396 — Suggestion preview (ghost text) disappears with incomplete trigger

[https://github.com/microsoft/vscode/issues/163396](https://github.com/microsoft/vscode/issues/163396)

## Summary

Still present on main: when an incomplete completion provider is re-triggered, the suggest widget clears its focused item during `showSuggestions()`, causing the ghost text preview to disappear until new results arrive and focus is restored.

## Change summary

Suggest preview ghost text disappears (flickers) when typing triggers a re-request from an incomplete completion provider.

## Best solution

Preserve the previous ghost text preview during the async gap of an incomplete re-trigger — either by retaining the old focused item in `showSuggestions()` when the widget is already open and re-triggering for incomplete, or by having the `SuggestWidgetAdaptor` hold onto the last valid `SuggestItemInfo` until a new focus event confirms the item is no longer valid.
