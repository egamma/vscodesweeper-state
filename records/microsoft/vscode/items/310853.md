Team-authored feature request collecting internal feedback on Agents window changes rendering — must stay open for maintainer judgment. The request covers flexible diff viewing (side-by-side vs. modal), responsive layout, and artifact presentation defaults.

## Change summary

Allow users to choose between side-by-side and modal diff rendering in the Agents window Changes panel, with responsive defaults based on available horizontal space.

## Best solution

Implement a user-facing toggle (in the Changes panel `...` menu) to switch between side-by-side and modal diff rendering, add responsive behavior that defaults to modal below a horizontal-space threshold, and ensure clicking a specific file focuses that file's diff.
