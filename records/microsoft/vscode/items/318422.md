## Summary

Team-authored issue reporting duplicate agent entries in the Copilot Chat picker when the same agent is contributed by both a VS Code extension and a Copilot CLI plugin. Keeping open for maintainer judgment.

## Change summary

Copilot Chat agent picker shows duplicate entries when the same agent is contributed by both a VS Code extension and a Copilot CLI plugin, with no UI affordance to distinguish or deduplicate them.

## Best solution
