# Issue #206101 — A way to open a file in hex editor from Terminal

[https://github.com/microsoft/vscode/issues/206101](https://github.com/microsoft/vscode/issues/206101)

## Summary

Valid feature request for a CLI `--use-editor` flag to open files with a specific custom editor. The related PR #206962 was closed unmerged, and the feature remains unimplemented on current main. Low community traction (1 upvote) so it stays open for the community process.

## Change summary

Allow specifying a custom editor (e.g. hex editor) when opening a file from the CLI via a flag like `--use-editor`.

## Best solution

Implement a `--use-editor=<editorId>` CLI argument that passes the editor ID through to the window service so the file opens with the requested custom editor. The work would touch the CLI argument parsing (src/vs/platform/environment/) and the file-opening path in the workbench. The closed PR #206962 could serve as a starting reference.
