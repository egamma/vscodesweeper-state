## Summary

The setting `chat.tools.terminal.terminalProfile.windows` is contributed by the Copilot extension, not VS Code core. The reported behavior — the first terminal command ignoring the configured profile — is owned by the extension, not the editor.

## Change summary

Chat's terminal tool ignores the configured `chat.tools.terminal.terminalProfile.windows` setting on the first command after restart, defaulting to PowerShell.

## Best solution
