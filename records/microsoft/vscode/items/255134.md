# Issue #255134 — Git - Add Git Tools for VS Code Copilot (git_status, git_diff, git_branch, git_log)

[https://github.com/microsoft/vscode/issues/255134](https://github.com/microsoft/vscode/issues/255134)

## Summary

Valid feature request to add dedicated git tools (git_status, git_diff, git_branch, git_log) for Copilot Chat so agents can query git state without terminal round-trips and user confirmation clicks. No such tools exist on current main, and no canonical issue was found that owns this work. Keeping open for the community process — only 2 upvotes so far, well below the 20-upvote Backlog promotion threshold.

## Change summary

Add built-in git tools to VS Code Copilot so agents can query repository status, diffs, branches, and log without requiring terminal interaction and user confirmation.

## Best solution

The git extension (`extensions/git/`) could register language-model tools via `vscode.lm.registerTool` to expose structured git status, diff, branch, and log data to chat participants — this is the natural place since VS Code already has a full Git integration layer. Alternatively, the existing terminal tool auto-approval for read-only git commands partially addresses the friction today. The request has low traction (2 upvotes); let the community process decide whether to promote it.
