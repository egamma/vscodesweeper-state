## Summary

Team-authored feature request for reducing git ceremony in the Agents window — auto-commit on PR creation and smart commit stacking. Keeping open for maintainer judgment.

## Change summary

Creating a PR from the Agent app should not require a manual commit step; Copilot should auto-commit and intelligently stack commits for longer explorations.

## Best solution

Implement a combined 'commit + create PR' gesture as the default in the Agents window, leveraging the existing auto-commit setting (currently off by default) and surfacing a 'Create PR' button when changes are ready. For longer sessions, add logic to partition work into logical stacked commits automatically.
