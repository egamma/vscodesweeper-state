## Summary

Valid, well-supported feature request (30 upvotes, Backlog milestone, help wanted) for interactive rebase UI in VS Code's built-in Git. The current git extension only has basic rebase-onto-branch and abort — no interactive rebase editor exists on main. A newer related issue #260637 specifically asks for the editor component but this broader issue is the canonical home for the full interactive rebase workflow. Keeping open.
