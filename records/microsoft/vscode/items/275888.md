# Issue #275888 — Let me hide Ask & Edit modes

[https://github.com/microsoft/vscode/issues/275888](https://github.com/microsoft/vscode/issues/275888)

## Summary

Team-authored feature request still open — Ask and Edit modes are unconditionally included in the builtin mode list on current main with no user setting to hide them. Two related issues were closed as duplicates of this one, making it the canonical tracker.

## Change summary

Allow users to hide the built-in Ask and Edit chat modes from the mode picker.

## Best solution

Implement a user-facing setting or visibility toggle (similar to `ICustomAgentVisibility`) for the built-in Ask and Edit modes so users who exclusively use Agent mode can declutter their mode picker. The work is in `src/vs/workbench/contrib/chat/common/chatModes.ts` (the `getBuiltinModes()` method) and whatever UI drives mode visibility.
