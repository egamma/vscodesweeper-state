## Summary

Valid bug — the diff overview ruler delegates pointer-down to the scrollbar (which applies scrollByPage), rather than jumping to the clicked position. Still unfixed on main; an open PR (#308020) exists but hasn't merged.

## Change summary

Clicking the diff overview ruler honors scrollByPage and scrolls one page instead of jumping to the clicked diff location.

## Best solution
