# Issue #158473 — Git - Use incrementing branch names / branch names based on date

[https://github.com/microsoft/vscode/issues/158473](https://github.com/microsoft/vscode/issues/158473)

## Summary

Team-authored feature request for richer branch-name generation strategies (incrementing numbers, date-based, extension-contributed commands). Current implementation only supports random dictionary-based names. Must keep open per team-author rule.

## Change summary

Allow branch name generation to use incrementing numbers, date-based patterns, or extension-contributed template commands instead of only random dictionary words.

## Best solution

Extend `generateRandomBranchName` in `extensions/git/src/repository.ts` and the `git.branchRandomName.dictionary` setting to support additional strategies — e.g. an `incrementing` or `date` dictionary entry — and consider exposing a Git extension API point so other extensions can contribute custom naming commands to the branch-name template.
