# Issue #62333 — [css] Add @media and @supports to CSS breadcrumbs

[https://github.com/microsoft/vscode/issues/62333](https://github.com/microsoft/vscode/issues/62333)

## Summary

Partially implemented: @media already appears in CSS breadcrumbs via the language service's document symbols. @supports is still missing — the parser has the node type but `collectDocumentSymbols` doesn't handle it. The fix belongs upstream in vscode-css-languageservice.

## Change summary

Request to show @media and @supports queries in CSS breadcrumbs for easier orientation in long stylesheets.

## Best solution

Add a `nodes.Supports` case to `collectDocumentSymbols` in vscode-css-languageservice (similar to the existing `nodes.Media` handler), then bump the dependency in this repo. The @media part is already done.
