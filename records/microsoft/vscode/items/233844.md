## Summary

Valid accessibility feature request with team endorsement (meganrogge agreed an icon makes sense) and assigned to benibenj. The current implementation in markersFileDecorations.ts only uses color to indicate problems on tabs — no icon-based alternative exists on main. Keeping open as it's already triaged and assigned.

## Change summary

Add a non-color-based visual indicator (e.g. warning icon) to editor tabs with problems, so colorblind users can distinguish them from modified tabs.

## Best solution
