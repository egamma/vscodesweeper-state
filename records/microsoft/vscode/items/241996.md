## Best solution

Implement quoted-path awareness in the terminal suggest completions provider (under `src/vs/workbench/contrib/terminalContrib/suggest/`) so that it detects when the cursor is inside quotes, strips them for the file-system lookup, and wraps accepted completions appropriately.
