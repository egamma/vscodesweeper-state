## Summary

Team-authored feature request (MEMBER) with active community interest and recent discussion — keep open for maintainer judgment.

## Change summary

Add a discoverable Find button to tree view titles so users don't have to know the Ctrl/Cmd+F shortcut after focusing.

## Best solution

Expose a `showFindControl` option in `TreeViewOptions` (like `showCollapseAll`) that adds a magnifying-glass button to view/title bars, triggering `list.find` on the correct tree. Address the API gaps RedCMD identified: context for find-widget availability, view-origin context for button presses, and `view == focusedView` when-clause support.
