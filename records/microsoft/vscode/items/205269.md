# Issue #205269 — Incorrect project tab names

[https://github.com/microsoft/vscode/issues/205269](https://github.com/microsoft/vscode/issues/205269)

## Summary

Correctly triaged as upstream. macOS native tab naming is entirely delegated to Electron's BrowserWindow.addTabbedWindow API; VS Code sets no custom tab titles in this path. The bug was re-confirmed in Feb 2025 and remains open upstream. Keeping open as-is.

## Change summary

macOS native tabs show incorrect names and ordering when a new tab is created via Window > New Tab.

## Best solution

Track and link the upstream Electron issue for native macOS tab naming/ordering. VS Code's code at `src/vs/platform/windows/electron-main/windowImpl.ts:1163` and `windowsMainService.ts:1619` is a thin passthrough to Electron — the fix must come from Electron or Chromium's macOS tab integration.
