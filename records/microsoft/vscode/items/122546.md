# Issue #122546 — Sign in quick pick dialog: focus movement could be improved

[https://github.com/microsoft/vscode/issues/122546](https://github.com/microsoft/vscode/issues/122546)

## Summary

Fixed on main: checkboxes were removed from the tab order in the quick pick list (commit e731b3430fd), so Tab no longer stops at each checkbox before reaching action buttons. Additionally, the old toggle system was replaced entirely with button-based toggles. Shipped in 1.107.0.

## Change summary

In the Settings Sync sign-in quick pick, pressing Tab moved focus to each checkbox individually instead of skipping to the action button.

## Best solution

Keep the shipped fix — checkboxes are now `tabIndex=-1` and navigable only via arrow keys; no further change needed.
