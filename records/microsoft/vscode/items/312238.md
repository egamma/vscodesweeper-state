## Summary

This request for full semver pre-release tags (e.g. 1.0.0-alpha.1) in extension versions is an upstream Marketplace concern already tracked at microsoft/vsmarketplace#264. VS Code core does not own extension version format constraints — that lives in the Marketplace and vsce tooling.
