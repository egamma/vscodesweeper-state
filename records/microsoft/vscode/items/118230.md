# Issue #118230 — Add "Copy Value" to the context menu in the Debug Console.

[https://github.com/microsoft/vscode/issues/118230](https://github.com/microsoft/vscode/issues/118230)

## Summary

Valid feature request on the 'On Deck' milestone — the gap remains on current main. The Copy Value action is registered for the Watch context menu but not for the Debug Console context menu; logged objects only get the truncated 'Copy' action today.

## Change summary

Add a 'Copy Value' context menu entry in the Debug Console so users can copy the full underlying value of logged objects, not just the truncated description.

## Best solution

Register the existing `COPY_VALUE_ID` action (or a console-specific variant) on `MenuId.DebugConsoleContext` in `debug.contribution.ts` / `repl.ts`, gated on appropriate context keys. The `tryEvaluateAndCopy` pattern in `debug.replCopy` already shows how to call `session.evaluate` with `clipboard` context — the gap is that it only works for `ReplEvaluationResult` (typed expressions), not for logged objects (see the `#187784` TODO in `repl.ts:1205`).
