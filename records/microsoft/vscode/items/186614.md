## Summary

Blocked on an upstream Chromium issue (https://issues.chromium.org/issues/375314974) that causes a perf regression when COI is enabled. The feature has been attempted and reverted twice. Milestone is 'On Deck' (hard keep-open). Must remain open until the upstream fix lands.

## Change summary

Enable Cross-Origin Isolation on desktop by default using COEP credentialless, without the startup perf regression caused by forced BrowsingInstance swaps in Chromium.

## Best solution

Wait for the upstream Chromium fix (https://issues.chromium.org/issues/375314974) to avoid the forced BrowsingInstance swap on COI-enabled origins navigating from about:blank. Once that ships in Electron, re-land the change from the reverted PR https://github.com/microsoft/vscode/pull/231665 and verify no perf regression.
