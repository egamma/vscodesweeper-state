# Issue #319274 — Multi suggestion failed. Atl + ] didn't work

[https://github.com/microsoft/vscode/issues/319274](https://github.com/microsoft/vscode/issues/319274)

## Summary

The reporter is following a GitHub Copilot tutorial but does not have the GitHub Copilot extension installed — their extensions list lacks `github.copilot`. The Alt+] keybinding exists in VS Code core and works correctly when an inline suggestion is visible, but without a Copilot (or other inline completion provider) extension, no suggestions appear to cycle through.

## Change summary

Alt+] does not cycle through multiple inline suggestions when following a GitHub Copilot tutorial.

## Best solution

The reporter needs to install the GitHub Copilot extension (`github.copilot`) from the Marketplace. Once installed and authenticated, inline suggestions will appear and Alt+] / Alt+[ will cycle through them. This is a usage question, not a VS Code core bug.
