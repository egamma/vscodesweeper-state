# Issue #72684 — [folding] Option to show both fold/unfold controls on mouseover only

[https://github.com/microsoft/vscode/issues/72684](https://github.com/microsoft/vscode/issues/72684)

## Summary

Keep open as a low-traction feature request already routed to editor folding; current `main` still only supports `always`, `never`, and `mouseover`, and `mouseover` keeps collapsed controls visible outside hover.

## Change summary

Add an `editor.showFoldingControls` mode that hides both expanded and collapsed folding controls until mouseover.

## Best solution

If accepted, add a fourth folding-control visibility mode in the editor options schema and folding decoration/CSS handling so collapsed and expanded controls can both be hover-only. This belongs in the editor folding area and needs a small product decision on the exact setting name and behavior.
