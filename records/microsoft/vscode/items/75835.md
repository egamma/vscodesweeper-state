# Issue #75835 — Allow QuickPicks to show right-aligned text like "recently opened" in the command palette

[https://github.com/microsoft/vscode/issues/75835](https://github.com/microsoft/vscode/issues/75835)

## Summary

Valid, unimplemented feature request — this is the canonical issue for right-aligned text on QuickPickItems (issue #199609 was duped here). The current API offers `description` (inline, not right-aligned) and `detail` (separate line), but no right-aligned slot. Keeping open for the community process.

## Change summary

Allow extensions to render right-aligned text on individual QuickPickItems, similar to the 'recently used' annotation in the command palette.

## Best solution

Add a right-aligned text property (e.g. `annotation` or `rightLabel`) to `QuickPickItem` in the extension API, with corresponding rendering in the quick-input widget. This would need a core change first (as TylerLeonhardt noted), then API exposure. The work touches `src/vs/platform/quickinput/` and `src/vscode-dts/vscode.d.ts`.
