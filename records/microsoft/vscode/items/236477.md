# Issue #236477 — Debug: Search inside collapsed nodes

[https://github.com/microsoft/vscode/issues/236477](https://github.com/microsoft/vscode/issues/236477)

## Summary

Valid, community-promoted feature request (26 upvotes, in Backlog) for deep search inside collapsed/unexpanded nodes in the debug variables view. The existing tree filter (added in 1.97.0 via commit d953d84d901) only matches already-visible items; searching recursively through unexpanded children is a distinct, unimplemented capability.

## Change summary

Allow searching/filtering inside collapsed (unexpanded) nodes in the debug Variables view to find deeply nested keys and values without manually expanding the tree.

## Best solution

Implement a recursive search mode in the debug Variables view that fetches and filters child variables of collapsed nodes, surfacing matches from deep in the tree. This would likely involve changes to `src/vs/workbench/contrib/debug/browser/variablesView.ts` and potentially the `VariablesDataSource` to support pre-fetching children for search. The existing `expressionAndScopeLabelProvider` filter covers only visible items; a new mechanism would need to walk the DAP variable hierarchy.
