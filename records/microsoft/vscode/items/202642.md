# Issue #202642 — When using monospaced Chinese fonts, the auto line-wrapping in simple mode is incorrect.

[https://github.com/microsoft/vscode/issues/202642](https://github.com/microsoft/vscode/issues/202642)

## Summary

This is the same as-designed behavior closed in #224493: the simple wrapping strategy uses a fast heuristic that can be inaccurate with CJK characters. The fix is to use `editor.wrappingStrategy: "advanced"` for pixel-accurate wrapping.

## Change summary

Monospaced Chinese font line-wrapping is incorrect in simple wrapping mode but works in advanced mode.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/224493 — this was closed as by-design. The simple algorithm intentionally trades accuracy for speed; users needing pixel-accurate CJK wrapping should use `editor.wrappingStrategy: "advanced"`.
