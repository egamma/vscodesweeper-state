# Issue #186561 — Copying text from the debug console doesn't work quite right

[https://github.com/microsoft/vscode/issues/186561](https://github.com/microsoft/vscode/issues/186561)

## Summary

The debug console now supports proper user text selection via commit dd86be1a954, which added a `userSelection` mode to the list view and adopted it in the REPL tree. This shipped in 1.97.0 and addresses the core complaint of not being able to select/copy text properly.

## Change summary

Copying text from the debug console fails to include the first character of a line because the tree list didn't support native text selection.

## Best solution

Keep the shipped fix (user selection support in the debug console list view, commit dd86be1a954, 1.97.0). No further change needed — if edge cases remain, they should be filed as new issues with current reproduction steps.
