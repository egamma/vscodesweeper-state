# Issue #209328 — Diff Editor - moved code block indicator interferes with menu rendering

[https://github.com/microsoft/vscode/issues/209328](https://github.com/microsoft/vscode/issues/209328)

## Summary

Team-authored issue; the moved-blocks-lines SVG overlay in the diff editor lacks proper z-index management, causing it to interfere with context menus and action buttons. No fix has landed on main.

## Change summary

The diff editor's moved code block indicator (arrow/lines SVG) overlaps and obscures context menus and inline action buttons (revert, stage).

## Best solution

Set a lower z-index on `.moved-blocks-lines` in `src/vs/editor/browser/widget/diffEditor/style.css` (or add `pointer-events: none` more broadly) so it renders beneath menus and gutter action widgets, which already use z-index 10001.
