# Issue #185999 — [css] add clamp() math function

[https://github.com/microsoft/vscode/issues/185999](https://github.com/microsoft/vscode/issues/185999)

## Summary

Valid feature request — the `clamp()` CSS math function is not offered in intellisense completions. The fix lives upstream in `vscode-css-languageservice` which provides the completion data. Already triaged to Backlog with correct area label.

## Change summary

CSS intellisense should offer the `clamp()` math function in completions, as it is widely supported by modern browsers.

## Best solution

Add `clamp()` (and potentially other modern CSS math functions like `min()`, `max()`) to the completion data in the upstream `microsoft/vscode-css-languageservice` package, then bump the dependency in VS Code. The syntax grammar already recognizes `clamp` for highlighting; only the language service completions are missing.
