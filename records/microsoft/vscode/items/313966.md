[https://github.com/microsoft/vscode/issues/313966](https://github.com/microsoft/vscode/issues/313966)

## Summary

The report describes indentation settings being overridden despite `detectIndentation: false`, but the user has 31 extensions installed (including cpptools) and hasn't tested with extensions disabled. A nearly identical issue (#246690) was closed as `*caused-by-extension`. The VS Code source correctly gates indentation detection on the setting — this is very likely an extension overriding the model options via the API.

## Change summary
