# Issue #120460 — [Gutter] Git Decorations bounding box and appearing

[https://github.com/microsoft/vscode/issues/120460](https://github.com/microsoft/vscode/issues/120460)

## Summary

Still present on main: the dirty-diff gutter decoration's click area overlaps the folding icon zone, causing accidental git menu triggers when clicking to fold. The code has an acknowledged TODO with a 3px tolerance check that doesn't fully prevent collisions.

## Change summary

Clicking the folding icon in the gutter accidentally triggers the git dirty-diff menu because their hit areas overlap.

## Best solution

Tighten the hit-test in `quickDiffWidget.ts` (line ~708) or adjust the layout so the dirty-diff decoration's clickable area does not extend into the folding zone. The hover visibility on mouseover of the fold area (CSS in the same file, lines 527–543) also needs scoping so the decoration only expands when the pointer is over the decoration column itself.
