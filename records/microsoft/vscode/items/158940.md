# Issue #158940 — Live preview extension contributed task does not include the "Task" tab description

[https://github.com/microsoft/vscode/issues/158940](https://github.com/microsoft/vscode/issues/158940)

## Summary

Still reproduces on main: the CustomExecution branch of _createTerminal omits the `type` property from the shell launch config, so extension-contributed tasks don't get the 'Task' tab description. The issue is from a team contributor, so keeping it open.

## Change summary

Extension-contributed tasks using CustomExecution don't show 'Task' as the terminal tab description, unlike shell/process tasks.

## Best solution

Add `type: TaskTerminalType` (i.e. `'Task'`) to the shell launch config object in the `CustomExecution` branch at line ~1534 of `src/vs/workbench/contrib/tasks/browser/terminalTaskSystem.ts`, matching the process-task branch.
