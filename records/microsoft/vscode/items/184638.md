# Issue #184638 — Git - Diff decorations not working

[https://github.com/microsoft/vscode/issues/184638](https://github.com/microsoft/vscode/issues/184638)

## Summary

Still active on current main: symlinked files inside a git repository do not get diff decorations because `provideOriginalResource()` explicitly returns `undefined` for symbolic links. The 'On Deck' milestone indicates the team is tracking this for resolution.

## Change summary

Git diff gutter decorations do not appear for files accessed through symbolic links inside a repository.

## Best solution

The quick diff provider in `extensions/git/src/quickDiffProvider.ts` needs to resolve the symlink target and provide the original resource for the real path rather than skipping symlinks entirely. The 'On Deck' milestone and lszomoru's ongoing symlink work suggest this is in progress.
