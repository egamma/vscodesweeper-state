# Issue #231474 — Zen mode does not exit when leaving fullscreen after restart

[https://github.com/microsoft/vscode/issues/231474](https://github.com/microsoft/vscode/issues/231474)

## Summary

Still reproduces on main: the `!restoring` guard at line 1475 of layout.ts prevents `transitionedToFullScreen` from being set when zen mode is restored after restart, so leaving fullscreen doesn't exit zen mode.

## Change summary

Zen mode does not exit when leaving fullscreen if VS Code was restarted while in zen mode + fullscreen.

## Best solution

When restoring zen mode (`restoring = true`), detect that the window is already fullscreen and set `zenModeExitInfo.transitionedToFullScreen = true` so the fullscreen-exit handler at line 559 correctly toggles zen mode off. The fix is localized to `toggleZenMode()` in `src/vs/workbench/browser/layout.ts`.
