# Issue #231474 — Zen mode does not exit when leaving fullscreen after restart

[https://github.com/microsoft/vscode/issues/231474](https://github.com/microsoft/vscode/issues/231474)

## Summary

Still reproduces on main: when zen mode is restored after restart with `restoring = true`, `zenModeExitInfo.transitionedToFullScreen` is never set (guarded by `if (!restoring)` at line 1475), so the fullscreen-exit check at line 559 never triggers zen mode toggle.

## Change summary

Zen mode does not exit when leaving fullscreen after a restart because the `transitionedToFullScreen` exit info is not persisted/restored across sessions.

## Best solution

During zen mode restoration (the `restoring = true` path at line 1191 of `src/vs/workbench/browser/layout.ts`), detect that the window is currently fullscreen and set `zenModeExitInfo.transitionedToFullScreen = true` so that leaving fullscreen later correctly exits zen mode.
