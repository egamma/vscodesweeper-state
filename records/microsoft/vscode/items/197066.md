# Issue #197066 — Configurable styling of the indent guide line

[https://github.com/microsoft/vscode/issues/197066](https://github.com/microsoft/vscode/issues/197066)

## Summary

Legitimate feature request with 19 upvotes and active community interest; extension workarounds exist but the request for native support is reasonable. Keeping open for the community process.

## Change summary

Allow configurable indent guide line styling (dashed, dotted, width) in the editor.

## Best solution

Given that multiple extensions already achieve this via CSS patching (Custom UI Style, apc), and the vision favors extensions for per-feature visual customization, this is a strong extension-candidate. However, if the team decides native support is warranted, the change would live in `src/vs/editor/browser/viewParts/indentGuides/` with new settings for line style and width. Until then, the extension path is the right recommendation for users.
