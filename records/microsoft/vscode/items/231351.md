# Issue #231351 — Configurable fast scroll to keys other than Alt

[https://github.com/microsoft/vscode/issues/231351](https://github.com/microsoft/vscode/issues/231351)

## Summary

This is still a valid feature request: current main continues to hard-code Alt for fast scrolling, while only the scroll multiplier is configurable. The issue is already routed to the editor scrollbar area and parked in Backlog, so there is no strong close reason.

## Change summary

Allow users to configure the keyboard modifier used for fast scroll, or disable/extend the current Alt-only behavior.

## Best solution

If accepted, add a configurable fast-scroll modifier through the scrollable element options and expose it through the editor/list settings that currently only pass `fastScrollSensitivity`; otherwise leave the feature request open for maintainer prioritization.
