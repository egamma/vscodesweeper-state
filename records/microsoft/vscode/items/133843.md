## Change summary

Improve Toggle Line Comment for JSX, HTML, and CSS selections that overlap existing block comments so the resulting text remains syntactically valid.

## Best solution

Track this in the editor comments backlog as a request for nesting-aware or escaping-aware block comment toggling for languages without line comment tokens.
