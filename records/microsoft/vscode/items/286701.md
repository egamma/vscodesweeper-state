# Issue #286701 — Inline suggestion toolbar does not disappear if it's displayed on the last line

[https://github.com/microsoft/vscode/issues/286701](https://github.com/microsoft/vscode/issues/286701)

## Summary

Confirmed visual bug still present on main — the inline suggestion toolbar fails to hide when the suggestion is on the last line. An open PR (#305672) exists but has not been merged.

## Change summary

The inline suggestion toolbar (onHover mode) stays visible when the mouse moves away from the last line, because the hover dismissal logic doesn't fire correctly when there's no content below to trigger a mouseleave.

## Best solution

Merge or iterate on the open fix PR (https://github.com/microsoft/vscode/pull/305672) which targets the hover-dismissal logic in the inline completions hints widget. The fix likely lives in `src/vs/editor/contrib/inlineCompletions/browser/hintsWidget/hoverParticipant.ts` or the hover controller's mouse-leave handling.
