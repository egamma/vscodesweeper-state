# Issue #191648 — Should json sort on save also apply to JsonL

[https://github.com/microsoft/vscode/issues/191648](https://github.com/microsoft/vscode/issues/191648)

## Summary

Valid feature request to extend JSON sort to JSONL files. No implementation has landed on main; the sort command in the json-language-features extension has no JSONL-specific handling. A team member acknowledged it as a reasonable enhancement.

## Change summary

The JSON sort command should also handle JSONL (JSON Lines) files by sorting keys within each individual JSON object on each line.

## Best solution

Extend the sort logic in `extensions/json-language-features` (server-side `json/sort` request handler) to detect JSONL documents and apply key sorting per-line rather than treating the whole file as a single JSON value.
