# Issue #262360 — Invisible autostash

[https://github.com/microsoft/vscode/issues/262360](https://github.com/microsoft/vscode/issues/262360)

## Summary

Confirmed on current main: the stash-parsing regex in git.ts only matches entries with `on <branch>: <description>` format, silently dropping autostash entries (which have the bare subject `autostash`). An open PR (#325762) is actively addressing this — keeping open for that PR to land.

## Change summary

Autostash entries created during failed pull/rebase operations are invisible in VS Code's stash list because the parsing regex doesn't match their format.

## Best solution

Land https://github.com/microsoft/vscode/pull/325762, which handles autostash conflicts during pull and surfaces autostash entries in the stash list. Review and merge that PR.
