# Issue #105487 — [cli] specify command that runs on window open

## Summary

This is a valid, community-promoted feature request (44 upvotes, in Backlog) for a CLI option to execute VS Code commands on window open. No implementation exists on current main. Keep open.

## Change summary

Request a `--run-command` CLI argument that executes specified VS Code commands (with arguments) when the window opens.

## Best solution

Implement a `--run-command` (or similar) CLI argument in `src/vs/platform/environment/common/argv.ts` that queues commands for execution after window activation, supporting arguments and sequential invocation. Related issue https://github.com/microsoft/vscode/issues/238786 covers similar ground and could potentially be unified.
