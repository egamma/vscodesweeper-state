# Issue #3130 — Allow customization of mouse shortcuts

[https://github.com/microsoft/vscode/issues/3130](https://github.com/microsoft/vscode/issues/3130)

## Summary

Massively popular feature request (1920 upvotes) for general mouse shortcut customization. Only a narrow subset (multiCursorModifier setting) has been addressed; the broad capability to bind arbitrary actions to mouse+modifier combinations remains unimplemented on main.

## Change summary

Allow users to customize mouse button + modifier key combinations (e.g. Ctrl+Click, Alt+Click) the same way keyboard shortcuts are customizable today.

## Best solution

Extend the keybindings system to support mouse button events as bindable inputs — likely touching `src/vs/workbench/services/keybinding` and `src/vs/platform/keybinding` — so users can remap actions like multi-cursor, go-to-definition, and peek to their preferred mouse+modifier combos. The existing `editor.multiCursorModifier` is a narrow workaround; the real ask is a general mouse-binding mechanism.
