# Issue #274039 — git diff PNG viewer errors out

[https://github.com/microsoft/vscode/issues/274039](https://github.com/microsoft/vscode/issues/274039)

## Summary

Already fixed on main and shipped in 1.120.0. The root cause — `localResourceRoots` retaining the query string from `git:` URIs — was resolved by stripping query and fragment from the dirname.

## Change summary

Opening a staged PNG in the git diff viewer errors out because the image preview webview denies the resource due to a query-string mismatch in localResourceRoots.

## Best solution

Keep the shipped fix (commit b6a00bc1d17, 1.120.0+). No further change needed. The open PR #300037 is now redundant and can be closed.
