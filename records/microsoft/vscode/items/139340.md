# Issue #139340 — Long text always wrap

[https://github.com/microsoft/vscode/issues/139340](https://github.com/microsoft/vscode/issues/139340)

## Summary

This appears fixed on current main: long-line-dominated files no longer force word wrapping unless accessibility support is enabled. The fix shipped in 1.103.0, so the issue can be closed as completed rather than with a triage-reason label.

## Change summary

Opening a file dominated by very long lines ignored `editor.wordWrap: off` and forced wrapping for performance reasons.
