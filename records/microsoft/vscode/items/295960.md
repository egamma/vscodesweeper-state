# Issue #295960 — The scroll bar and active item indicator are rendered on top of the terminal inline chat suggest

[https://github.com/microsoft/vscode/issues/295960](https://github.com/microsoft/vscode/issues/295960)

## Summary

Still reproduces on main: the terminal inline chat suggest widget is rendered behind the scrollbar and active item indicator due to z-index stacking. An open PR (#295968) exists but has not been merged.

## Change summary

The scrollbar and active item indicator render on top of the terminal inline chat suggest widget, obscuring it.

## Best solution

Review and land (or rework) the open PR https://github.com/microsoft/vscode/pull/295968 — the fix likely needs to adjust z-index layering in `src/vs/workbench/contrib/terminal/browser/media/terminal.css` or the terminal chat widget CSS so the suggest popup appears above the scrollbar (z-index 11). The PR's approach of bumping the entire terminal container's z-index to 12 may need validation against other stacking contexts.
