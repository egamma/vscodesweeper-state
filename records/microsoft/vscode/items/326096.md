# Issue #326096 — [Terminal] Manually restart action

[https://github.com/microsoft/vscode/issues/326096](https://github.com/microsoft/vscode/issues/326096)

## Summary

Valid feature request: the terminal Relaunch command exists (`TerminalCommandId.Relaunch`) but is not surfaced in the terminal tab context menu or terminal instance context menu, forcing users to hover or use the command palette. No duplicate found.

## Change summary

Add a 'Relaunch' (restart) action to the terminal tab context menu and optionally the terminal instance context menu.

## Best solution

Register the existing `TerminalCommandId.Relaunch` command as a menu item in `MenuId.TerminalTabContext` (and possibly `MenuId.TerminalInstanceContext`) in `src/vs/workbench/contrib/terminal/browser/terminalMenus.ts`, alongside the existing Kill entry. This is a small, localized change — the command already exists and works.
