## Summary

This feature has been implemented on main and shipped in 1.100. Users can now toggle the visibility of individual quick diff providers for gutter decorations via a context menu, and the selection is persisted.

## Change summary

Allow users to choose which SCM quick diff provider is used for gutter decorations when multiple providers are available.

Keep the shipped implementation — quick diff provider visibility toggling via the gutter context menu (SCMQuickDiffDecorations menu), with persisted state in the QuickDiffService. No further change needed.
