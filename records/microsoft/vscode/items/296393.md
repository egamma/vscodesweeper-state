# Issue #296393 — Add setting to auto-expand tool calls and edits in chat view

[https://github.com/microsoft/vscode/issues/296393](https://github.com/microsoft/vscode/issues/296393)

## Summary

Team-authored feature request (MEMBER). The codebase currently has `chat.tools.autoExpandFailures` for failures only, and conditional auto-expand for terminal output and confirmations, but no general setting to expand all tool calls/edits by default. The requested capability is a real gap.

## Change summary

Add a setting to automatically expand all tool calls, edits, and progress items in the chat view by default.

## Best solution

Add a new `chat.tools.autoExpandAll` (or similar) boolean setting in `chat.shared.contribution.ts` and wire it into the collapsible wrapper logic in the tool invocation parts (`chatSimpleToolProgressPart.ts`, `chatTerminalToolProgressPart.ts`, `chatInputOutputMarkdownProgressPart.ts`) and `chatSubagentContentPart.ts` so that when enabled, all collapsible tool/edit items start expanded. The existing `autoExpandFailures` setting could become a sub-case.
