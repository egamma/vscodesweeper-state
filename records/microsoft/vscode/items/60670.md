# Issue #60670 — [folding] show folding actions at the end of folding range as well

[https://github.com/microsoft/vscode/issues/60670](https://github.com/microsoft/vscode/issues/60670)

## Summary

This is a valid, well-supported feature request (28 upvotes, above the 20-upvote threshold) asking for folding controls at the end of folding ranges. The feature has not been implemented on current main — folding controls still only appear at the start line in the gutter. Multiple duplicates have been closed in favor of this issue, confirming it as the canonical tracker.

## Change summary

Request to show a folding action/icon at the closing line of a folding range so users don't have to scroll up to the start to collapse.

## Best solution

Add an option (or extend `showFoldingControls`) to render a fold indicator at the last line of each folding range, likely in `src/vs/editor/contrib/folding/browser/foldingDecorations.ts` and `folding.ts`. Related issue https://github.com/microsoft/vscode/issues/19690 (show folding lines) may share implementation work for visualizing the full extent of a range.
