# Issue #100868 — Debug console: introduce ctrl + up / down to focus output

[https://github.com/microsoft/vscode/issues/100868](https://github.com/microsoft/vscode/issues/100868)

## Summary

Valid accessibility feature request from a contributor. Ctrl+Up/Down to focus debug console output (mirroring terminal behavior) is not yet implemented on main. Hard keep-open: author is a CONTRIBUTOR.

## Change summary

Add Ctrl+Up/Down keyboard shortcut in the debug console to move focus between the input and the output tree, matching the terminal's existing behavior.

## Best solution

Implement Ctrl+Up/Down keybindings in the REPL view (src/vs/workbench/contrib/debug/browser/repl.ts) to shift focus from the input editor to the output tree and vice versa, consistent with how the terminal handles the same shortcut. The existing showPreviousValue/showNextValue history navigation via plain Up/Down and Shift+Tab would remain unchanged.
