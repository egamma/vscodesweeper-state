## Summary

Team-authored feature request for onType indentation rules analogous to onEnter rules; must remain open for maintainer judgment.

## Change summary

Request to add onType rules (inspecting previous line, before/after range text) to control outdentation when characters like `{` are typed, avoiding regressions from the current increaseIndentationPattern approach.

## Best solution

Design and implement an onType rule system in the editor auto-indent contribution (parallel to onEnterRules) that allows language grammars to specify context-aware indentation adjustments on arbitrary character input, resolving the regression path described in https://github.com/microsoft/vscode/issues/224584.
