## Summary

The agent host shell resolution already enumerates pwsh.exe installations (PSCore stable, MSIX, .NET tool, preview) and falls back to Windows PowerShell 5.1 via `findWinPS()`. The reporter resolved it themselves via a settings path change. A second commenter reports still seeing it but provides no details. Requesting info to determine if there's a real remaining gap.
