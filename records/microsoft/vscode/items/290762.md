# Issue #290762 — Custom Agents: Models in defined in custom agent do not lead to different models showing in models picker, should they?

[https://github.com/microsoft/vscode/issues/290762](https://github.com/microsoft/vscode/issues/290762)

## Summary

Team-authored design question about whether custom agent model definitions should constrain the models picker UI. Keeping open for maintainer judgment.

## Change summary

Models defined in a custom agent/mode don't filter the models picker to only those models, and hiding a mode-specified model in 'Manage Models' removes it from the picker even though the mode declares it.

## Best solution

Decide on the intended interaction between custom mode model lists and the picker UI — whether the picker should be scoped to only declared models when a mode is active, and whether 'Manage Models' visibility should override mode declarations. Likely touches chat model picker and custom modes contribution code.
