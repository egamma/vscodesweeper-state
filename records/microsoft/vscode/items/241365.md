## Summary

Team-authored issue (MEMBER); ghost text in HC Dark mode uses only a border for differentiation with no distinct foreground color, making it hard to distinguish. Still present on main — no changes to the HC Dark ghost text styling since the issue was filed.

## Change summary

Ghost text in High Contrast Dark theme is difficult to visually distinguish from regular editor text.

## Best solution
