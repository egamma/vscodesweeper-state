# Issue #283982 — Notifications are disruptive when using AI Chat window

[https://github.com/microsoft/vscode/issues/283982](https://github.com/microsoft/vscode/issues/283982)

## Summary

Configurable notification positions landed in commit ff21980e60c and shipped in 1.110, allowing users to move notifications to bottom-left or top-right to avoid overlapping the chat box. This resolves the reported disruption.

## Change summary

Notifications in the bottom-right corner overlap and obscure the AI Chat input area, forcing users to dismiss them manually.

## Best solution

Keep the shipped configurable-position feature (bottom-right, bottom-left, top-right); users can move notifications out of the chat area via the `workbench.notifications.position` setting. No further change needed.
