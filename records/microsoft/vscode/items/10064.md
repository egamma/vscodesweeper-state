# Issue #10064 — Git - Support git-bash ssh agent for Windows

[https://github.com/microsoft/vscode/issues/10064](https://github.com/microsoft/vscode/issues/10064)

## Summary

Still a valid, unimplemented feature request with 30 upvotes — VS Code's git extension doesn't detect or forward the git-bash ssh-agent socket on Windows. Stays open as a community-promoted Backlog item.

## Change summary

Automatically detect and use the git-bash ssh-agent on Windows so SSH key authentication works without manual SSH_AUTH_SOCK configuration.

## Best solution

The git extension could detect the git-bash ssh-agent socket (typically via a named pipe or SSH_AUTH_SOCK) on Windows and inject it into the git process environment. The `help wanted` label is appropriate — this is a well-scoped contribution opportunity in `extensions/git/src/askpass.ts` or the git environment setup.
