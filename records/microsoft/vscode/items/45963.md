# Issue #45963 — [icon themes] Expose API to provide a dynamic icon theme.

[https://github.com/microsoft/vscode/issues/45963](https://github.com/microsoft/vscode/issues/45963)

## Summary

The request for an official API to provide dynamic icon themes remains unaddressed. The unofficial `_watch: true` property (commit 24e8857cc5b) provides file-watching-based reload but was never promoted to a stable public API, and the broader ask for a programmatic reload/generation function is still open.

## Change summary

Request for a public extension API to dynamically reload or generate icon themes without requiring a full editor restart.

## Best solution

Either promote the existing `_watch` mechanism to a stable documented API, or design a richer API (e.g., a callback-based icon theme provider) that lets extensions programmatically trigger theme reloads. The relevant code lives in `src/vs/workbench/services/themes/browser/workbenchThemeService.ts` and `src/vs/workbench/services/themes/browser/fileIconThemeData.ts`.
