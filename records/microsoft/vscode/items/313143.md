# Issue #313143 — SSH connection

[https://github.com/microsoft/vscode/issues/313143](https://github.com/microsoft/vscode/issues/313143)

## Summary

Verified duplicate of #251777, which tracks the same WebAssembly/undici out-of-memory crash during Remote-SSH server installation on memory-constrained remote machines (HPC clusters with low RLIMIT_AS). The canonical issue is open, labeled as a regression, and actively investigated by a team member.

## Change summary

Remote-SSH server installation fails in an infinite loop on memory-constrained remote machines because Node.js v22.x's bundled undici module cannot instantiate its WebAssembly llhttp parser.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/251777 — the team is tracking an upstream Node.js fix (https://github.com/nodejs/node/pull/60788) and plans to add a workaround to auto-disable wasm guard regions when RLIMIT_AS is too low.
