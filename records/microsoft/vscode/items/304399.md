# Issue #304399 — Copilot ripgrep EACCES error

[https://github.com/microsoft/vscode/issues/304399](https://github.com/microsoft/vscode/issues/304399)

## Summary

The EACCES error comes from the Copilot extension's bundled ripgrep binary losing execute permissions on a specific remote Linux server — this is either a Copilot extension packaging issue or a server environment issue, not a VS Code core bug.

## Change summary

Copilot extension fails to spawn ripgrep on a remote Linux server due to EACCES (permission denied) on the rg binary.

## Best solution

This belongs in the Copilot extension's issue tracker (https://github.com/microsoft/vscode-copilot-release/issues). The user can work around it by running `chmod +x ~/.vscode-server/cli/servers/Stable-07ff9d6178ede9a1bd12ad3399074d726ebe6e43/server/node_modules/@vscode/ripgrep/bin/rg` on the remote server.
