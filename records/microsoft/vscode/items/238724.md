# Issue #238724 — Diff viewer: misaligned lines with inlays + word wrap

[https://github.com/microsoft/vscode/issues/238724](https://github.com/microsoft/vscode/issues/238724)

## Summary

Fixed on main: PR #248526 disables variable line heights in diff editors, preventing inlay-hint-induced misalignment when word wrap is active. Shipped in 1.101.0.

## Change summary

Diff editor lines become misaligned when inlay hints and word wrap are both active, because inlay decorations affect wrapping differently on the left vs right side.

## Best solution

Keep the shipped fix (commit 0dbc0a9de26, VS Code 1.101.0) which sets `allowVariableLineHeights = false` in diff editors, ensuring consistent line height calculations across both sides.
