# Issue #319362 — Add a reload file menu option, and keyboard shortcut

[https://github.com/microsoft/vscode/issues/319362](https://github.com/microsoft/vscode/issues/319362)

## Summary

The requested capability already exists: the "Revert File" command (workbench.action.files.revert) is available in the Command Palette and the File menu. It reloads the file from disk, discarding unsaved changes. A default keybinding is not assigned by design, but users can bind it via Preferences: Open Keyboard Shortcuts.

## Change summary

Add a menu option and keyboard shortcut to reload/revert a file from disk.

## Best solution

Keep current behavior: "File > Revert File" and the Command Palette entry already provide this. Users who want a keybinding (e.g. Cmd+R) can assign one through the Keyboard Shortcuts editor — this is the standard VS Code pattern for commands without a default binding.
