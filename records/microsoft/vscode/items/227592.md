## Summary

Valid feature request for JSON IntelliSense to suggest object completions (e.g. `{ ... }`) alongside string enum completions when the schema uses `anyOf`. Already triaged into Backlog; no fix found on main. The completion logic lives in the upstream `vscode-json-languageservice` package.

## Change summary

JSON IntelliSense does not suggest object schema alternatives when string enums are also present in an `anyOf` schema.

## Best solution
