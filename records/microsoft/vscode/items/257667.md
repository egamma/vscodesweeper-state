# Issue #257667 — Make agent talk less about tools they need to activate

[https://github.com/microsoft/vscode/issues/257667](https://github.com/microsoft/vscode/issues/257667)

## Summary

Team-authored issue requesting that the chat agent avoid verbose announcements about tool activation. Keeping open for maintainer judgment.

## Change summary

The chat agent verbosely announces tool activation/enabling, which is confusing and exposes implementation details to the user.

## Best solution

Adjust the agent's system prompt or tool-call messaging to suppress or soften tool-activation announcements — use language like 'retrieving' or 'discovering' instead of 'enabling/activating', or suppress the announcement entirely when tools are auto-activated.
