# Issue #237234 — No hint shown when hovering over type in debugger variables view

[https://github.com/microsoft/vscode/issues/237234](https://github.com/microsoft/vscode/issues/237234)

## Summary

Still reproduces on main: the type span element in the debug variables view has no hover/tooltip registered, unlike the name and value elements. No fix has landed.

## Change summary

Hovering over the type column in the debugger variables view shows no tooltip, making it impossible to see the full type when truncated.

## Best solution

Register a managed hover on the `type` span in `AbstractExpressionsRenderer.renderTemplate()` (src/vs/workbench/contrib/debug/browser/baseDebugView.ts ~line 158) and update it in `debugExpressionRenderer.ts` when `data.type.textContent` is set, so the full type string is visible on hover.
