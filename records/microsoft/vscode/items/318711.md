# Issue #318711 — Regression in VS Code 1.122.0: Remote Extension Host WebSocket repeatedly disconnects in WSL, breaking all AI extensions (Copilot, Kilo, etc.)

## Summary

This is the same root cause as #321901 — process.report was blocking the extension host, causing unacknowledged WebSocket messages to pile up and trigger socket timeouts in WSL. Fixed by disabling process.report in PR #321998, shipped in 1.125.1.
