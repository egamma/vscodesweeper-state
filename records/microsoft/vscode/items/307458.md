# Issue #307458 — Disable auto-adding highlighted text to chat

[https://github.com/microsoft/vscode/issues/307458](https://github.com/microsoft/vscode/issues/307458)

## Summary

Valid feature request for more granular control over implicit context. The existing `chat.implicitContext.enabled` setting can disable all implicit context (set to `never`), but there is no way to keep file-level context while disabling auto-attachment of highlighted text/selections specifically — the suggestedContext flow documents that selections are always included.

## Change summary

Request to add an option to disable the automatic inclusion of highlighted/selected text as context when opening agent chat.

## Best solution

Either document the existing `chat.implicitContext.enabled` = `never` setting more prominently (if disabling all implicit context satisfies the request), or add a dedicated `chat.implicitContext.includeSelection` boolean setting in `chatImplicitContext.ts` that gates the selection-specific path (lines 191–193) separately from the file/visible-range context.
