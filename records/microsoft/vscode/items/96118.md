# Issue #96118 — Allow to drag the entire panel/sidebar to move it left or right

[https://github.com/microsoft/vscode/issues/96118](https://github.com/microsoft/vscode/issues/96118)

## Summary

Valid feature request — the panel can be repositioned via commands (left/right/top/bottom) but there is no drag-and-drop gesture to move the entire panel container to a different layout position. The capability remains unimplemented on current main.

## Change summary

Allow dragging the entire panel (or sidebar) as a unit to reposition it left/right/bottom in the workbench layout, rather than only through commands.

## Best solution

Implement a drop-target overlay on the workbench edges that accepts the panel title bar drag and triggers `setPanelPosition()` — likely in `src/vs/workbench/browser/parts/paneCompositePart.ts` and the layout DnD infrastructure. The existing `CompositeDragAndDropObserver` handles individual view drags; a new handler for the container-level gesture is needed.
