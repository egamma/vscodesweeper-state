Active team design discussion with multiple members engaged on improving view DnD behavior; no implementation has landed on main yet. Keeping open as an unresolved under-discussion feature request.

## Change summary

Change view drag-and-drop so dropping into a sidebar/panel bar defaults to moving the view as a separate container rather than splitting, aligning with editor group DnD behavior.

## Best solution
