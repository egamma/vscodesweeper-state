# Issue #311874 — Inline Skill Picker with Fuzzy Search while typing in Chat

[https://github.com/microsoft/vscode/issues/311874](https://github.com/microsoft/vscode/issues/311874)

## Summary

Legitimate feature request for an inline fuzzy-search skill picker in the chat input. Not implemented on current main — the existing skill picker uses a standard quick pick dialog, not an inline dropdown with real-time fuzzy matching while typing. Should remain open for product discussion.

## Change summary

Request for an inline dropdown in Chat input that suggests matching skills via fuzzy search as the user types, allowing explicit skill attachment.

## Best solution

This is a UX enhancement for the chat customizations/skills system. The product team should evaluate whether inline skill suggestions (like @-mention completions) would improve discoverability and reduce mismatch between user intent and keyword-based skill matching. The chat widget input area (`src/vs/workbench/contrib/chat/browser/widget/`) and the prompt syntax pickers (`src/vs/workbench/contrib/chat/browser/promptSyntax/pickers/`) are the relevant areas.
