# Issue #168746 — [Accessibility] Word wrap does not work in diff view (F7 and Shift+F7)

[https://github.com/microsoft/vscode/issues/168746](https://github.com/microsoft/vscode/issues/168746)

## Summary

Still unimplemented on main: the accessible diff viewer (F7/Shift+F7) renders lines via renderViewLine2 with no word-wrap support. The feature request remains valid and belongs in the accessibility/diff-editor area.

## Change summary

Word wrap does not apply in the accessible diff viewer (F7/Shift+F7) when screen reader mode is on.

## Best solution

Add word-wrap support to the accessible diff viewer's line rendering in `src/vs/editor/browser/widget/diffEditor/components/accessibleDiffViewer.ts`. The `_getLineHtml` method currently renders each line as a single unbroken span; it would need to split long lines according to the editor's wordWrap setting and container width.
