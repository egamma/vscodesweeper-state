# Issue #168746 — [Accessibility] Word wrap does not work in diff view (F7 and Shift+F7)

[https://github.com/microsoft/vscode/issues/168746](https://github.com/microsoft/vscode/issues/168746)

## Summary

Valid accessibility feature request. The accessible diff viewer (F7/Shift+F7) uses fixed-height lines with `white-space: pre` CSS and no wrapping logic in its rendering — word wrap is not supported on current main.

## Change summary

Add word wrap support to the accessible diff viewer (F7/Shift+F7) so long lines wrap when the word wrap setting is enabled.

## Best solution

Add wrapping support to the accessible diff viewer in `src/vs/editor/browser/widget/diffEditor/components/accessibleDiffViewer.ts` — the `_getLineHtml` rendering and the `.diff-review-cell` CSS (`white-space: pre`, fixed `height`) need to account for the user's word wrap setting. This is a localized change to the accessible diff viewer component.
