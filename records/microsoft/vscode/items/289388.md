# Issue #289388 — Interleaved thinking block descriptions are misleading

[https://github.com/microsoft/vscode/issues/289388](https://github.com/microsoft/vscode/issues/289388)

## Summary

Team-authored polish issue (Tyriar is a CONTRIBUTOR) about misleading interleaved thinking block descriptions that conflate past and future actions. The thinking title generation code in chatThinkingContentPart.ts has seen heavy recent work but no commit specifically addresses this framing issue. Keeping open per hard keep-open rule.

## Change summary

Interleaved thinking block descriptions misleadingly describe both what already happened and what is about to happen, instead of accurately reflecting only current/past state.

## Best solution

Update the thinking block title/description generation in `src/vs/workbench/contrib/chat/browser/widget/chatContentParts/chatThinkingContentPart.ts` to use past-tense or present-progressive phrasing (e.g. 'Summarizing what's happened', 'Taking notes') rather than descriptions that conflate completed and upcoming actions.
