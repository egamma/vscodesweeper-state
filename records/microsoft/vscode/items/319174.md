# Issue #319174 — When changing display language and restarting, only one window is reopened

[https://github.com/microsoft/vscode/issues/319174](https://github.com/microsoft/vscode/issues/319174)

## Summary

Team-authored bug report about incomplete window restoration after a display language change relaunch on macOS. The relaunch code sets `wasRestarted` which should force `restoreWindows: 'all'`, but the reporter sees only one window reopened. Keeping open for maintainer investigation.

## Change summary

Changing the display language and restarting only reopens one window instead of all previously open windows on macOS.

## Best solution

Investigate whether `windowsStateHandler.state.openedWindows` is correctly populated before the relaunch quit sequence completes, or whether macOS-specific behavior (e.g. app activation after relaunch) causes the initial open to bypass the `doGetPathsFromLastSession` path. Likely files: `src/vs/platform/windows/electron-main/windowsMainService.ts` and `src/vs/platform/windows/electron-main/windowsStateHandler.ts`.
