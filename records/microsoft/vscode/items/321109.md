## Summary

The report describes modal "Cannot reconnect" dialogs across multiple remote-SSH windows blocking each other and requiring dismissal in a specific order. The dialog is in core VS Code (src/vs/workbench/contrib/remote/browser/remote.ts:1008), but the cross-window ordering behavior is an Electron modal dialog characteristic. Previous identical reports (https://github.com/microsoft/vscode/issues/237884, https://github.com/microsoft/vscode/issues/233890) were both closed as info-needed. The current report lacks reproduction steps precise enough to distinguish a VS Code bug from expected Electron modal behavior, and there are no upvotes or confirmed reproductions from other commenters.

## Change summary
