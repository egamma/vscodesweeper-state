# Issue #237445 — Recursive variable font is rendered with wrong space width

[https://github.com/microsoft/vscode/issues/237445](https://github.com/microsoft/vscode/issues/237445)

## Summary

Confirmed upstream Chromium bug: variable font Recursive renders spaces too narrow in Chromium's text shaping engine. VS Code inherits this via Electron; the fix must come from Chromium.

## Change summary

The Recursive variable font renders space characters too narrow in the VS Code editor compared to native applications.

## Best solution

Track the upstream Chromium bug (https://issues.chromium.org/issues/412384505) and close once Chromium ships a fix that lands in a VS Code Electron update. No VS Code-side workaround is practical since the issue is in Chromium's font shaping for variable fonts.
