# Issue #283192 — Can the chat window context support reading PDF and OFFICE format files?

[https://github.com/microsoft/vscode/issues/283192](https://github.com/microsoft/vscode/issues/283192)

## Summary

Valid feature request for PDF/Office file support in chat context. Only 4 upvotes so far — well below the 20-upvote backlog threshold. No duplicate found. Keeping open for the community process.

## Change summary

Allow attaching PDF and Office format files (DOCX, etc.) as context in the VS Code chat window.

## Best solution

This likely belongs as an extension or MCP tool rather than core — PDF/DOCX parsing requires heavyweight libraries (pdf.js, mammoth, etc.) that conflict with the lean-core value. An MCP server or extension that extracts text from these formats and provides it as context would be the cleanest path. Alternatively, the chat attachment system could be extended with a pluggable binary-format decoder API so extensions supply format support.
