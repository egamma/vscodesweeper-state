# Issue #55146 — "Quick Open" open all/multi-select files

[https://github.com/microsoft/vscode/issues/55146](https://github.com/microsoft/vscode/issues/55146)

## Summary

Valid, unimplemented feature request — multi-select in the Quick Open file picker is not yet supported on main. This is the canonical issue (others were duped here). With 8 upvotes it's below the 20-upvote backlog promotion threshold but still a reasonable request.

## Change summary

Allow selecting and opening multiple files at once from the Quick Open (Ctrl+P) file picker via multi-select or an 'open all' action.

## Best solution

Implement multi-select support in the file Quick Open picker (leveraging the existing `canSelectMany` infrastructure in QuickPick) so users can Ctrl+click or Shift+click to open several files at once. The change would likely live in `src/vs/workbench/contrib/quickaccess/browser/` and the quick-input layer. Alternatively, this could remain community-driven via extensions like 'Open Matching Files'.
