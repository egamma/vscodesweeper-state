# Issue #237582 — Find widget looks different at different places

[https://github.com/microsoft/vscode/issues/237582](https://github.com/microsoft/vscode/issues/237582)

## Summary

The find widget intentionally differs between editor (full FindWidget with replace) and terminal/output (SimpleFindWidget, search-only). A team member confirmed this is by design. The reporter's follow-up asking if Output and Terminal could match is already true — both use SimpleFindWidget.

## Change summary

The find widget has different appearances in the editor, output panel, and terminal.

## Best solution

Close as designed — the editor find widget includes replace functionality (the expand arrow) which terminal/output don't need. Output and Terminal already share the same SimpleFindWidget base. Visual consistency beyond this would be a separate polish request with a concrete proposal.
