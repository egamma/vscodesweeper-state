# Issue #279184 — Provide the ability to disable extension specific instructions in agent mode

[https://github.com/microsoft/vscode/issues/279184](https://github.com/microsoft/vscode/issues/279184)

## Summary

Team-authored feature request; must be kept open for maintainer judgment. The request for user control over extension-contributed instructions remains valid, though the immediate example may be addressable by the extension fixing its `applyTo` scope.

## Change summary

Users cannot disable or remove extension-specific instructions that are automatically attached to agent-mode prompts.

## Best solution

Two-pronged: (1) extensions like Azure should fix their instruction scope (no blanket `applyTo: '**'`), and (2) the chat prompts UI should let users toggle off individual extension-contributed instructions from the context, similar to how attached files can be removed.
