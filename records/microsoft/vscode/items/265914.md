# Issue #265914 — Chat: Support managing tool approvals via command palette

[https://github.com/microsoft/vscode/issues/265914](https://github.com/microsoft/vscode/issues/265914)

## Summary

Valid feature request for granular tool approval management via command palette. Not yet implemented on main — the only existing mechanism is a full reset command. Keeping open as a reasonable UX improvement for the chat tools system.

## Change summary

Allow users to view and individually manage (revoke/grant) tool approvals through the command palette, rather than only resetting all at once.

## Best solution

Add a command palette entry (e.g. 'Chat: Manage Tool Approvals') that shows a quick-pick list of tools with their current approval state, letting users toggle individual approvals. This likely belongs in `src/vs/workbench/contrib/chat/browser/` alongside the existing tool confirmation infrastructure.
