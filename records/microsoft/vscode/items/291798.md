Valid feature request for a manual drag-to-resize handle on the chat prompt input box. The input currently auto-grows with content but has no user-controlled resize mechanism. Only 2 upvotes and no milestone — keeping open for the community process.

## Change summary

Allow users to manually drag-resize the chat prompt input box for easier editing and reviewing of long prompts.
