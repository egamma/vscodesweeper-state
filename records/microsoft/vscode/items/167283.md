# Issue #167283 — Expose User tasks and all task commands in vscode Web

[https://github.com/microsoft/vscode/issues/167283](https://github.com/microsoft/vscode/issues/167283)

## Summary

The web-commands part of this request appears addressed (TaskExecutionSupportedContext now includes CustomExecutionSupportedContext, so task commands show in web when custom execution is available). However, the global/user-scoped tasks for extensions part remains unimplemented, and an open PR (#178597) references this issue — keeping open until that PR is resolved.

## Change summary

Expose task commands in VS Code Web and allow extensions to provide global (user-scoped) tasks shared across workspaces.

## Best solution

Review and land (or close) the open PR https://github.com/microsoft/vscode/pull/178597 which implements extension-provided global tasks. The web-commands portion is already working via CustomExecutionSupportedContext; the remaining gap is the global task scope for the extension API.
