# Issue #181078 — kernel quick pick flashes

[https://github.com/microsoft/vscode/issues/181078](https://github.com/microsoft/vscode/issues/181078)

## Summary

Still reproduces on main: the kernel picker creates a new QuickPick instance when navigating between levels (hide → recursive showQuickPick), causing the visible flash. No fix has landed for this specific issue.

## Change summary

Notebook kernel quick pick flashes (hides and re-shows) when navigating between picker options/levels.

## Best solution

Refactor `notebookKernelQuickPickStrategy.ts` to reuse a single QuickPick instance across navigation levels (updating items/title in place) rather than hiding and recursively creating a new one — similar to how multi-step quick picks work elsewhere in VS Code.
