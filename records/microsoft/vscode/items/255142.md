## Change summary

Allow chatmode files to opt out of project context (instruction files, workspace context) so that generic questions aren't answered with project-specific framing.

## Best solution

Add a frontmatter attribute (e.g. `instructions: false` or `isolated: true`) to chatmode/prompt files that suppresses automatic injection of instruction files and workspace context.
