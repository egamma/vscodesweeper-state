## Change summary

Allow the AI agent to programmatically push a running terminal command into the background so it can continue other work without waiting for long-running sync commands.

## Best solution

Implement a tool or API surface that lets the agent mark an active sync terminal session as background, enabling it to proceed with other tasks and check back later.
