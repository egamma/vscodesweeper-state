# Issue #169166 — Multi-Process Debugger isn't working on VSC 1.74

[https://github.com/microsoft/vscode/issues/169166](https://github.com/microsoft/vscode/issues/169166)

## Summary

Still an active bug being worked on — compound launch configs with shared preLaunchTasks can race, causing the second config to skip the task. A team member reproduced and identified the root cause; work is ongoing under issue #247414 with a recent WIP commit (49afe3b173a, April 2025).

## Change summary

Compound debug launch configurations skip preLaunchTask execution for the second config when both reference the same task.

## Best solution

Active work on https://github.com/microsoft/vscode/issues/247414 (commit 49afe3b173a adds getBusyTasks logic to debugTaskRunner.ts). This issue could be closed as a duplicate of #247414 once that lands, or kept as the original report. The fix belongs in `src/vs/workbench/contrib/debug/browser/debugTaskRunner.ts` and `src/vs/workbench/contrib/tasks/browser/terminalTaskSystem.ts`.
