# Issue #230975 — Debug toolbar trapped in issue reporter

## Summary

The issue reporter was refactored from a full-screen overlay (which shared z-index 2520 with the debug toolbar, causing the toolbar to render trapped inside it) into an editor-tab wizard. This architectural change resolves the reported visual bug — the debug toolbar now floats above the editor area normally.
