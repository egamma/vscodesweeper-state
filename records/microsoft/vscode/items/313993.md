# Issue #313993 — Source control doesn't show changes for png images

[https://github.com/microsoft/vscode/issues/313993](https://github.com/microsoft/vscode/issues/313993)

## Summary

Duplicate of #299774, which was fixed by commit b6a00bc1d17 (shipped in 1.120.0). The root cause was that the media-preview webview's localResourceRoots included the query/fragment from git: URIs, preventing image loading.

## Change summary

Image diff in Source Control shows 'An error occurred while loading the image' instead of rendering the previous version of modified PNG files.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/299774 — the fix in commit b6a00bc1d17 (stripping query/fragment from localResourceRoots for git: URIs) resolves this exact symptom, shipped in 1.120.0.
