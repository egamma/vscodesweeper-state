## Summary

Feature request to allow hiding/detaching the chat panel from the sessions sidebar, letting agent sessions live exclusively in editor tabs. No duplicate found; this is a distinct UX customization ask about the sessions/chat layout introduced in 1.109. Route to the chat area for the team to evaluate.

## Change summary

Allow users to disable or detach the chat panel from the sessions sidebar so agent sessions can be used independently in editor tabs without snapping back to the chat panel.

## Best solution

The team should evaluate whether the sessions sidebar can operate independently of the chat panel — either via a setting to hide the chat panel or by decoupling session navigation from the panel entirely. The bug described (sessions snapping back to the chat panel on input) should be filed or tracked separately if reproducible.
