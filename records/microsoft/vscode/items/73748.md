# Issue #73748 — Be able disable task detection from tasks.json or override tasks execution

[https://github.com/microsoft/vscode/issues/73748](https://github.com/microsoft/vscode/issues/73748)

## Summary

Team-authored feature request on the 'On Deck' milestone requesting the ability to disable or override task detection from tasks.json based on file scheme. No implementation found on main; the gap remains.

## Change summary

Allow extensions to disable tasks.json-based task detection or override task execution when the workspace uses a non-file scheme (e.g. remote/Live Share).

## Best solution

Implement an API for extensions to opt in/out of tasks.json detection per scheme, or allow overriding task execution with the originating URI. This belongs in the tasks contribution area (src/vs/workbench/contrib/tasks/) and relates to the existing CustomExecution API work tracked in #33523.
