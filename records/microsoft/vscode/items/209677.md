# Issue #209677 — Inline editing of debug variables

[https://github.com/microsoft/vscode/issues/209677](https://github.com/microsoft/vscode/issues/209677)

## Summary

Team-authored feature request for inline debug variable editing — must stay open for maintainer judgment. The capability does not exist on current main and remains a valid enhancement proposal.

## Change summary

Allow editing debug variables inline in the editor (via clickable inline values or debug hover edit action) instead of requiring the REPL or Variables view.

## Best solution

Implement an inline edit experience for debug variables using the EvaluatableExpressionProvider and DAP's setVariable/setExpression requests. The overlay input box pattern from inline chat could be reused. Work would likely touch `src/vs/workbench/contrib/debug/browser/` — the inline values decoration provider and the debug hover widget. This is a team-owned design decision on the debug UX.
