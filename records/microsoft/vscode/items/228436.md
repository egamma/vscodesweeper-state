# Issue #228436 — Editor scrolls horizontally with wrapping enabled

[https://github.com/microsoft/vscode/issues/228436](https://github.com/microsoft/vscode/issues/228436)

## Summary

Keep open and route to the editor area; the report describes a plausible editor layout bug and current main still contains minimap/viewport-wrapping content-width logic that can create horizontal scroll space.

## Change summary

With word wrap enabled, the editor still allows horizontal scrolling, especially when the right-side minimap and vertical scrollbar are visible.

## Best solution

The editor layout should avoid adding horizontal scroll width that is caused only by the right minimap or vertical scrollbar padding when viewport wrapping is enabled, likely around `src/vs/editor/common/viewLayout/viewLayout.ts` and `src/vs/editor/browser/viewParts/editorScrollbar/editorScrollbar.ts`.
