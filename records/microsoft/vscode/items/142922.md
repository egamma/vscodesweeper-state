# Issue #142922 — Debug Console Filter is a bit confusing

[https://github.com/microsoft/vscode/issues/142922](https://github.com/microsoft/vscode/issues/142922)

## Summary

Some sub-requests have been partially addressed (the 'Showing X of Y' badge exists, #132995 and #199872 were closed into this), but the core remaining ask — a Find/search mode in the Debug Console that navigates to matches without hiding non-matching lines — is still unimplemented. With 8 upvotes and active community interest, this should stay open.

## Change summary

Improve the Debug Console filter UX: add a 'Find' mode that highlights and navigates matches without hiding output, make the active filter more visually prominent, and add a clear-filter button.

## Best solution

The main remaining gap is a Find/search mode for the Debug Console (navigate matches without filtering out non-matches). This would live in `src/vs/workbench/contrib/debug/browser/repl.ts` and the `replFilter.ts` infrastructure. The filter badge ('Showing X of Y') and the clear-on-reopen behavior (#141682) are already shipped. A maintainer should confirm whether the 'highlight matches' idea from #132995 discussion is also still desired, and scope the remaining work to the Find mode.
