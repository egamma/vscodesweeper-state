# Issue #318548 — Link provider runs for test results diff view, but not test results output view

[https://github.com/microsoft/vscode/issues/318548](https://github.com/microsoft/vscode/issues/318548)

## Summary

Verified duplicate of #243469: the test results output view uses a detached terminal, and external link providers are explicitly skipped for detached terminals in the link contribution (line 72 of terminal.links.contribution.ts). Issue #243469 is open, labeled bug/testing, and assigned — it tracks the same root cause.

## Change summary

Extension-provided terminal link providers (e.g. Dart's `package:` URI handler) do not fire for the test results output view, only for the diff view and the regular terminal.

## Best solution

Consolidate on https://github.com/microsoft/vscode/issues/243469 which is already assigned and tracks enabling terminal link providers in the test output view (detached terminal). The fix involves wiring up `externalProvideLinksCb` for detached terminals in `TerminalLinkContribution.xtermReady()`.
