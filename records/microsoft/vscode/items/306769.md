## Summary

The "Copy Changes (Patch)" context menu item does not exist in VS Code's built-in Git extension — it is contributed by a third-party extension. The built-in `scm/resourceGroup/context` menu for staged changes only offers Unstage All and View Staged Changes; no patch-copy command is registered.

## Change summary

Right-clicking "Staged Changes" header and selecting "Copy Changes (Patch)" sometimes includes unstaged changes.

## Best solution

The reporter should file this issue against the extension that contributes the "Copy Changes (Patch)" command (likely GitLens or a similar Git enhancement extension). The built-in Git extension does not provide this command.
