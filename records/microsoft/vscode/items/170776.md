## Summary

## Change summary

Detect Indentation from Content should ignore or discount wrapped Doxygen comment continuation lines so a 3-space C++ file is not guessed as 4-space.

## Best solution

Improve `src/vs/editor/common/model/indentationGuesser.ts` to avoid treating block-comment continuation alignment as indentation evidence, and add a regression case under `src/vs/editor/test/common/model/textModel.test.ts` for the reported C++/Doxygen sample.
