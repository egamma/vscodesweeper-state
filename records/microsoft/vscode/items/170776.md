# Issue #170776 — Detect indentation from content confused by Doxygen comments

[https://github.com/microsoft/vscode/issues/170776](https://github.com/microsoft/vscode/issues/170776)

## Summary

Valid enhancement request: the indentation guesser's alignment-detection heuristic doesn't cover Doxygen/block-comment continuation lines. The issue is correctly labeled and milestoned in Backlog; no fix has landed on main.

## Change summary

Indentation detection is confused by multi-line Doxygen comments whose continuation lines are aligned to the opening `/**`, skewing the tab-size guess.

## Best solution

Extend the `looksLikeAlignment` heuristic in `src/vs/editor/common/model/indentationGuesser.ts` to recognize block-comment continuation patterns (lines starting with whitespace + `*` or aligned to a `/**` opener) and skip them, similar to how comma-alignment lines are already skipped.
