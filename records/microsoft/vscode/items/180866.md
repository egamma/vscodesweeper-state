# Issue #180866 — undo image paste in rendered notebook markdown will just remove the attachment

[https://github.com/microsoft/vscode/issues/180866](https://github.com/microsoft/vscode/issues/180866)

## Summary

Team-authored bug on the On Deck milestone; the root cause in the undo/redo groupOrder logic is still present on main. Keeping open.

## Change summary

Undoing an image paste in a rendered notebook markdown cell only reverts the metadata (attachment) but not the cell text, leaving a broken image link.

## Best solution

Fix `_findClosestUndoElementInGroup` in `src/vs/platform/undoRedo/common/undoRedoService.ts` to undo ALL elements in the same group (regardless of groupOrder) when the trigger resource's element has a lower groupOrder, or rethink how notebook cell vs notebook-level undo stacks interact when `notebook.undoRedoPerCell` is enabled and a workspace edit touches both.
