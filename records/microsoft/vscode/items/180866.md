# Issue #180866 — undo image paste in rendered notebook markdown will just remove the attachment

[https://github.com/microsoft/vscode/issues/180866](https://github.com/microsoft/vscode/issues/180866)

## Summary

Keep open: this is a team-authored notebook markdown bug with an assigned `On Deck` milestone, both of which require explicit maintainer judgment rather than cleanup closure.

## Change summary

Undoing after pasting an attached image into a rendered notebook markdown cell removes only the attachment metadata and leaves a broken image link.

## Best solution

Notebook and undo/redo owners should decide the correct fix for grouped undo elements across the cell text model and notebook document, preserving both markdown text and attachment metadata consistently when undoing from rendered markdown.
