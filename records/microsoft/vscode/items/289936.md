## Summary

The ext API cleanup landed in PR #290592, but the internal dual-format representation (`hasValidDiff` and `getAgentChangesSummary` handling both array and object formats) still exists on current main and is actively used — the debt remains open.

Unify the dual-format `hasValidDiff` and `getAgentChangesSummary` checks so the changes representation uses a single format internally.
