# Issue #24389 — SCM - Support workspace diff navigation

[https://github.com/microsoft/vscode/issues/24389](https://github.com/microsoft/vscode/issues/24389)

## Summary

Partially addressed by the Multi-Diff Editor (Alt+F5 cycles across files), but the seamless IntelliJ-style "press one key anywhere to cycle all SCM changes" workflow is not yet fully realized. With 113 upvotes and strong community demand, this stays open as a valid feature request.

## Change summary

Add keyboard-driven navigation to cycle through all SCM changes across all modified files without requiring the user to open a specific editor first.

## Best solution

The Multi-Diff Editor already provides cross-file change navigation (multiDiffEditor.goToNextChange, Alt+F5) when viewing SCM changes. The remaining gap is a global command that works from any context — opening the multi-diff editor if needed and cycling through changes IntelliJ-style. This likely belongs in `src/vs/workbench/contrib/scm/` as a command that opens the multi-diff view and starts navigating, or extends the existing quick-diff widget to chain across files. Issue https://github.com/microsoft/vscode/issues/145009 covers a subset and could be consolidated here.
