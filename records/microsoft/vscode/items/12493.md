# Issue #12493 — [icon-themes] Support for globs in file associations (Icon themes)

[https://github.com/microsoft/vscode/issues/12493](https://github.com/microsoft/vscode/issues/12493)

## Summary

Still an unimplemented feature request. The icon theme schema explicitly disallows patterns/wildcards in file associations, and no glob support has been added on current main.

## Change summary

Allow glob patterns in icon theme file associations so themes can map files like `web.*.config` or `*.test.js` to specific icons.

## Best solution

Extend the file icon theme schema in `src/vs/workbench/services/themes/common/fileIconThemeSchema.ts` and the resolution logic in `src/vs/workbench/services/themes/browser/fileIconThemeData.ts` to support a new `filePatterns` (or similar) mapping that accepts glob patterns. This would complement the existing `fileNames` and `fileExtensions` lookups.
