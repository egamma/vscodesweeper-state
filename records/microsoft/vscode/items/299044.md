[https://github.com/microsoft/vscode/issues/299044](https://github.com/microsoft/vscode/issues/299044)

## Summary

Plan mode tool-call enforcement for Claude Sonnet 4.6 is implemented in the Copilot Chat extension (extensions/copilot/src/extension/agents/vscode-node/planAgentProvider, extensions/copilot/src/extension/chatSessions/claude/), not in VS Code core. The reporter is on extension version 0.37.9 which is significantly outdated; the current codebase shows substantial plan-mode enforcement logic that restricts tools to read-only operations.

## Change summary

Claude Sonnet 4.6 executes file edits and terminal commands in Plan mode instead of producing a text-only plan.
