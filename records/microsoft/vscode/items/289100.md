## Summary

Detailed report of PSReadLine history navigation cursor not resetting with shell integration enabled. ## Change summary

PSReadLine Up Arrow history navigation shows old commands instead of recent ones when VS Code shell integration is enabled, because the navigation cursor is not reset after interactive command execution.
