## Summary

Team-authored feature request for enabling save participants (formatters, etc.) during AI edit sessions via a shadow-model approach. Not yet implemented on main — save participants still run only on accept. Keeping open per team-authored rule.

## Change summary

Run save participants (e.g. formatters) during AI edit sessions so the LM doesn't chase formatting-related linter errors, using a shadow model to avoid content divergence.

## Best solution

Implement the shadow-model approach described in the issue: maintain an unformatted mirror for continued LM edits while running save participants on the real model after each edit, so on-disk and visible content stays formatted.
