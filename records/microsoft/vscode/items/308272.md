## Summary

Team-authored bug with milestone 'On Deck' — keeping open. The thinking-effort config button's managed hover (line 1261 of chatModelPicker.ts) appears to overlap the badge icon's hover region, so hovering the warning badge shows the config tooltip instead of the warning text.

## Change summary

Hovering the warning badge icon on a model in the chat model picker shows the thinking effort tooltip instead of the model's warning message.

## Best solution
