# Issue #325950 — Chat input should insert file and code references inline

[https://github.com/microsoft/vscode/issues/325950](https://github.com/microsoft/vscode/issues/325950)

## Summary

Feature request to replace attachment chips with Cursor-style inline @file references in chat input. An open PR (#325949) from the same author is linked, so this should stay open for maintainer review. Related issues exist (#322946, #297190) but none is established as a canonical — routing to the chat-input area for triage.

## Change summary

Insert file, folder, and code-range references inline in the chat input (e.g. @src/file.ts:2-4) instead of showing them as attachment chips above the input.
