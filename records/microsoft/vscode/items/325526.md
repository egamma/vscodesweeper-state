# Issue #325526 — Copilot Chat lacks paste as plain text shortcut/ability

[https://github.com/microsoft/vscode/issues/325526](https://github.com/microsoft/vscode/issues/325526)

## Summary

Valid feature request: the chat input widget converts rich clipboard content (e.g. hyperlinks) into Markdown on paste, and there is no 'paste as plain text' shortcut or menu entry to bypass this. No existing implementation or canonical issue found.

## Change summary

Add a 'paste as plain text' action (Cmd/Ctrl+Shift+V) to the Copilot Chat input so users can paste without rich-text-to-Markdown conversion.

## Best solution

Add a paste-as-plain-text keybinding and/or context-menu entry in the chat input widget (`src/vs/workbench/contrib/chat/browser/widget/input/chatInputPart.ts`). The editor already has `pasteAs` infrastructure; wiring a plain-text variant into the chat input's Monaco editor instance should be a contained change. This may also benefit the main editor — consider a shared command.
