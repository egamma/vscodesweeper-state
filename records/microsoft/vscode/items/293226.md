# Issue #293226 — Show premium agent features locked for free sku

[https://github.com/microsoft/vscode/issues/293226](https://github.com/microsoft/vscode/issues/293226)

## Summary

Team-authored feature request to improve the UX for free-tier users by showing premium agent features (Cloud, Codex, Claude) as locked or greyed out with upgrade messaging, rather than appearing fully available. Keeping open per team-member rule.

## Change summary

Free Copilot users see premium agent targets (Cloud, Codex, Claude) as available in the session target dropdown with no indication they require a paid plan.

## Best solution

Add entitlement-aware UI to the session/agent target dropdown: show a lock icon or greyed-out state for premium-only agents when the user is on the free SKU, with a hover/tooltip prompting upgrade to Copilot Pro. This likely involves the chat entitlement service and the agent/session target picker UI.
